# 策略分類與評估標準分析

## 關鍵洞察

**不同類型的策略評估方式完全不同！不能一概而論。**

## 策略類型分類

### 1. 趨勢跟踪策略（Trend Following）

**代表策略：** Supertrend, MACD

**核心特點：**
- 漲了繼續漲，跌了繼續跌
- 在**趨勢市場**中表現優異
- 在**震盪市場**中會頻繁止損

**市場適應性：**
- ✅ 牛市：表現優異
- ✅ 熊市：可以做空
- ❌ 震盪市：頻繁虧損

**評估標準（優先順序）：**

| 指標 | 重要度 | 理想值 | 說明 |
|------|--------|--------|------|
| **總收益率** | ⭐⭐⭐⭐⭐ | 越高越好 | 趨勢策略的核心目標 |
| **最大回撤** | ⭐⭐⭐⭐⭐ | 越小越好 | 單次虧損可能很大 |
| **夏普比率** | ⭐⭐⭐⭐ | > 1.0 | 風險調整後收益 |
| **盈虧比** | ⭐⭐⭐⭐ | > 2.0 | 盈利交易 / 虧損交易 |
| **趨勢持續性** | ⭐⭐⭐⭐ | 長 | 平均持倉時長 |
| **勝率** | ⭐⭐ | > 40% | 趨勢策略勝率通常較低 |

**典型回測模式：**
```
收益曲線：長期上漲，偶爾大跌
交易頻率：低到中等
勝率：30-50%（正常）
盈虧比：2:1 到 4:1
```

**關鍵風險：**
- 趨勢反轉時虧損巨大
- 震盪市頻繁止损
- 需要快速識別市場狀態

---

### 2. 均值回歸策略（Mean Reversion）

**代表策略：** RSI, Bollinger, 協整對（Cointegration）

**核心特點：**
- 價格偏離均值後會回歸
- 在**震盪市場**中表現優異
- 在**趨勢市場**中容易逆勢虧損

**市場適應性：**
- ✅ 震盪市：表現優異
- ❌ 牛市：容易踏空
- ❌ 熊市：容易接飛刀
- ⚠️ 弱趨勢：表現一般

**評估標準（優先順序）：**

| 指標 | 重要度 | 理想值 | 說明 |
|------|--------|--------|------|
| **勝率** | ⭐⭐⭐⭐⭐ | > 70% | 均值回歸的核心目標 |
| **交易頻率** | ⭐⭐⭐⭐ | 适中 | 過高 = 噪音，過低 = 錯失機會 |
| **回歸速度** | ⭐⭐⭐⭐ | 5-20 天 | 半衰期 |
| **殘差穩定性** | ⭐⭐⭐⭐ | 高 | ADF 檢驗 p-value < 0.05 |
| **夏普比率** | ⭐⭐⭐ | > 0.8 | 風險調整後收益 |
| **總收益率** | ⭐⭐ | 中等 | 穩定但不是爆發式 |

**典型回測模式：**
```
收益曲線：穩定上升，回撤小
交易頻率：中等
勝率：70-90%（正常）
盈虧比：1:1 到 1.5:1
```

**關鍵風險：**
- 強趨勢中逆勢虧損
- 需要識別市場狀態
- 半衰期過長會延遲回歸

---

### 3. 動量策略（Momentum）

**代表策略：** 動量輪動、強者恆強

**核心特點：**
- 強者恆強，弱者恆弱
- 基於相對表現選股
- **市場中性**（多強空弱）

**市場適應性：**
- ✅ 牛市：選出強勢股
- ✅ 熊市：多強空弱，可能獲利
- ✅ 震盪市：相對表現穩定

**評估標準（優先順序）：**

| 指標 | 重要度 | 理想值 | 說明 |
|------|--------|--------|------|
| **動量持續性** | ⭐⭐⭐⭐⭐ | 3-6 個月 | 動量持續時長 |
| **換手率** | ⭐⭐⭐⭐ | < 30% / 月 | 過高會增加交易成本 |
| **信息比率** | ⭐⭐⭐⭐ | > 1.0 | 超額收益穩定性 |
| **市場適應性** | ⭐⭐⭐ | 高 | 在不同市場中表現 |
| **總收益率** | ⭐⭐⭐ | 中高 | 穩定的超額收益 |
| **勝率** | ⭐⭐ | > 55% | 略高於隨機 |

**典型回測模式：**
```
收益曲線：穩定上升，低波動
交易頻率：低到中等
勝率：55-65%（正常）
盈虧比：1:1 到 2:1
```

**關鍵風險：**
- 動量崩潰（短期反轉）
- 交易成本高（換手）
- 需要多元化資產池

## 新的分析結論

### 問題 1：市場狀態未知

所有策略都只在**單一資產（台積電）**和**單一市場狀態（可能強多頭）**下測試。

**缺失驗證：**
- ❌ 震盪市表現
- ❌ 熊市表現
- ❌ 不同資產的穩定性

### 問題 2：策略類型不匹配

- **Supertrend**：趨勢跟踪，勝率 100% 不合理
- **RSI**：均值回歸，表現符合預期
- **MACD**：趨勢跟踪，表現一般

### 問題 3：人工成本考量

- **Supertrend**：5 筆，成本低 ✅
- **RSI**：7 筆，成本中等 ✅
- **MACD**：27 筆，成本高 ❌

## 改進建議

### 1. 市場狀態分離

```python
# 測試策略在不同市場狀態下的表現
def backtest_by_regime(strategy, price_data, regime_labels):
    """
    按市場狀態分離回測

    Args:
        strategy: 策略
        price_data: 價格數據
        regime_labels: 市場狀態標籤

    Returns:
        dict: 不同市場狀態下的回測結果
    """
    results = {}

    for regime in ['bull_weak', 'bull_medium', 'bull_strong',
                   'bear_weak', 'bear_medium', 'bear_strong',
                   'sideways_weak', 'sideways_medium', 'sideways_strong']:

        # 選擇該市場狀態的數據
        regime_data = price_data[regime_labels == regime]

        # 回測
        regime_result = strategy.backtest(regime_data)

        results[regime] = regime_result

    return results
```

### 2. 多資產測試

- 測試策略在 10+ 個資產上的表現
- 評估策略的穩定性
- 避免單一資產的幸運者偏差

### 3. 動態策略組合

```python
class RegimeBasedStrategy:
    """基於市場狀態的動態策略"""

    def __init__(self):
        self.trend_strategies = {
            'bull_strong': 'supertrend',
            'bull_medium': 'macd',
            'bear_strong': 'supertrend',
        }

        self.mean_reversion_strategies = {
            'sideways_weak': 'rsi',
            'sideways_medium': 'rsi',
            'bull_weak': 'rsi',
        }

    def select_strategy(self, regime):
        """根據市場狀態選擇策略"""

        if regime in self.trend_strategies:
            return self.trend_strategies[regime]
        elif regime in self.mean_reversion_strategies:
            return self.mean_reversion_strategies[regime]
        else:
            return 'neutral'
```
