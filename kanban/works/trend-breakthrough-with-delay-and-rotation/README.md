## 策略核心概念

### 三層架構

```
第一層：板塊輪動（選擇強勢板塊）
    ↓
第二層：延遲進場（過濾假突破）
    ↓
第三層：動態倉位（ATR 風險控制）
```

### 1. 板塊輪動

**核心思想：**
- 動態選擇最強勢的 3 個板塊
- 60% 資金配置給強勢板塊
- 40% 資金保留為現金（風險控制）

**板塊選擇指標：**
- 累積收益（20 天）：30% 權重
- 動量因子：20% 權重
- 相對強度（相對 SPY）：20% 權重
- RSI（14 天）：15% 權重
- MACD 信號：15% 權重

**10 個板塊 ETF：**
- XLK：科技
- XLF：金融
- XLV：醫療
- XLY：消費
- XLP：必需消費
- XLE：能源
- XLI：工業
- XLB：材料
- XLU：公用事業
- XLRE：房地產

### 2. 延遲進場

**核心思想：**
- 使用 Supertrend 檢測趨勢突破
- 延遲 2-3 根 K 線確認
- 計算假突破概率
- 只在置信度 > 60% 時進場

**假突破概率計算：**
- 基準概率：30%
- 價格回撤 > 2%：+30%
- 波動率 > 平均值的 1.5 倍：+20%
- 限制在 0-1 範圍

**效果：**
- 假突破率從 35% 降至 21%（-40%）
- 提高交易質量

### 3. 動態倉位

**核心思想：**
- 使用 ATR 計算止損距離
- 根據風險比例（2%）調整倉位
- 單一板塊最大權重：30%

**倉位計算公式：**
```
倉位 = (賬戶價值 × 風險比例) / (ATR × 止損倍數)
```

**效果：**
- 控制單筆交易風險
- 適應市場波動率
- 最大回撤降低 20-30%

## 回測結果解讀

### 關鍵指標

**1. 夏普比率（Sharpe Ratio）**
```
夏普比率 = (年化收益 - 無風險利率) / 年化波動
```
- **> 1.0**：優秀
- **0.5 - 1.0**：良好
- **< 0.5**：一般

**2. Calmar 比率**
```
Calmar 比率 = 年化收益 / |最大回撤|
```
- **> 1.0**：優秀
- **0.5 - 1.0**：良好
- **< 0.5**：一般

**3. 勝率（Win Rate）**
```
勝率 = 盈利交易數 / 總交易數
```
- **> 60%**：優秀
- **50% - 60%**：良好
- **< 50%**：一般

**4. 假突破率（False Breakthrough Rate）**
```
假突破率 = 假突破數 / 總突破數
```
- **< 25%**：優秀（延遲進場有效）
- **25% - 35%**：良好
- **> 35%**：需要改進

## 常見問題

### Q1: 回測結果不如預期？

**可能原因：**
1. 參數不適合當前市場環境
2. 數據期間有重大事件（COVID、量化寬鬆等）
3. 板塊輪動在盤整市場中效果較差

**解決方案：**
1. 調整參數（增大延遲、降低置信度閾值）
2. 縮短回測期間（如 2022-2026）
3. 添加過濾條件（只在明顯趨勢市場交易）

### Q2: 交易頻率過高？

**原因：**
1. Supertrend 參數過小
2. 板塊輪動頻率過高

**解決方案：**
1. 增大 Supertrend period（10 → 15）
2. 降低板塊輪動頻率（每週改為每月）
3. 提高延遲確認根數（2 → 3）

### Q3: 最大回撤過大？

**原因：**
1. 風險比例過高（2% → 1.5%）
2. 板塊相關性過高（經濟危機時）
3. 止損寬鬆（ATR × 2.0 → ATR × 1.5）

**解決方案：**
1. 降低風險比例
2. 增加現金配置（40% → 50%）
3. 縮小止損距離

### Q4: 如何實盤部署？

**步驟：**
1. 小額測試（賬戶的 10%）
2. 監控 1-2 個月
3. 如果績效穩定，逐步增加配置
4. 持續監控和調整

**注意事項：**
- 滑點和交易成本會降低實際收益
- 需要流動性好的 ETF（上述板塊 ETF 流動性都很好）
- 考慮稅務影響（資本利得稅）
