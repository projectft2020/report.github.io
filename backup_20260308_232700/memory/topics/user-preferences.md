## 🔧 實務工具範例：DHRI

### 來源理論
- **基於：** t005 尾部風險對沖報告（2,907 行）
- **簡化成果：** 3 種計算方法

### 三種計算方法

#### 方法一：ultra_simple（推薦）
- **輸入：** 僅 VIX
- **計算：** 簡單公式
- **優點：** 30 秒查詢
- **缺點：** 精度較低

#### 方法二：simple
- **輸入：** VIX + 波動率
- **計算：** 中等複雜度
- **優點：** 3 分鐘執行
- **缺點：** 需要計算波動率

#### 方法三：precise
- **輸入：** VIX + 波動率 + 趨勢 + 市場狀態
- **計算：** 高複雜度
- **優點：** 最精確
- **缺點：** 計算時間較長

### 輸出格式
- **DHRI (0-100)** - 0 = 最小避險，100 = 最大避險
- **避險比例 (0-50%)** - 實際交易比例

### 代碼
- **類名稱：** DailyHedgeRatioIndicator
- **使用：** 可直接導入使用
