## 研究摘要

本研究深入分析了 Nguyen 等人於 2025 年 7 月發表在 *Journal of Risk and Financial Management* 的論文《DASF-Net: A Multimodal Framework for Stock Price Forecasting with Diffusion-Based Graph Learning and Optimized Sentiment Fusion》。DASF-Net 是一個創新的多模態框架，結合了擴散基礎的圖學習和優化的情感融合技術，用於股價預測。該方法在 12 支 S&P 500 大型股的 4 年數據集上，相比先進基線模型實現了高達 91.6% 的均方誤差（MSE）相對降低。研究發現，透過雙重金融圖（產業圖和基本面圖）的擴散過程結合 FinBERT 情感嵌入，並使用多頭注意力機制進行自適應融合，能夠顯著提升金融預測的魯棒性和準確性。

## 關鍵發現

### 1. 核心技術創新

**擴散基礎的圖學習**  
- **雙重金融圖構建**: DASF-Net 建構兩個互補的金融圖：
  - **產業圖（Industry Graph, IG）**: 基於靜態的行業關係，若兩支股票屬於同一行業部門則建立邊
  - **基本面圖（Fundamental Graph, FG）**: 基於動態的報酬相似性，使用報酬序列的絕對餘弦相似性計算邊權重

- **擴散過程表示學習**: 使用擴散過程替代傳統 GNN 的局部消息傳遞，擴大接收場並減少過度平滑問題
  ```
  H_{t+1} = α · Adj^g · H_t + (1-α) · H_0
  ```
  其中 α 控制局部和全局信息的平衡

**優化的情感聚合窗口**  
- 經驗性確定最佳情感聚合窗口為 **3 天**（T_opt = 3）
- 與固定窗口方法相比，這種優化減少了時間偏差，提升了預測準確性
- 使用 FinBERT（領域專用的語言模型）從金融新聞中提取情感嵌入

**自適應多模態融合**  
- 採用多頭注意力機制（MHA）動態融合三種表示：
  - 產業圖結構表示（H_IG）
  - 基本面圖結構表示（H_FG）
  - 情感知覺表示（H_SA）
- 相較於靜態融合方法，多頭注意力機制在波動期間提升了 10% 的性能

### 2. 實驗性能表現

**數據集與設定**  
- 使用 FNSPID（Financial News and Stock Price Integration Dataset）
- 涵蓋 12 支 S&P 500 大型股
- 時間範圍：2020 年 1 月至 2023 年 12 月（包含 COVID-19 疫情和經濟復甦階段）
- 預測期限：1 天、2 天、3 天

**基線模型比較**  
- LSTM + CNN: CNN 用於空間特徵提取，LSTM 用於時間建模
- Multi-GCGRU: 整合圖卷積網絡（GCN）與門控循環單元（GRU）
- Sentiment + LSTM: LSTM 模型結合金融新聞的情感特徵
- MGAR: 融合多種圖結構嵌入的多模態框架

**性能優勢**  
| 預測期限 | DASF-Net MSE | 相較 MGAR 降低 | 相較 Sentiment+LSTM 降低 |
|---------|--------------|---------------|-------------------------|
| 1 天    | 3.8 × 10⁻⁴  | 91.6%         | 94.7%                   |
| 2 天    | 6.2 × 10⁻⁴  | 87.2%         | 91.4%                   |
| 3 天    | 1.1 × 10⁻³  | 54.2%         | 66.7%                   |

- 統計顯著性：p < 0.01（配對 t 檢驗）
- 相較於僅使用產業圖（MSE: 7.3 × 10⁻⁴）或基本面圖（MSE: 3.4 × 10⁻³）的變體，完整模型（包含情感）在 1 天預測上 MSE 降低 17.4%
- 與 Multi-GCGRU 相比，特徵相似性降低 12%，有效緩解了過度平滑問題

### 3. 技術細節深度分析

**雙重圖構建的理論基礎**  
DASF-Net 的雙重圖設計體現了金融市場的雙重特性：
1. **結構性關係**: 產業圖捕捉股票間固有的行業歸屬關係，反映相似商業模式和市場暴露
2. **行為性關係**: 基本面圖捕捉股票報酬模式的動態相似性，反映市場參與者的投資行為

這種設計克服了單一圖結構的局限性，能夠同時建模靜態的部門歸屬和動態的市場互動。

**擴散過程的數學原理**  
擴散過程模擬信息在圖上的傳播機制，其核心優勢在於：
- **擴大接收場**: 允許信息跨越多跳傳播，捕捉更高階的股票間依賴
- **保持節點個體性**: 透過保留初始特徵 H_0 的權重（1-α），避免表示過度收斂
- **計算效率**: 相較於全連接的 GNN，擴散過程透過稀疏化技術保持計算效率

**情感窗口優化的實證發現**  
研究對不同情感聚合窗口（T_n = 1, 2, 3, 4, 5 天）進行了系統性分析：
- **T_n = 1 天**: MSE = 4.1 × 10⁻⁴，MAE = 1.7 × 10⁻²（窗口過窄，缺乏足夠上下文）
- **T_n = 3 天**: MSE = 3.8 × 10⁻⁴，MAE = 1.5 × 10⁻²（最佳性能）
- **T_n = 5 天**: 性能開始下降（窗口過寬，引入無關噪聲）

這表明 3 天窗口能夠捕捉市場對新聞的反應延遲，同時避免信號稀釋。

**多頭注意力的融合機制**  
多頭注意力機制在 DASF-Net 中的關鍵作用：
- **模態權重動態調整**: 根據市場條件動態調整結構特徵和情感特徵的權重
- **跨模態交互建模**: 捕捉不同模態之間的複雜交互關係
- **魯棒性增強**: 在高波動期間能夠優先考慮情感信號，在穩定期間關注股票間結構關係

## 詳細技術分析

### 4. 架構設計

#### 4.1 整體架構

DASF-Net 包含五個核心組件：

1. **雙重金融圖構建**
2. **擴散基礎的結構表示學習（P-Rep）**
3. **情感感知表示提取（SA-Rep）與優化的時間聚合**
4. **多頭注意力機制的自適應多模態融合**
5. **LSTM 時間預測模組**

#### 4.2 產業圖（Industry Graph）構建

產業圖捕捉靜態的行業歸屬關係：

```
e_{ij}^{IG} = {
  1, if stocks i and j belong to the same industry sector
  0, otherwise
}
```

這種二值邊設計體現了行業歸屬的離散特性，反映相同行業股票間的固有相似性。

#### 4.3 基本面圖（Fundamental Graph）構建

基本面圖捕捉動態的報酬相似性：

**步驟 1: 計算報酬序列**
對於每支股票 i，計算回顧期間 T_p 內的報酬序列 n_i ∈ ℝ^{T_p}：

```
r_t^i = (close_t^i - close_{t-1}^i) / close_{t-1}^i
```

**步驟 2: 計算邊權重**
使用報酬序列的絕對餘弦相似性：

```
e_{ij}^{FG} = cos(n_i, n_j) = ⟨n_i, n_j⟩ / (||n_i|| · ||n_j||)
```

這種設計確保邊權重反映股票報酬行為的相關程度，不考慮方向（正向或負向相關）。

#### 4.4 擴散過程實現

擴散過程的迭代公式：

```
H_{t+1} = α · Adj^g · H_t + (1-α) · H_0
```

其中：
- H_0: 初始特徵矩陣（報酬序列）
- Adj^g: 圖 g 的鄰接矩陣
- H_t: 擴散步驟 t 的特徵矩陣
- α: 擴散率，控制局部和全局信息的平衡

擴散步數 T_diff 的選擇影響模型的接收場大小，需要在信息傳播深度和計算效率之間取得平衡。

#### 4.5 情感表示提取

**步驟 1: FinBERT 情感評分**
對於每篇新聞文章 a_z，使用 FinBERT 提取情感分數：

```
s_z = FinBERT(a_z)
```

**步驟 2: 統計特徵提取**
將當天所有相關新聞的情感分數壓縮為 5 維統計特徵：

```
Stat_t^i = [min(S_t^i), max(S_t^i), mean(S_t^i), σ(S_t^i), Z_t]
```

其中 S_t^i 是股票 i 在第 t 天的情感分數向量，Z_t 是新聞數量。

**步驟 3: 時間聚合與嵌入**
使用優化的 3 天窗口聚合情感信息：

```
I_t^i = [Stat_{t-2}^i, Stat_{t-1}^i, Stat_t^i] ∈ ℝ^{3×5}
```

展平並通過全連接層產生固定長度的情感感知嵌入：

```
h_{SA,i} = FC(I_t^i) ∈ ℝ^{32}
```

#### 4.6 多頭注意力融合

多頭注意力機制融合三種表示：

**投影矩陣學習**:
```
Q = H_IG · W_Q^{IG} + H_FG · W_Q^{FG} + H_SA · W_Q^{SA}
K = H_IG · W_K^{IG} + H_FG · W_K^{FG} + H_SA · W_K^{SA}
V = H_IG · W_V^{IG} + H_FG · W_V^{FG} + H_SA · W_V^{SA}
```

**注意力計算**:
對於每個注意力頭 h：
```
head_h = softmax(Q_h · K_h^T / √d_k) · V_h
```

**融合輸出**:
```
H_final = Concat(head_1, ..., head_{n_heads}) · W_O
```

## 應用價值評估

### 5. 金融領域應用潛力

**量化交易策略**  
- **趨勢預測**: DASF-Net 能夠準確預測短期（1-3 天）股價走勢，為量化交易策略提供信號
- **風險管理**: 透過捕捉股票間結構關係，有助於識別系統性風險和投資組合分散化
- **情感套利**: 結合情感分析，能夠捕捉市場對新聞的反應，實現情感套利機會

**投資組合管理**  
- **多資產配置**: 擴散過程能夠識別不同股票間的隱藏關係，優化投資組合配置
- **行業輪動**: 產業圖和基本面圖的結合能夠識別行業間的聯動效應
- **動態調整**: 每日重新估計圖邊權重，確保模型對市場制度變化的敏感性

**金融科技應用**  
- **智能投顧**: 整合結構和情感信息，提供個性化的投資建議
- **風險監控**: 實時監控市場情緒和結構變化，預警潛在風險
- **市場分析工具**: 提供多維度的市場洞察，輔助投資決策

### 6. 技術創新價值

**擴散學習的應用推廣**  
- **超越 GNN 限制**: 擴散過程克服了傳統 GNN 的接收場限制和過度平滑問題
- **計算效率**: 透過稀疏化技術保持計算效率，適合大規模金融圖
- **魯棒性**: 對圖結構中的噪聲具有較強的抵抗能力

**情感分析的優化**  
- **經驗性窗口選擇**: 3 天窗口的確定提供了情感聚合的最佳實踐參考
- **領域專用模型**: 使用 FinBERT 這類金融領域專用語言模型，提升了情感分析的準確性
- **多維度統計**: 不僅使用平均情感，還考慮情感的分佈特徵（最小值、最大值、標準差）

**多模態融合的創新**  
- **自適應權重**: 多頭注意力機制允許模型根據市場條件動態調整不同模態的權重
- **互動建模**: 捕捉不同模態之間的交互關係，而非簡單的特徵拼接
- **可解釋性**: 注意力權重提供了模型決策的可解釋性

## 局限性分析

### 7. 方法論局限性

**圖構建的假設**  
- **靜態產業歸屬**: 產業圖基於靜態的行業分類，可能無法捕捉企業業務多樣化的現實
- **報酬相似性**: 基本面圖基於歷史報酬相似性，可能存在"相關性不代表因果性"的問題
- **圖稀疏性**: 僅考慮連通分量中最大的部分，可能遺漏重要的邊緣股票關係

**情感分析的挑戰**  
- **新聞來源限制**: 僅使用特定金融新聞網站，可能忽略社交媒體等其他重要信息源
- **情感窗口固定**: 3 天窗口是經驗性確定的，可能不適用所有市場環境或股票類型
- **情感語義的複雜性**: 金融文本的情感表達往往隱含且複雜，FinBERT 可能無法完全捕捉

**時間預測的局限**  
- **短視界預測**: 僅預測 1-3 天，不適合長期投資決策
- **非平穩性**: 金融數據的非平穩特性可能導致模型性能隨時間退化
- **外生衝擊**: 難以預測突發外生事件（如地緣政治事件、自然災害）對市場的影響

### 8. 實際應用局限性

**數據要求**  
- **高質量數據**: 需要完整的股價和新聞數據，數據缺失會影響性能
- **實時性要求**: 每日重新估計圖邊權重，需要高效的數據處理流程
- **計算資源**: 多頭注意力和擴散過程需要相當的計算資源

**市場適用性**  
- **市場依賴**: 在 S&P 500 市場表現良好，但在其他市場（如新興市場）的適用性需要驗證
- **制度變化**: 市場制度變化（如政策轉變、監管變化）可能影響模型穩定性
- **黑天鵝事件**: 難以處理極端市場條件下的預測任務

**模型可解釋性**  
- **黑盒特性**: 深度學習模型的本質使其決策過程難以完全解釋
- **注意力權重**: 雖然注意力權重提供了一定可解釋性，但整體架構仍然複雜
- **因果推斷**: 模型捕捉相關性而非因果性，可能導致錯誤的因果推斷

### 9. 擴展性考量

**規模化挑戰**  
- **股票數量限制**: 實驗僅使用 12 支股票，擴展到全市場可能需要額外優化
- **新聞數量增長**: 隨著股票數量增加，需要處理的新聞數量呈線性增長
- **圖規模**: 大規模圖的擴散過程計算複雜度需要優化

**跨市場應用**  
- **市場差異**: 不同市場的結構和情感動態可能存在顯著差異
- **語言差異**: FinBERT 主要針對英文文本，非英語市場的應用需要適應
- **監管差異**: 不同市場的監管環境可能影響模型的有效性

## 實施建議

### 12. 技術實施

**數據管道設計**  
1. **股價數據處理**
   - 使用 min-max 標準化確保不同價格範圍股票的可比性
   - 處理缺失數據（前向填充或插值）
   - 計算報酬序列作為圖節點特徵

2. **新聞數據處理**
   - 實時抓取金融新聞
   - 使用 FinBERT 進行情感分析
   - 處理缺失數據（傳播最近可用情感分數）

3. **圖構建流程**
   - 每日更新產業圖（靜態）
   - 每日重新估計基本面圖（動態）
   - 確保圖連通性（過濾最大連通分量）

**模型訓練策略**  
1. **超參數優化**
   - 使用網格搜索和隨機搜索的組合
   - 在驗證集上選擇最佳配置
   - 擴散率 α、擴散步數 T_diff、情感窗口 T_opt

2. **訓練配置**
   - 損失函數：均方誤差（MSE）
   - 優化器：Adam
   - 學習率：0.001-0.01
   - 批次大小：根據可用的 GPU 記憶體調整

3. **驗證策略**
   - 時間序列交叉驗證
   - 確保測試集在訓練集之後
   - 監控過擬合和模型穩定性

**部署架構**  
1. **推理流程**
   ```
   實時數據 → 圖構建 → 擴散過程 → 情感提取 → 注意力融合 → LSTM 預測 → 輸出
   ```

2. **系統組件**
   - 數據收集模組
   - 圖構建模組
   - 特徵提取模組

## 結論

DASF-Net 代表了金融時間序列預測領域的一個重要進展。透過創新性地結合擴散基礎的圖學習、優化的情感聚合和多模態融合技術，該方法在短期股價預測任務上取得了顯著的性能提升。

**核心優勢**：
1. 擴散過程有效克服了傳統 GNN 的接收場限制和過度平滑問題
2. 3 天情感聚合窗口的經驗性確定提供了情感整合的最佳實踐
3. 多頭注意力機制實現了自適應的多模態融合
4. 在 S&P 500 大型股上實現了高達 91.6% 的 MSE 相對降低

**主要局限性**：
1. 圖構建依賴於靜態產業分類和歷史報酬相似性
2. 情感分析僅使用有限的新聞來源和固定窗口
3. 短視界預測（1-3 天）限制了其在長期投資決策中的應用
4. 模型的黑盒特性影響了其可解釋性和因果推斷能力

**未來展望**：
儘管存在局限性，DASF-Net 的核心思想——結合結構和情感信息進行金融預測——具有廣闊的應用前景。未來研究可以在動態圖學習、多源情感整合、跨市場驗證等方面進一步探索。隨著金融科技的發展和數據可用性的提高，類似 DASF-Net 的多模態預測框架有望在量化交易、風險管理和智能投顧等領域發揮重要作用。

對於 Quantocracy 社群來說，DASF-Net 提供了一個有價值的參考框架，特別是在如何整合圖結構學習和情感分析方面。實踐者可以基於此框架，結合自己的數據和領域知識，開發適合特定市場或資產類別的定製化預測模型。
