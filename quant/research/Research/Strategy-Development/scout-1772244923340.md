# 論文研究報告：基於間隙的近乎極小化最優強化學習邊界與線性函數逼近

## 研究摘要

本論文《Gap-Dependent Bounds for Nearly Minimax Optimal Reinforcement Learning with Linear Function Approximation》由 Haochen Zhang、Zhong Zheng 和 Lingzhou Xue 於賓夕法尼亞州立大學完成（2026年2月，arXiv:2602.20297）。這項工作首次為近乎極小化最優的 LSVI-UCB++ 算法建立了間隙依賴的遺憾上界，顯著改善了特徵維度 d 和時間長度 H 的依賴關係。此外，作者還利用 LSVI-UCB++ 的低策略切換特性，提出了並發變體，實現了線性加速的多智能體在線學習。

## 主要發現

1. **改進的間隙依賴遺憾邊界** — 對 LSVI-UCB++ 算法建立了 Õ(d²H⁴/Δmin) 的預期遺憾邊界，相較於之前的 Õ(d³H⁵/Δmin) 有顯著改進 | 來源: arXiv:2602.20297

2. **並發強化學習算法** — 提出了 Concurrent LSVI-UCB++，在具有線性函數逼近的多智能體在線強化學習中實現了線性加速 | 來源: arXiv:2602.20297

3. **PAC 樣本複雜度改進** — 尋找 ε-最優策略所需回合數從 Õ(1/ε²) 改進至 Õ(1/ε) | 來源: arXiv:2602.20297

4. **技術創新** — 引入了代理矩陣和遞歸方差估計技術來控制部分和 | 來源: arXiv:2602.20297

## 詳細分析

### 1. 研究背景與動機

強化學習（RL）在處理大型狀態和動作空間時，函數逼近成為關鍵技術。線性函數逼近是一種廣泛採用的方法，其中值函數被表示為已知特徵的線性函數。

LSVI-UCB++ 算法（He et al., 2023）是當前最先進的算法之一，在線性 MDP 設置下實現了近乎極小化最優的遺憾邊界 Õ(d√(H³K))，其中：
- d 是特徵維度
- H 是時間長度（每個回合的步數）
- K 是回合數

然而，在現實應用中，當存在正的次優間隙（即最優動作比次優動作明顯更好）時，RL 算法的性能通常優於其最壞情況保證。這就是所謂的「間隙依賴分析」。

**核心問題：** 能否為近乎極小化最優的算法建立改進的間隙依賴遺憾上界？

### 2. 核心技術貢獻

#### 2.1 LSVI-UCB++ 算法回顧

LSVI-UCB++ 基於「面對不確定性的樂觀原則」，使用加權嶺迴歸和悲觀主義技術。算法的核心步驟包括：

1. **加權嶺迴歸估計器：**
   ```
   ŵ_{k,h} = argmin_w λ||w||² + Σ_{i=1}^{k-1} σ̄_{i,h}^{-2}(w^Tφ(s_h^i,a_h^i) - V_{k,h+1}(s_{h+1}^i))²
   ```

2. **調整後的估計方差：**
   ```
   σ̄_{k,h}² = max{σ_{k,h}², H, 2d³H²||φ(s_h^k,a_h^k)||_{Σ_{k,h}^{-1}}}
   ```

3. **樂觀估計值函數：**
   ```
   Q_{k,h}(s,a) ≈ r_h(s,a) + ŵ_{k,h}^T φ(s,a) + β||φ(s,a)||_{Σ_{k,h}^{-1}}
   ```

其中，β 的選擇為 Θ̃(√d)，使 Q_{k,h}(s,a) 以高概率成為 Q_h^*(s,a) 的上界。

#### 2.2 主要理論結果

**定理 4.1：** 對於任何線性 MDP，如果設置參數 λ = 1/H²，置信半徑 β, β̄, β̃ 為適當值（具體公式見論文），且失敗概率 δ = 1/(18T)，則算法 1（LSVI-UCB++）在前 T 步的預期遺憾滿足：

```
E[Regret(T)] ≤ O(d²H⁴/Δ_min · ι₁³ + d⁶H⁷ · ι₁²)
```

其中：
- Δ_min 是最小間隙
- ι₁ = log(1 + dHK/Δ_min)

**關鍵改進：** 與現有的間隙依賴遺憾邊界相比（見表 1），本論文將對 d 和 H 的依賴從 Õ(d³H⁵/Δ_min) 改進至 Õ(d²H⁴/Δ_min)。

| 算法 | 預期遺憾邊界 |
|------|-------------|
| LSVI-UCB | Õ(d³H⁵/Δ_min) (He et al., 2021a) |
| UCRL-VTR | Õ(d²H⁵/Δ_min) (He et al., 2021a) |
| LSVI-UCB | Õ(d³H⁵/Δ_min) (Papini et al., 2021) |
| Cert-LSVI-UCB | Õ(d³H⁵/Δ_min) (Zhang et al., 2024a) |
| **LSVI-UCB++** | **Õ(d²H⁴/Δ_min) (本文)** |

#### 2.3 PAC 樣本複雜度改進

**推論 4.2：** 作為定理 4.1 的直接推論，獲得了間隙依賴的 PAC（大約正確）樣本複雜度邊界。對於任何線性 MDP，失敗概率 δ ∈ (0,1) 和精度參數 ε > 0，運行算法 1，在概率至少 1-δ 下，經過：

```
K = Õ(d²H⁴/(Δ_min δε) + d⁶H⁷/(δε))
```

回合後，輸出策略 π̂ = Σ_{k=1}^K π_k/K 是一個 ε-最優策略。

**改進：** 與表格型 RL 和線性函數逼近 RL 中的最壞情況 PAC 邊界（通常為 Õ(1/ε²)）相比，本結果將對精度 ε 的依賴改進至 Õ(1/ε)，意味著可以用更少的樣本學習 ε-最優策略。

### 3. 並發強化學習擴展

#### 3.1 問題設定

考慮 M 個並行智能體的設置，每個智能體與同一個獨立的分回合 MDP 副本交互。在每個回合內，M 個智能體同步執行，不進行通信。信息交換和策略更新僅在所有智能體完成回合後允許。

並發回合定義為 M 個智能體同時完成一個回合並通信以更新其策略的時間段。

#### 3.2 並發 LSVI-UCB++ 算法

**算法 2：並發 LSVI-UCB++**

核心思想：通過將單個並發回合中收集的 M 個回合視為沒有中間策略更新的 M 個連續回合，來模擬單智能體 LSVI-UCB++ 算法。

步驟：
1. 所有智能體遵循由當前值估計確定的相同策略 π^k
2. 依次將收集的軌跡饋入 LSVI-UCB++
3. 當觸發更新時（算法 1 的第 8 行），更新估計的值函數和策略
4. 跳過當前回合中的剩餘軌跡

**定理 4.3：** 給定 M 個並行智能體，任何 δ ∈ (0,1) 和 ε > 0，在概率至少 1-δ 下，並發 LSVI-UCB++ 最多需要：

```
Õ(dH + d²H⁴/(MΔ_min δε) + d⁶H⁷/(Mδε))
```

並發回合來學習 ε-最優策略。

**線性加速：** 定理 4.3 意味著並發 LSVI-UCB++ 在智能體數量 M 上實現了線性加速。據作者所知，這是首次為線性函數逼近的在線多智能體強化學習提供間隙依賴樣本複雜度上界。

### 4. 技術創新與證明技術

為了將 LSVI-UCB++ 的最壞情況保證細化為間隙依賴邊界，作者引入了新的技術來控制獎勵部分和以及估計方差的部分和。

#### 4.1 控制獎勵部分和

**引理 5.3：** 引入了一個代理矩陣，該矩陣具有單步遞歸結構，允許獲得緊密邊界。

具體來說，通過構造一個代理矩陣 𝐒_k,h，該矩陣滿足遞歸關係：

```
𝐒_{k+1,h} = 𝐒_{k,h} + ...
```

從而能夠獲得獎勵部分和的精確估計。

#### 4.2 控制估計方差部分和

**引理 5.4：** 建立了不同步驟之間值函數估計誤差的部分和的遞歸關係。

通過建立值函數估計誤差 V_{k,h}(s) - V_h^*(s) 與估計值 V_{k,h}(s) - V̌_{k,h}(s) 之間的聯繫，獲得了估計方差的上界。

**引理 C.4 和 C.5：** 提供了遞歸關係的詳細證明，這些關係最終產生了估計方差的上界。

### 5. 應用價值評估

#### 5.1 高維環境中的樣本效率

在機器人操作和抓取任務中，狀態空間通常由高維連續觀測組成，包括機器人關節配置、物體位置等環境變量。將對 d 和 H 的理論依賴從 Õ(d³H⁵/Δ_min) 改進至 Õ(d²H⁴/Δ_min) 意味著：

- 在高維特徵空間中樣本效率顯著提高
- 長時間範圍任務的學習效率得到改善
- 理論改進直接增強了算法在實際應用中的可行性

#### 5.2 醫療保健應用

在醫療保健應用中，如優化細胞生長條件或順序治療規劃，智能體必須處理豐富的生物狀態表示和長序列的決策。改進的邊界使得：

- 在複雜醫療決策中所需的樣本數減少
- 更適合處理長時間範圍的醫療治療序列
- 為個性化醫療決策提供了更好的理論基礎

#### 5.3 多智能體協作

並發 LSVI-UCB++ 的線性加速特性使得：

- 在單個智能體的數據收集能力本質上受限且需要多智能體協作的現實應用中尤為有價值
- 通過並行探索顯著提高樣本效率
- 由於 LSVI-UCB++ 的低策略切換特性，通信成本得以有效控制

### 6. 局限性與未來方向

#### 6.1 局限性

1. **依賴線性 MDP 假設：** 算法理論保證基於線性 MDP 假設，這要求轉移動態和獎勵函數相對於已知特徵是線性的。在實際應用中，這種假設可能不成立。

2. **間隙未知：** 間隙依賴邊界中的 Δ_min 假設已知或可估計。在實踐中，實際間隙可能未知，需要額外的方法來估計。

3. **對數因子的隱藏：** 符號 Õ(·) 隱藏了對數因子，這些因子在某些情況下可能顯著。

4. **高概率保證 vs 預期保證：** 定理 4.1 提供的是預期遺憾邊界，而一些先前的工作提供了高概率常數遺憾（如 Papini et al., 2021 和 Zhang et al., 2024a）。

5. **通信成本：** 並發算法雖然實現了線性加速，但仍需要定期同步，在分布式環境中可能產生通信瓶頸。

#### 6.2 未來研究方向

1. **非線性函數逼近：** 擴展到更一般的函數逼近設置，如神經網絡逼近。

2. **自適應間隙估計：** 開發在線估計間隙 Δ_min 的方法，使算法能夠自適應調整探索策略。

3. **異步並發學習：** 擴展並發算法以支持異步通信和策略更新，進一步降低通信成本。

4. **非時間齊次設置：** 將結果擴展到非時間齊次的線性 MDP。

5. **實驗驗證：** 雖然論文提供了理論保證，但需要更多實驗來驗證算法在實際任務中的性能。

### 7. 與相關工作的比較

#### 7.1 近似最優 RL

在表格型 RL 中，算法通常分為基於模型和無模型方法：

- **基於模型：** 顯式估計轉移和獎勵模型，並基於學習的模型進行規劃（如 Azar et al., 2017; Dann et al., 2019）
- **無模型：** 直接維護值函數估計並貪婪行動（如 Jin et al., 2018; Yang et al., 2021）

在線性函數逼近 RL 中：

- **線性 MDP：** Yang and Wang (2019); Jin et al. (2020) 提出 LSVI-UCB，遺憾邊界 Õ(√(d³H³T))；He et al. (2023) 提出 LSVI-UCB++，改進至 Õ(d√(H³K))
- **線性混合 MDP：** Ayoub et al. (2020) 提出 UCRL-VTR，遺憾邊界 Õ(d√(H⁴K))；Zhou et al. (2021a) 開發了非齊次線性混合 MDP 的近似最優算法

#### 7.2 間隙依賴 RL

在表格型 RL 中：

- 早期工作建立了漸近對數遺憾邊界（Auer and Ortner, 2007; Tewari and Bartlett, 2008）
- 後期推導了非漸近邊界（Simchowitz and Jamieson, 2019; Dann et al., 2021; Yang et al., 2021）

在線性函數逼近 RL 中：

- He et al. (2021a) 開創了這一方向，顯示 LSVI-UCB 在線性 MDP 下實現 Õ(d³H⁵/Δ_min)，UCRL-VTR 在線性混合 MDP 下實現 Õ(d²H⁵/Δ_min)
- Papini et al. (2021) 和 Zhang et al. (2024a) 改進了這些結果，顯示 LSVI-UCB 和 Cert-LSVI-UCB 可以實現與總步數 T 無關的常數遺憾

#### 7.3 線性函數逼近的 MARL

- Dubey and Pentland (2021) 提出 Coop-LSVI，將 LSVI-UCB 從單智能體擴展到協作多智能體並行 RL
- Min et al. (2023) 引入異步變體，在保持相同遺憾邊界的同時改進通信效率
- Hsu et al. (2024) 開發了兩個隨機探索算法，實現相同的遺憾和通信保證

本論文的貢獻在於首次為線性函數逼近的在線 MARL 提供間隙依賴樣本複雜度邊界。

### 8. 結論

本論文通過為近乎極小化最優的 LSVI-UCB++ 算法建立改進的間隙依賴遺憾邊界，解決了一個重要的開放問題。主要貢獻包括：

1. **理論改進：** 將特徵維度 d 和時間長度 H 的依賴從 Õ(d³H⁵/Δ_min) 改進至 Õ(d²H⁴/Δ_min)
2. **PAC 樣本複雜度：** 尋找 ε-最優策略所需回合數從 Õ(1/ε²) 改進至 Õ(1/ε)
3. **多智能體擴展：** 提出並發 LSVI-UCB++，實現線性加速，為線性函數逼近的在線 MARL 提供首次間隙依賴樣本複雜度邊界
4. **技術創新：** 引入代理矩陣和遞歸方差估計技術

這些貢獻不僅具有重要的理論意義，還在機器人、醫療保健等需要高維特徵和長時間範圍決策的實際應用中具有顯著的實踐價值。
