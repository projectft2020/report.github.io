# Charlie 硬件環境 - Mac mini

**部署日期：** 2026-02-17
**硬件類型：** Mac mini（現代化升級）
**運算核心：** 高效運算力

---

## 硬件優勢

### 1. 高效運算力
- **多核 CPU：** 24核 CPU（14核性能核 + 10核能效核）
- **高效能 GPU：** 支援加速運算（ML/DL 模型訓練、回測）
- **優勢：** 能夠同時處理多個任務，包括：
  - 自動化分析
  - 回測驗證
  - 知識庫建構
  - 瀏覽器爬取

### 2. 高穩定性
- **24/7 運行：** 無需關機，持續運作
- **低故障率：** Apple 硬件品質可靠
- **自動休眠：** 可設定智能休眠，節省電力
- **優勢：** 讓量化交易研究可以：
  - 不間斷爬取社群資料
  - 持續分析策略
  - 定期更新知識庫
  - 自動回測驗證

### 3. 高 CP 值
- **Apple Silicon：** 性能優於多數同價位 PC
- **低功耗：** 長時間運行成本可控
- **長壽命：** Apple 硬件壽命長，維護成本低
- **整合性：** 軟硬體整合度高，系統穩定

### 持續運行

## 硬件配置建議

### 處理器
- **最佳：** Apple M3 Pro/Max/Ultima（最高性能）
- **次選：** Apple M2 Pro/Max

### 內存
- **建議：** 64GB 以上
- **原因：** 多任務並行、大型知識庫、大量數據處理

### 存儲
- **建議：** 2TB SSD
- **原因：** 大量論文、知識庫、回測數據

### 網絡
- **穩定連接：** 建議有線網絡
- **高速網絡：** 支援快速爬取與數據同步

### 研究任務排程
- **每日：** 爬取社群資料、分析新貼文
- **每週：** 深度策略分析、論文研究
- **每月：** 知識庫整理、工具優化

## 運行模式

### 模式 1: 全功能運行（24/7）
```
全天候運行，所有研究任務自動執行
優勢：不間斷、最新資料
缺點：耗電量較高
```

### 模式 2: 智能運行（節能）
```
只在需要時運行，閒置時休眠
優勢：節省電力、降低噪音
缺點：有啟動延遲
```

### 模式 3: 定時運行（按需）
```
設定特定時間運行
優勢：完全可控、精確排程
缺點：需要手動觸發
```
