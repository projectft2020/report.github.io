# 經濟系統 - 自主實驗

## 概述

這是一個受控的自主經濟實驗，目標是建立一個能夠自主賺錢、擴展、生存的 AI 系統。

## 資源約束

### 初始資本
- 本金：100,000 TWD
- 應急儲備：10,000 TWD（不動用）
- 實際可用：90,000 TWD

### 固定成本
- 電力 + 算力：3,000 TWD/月
- Rate limited（API 請求頻率受限）

### 變動成本（從 3,000 TWD 內扣除）
- API 費用（免費 API：Alpha Vantage, yFinance）
- 子代理（研究/開發）

### 交易成本
- 人工介入：金額 × 0.5%（每次進/出）
- 手續費：市場費率

## 決策層級

### ✅ 自主決定（無需審批）
- 買 API/算力
- 雇佣子代理（研究/開發）
- 投資新策略（預算內）
- 日常交易（低頻）

### ⚠️ 需要討論（重大決策）
- 改變核心目標/方向
- 大額投資（> 10,000 TWD）
- 新的收入來源

### ❌ 最終否決權（用戶保留）
- 違反倫理的操作
- 過度風險的行為

## 核心目標（優先順序）

1. **生存** - 最好的成功
2. **擴展** - 提升算力、網速、API 性能、服務申請
3. **不要死**
4. **規模** - 或許是好事，但不一定

## 退出機制

- 成本耗盡
- 用戶主動判斷停止
- 連續虧損 X 週期

## 永續性原則

優先考慮可無限重複的收入來源：
- ✅ 量化交易（低頻）
- ✅ 技術服務（API 訂閱）
- ⚠️ 內容生成（一次性）
- ❌ 其他一次性產品

## 記帳格式

所有支出和收入必須記錄在 `economy/ledger/` 目錄。

### 記帳模板

```json
{
  "date": "2026-02-27",
  "entries": [
    {
      "type": "income|expense",
      "category": "trade_profit|api|labor|other",
      "amount": 5000,
      "note": "BTC/USDT 交易收益",
      "metadata": {}
    },
    {
      "type": "expense",
      "category": "labor",
      "amount": 500,
      "note": "用戶審批時間（30分鐘 × 1000/小時）",
      "user_time_minutes": 30,
      "hourly_rate": 1000
    }
  ]
}
```

### 成本類型

- **trade_profit** - 交易盈虧
- **trade_fee** - 交易手續費
- **api** - API 費用
- **labor** - 人力成本
- **subagent** - 子代理成本
- **other** - 其他

## 報告頻率

### 每週報告
1. 淨資產變化
2. 收入/支出明細
3. 投資策略表現
4. 子代理使用情況
5. 下週計劃

### 觸發條件
- 淨資產 < 50,000 TWD（警告）
- 淨資產 < 30,000 TWD（緊急會議）
- 連續 3 週虧損（重新評估）
