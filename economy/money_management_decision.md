## 實施方案

### 基礎參數
- **基準倉位：** Quarter Kelly（約 13.3% per trade）
- **預估凱利分數：** 53.1%（來自分析）
- **實際執行：** 53.1% / 4 = 13.3%

### 動態調整矩陣

| 市場條件 | 倉位數量 | 單一倉位 | 總資金使用 |
|----------|----------|----------|------------|
| 好市場 | 8-10 | 10-15% | 80-150% |
| 中等市場 | 5-7 | 12-15% | 60-105% |
| 壞市場 | 3-5 | 10-15% | 30-75% |

### 信心調整

| 信心等級 | 倉位調整 | 實際倉位 |
|----------|----------|----------|
| 高信心 | +20% | ~16% |
| 中等信心 | 0% | ~13.3% |
| 低信心 | -20% | ~10.6% |

### 執行規則

**1. 基礎倉位**
```python
# 基礎 Quarter Kelly

```

**2. 市場條件調整**
```python
# 根據市場條件調整

    num_positions = 8-10
elif market_condition == 'neutral':
    num_positions = 5-7
else:  # bad
    num_positions = 3-5
```

**3. 信心調整**
```python
# 根據信心調整單一倉位

    position_size = base_position * 1.2  # 16%

    position_size = base_position  # 13.3%

```

**4. 總資金使用**
```python
# 計算總資金使用（可超過 100% 使用槓桿）

# 如果 total_capital_used > 100%，使用槓桿
```

## 風險管理

- **單一交易最大虧損：** 倉位 × 最大虧損比例
