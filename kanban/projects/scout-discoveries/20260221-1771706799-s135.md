## Detailed Analysis

### Theoretical Framework for Error Propagation

The main paper (arXiv:2602.16601) establishes a comprehensive theoretical framework for understanding error propagation in score-based diffusion models under recursive training. The framework analyzes a realistic pipeline where each training round uses a combination of synthetic data and fresh samples from the target distribution, obtaining upper and lower bounds on accumulated divergence between generated and target distributions.

#### Key Mathematical Components

The research tracks two primary quantities:
- **Accumulated divergence**: Dᵢ = χ²(p̂ⁱ∥p_data) measuring divergence between the i-th generation model and true data distribution
- **Intra-generation divergence**: Iᵢ = χ²(p̂ⁱ⁺¹∥qᵢ) measuring error introduced in one training round

The training mixture is defined as:
qᵢ = α·p_data + (1-α)·p̂ⁱ

where α represents the proportion of fresh samples and (1-α) represents synthetic samples.

### Fundamental Theoretical Results

#### Intra-generation Divergence Control

The research establishes a crucial two-sided bound for intra-generation divergence:

**Theorem 3.4**: Under appropriate assumptions, for sufficiently small score errors:
χ²(p̂ⁱ⁺¹∥qᵢ) ≍ ε⋆,ᵢ²

This shows that intra-generational divergence is controlled by pathwise score-error energy and they are equivalent up to constants, accounting for observability and tail effects.

#### Error Observability Coefficient

A key innovation is the **observability coefficient** ηᵢ, which quantifies how visible score errors are at the endpoint:

ηᵢ = Varℙᵢ⋆(𝔼[M_Tᵢ∣Y_{t₀}ⁱ,⋆]) / ε⋆,ᵢ²

When ηᵢ > 0, score errors leave detectable imprints on the terminal distribution, which is typically the case for state-dependent perturbations in practical neural networks.

#### Long-term Error Accumulation

The research identifies two distinct regimes for error accumulation:

**Proposition 4.1**: 
1. If ∑ᵢ≥₀ ε⋆,ᵢ² = ∞, then ∑ᵢ≥₀ Dᵢ = ∞ and (Dᵢ) cannot converge to 0
2. If ε⋆,ᵢ² ≥ ε̄ > 0 for large i, then lim supᵢ→∞ Dᵢ ≥ α·η̄·ε̄/(16(1+(1-α)²))

For controlled errors where ∑ε⋆,ₖ² < ∞, the accumulated divergence follows a **geometrically-discounted decomposition**:

χ²(p̂ᴺ⁺¹∥p_data) + C_bias ≍ ∑ₖ=₀ᴺ (1-α)²⁽ᴺ⁻ᵏ⁾ · ε⋆,ₖ²

This shows that errors from m generations in the past are suppressed by a factor (1-α)²ᵐ, quantifying the effect of fresh data proportion α.

### Related Theoretical Developments

#### Modular Error Framework (arXiv:2308.05021)

Foundational work establishes that diffusion models suffer from error propagation through three interconnected elements:
1. **Modular error**: errors within individual components
2. **Cumulative error**: accumulated errors across the model
3. **Propagation equation**: relates modular and cumulative errors

This framework provides theoretical confirmation that DMs are indeed affected by error propagation, unlike some sequential models like Conditional Random Fields.

#### Recursive Stability (arXiv:2502.18865)

The concept of **recursive stability** provides the first theoretical generalization analysis explaining discrepancies in STL outcomes. Key findings include:
- Model architecture significantly influences STL success
- Even constant-sized proportions of real data ensure convergence
- The framework extends to transformers in in-context learning
- Provides insights into optimal synthetic data sizing

#### Data Accumulation vs. Replacement (arXiv:2404.01413)

This research demonstrates a crucial distinction:
- **Data replacement**: Leads to model collapse as test error increases with iterations
- **Data accumulation**: Results in finite upper bounds on test error, preventing collapse

Using an analytically tractable framework with linear models, the authors prove that when data accumulate, the test error has a finite upper bound independent of iteration count, meaning model collapse no longer occurs.

### Practical Implications

#### Fresh Data Proportion Optimization

The theoretical results provide guidance for optimizing the fresh data proportion α:
- **Low α (α ≈ 0.1)**: Errors persist across many generations, creating wide bands of contributions
- **Medium α (α ≈ 0.5)**: Intermediate regime with moderate memory decay  
- **High α (α ≈ 0.9)**: Short memory structure where only recent generations dominate

#### Model Architecture Considerations

The research suggests that model architecture plays a crucial role in determining recursive stability. architectures with better error observability (ηᵢ > 0) and lower score error energies ε⋆,ᵢ² will be more stable under recursive training.

#### Mitigation Strategies

Several strategies emerge from the theoretical analysis:
1. **Data accumulation**: Rather than replacing, accumulate synthetic data alongside real data
2. **Fresh data injection**: Maintain sufficient α > 0 to ensure geometric error decay
3. **Score error minimization**: Reduce ε⋆,ᵢ² through improved score estimation techniques
4. **Architecture design**: Choose architectures with favorable observability properties
