# 复杂度计算公式

## 完整公式

```
复杂度分数 = 输入文件数 + 任务类型系数 + 模型系数 + (依赖数 × 0.5) + (关键词数 × 0.5)
```

## 系数表

### 任务类型系数
| 任务类型 | 系数 |
|---------|-------|
| automation | 1 |
| research | 2 |
| analyst | 3 |
| creative | 1 |

### 模型系数
| 模型 | 系数 |
|------|-------|
| zai/glm-4.5 | 0 |
| zai/glm-4.7 | 1 |
| zai/glm-4.7-flash | 0 |

## 分数 → 等级 → 时间 → 模型

| 分数范围 | 等级 | 预估时间 | 推荐模型 |
|---------|-------|----------|---------|
| ≤2 | 1 | 3-8 分钟 | GLM-4.5 |
| 3-4 | 2 | 8-18 分钟 | GLM-4.5 |
| 5-7 | 3 | 15-35 分钟 | GLM-4.7 |
| >7 | 4 | 30-50 分钟 | GLM-4.7 + 分解 |

## 示例计算

### 示例 1：简单任务
- 输入文件：0 个
- 任务类型：automation (系数 1)
- 模型：GLM-4.5 (系数 0)
- 依赖数：0
- 关键词数：0

**分数 = 0 + 1 + 0 + (0 × 0.5) + (0 × 0.5) = 1**
**等级：1 → 时间：3-8 分钟 → 模型：GLM-4.5**

### 示例 2：中等任务
- 输入文件：2 个
- 任务类型：analyst (系数 3)
- 模型：GLM-4.7 (系数 1)
- 依赖数：1
- 关键词数：2

**分数 = 2 + 3 + 1 + (1 × 0.5) + (2 × 0.5) = 7**
**等级：3 → 时间：15-35 分钟 → 模型：GLM-4.7**

### 示例 3：复杂任务
- 输入文件：4 个
- 任务类型：analyst (系数 3)
- 模型：GLM-4.7 (系数 1)
- 依赖数：2
- 关键词数：3

**分数 = 4 + 3 + 1 + (2 × 0.5) + (3 × 0.5) = 9.5**
**等级：4 → 时间：30-50 分钟 → 模型：GLM-4.7 + 分解**
