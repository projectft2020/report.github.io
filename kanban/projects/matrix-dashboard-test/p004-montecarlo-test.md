### 1.2 測試方法

**幾何布朗運動模型 (GBM):**

```
dS = μS dt + σS dW

其中:
- S: 資產價格
- μ: 預期收益率
- σ: 波動率
- dW: 維納過程增量 (標準布朗運動)
- dt: 時間步長
```

**離散化實現:**

```
S(t+dt) = S(t) × exp((μ - 0.5σ²)dt + σ√dt × Z)

其中 Z ~ N(0, 1) 為標準正態隨機變量
```

### 11.2 數學參考

**幾何布朗運動 (GBM) 解析:**

```
S(t) = S0 × exp((μ - 0.5σ²)t + σW(t))

其中 W(t) 是維納過程:
W(t) ~ N(0, t)
```

**期望值與方差:**

```
E[S(t)] = S0 × exp(μt)
Var[S(t)] = S0² × exp(2μt) × (exp(σ²t) - 1)
```
