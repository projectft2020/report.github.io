## 研究摘要

本研究針對離散狀態空間的生成建模，提出了一個新穎的離散去噪擴散框架。該框架結合了樣本高效估計器（NeurISE）用於單點條件機率，以及輪循噪聲和去噪動力學，為離散生成建模提供了一個更高效的解決方案。實驗結果表明，該方法在多個基準測試中優於現有的 ELBO 和基於 score 的方法，包括 D3PM 和 SEDD。

**核心貢獻：**
- 將單點條件機率作為參數化逆擴散過程的基礎對象，而非近似離散 score 函數
- 使用 NeurISE 估計擴散動力學中的條件機率
- 提供了總變差誤差傳播界限，量化局部不準確性在採樣步驟中的累積
- 證明了在輪循噪聲方案下，自回歸生成作為硬噪聲極限的逆時採樣器

## 主要發現

### 1. 核心技術創新

**單點條件機率方法**
- 傳統離散擴散模型嘗試近似離散 score 函數
- 本文直接將單點條件機率 P(σu|σ-u) 作為參數化對象
- 這種方法更自然地契合離散空間的結構，避免了連續鬆弛的問題

**NeurISE（神經交互篩選估計器）**
- 原創於 Jayakumar 等人 (2020)
- 樣本高效且近似高效的離散條件機率學習方法
- 通過神經網絡參數化部分能量函數來學習局部條件機率

**輪循噪聲方案**
- 每個時間步只選擇一個坐標進行噪聲添加
- 相比同時對所有變量添加噪聲的方法，需要學習的比率數量更少
- 提供了從擴散到自回歸生成的自然橋樑

### 2. 理論貢獻

**總變差誤差傳播定理**
對於近似逆鏈的輸出分佈，誤差界限為：

```
||μ̂₀ - μ₀||_TV ≤ δ_T (混合誤差) + T·η (逆核估計誤差)
```

其中：
- δ_T: 前向過程混合到噪聲分佈的誤差
- η: 每個逆核估計的誤差上界
- T: 總時間步數

**關鍵洞察：**
- 擴散模型的採樣難度來自兩個因素：(i) 學習誤差，(ii) 前向過程的混合性
- 這與 Langevin 或 Glauber 動力學 MCMC 採樣器不同，後者收斂速率更直接地依賴於數據分佈的性質

### 3. 自回歸極限

**硬噪聲極限（ε = 0）**
- 當噪聲參數 ε = 0 時，每個坐標完全隨機化
- 逆過程在這種極限下恢復為自回歸生成
- 每個逆步驟變成「從單點條件機率重新採樣一個坐標」
- 按固定順序展開更新得到自回歸採樣

**與現有工作的比較：**
- Ou 等人 (2024) 從目標層面連接吸收擴散與任意階自回歸模型
- 本文提供了更直接的有限步採樣器崩塌，通過特定輪循構建

## 詳細分析

### 3.1 離散擴散框架

**問題設定**
- 設 Σ 為離散字母表，|Σ| = p
- 配置空間 Σ^q，包含 q 個離散變量或坐標
- 目標：從分佈 μ 的訓練樣本構建擴散生成模型

**前向過程（輪循噪聲）**
- 固定隨機化參數 ε ∈ [0,1]
- 在每個時間步 n ∈ {1,…,T}，以輪循順序選擇坐標 u = ((n-1) mod q) + 1
- 以概率 ε 保持坐標不變，以概率 1-ε 從 Σ 均勻隨機採樣

前向轉移核為：

```
k_n(σ, σ̃) = {
  (1-ε)/p, if σ_-u = σ̃_-u, σ_u ≠ σ̃_u
  (1-ε)/p + ε, if σ = σ̃
  0, otherwise
}
```

**二進制情況（Σ = {-1,1}）**

```
k_n(σ, σ̃) = {
  (1-ε)/2, if σ_-u = σ̃_-u, σ_u ≠ σ̃_u
  (1+ε)/2, if σ = σ̃
  0, otherwise
}
```

### 3.2 逆過程

**貝葉斯規則構建**

逆轉移核通過貝葉斯規則構建：

```
k_n^{rev}(σ, σ̃) = k_n(σ̃, σ) · μ_n(σ) / Σ_{σ̂∈Σ^q} k_n(σ̃, σ̂) · μ_n(σ̂)
```

這可以簡化為條件機率的比率：

```
μ_n(σ̃)/μ_n(σ) = μ_n(σ̃_u | σ_-u) / μ_n(σ_u | σ_-u)
```

**二進制情況的逆轉移**

```
k_n^{rev}(σ, σ̃) = (1-ε) · μ_n(σ) / [(1-ε)μ_n(σ) + (1+ε)μ_n(σ̃)]
```

### 3.3 使用 NeurISE 學習條件機率

**中心指示嵌入**

對於每個符號 s, r ∈ Σ，定義：

```
Φ_s(r) = {
  1 - 1/q, if r = s
  -1/q, if r ≠ s
}
```

**能量分解假設**

假設 μ_n(σ) ∝ exp(H(σ))，其中 H: Σ^q → ℝ 是哈密頓量函數。

對於任意坐標 u，存在分解：

```
H(σ) = H_-u(σ_-u) + H_u(σ)
```

其中 H_-u 不依賴於 σ_u。

**條件機率表示**

```
μ_n(σ_u | σ_-u) = exp(H_u(σ)) / Σ_{r∈Σ} exp(H_u(σ_u=r, σ_-u))
```

**NeurISE 參數化**

使用神經網絡 NN_θ: ℝ × ℝ^q × ℝ^{q-1} → ℝ^p 接受參數 (t, u, σ_-u)：

```
Ĥ_u(σ; w) = ⟨Φ(σ_u), NN_θ(t, u, σ_-u)⟩
```

**損失函數**

對於每個位置 u：

```
ℒ_u(θ) = (1/TN) Σ_{s=1}^T Σ_{n=1}^N exp(-⟨Φ((X_s^n)_u), NN_θ(t, u, (X_s^n)_-u)⟩)
```

**學習的條件分佈**

```
μ̂_n(σ_u | σ_-u) = exp(⟨Φ(σ_u), NN_u(σ_-u)⟩) / Σ_{r∈Σ} exp(⟨Φ(r), NN_u(σ_-u)⟩)
```

### 3.4 理論分析

**定理 1：誤差傳播界限**

設 {X_n}_{n=0}^T 為 Σ^q 上的馬爾可夫鏈，前向轉移核為 k_n。假設存在 δ_T ∈ [0,1] 使得：

```
||μ_T - μ_noise||_TV ≤ δ_T
```

設 {k̂_n^{rev}}_{n=0}^{T-1} 為近似逆核，使得對所有 n：

```
sup_{σ∈Σ^q} ||k̂_n^{rev}(·, σ) - k_n^{rev}(·, σ)||_TV ≤ η
```

則輸出分佈滿足：

```
||μ̂_0 - μ_0||_TV ≤ δ_T + Tη
```

**推論：初始化誤差**

如果從近似噪聲分佈 μ̂_noise 初始化，使得：

```
||μ̂_noise - μ_noise||_TV ≤ γ
```

則：

```
||μ̃_0 - μ_0||_TV ≤ δ_T + Tη + γ
```

**實際含義：**

1. **混合誤差（δ_T）**：前向過程向噪聲分佈混合的程度
2. **逆核估計誤差（η）**：每個逆步驟的學習誤差
3. **噪聲採樣誤差（γ）**：從噪聲分佈採樣的誤差

這解釋了為什麼屏蔽擴散模型在實踐中表現更好：吸收狀態可以設置 γ = 0。

### 3.5 實驗結果

#### 實驗 1：Edwards-Anderson (EA) 模型
- **數據集：** 25 個二進制變量的 EA 模型（5×5 格子）
- **評估指標：** 總變差（TV）、交叉相關誤差、訓練時間
- **結果：**
  - NeurISE 擴散顯示最快的 TV 距離衰減
  - 優於 SEDD 方法
  - D3PM 在低樣本數時表現良好，但隨訓練集增加性能下降
  - 交叉相關誤差顯示類似趨勢
  - 硬噪聲情況在小訓練集下均勻優於其他方案

#### 實驗 2：MNIST
- **數據集：** 二值化 MNIST（784 像素）
- **評估指標：** MMD、平均交叉相關
- **結果：**
  - NeurISE 學習的條件機率達到最低 MMD 和交叉相關誤差
  - 與 EA 模型相比，D3PM 達到更好的比較性能
  - 這表明 D3PM 擅長重現數據集的低階投影，但在真實分佈學習方面，NeurISE 擴散成功

#### 實驗 3：D-Wave 數據集
- **數據集：** D-Wave Advantage 量子退火器產生的數據（2000 量子位）
- **訓練樣本：** 8×10^4
- **測試樣本：** 2×10^4
- **結果：** NeurISE 擴散在所有指標中顯示優勢

#### 實驗 4：多字母 Potts 模型
- **設置：** EA 模型的 Potts 版本，Σ = {0,1,…,p-1}
- **測試配置：** L=2（q=4）和 L=3（q=9）
- **結果：** TV 誤差隨訓練樣本數增加而下降

#### 實驗 5：GHZ 狀態的量子層析成像
- **數據集：** 四結果測量（p=4）的 GHZ 狀態模擬
- **系統：** 20 個量子位
- **評估指標：** 推廣到多字母情況的交叉相關
- **結果：** 交叉相關誤差在 10^4 個樣本後顯著下降，表明模型能夠學習這個量子狀態的忠實生成模型

## 局限性分析

### 1. 計算複雜度
- **問題：** 每個時間步需要學習每個坐標的條件機率
- **影響：** 對於高維數據（如 MNIST 的 784 像素），計算成本可能顯著
- **對比：** 與同時更新所有坐標的方法相比，輪循方案在每步需要估計的參數更少，但需要更多步驟

### 2. 網絡架構依賴性
- **問題：** 實驗中使用了 MLP 作為基礎架構
- **影響：** 更複雜的架構（如 Transformer）可能帶來不同結果
- **未來方向：** 探索不同架構對性能的影響

### 3. 噪聲參數 ε 的選擇
- **問題：** 實驗未顯示軟噪聲顯著優於硬噪聲
- **影響：** 噪聲參數的最優選擇仍不明確
- **未來方向：** 系統研究 ε 對性能的影響

### 4. 大規模驗證
- **問題：** 主要實驗在相對小的系統上進行（25 變量、2000 量子位）
- **影響：** 在更大規模（如大型語言模型）上的泛化能力未知
- **未來方向：** 在更大規模數據集上驗證

### 5. 與其他方法的理論比較
- **問題：** 與 Ou 等人 (2024) 的比較主要是概念層面的
- **影響：** 實際性能比較需要更多實驗
- **未來方向：** 系統比較不同離散擴散方法

### 6. 訓練穩定性
- **問題：** 條件機率估計的訓練穩定性未詳細討論
- **影響：** 實際應用中可能需要額外的正則化技術
- **未來方向：** 研究訓練動態和穩定性問題

### 7. 類別不平衡處理
- **問題：** 未明確討論如何處理類別不平衡數據
- **影響：** 在現實世界不平衡數據集上可能表現不同
- **未來方向：** 探索不平衡數據的處理策略

## 與現有方法的比較

### 與 D3PM (Austin et al., 2021) 的對比

**D3PM 特點：**
- 優化變分下界（VLB）對數似然
- 支持吸收和均勻擴散
- 同時噪聲所有變量

**比較：**
- NeurISE 擴散在 EA 模型上表現更好
- D3PM 在 MNIST 上表現接近
- D3PM 更適合低階投影（如 MMD），NeurISE 擴散更適合真實分佈學習

### 與 SEDD (Lou et al., 2024) 的對比

**SEDD 特點：**
- 學習離散版本的 score
- 使用 score-entropy 函數確保 score 的非負性
- 配置需要 one-hot 編碼

**比較：**
- NeurISE 擴散在所有測試中優於 SEDD
- NeurISE 擴散不需要 one-hot 編碼
- NeurISE 擴散直接學習條件機率，更自然

### 與 RADD (Ou et al., 2024) 的對比

**RADD 特點：**
- 連接吸收擴散與任意階自回歸模型
- 在無窮噪聲極限下，吸收擴散損失等於 AO-ARM 訓練目標
- 通過代數變換和訓練損失極限建立等價性

**比較：**
- NeurISE 擴散提供更直接的有限步採樣器崩塌
- 通過特定輪循構建，而非代數變換
- 兩者都揭示了擴散和自回歸生成之間的關係

## 技術細節深入

### NeurISE 的樣本效率

**原理：**
- 通過中心指示嵌入將離散狀態映射到實向量空間
- 條件機率的比率可以通過能量函數的內積表示
- 避免了直接估計高維聯合分佈

**優勢：**
1. 局部性：只需學習局部條件機率
2. 效率：能量函數的參數化更高效
3. 通用性：適用於任意離散字母表

### 輪循方案的效率

**相比同時噪聲方案：**
- 每步只需學習一個坐標的條件機率
- 總共需要學習的參數數量更少
- 更容易在高維空間中擴展

**潛在缺點：**
- 需要更多時間步（至少 q 步才能覆蓋所有坐標）
- 採樣時間可能增加

### 硬噪聲極限的自回歸性質

**證明：**
1. 當 ε = 0，每個坐標完全隨機化
2. 逆過程從均勻分佈開始
3. 每個步驟從條件機率採樣一個坐標
4. 展開後得到自回歸因子化

**意義：**
- 提供了擴散和自回歸的統一視角
- 解釋了為什麼擴散模型在某些情況下接近自回歸性能

## 結論

本文提出了一個新穎的離散擴散框架，通過結合 NeurISE 和輪循噪聲方案，為離散狀態空間的生成建模提供了樣本高效的解決方案。理論分析提供了清晰的誤差傳播界限，實驗結果在多個基準測試中顯示出優於現有方法的性能。

**主要優勢：**
- 樣本效率高，特別在小訓練集下
- 理論基礎紮實，誤差分析清晰
- 自然保持離散結構
- 連接擴散和自回歸生成

**主要局限：**
- 計算複雜度在高維數據上可能較高
- 需要更多在大規模數據集上的驗證
