## Research Summary

本研究分析了由挪威奧斯陸大學研究團隊於 2026 年 2 月發表的論文「Dirichlet Scale Mixture Priors for Bayesian Neural Networks」（貝葉狄利克雷特尺度混合先驗）。該論文提出了一種新型的貝葉神經網絡先驗分佈類別（DSM），透過三層層級的結構化稀疏誘導機制，解決了當前 BNN 中先驗設定困難的問題。研究顯示 DSM 先驗能在相關的中小數據場景中有效產生稀疏網絡、提升對抗性攻擊的穩健性，並在保持競爭性預測性能的同時大幅減少有效參數數量。

## Detailed Analysis

### 1. 研究背景與問題陳述

貝葉神經網絡（Bayesian Neural Networks，BNNs）透過在網絡參數上放置先驗分佈 p(Θ)，並推斷後驗分佈 p(Θ|y)，將推斷擴展至單點估計之外。然而，BNN 面臨一個核心困難：如何選擇適當的先驗分佈。

由於指定先驗分佈的困難性，許多 BNN 應用為求方便而選擇最簡單的先驗——各向同性高斯分佈 p(Θ) = N(0,α²I)，其中 α 通常被選擇為層寬平方根的倒數。高斯先驗被認為是「無信息」的，且具有便於抽樣的特性。

然而，近期研究發現：
- **冷後驗效應（Cold Posterior Effect）**：Wenzel 等人 (2020) 和 Tran 等人 (2022) 發現後驗降溫顯著改善性能，這表明可能似然函數或先驗分佈設定不當。
- **先驗的關鍵作用**：Fortuin 等人 (2022) 研究表明全連接層經隨機梯度下降法訓練後呈現重尾分佈，建議在 BNN 中反映這種性質。
- **稀疏性的重要性**：Lottery Ticket Hypothesis (Frankle 和 Carbin, 2019) 表明過參數化網絡中存在性能相當的子網絡，獲取這些網絡雖非易事。

### 2. DSM 先驗的核心技術框架

#### 2.1 階層結構定義

Dirichlet Scale Mixture (DSM) 先驗將經典的「全局-局部」尺度混合先驗擴展至三層層級結構：

```
w_jk | τ, λ_j, ξ_jk ~ N(0, τ²λ_j²ξ_jk)
(ξ_j1, ..., ξ_jp) ~ Dir(α₁, ..., α_p) ∈ Δ^(p-1)
λ_j ~ P_λ
τ ~ P_τ
```

其中：
- **τ（全局尺度）**：控制整體收縮強度，τ > 0
- **λ_j（組別尺度）**：每個參數組的特定尺度，λ_j > 0
- **ξ_j（狄利克雷特分配向量）**：單體值隨機向量，滿足 Σₖ=1ᵖ ξ_jk = 1
- **α（濃度參數）**：控制組內稀疏程度，當對稱時 α_k = α

#### 2.2 神經網絡中的分組策略

DSM 先驗的核心創新在於利用狄利克雷特分佈以「對神經網絡結構自然」的方式分配固定方差預算：

- **在線性迴歸中**：由於沒有可類比的架構分組，將每個係數視為獨立組，並分配個別局部尺度。
- **在神經網絡中**：有意義的分組自然來自架構本身。所有映射到同一隱藏層節點的權重共享組別尺度 λ_j，並使用狄利克雷特組件 ξ_jk 控制輸入權重之間的方差分配。

這種設計的直覺是：整個節點可透過組別尺度被強烈收縮，而組內的個別權重在局部層級被進一步收縮。

#### 2.3 Dirichlet Student's t 先驗變體

論文主要關注使用 half-Cauchy 分佈作為 τ 的先驗，並讓 λ_i 遵循 half-Student's t 分佈：

```
w_jk | τ, λ_j, ξ_jk ~ N(0, τ²λ_j²ξ_jk)
(ξ_j1, ..., ξ_jp) ~ Dir(α, ..., α) ∈ Δ^(p-1)
λ_j ~ t⁺_ν(0, 1)
τ ~ C⁺(0, τ₀²)
```

- 當 ν 較小時，分佈呈現重尾特性，對大多數抽樣強烈收縮，同時允許少數取大值。
- 當 ν 增大時，尾變輕，收縮更趨向均勻。
- ν = 1 時，等價於 Dirichlet horseshoe 先驗（Nagel 等人, 2024）。

#### 2.4 軟正則化技術

重尾尺度先驗（如 half-Cauchy 和低自由度 Student's t）可能產生極大的局部尺度，導致數值不穩定和後驗抽樣混合緩慢。論文採用了 Piironen 和 Vehtari (2017) 的軟正則化技術：

```
λ̃_j² = c²λ_j² / (c² + τ²λ_j²)
c² ~ InvGamma(a, b)
```

其中超參數 c 控制截斷程度：
- 當 c² ≫ τ²λ_j² 時，先驗約化為原始重尾形式。
- 當 c² ≪ τ²λ_j² 時，局部變異數受 c² 邊界。

實驗中設定 ν_c² = 4, s² = 2，對應於中度重尾的 slab 分佈。

### 3. 理論性質

#### 3.1 相關性結構分析

研究透過變異數組件分析 DSM 先驗誘導的相關性：

權重 w_jk 的先驗變異數為：
```
Var(w_jk | λ̃_j², ξ_j) = τ²λ̃_j² ξ_jk
```

透過定義 V_jk = λ̃_j² ξ_jk 並計算協方差，研究導出：

```
Cov(V_jk, V_jl) = [1 / (p²(pα + 1))] × [αp Var(λ̃_j²) - E[λ̃_j²]²]
```

相關性的符號由變異數係數 CV²(λ̃_j²) 決定：

- **負相關**：當 CV²(λ̃_j²) < 1/(pα) 時
- **零相關**：當 CV²(λ̃_j²) = 1/(pα) 時
- **正相關**：當 CV²(λ̃_j²) > 1/(pα) 時

重要發現：
- 重尾先驗傾向增加變異數，從而促進正相關。
- 輕尾先驗偏愛負相關。
- Dirichlet-Laplace 先驗（Bhattacharya 等人, 2015）是特殊情況，其中 CV²(λ̃_j²) = 1/(pα)，恢復獨立且不相關的情況。

#### 3.2 線性迴歸中的收縮行為

論文首先在線性迴歸設定中研究收縮行為：

**標準 Horseshoe 先驗的收縮因數**：
```
κ_j = 1 / (1 + nσ⁻²τ²s_j²λ_j²)
```

其中 z_j = √n σ⁻¹τ s_j。κ_j = 1 表示完全收縮，κ_j = 0 表示無收縮。

**有效非零係數數量**：
```
m_eff = Σⱼₖ₁ᵖ (1 - κ_j)
```

這實際計算 κ_j 接近零的次數，可用作為有效模型大小的指標。

**Dirichlet Horseshoe 的收縮**：
研究導出了 Dirichlet Horseshoe 先驗下 κ_j 的邊際先驗分佈，並證明其收縮比標準 horseshoe 更激烈（更多質量集中在 κ ≈ 1 附近）。

#### 3.3 神經網絡中的矩陣值收縮

研究將純量收縮分析推廣至神經網絡。考慮單隱藏層 BNN 並在參考點 (𝐰₁,₀, 𝐛₁,₀, 𝐰_L,₀, b_L,₀) 附近線性化：

**線性化後的高斯模型**：
```
y* | 𝐰₁ ~ N(J_𝐰 𝐰₁, Σ_y)
𝐰₁ ~ N(0, τ²Ψ)
```

其中：
- Ψ ∈ ℝ^(pH × pH) 是對角矩陣，條目 Ψ_(k,j),(k,j) = λ_j² ξ_kj
- J_𝐰 是網絡輸出對向量化輸入權重的雅可比矩陣
- Σ_y 是線性化似然的邊際共變異數

**收縮矩陣**：
```
K = (P + S)⁻¹S = I - (P + S)⁻¹P
```

其中 P = τ⁻²Ψ⁻¹，S = J_𝐰⊤Σ_y⁻¹J_𝐰。

這個矩陣 K 推廣了線性迴歸中的純量收縮因數 κ，支配著普通最小二乘估計量 ĥ𝐰 的收縮。

**廣義特徵方向上的收縮**：
透過將矩陣 G = P⁻¹/²SP⁻¹/² 對角化為 G = UΩU⊤，其中 Ω = diag(ω₁, ..., ω_pH)，收縮沿著廣義特徵向量作用：

```
κ(𝐮_j) = 1 / (1 + τ²ψ_eff²(𝐮_j))
ψ_eff²(𝐮_j) = (𝐮_j⊤Ψ⁻¹𝐮_j)⁻¹
```

每個廣義特徵方向 𝐮_j 類比於具有有效局部尺度 ψ_eff 和有效數據項 𝐮_j⊤S𝐮_j 的純量問題。

**跡蹟對比**：
```
tr((P + S)⁻¹S) = tr((I + G)⁻¹G) = Σⱼₖ₁ᴴH [ω_j / (1 + ω_j)]
```

這提供了對 m_eff 的直接類比，並在非對角 S 的情況下仍然有效。

研究指出：
- 神經網絡中，收縮沿廣義特徵方向而非座標方向作用
- 由於每個隱藏單元僅依賴於自己的輸入權重，Jacobian 具有單元結構
- S 矩陣接近塊對角，塊對應於饋入單個隱藏節點的權重集合
- 許多廣義特徵向量 𝐮_j 實際上局域於單個隱藏單元內

### 5. 應用價值評估

#### 5.1 主要優勢

1. **結構化的稀疏誘導**：三層層級的正則化（全局、組別、局部）提供了比單層結構更豐富的正則化機制，能夠在神經網絡的自然分組結構上操作。

2. **隱式特徵選擇**：透過狄利克雷特分配的競爭機制，先驗自動在組內的權重之間進行特徵選擇，無需人工干預。

3. **減緩冷後驗效應**：重尾先驗機制與近期發現一致，能夠減緩冷後驗效應，提供比常用高斯先驗更原則性的替代方案。

4. **更適合剪枝**：DSM 先驗產生的網絡具有更清晰的分組結構，使得權重剪枝更容易實施，從而減少模型大小和計算成本。

5. **對抗性穩健性**：稀疏性和正則化通常能夠提升模型對對抗性攻擊的抵禦能力，DSM 先驗在實驗中展現了這種特性。

6. **中小數據場景優勢**：在相關的中小數據場景中，DSM 先驗的優勢最為顯著，這些場景通常是實際應用中常見的。

#### 5.2 實際應用場景

1. **資源受限環境**：在計算資源受限的設備上部署模型時，稀疏網絡可以顯著降低存儲和計算需求。

2. **邊緣計算**：移動和嵌入式設備上的機器學習應用受益於稀疏模型。

3. **高風險決策**：金融、醫療等領域需要不僅準確的預測，還需要估計不確定性。BNN 結合後驗框架提供自然的不確定性量化，DSM 先驗的稀疏性進一步提升解釋性。

4. **對抗性防禦**：自動駕駛、金融詐欺檢測等需要檢測異常輸入的場景中，模型穩健性至關重要。

### 6. 局限性與挑戰

#### 6.1 計算複雜度

1. **三層層級層次化**：相較於簡單的高斯先驗，DSM 先驗引入了額外的超參數（τ、各組的 λ_j、狄利克雷特組件 ξ_j），增加了模型複雜度和調優難度。

2. **後驗抽樣挑戰**：三層層級的層次結構和狄利克雷特分配約束可能導致後驗抽樣的混合緩慢，特別是在大規模神經網絡中。

3. **超參數敏感性**：濃度參數 α、自由度 ν、正則化參數 c 等需要小心調優，不當的設置可能影響模型性能。

#### 6.2 理論挑戰

1. **近似誤差**：神經網絡的線性化近似在某些情況下可能不準確，特別是在非線性區域或深度網絡中。

2. **相關性解釋**：誘導的負相關或正相關的實際影響和物理意義需要更深入的研究。

3. **收縮理論推廣**：從線性迴歸到神經網絡的收縮理論推廣需要更強的假設，實際應用中可能不完全滿足。

#### 6.3 實踐限制

1. **實驗範圍限制**：論文實驗主要集中在線性迴歸、單隱藏層 BNN 和特定數據集，在深度網絡、大型語言模型等現代架構上的驗證有限。

2. **性能基準**：雖然 DSM 先驗在稀疏性和穩健性方面表現優異，但在預測精度上是否一致優於其他方法需要更廣泛的實驗驗證。

3. **實現複雜度**：DSM 先驗的實現比標準高斯先驗更複雜，需要開發新的後驗抽樣算法或變分推斷方法。

### 7. 未來發展方向

#### 7.1 理論研究方向

1. **更深入的收縮理論**：建立更精確的神經網絡收縮理論，減少對線性化近似的依賴。
