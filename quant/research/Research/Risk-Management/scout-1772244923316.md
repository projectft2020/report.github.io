## 研究摘要

本論文《Amortized Bayesian inference for actigraph time sheet data from mobile devices》由 UCLA 的 Daniel Zhou 和 Sudipto Banerjee 撰寫，於 2026 年 2 月發表於 arXiv（論文 ID: 2602.20611）。該研究提出了一種將攤銷貝葉斯推斷（Amortized Bayesian Inference, ABI）應用於可穿戴設備加速度計時間表數據的新型框架。論文結合了分層動態線性模型（Hierarchical Dynamic Linear Model）和 BayesFlow 神經網絡框架，實現了對來自 PASTA-LA 研究的真實加速度計數據的高效分析和推斷。該方法不僅能夠進行概率性數據插補，還能夠統計學習解釋變量對加速度幅值（MAG）的時變影響。

## 主要發現

1. **攤銷貝葉斯推斷框架** — 論文成功將 BayesFlow 的 ABI 方法應用於可穿戴設備數據分析，通過在合成數據上訓練可逆神經網絡（INNs），實現了快速後驗採樣，避免了為每個新數據集重新運行昂貴的貝葉斯推斷程序 | 來源: [arXiv:2602.20611](https://arxiv.org/abs/2602.20611)

2. **加速度計時間表格式** — 剛新設計的「加速度計時間表」數據結構能夠高效組織主體、日期和時間點的三維數據，支持基於動態線性模型的離散時間分析，並提供概率性插補框架 | 來源: [arXiv:2602.20611](https://arxiv.org/abs/2602.20611)

3. **分層動態線性模型** — 論文採用帶有前向濾波後向採樣（FFBS）算法的動態線性模型來建模加速度幅值（MAG）的時變模式，利用共軛性實現高效推斷 | 來源: [arXiv:2602.20611](https://arxiv.org/abs/2602.20611)

4. **PASTA-LA 數據集應用** — 研究基於 UCLA 的 PASTA-LA 研究（460 個體，2017-2018 年），通過預處理將數據轉化為 5-20 分鐘的軌跡，將稀疏性從 9.0% 降至 45.2% 的覆蓋率 | 來源: [arXiv:2602.20611](https://arxiv.org/abs/2602.20611)

5. **遷移學習能力** — 訓練後的神經網絡能夠推廣到應用領域內的不同數據集而無需過度擬合，實現了真正的遷移學習，適用於實時健康監護系統 | 來源: [BayesFlow 文檔](https://bayesflow.org/main/user_guide/introduction.html)

## 詳細分析

### 1. 研究背景與動機

**可穿戴設備的興起：**
- 可穿戴技術的快速發展推動了健康數據庫的建立，這些數據庫包含用於研究活動模式和健康結果的人體運動數據
- 加速度計是小型便攜式設備，配備運動傳感器或加速度計，能夠檢測身體在不同軸線上的加速度並收集大量數據
- 這些設備越來越普及，因其成本較低、準確性高，並且廣泛應用於智能手機、智能手表和其他可穿戴設備中

**統計挑戰：**
- 高分辨率加速度計數據的統計分析方法取決於具體的推斷背景
- 傳統貝葉斯推斷需要為每個新數據集重複計算，計算成本高昂
- 人工智能框架的興起要求方法必須與遷移學習和攤銷計算相一致

**科學目標：**
- 建立一個可解釋的人類交互系統，考慮環境因素並預先構建鍛煉路徑
- 提供數值基準供人們作為鍛煉目標
- 減少或消除預先決定鍛煉路線的開銷

### 2. 數據預處理與加速度計時間表

**PASTA-LA 研究概覽：**
- 研究機構：UCLA Fielding School of Public Health
- 研究對象：460 名受試者
- 數據收集期間：2017 年 5 月至 2018 年 6 月
- 地點：洛杉磯 Westwood 區
- 數據來源：在線問卷、GPS 設備（Global-Sat DG-500）、便攜式加速度計單元（Actigraph GT3X+）
- 結果數據結合：生物和生計因素（年齡、身高、體重、種族、性別、BMI）、地理空間坐標、時間測量、地理測量（緯度、經度、高度、距離家庭/工作場所的距離）以及 MAG 加速度計測量

**數據預處理步驟：**

1. **軌跡定義：**
   - 當一個軌跡的結束時間和同一日期下一個軌跡的開始時間之間大於 3 分鐘時，將其定義為不同的軌跡
   - 過濾掉短於 5 分鐘的軌跡，以最小化短暫活動的影響（如短距離步行）
   - 過濋掉長於 22 分鐘的軌跡
   - 將剩餘軌跡在 20-22 分鐘之間的截斷至 20 分鐘標記

2. **活動閾值：**
   - 過濋掉 MAG 小於 0.05 的時間點，以防止少量不屬於運動的活動（如駕駛或在鄰裡騎行電動滑板車）
   - 最終考慮的運動軌跡包括 5-20 分鐘的活動，以 20 秒為時間步長，以相對一致的速率進行至少最小活動

3. **數據變換：**
   - 將 MAG 對數化（log(MAG)），因為 MAG 本質上是非負的，而動態線性模型設計用於正負結果
   - 移除小於 0.05 的 MAG 也解決了取對數的數值問題（即 log(0)）

4. **稀疏性改善：**
   - 在絕對時間步長處理下，數據將覆蓋每個受試者覆蓋日期中所有可能時間步長的約 9.0%
   - 在相對時間步長設置下，數據將覆蓋考慮的所有時間點的 45.2%
   - 此設置將受試者數量減少至 92 人

**加速度計時間表格式：**

時間表的核心創新是將受試者與他們活動的日期配對，這樣我們只記錄我們期望他們活動的日期的受試者活動，並進一步按起始時間點對軌跡進行索引。這個（受試者、日期、第一時間步）元組足以索引數據集中的每個軌跡。

**插補模式：**

加速度計時間表支持離散時間分析，並將個人運動的特定日期與該日期運動的人分組。完整的插補程序包括三個步驟：

1. **計算後驗：** 通過使用觀測協變量和參數對觀測數據進行回歸，計算後驗 𝜽₁:T ∣ 𝒚₁:T,o
2. **解析插補：** 解析插補未觀測的協變量，生成 𝑿_t,u*
3. **合成結果：** 從後驗中採樣 L 個向量，並合成結果以插補未觀測結果的時間步驟

### 3. 動態線性模型

**模型結構：**

動態線性模型（DLM）是論文的核心統計框架，其形式如下：

```
𝒚_t = 𝑿_t 𝜷_t + 𝝂_t ; 𝝂_t ~ ind 𝒩_n_t(0, σ²𝑽_t)
𝜷_t = 𝑮_t 𝜷_{t-1} + 𝝎_t ; 𝝎_t ~ ind 𝒩_p(0, σ²𝑾_t)
𝜷_0 ~ 𝒩_p(𝒎₀, σ²𝑴₀) ; σ⁻² ~ 𝒢(a₀, b₀)
```

其中：
- 𝒚_t 是時間 t 的觀測結果（log(MAG)）
- 𝑿_t 是觀測矩陣
- 𝜷_t 是時間 t 的狀態向量（固定效應）
- 𝑮_t 是狀態轉移矩陣
- 𝝂_t 是觀測噪聲
- 𝝎_t 是狀態噪聲
- σ² 是尺度方差
- 𝑽_t 和 𝑾_t 是相關矩陣

**DLM 的優勢：**
1. **線性性質：** 使模型相對容易理解和實現
2. **馬可夫性質：** 使推斷能夠從最新時間點恢復，而不是整個擬合歷史
3. **共軛性：** 每個序列項的分布在時間上保持在同一分布族中，僅參數不同

**前向濾波後向採樣（FFBS）算法：**

FFBS 是獲取 𝜷₁:T, σ² ∣ 𝒚₁:T 後驗分布的既定解決方案：

1. **前向濾波（Kalman 濾波器）：**
   - 按順序接收數據
   - 根據數據和現有參數估計更新下一時間步的參數
   - 計算過濾分布參數

2. **後向採樣：**
   - 獲取每個 𝜷_t | 𝒚₁:T 在時間 t 的樣本
   - 利用所有時間的數據進一步精煉 𝜷_t 的後驗均值和方差
   - 使用前向濾波器的係數計算平滑參數

### 4. 攤銷貝葉斯推斷（ABI）

**核心概念：**

在遷移學習框架的一種解釋中，目標是構建一個將數據集 𝒚 作為輸入並構建參數估計的後驗樣本 𝜽 ∣ 𝒚 作為輸出的系統。該系統在其應用領域內的不同數據集之間推廣，而不會過度擬合到任何一個數據集。

**「攤銷」的含義：**
- 指訓練神經網絡系統的高時間成本，該成本隨後在通過訓練後的系統傳遞多個數據集以產生低平均運行時間時被平均分攤
- 評估 f_ϕ(𝒚⁽ᵈ⁾) 不再涉及標準機器學習框架中訓練步驟的高時間成本，在該框架中我們單獨且從頭開始訓練每個數據集 𝒚⁽ᵈ⁾
- 訓練和運行系統的平均時間成本在訓練後系統運行的次數或不同設置 d=1,…,D 上減少（「攤銷」）

**BayesFlow 框架：**

BayesFlow 是實現 ABI 的 Python 庫，其核心組件包括：

1. **Simulators（模擬器）：**
   - 高級包裝器，用於將先驗、模擬器和元函數粘合在一起
   - 生成建模場景中所有感興趣的量

2. **Adapters（適配器）：**
   - 實用程序，將模擬器生成的數據預處理為神經近似器更友好的格式

3. **Networks（網絡）：**
   - 用於各種任務的核心神經架構
   - FlowMatching 架構用於近似分布
   - DeepSet 用於學習排列不變的摘要表示（嵌入）

4. **Approximators（近似器）：**
   - 高級包裝器，將各種網絡連接在一起
   - 指示它們在推斷管道中的特定目標

**歸一化流（Normalizing Flows）：**

ABI 使用可逆神經網絡（INNs）通過歸一化流來近似後驗分布：

1. **變分近似：**
   - 通過逆向流 p(f_ϕ⁻¹(z; y)) 追求對真實後驗 p(𝜽 ∣ 𝒚) 的變分近似
   - 通過找到最小化兩者之間 KL 散度的參數 𝜙

2. **目標函數：**
   ```
   ϕ̂ = argmin_ϕ (1/M) ∑_{m=1}^M (1/2 ||f_ϕ(𝜽⁽ᵐ⁾; 𝒚⁽ᵐ⁾)||² - log|det J_fϕ⁽ᵐ⁾|)
   ```
   其中 J_fϕ 是可逆網絡在正向方向的 Jacobian

3. **訓練算法：**
   - 在每個訓練迭代中，生成 M 個先驗和合成數據
   - 將 (𝜽⁽ᵐ⁾, 𝒚⁽ᵐ⁾) 通過推斷網絡的正向方向
   - 計算損失並通過反向傳播更新神經網絡參數

4. **採樣算法：**
   - 採樣潛在變量實例 z⁽ˡ⁾ ~ 𝒩_D(0, I)
   - 將 (𝒚⁽ᵒᵇˢ⁾, z⁽ˡ⁾) 通過推斷網絡的逆向方向
   - 返回後驗樣本 𝜽⁽¹ᴸ⁾

**可逆性關鍵：**

變換後的變量 f_ϕ̂(𝜽; 𝒚) 在概率上獨立於 𝒚，盡管它將 𝒚 作為參數。這對於採樣方案通過逆向變換標準正態隨機變量發揮作用至關重要。在完美收斂的情況下：
- f_ϕ̂(𝜽; 𝒚) ⟂ 𝒚
- f_ϕ̂⁻¹(z; 𝒚) ≡_d 𝜽 ∣ 𝒚

### 5. 正規-正態-伽馬示例

論文提供了具體的數學示例來說明 ABI 框架的應用：

**模型設定：**
```
𝒚 ∣ 𝜽 = 𝒚 ∣ 𝜷, σ² ~ 𝒩_n(𝑿𝜷, σ²𝑽)
𝜷, σ⁻² ~ 𝒩𝒢_p(𝒎, 𝑴, a₀, b₀)
```

**真實後驗：**
```
𝜷, σ⁻² ∣ 𝒚 ~ 𝒩𝒢_p(𝒎_post, 𝑴_post, a_post, b_post)
```

其中：
```
𝑴_post = (𝑿^T 𝑽⁻¹ 𝑿 + 𝑴⁻¹)⁻¹
𝝁_post = 𝑴_post(𝑿^T 𝑽⁻¹ 𝒚 + 𝑴⁻¹ 𝒎)
a_post = a₀ + p
b_post = b₀ + 1/2{𝒚^T 𝑽⁻¹ 𝒚 + 𝒎^T 𝑴⁻¹ 𝒎 - 𝒎_post^T 𝑴_post⁻¹ 𝒎_post}
```

**轉換到標準正態：**

1. **對於 𝜷：**
   ```
   f̃(𝜷, ·; 𝒚) = 𝑳_post⁻¹(𝜷 - 𝒎_post)
   ```
   其中 𝑳_post 是後驗協方差矩陣 𝑴_post 的 Cholesky 分解

2. **對於 σ²：**
   ```
   Φ⁻¹(Γ(σ⁻², a_post, b_post))
   ```
   其中 Γ(σ⁻², a, b) 是形狀為 a 和比率為 b 的伽馬隨機變量的累積分布函數（CDF），Φ 是標準正態隨機變量的 CDF

**神經網絡近似：**

訓練通過 ABI 旨在求解參數 𝜙，使得推斷網絡 f_ϕ 最佳近似 f̃。具體步驟：

1. 從正態-伽馬先驗採樣 𝜷⁽ᵐ⁾, σ⁻²⁽ᵐ⁾
2. 從模型生成合成數據樣本 𝒚⁽ᵐ⁾ ∣ 𝜷⁽ᵐ⁾, σ²⁽ᵐ⁾
3. 將 𝜷⁽ᵐ⁾, σ⁻²⁽ᵐ⁾ 和 𝒚⁽ᵐ⁾ 傳遞給目標函數中的單個 Monte Carlo 項
4. 在訓練循環結束時，通過反向傳播更新 𝜙

**設計屬性：**
- 𝔼[f_ϕ(𝜷, σ²; 𝒚)] = 𝟎
- Var[f_ϕ(𝜷, σ²; 𝒚)] = 𝑰
- 轉換變量的分布也是正態的
- 盡管縮放數量 f_ϕ(𝜷, σ²; 𝒚) 接收數據 𝒚 作為參數，但通過歸一化流變換的結果變量最終不依賴於數據

### 6. 應用於加速度計時間表

**數據插補：**

論文展示了如何將 ABI 框架應用於 PASTA-LA 數據集：

1. **協變量插補：**
   - 依賴於主體特定遍歷路徑的協變量（如 GPS 坐標）可以線性插值
   - 依賴於地理坐標的協變量可以從數據集中的相應緯度-經度對直接獲取
   - 如果坐標對不存在於數據集中，則使用距離的加權高斯平均值對一組半徑內的位置進行平均
   - 主體特定信息（如年齡）從數據集中的記錄信息直接獲取

2. **結果插補：**
   - 從後驗採樣 L 個向量 𝜷_t⁽ˡ⁾
   - 生成合成結果 𝒚_t,u*⁽¹ᴸ⁾ ~ 𝒩_{n_t}(𝑿_t,u* 𝜷_t, σ²𝑽_{t,uu})
   - 通過取跨 L 個樣本的 𝒚_t,u*⁽¹ᴸ⁾ 的分位數計算每個 𝒚_t,u 的可信區間

**時變影響學習：**

除了實現加速度計時間表的概率性插補外，該方法還能夠統計學習解釋變量對一組受試者的加速度幅值（MAG）的時變影響。這使得研究人員能夠：

- 識別影響物理活動的環境因素
- 理解個人屬性如何與環境因素相互作用
- 開發針對性健康干預措施

**遷移學習框架：**

論文強調，為可穿戴設備數據設計的遷移學習框架是一個新穎的貢獻。該框架：

1. **泛化能力：** 訓練後的網絡能夠在其應用領域內的不同數據集之間推廣
2. **無過度擬合：** 不會過度擬合到任何特定數據集
3. **快速推理：** 一旦訓練完成，新數據集的推理速度很快
4. **解釋性：** 保持貝葉斯方法的不確定性量化能力

- 使用 FFBS 算法的精確貝葉斯分析提供了性能評估的基準

## 局限性與挑戰

### 1. 方法論局限性

**收斂問題：**
- 完美收斂是不現實的，總是存在實踐中的誤差
- 文中指出：「算法可以收斂到足夠接近，使得誤差變得可忽略」
- Monte Carlo 誤差和可逆網絡的誤差可能影響結果準確性

**網絡規格不確定性：**
- 雖然存在神經網絡的通用近似定理，但所需規格並非預先已知，必須通過實驗確定
- 需要經驗性地測試不同的網絡架構
- 網絡設計的選擇可能影響性能

**分布假設：**
- 動態線性模型假設線性和正態性
- 實際數據可能違反這些假設
- 需要進行模型診斷和驗證

**超參數選擇：**
- 先驗分布和超參數的選擇需要領域知識
- 不恰當的選擇可能影響推斷結果
- 敏感性分析是必要的

### 2. 數據局限性

**樣本規模限制：**
- PASTA-LA 研究最初有 460 名受試者，但經過預處理後減少至 92 人
- 較小的樣本規模可能限制發現的普遍性
- 需要更多的外部驗證

**數據稀疏性：**
- 儘管經過預處理，數據仍然存在稀疏性問題
- 缺失數據的插補可能引入偏差
- 插補方法的假設需要仔細評估

**時間範圍限制：**
- 數據收集僅限於兩個各為一週的期間
- 季節性變化可能未被充分捕獲
- 長期趨勢和模式可能被遺漏

**地理限制：**
- 研究僅在洛杉磯 Westwood 區進行
- 結果可能不推廣到其他地理環境
- 氣候、地形和城市設計等因素可能影響可轉移性

**設備限制：**
- 使用特定的加速度計設備（Actigraph GT3X+）

- 跨設備校準是一個挑戰

- Jacobian 的計算在計算上是昂貴的
- 網絡架構的優化需要專業知識

- 對數變換和逆變換可能引發數值問題
