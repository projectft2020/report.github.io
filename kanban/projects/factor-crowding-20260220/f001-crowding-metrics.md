## Executive Summary

設計了一套完整的因子擁擠度指標系統，整合資金流動、估值水平和波動率三個核心維度，建立 0-100 分的擁擠度評分模型。系統包含早期預警機制（擁擠度 > 80 或 < 20 時觸發）和歷史回測框架，為組合管理提供過熱/過冷因子的識別能力。

**核心指標：**
- 因子暴露度指數 (Factor Exposure Index, FEI)
- 因子估值指數 (Factor Valuation Index, FVI)
- 因子波動率指數 (Factor Volatility Index, FVIol)
- 綜合擁擠度評分 (Comprehensive Crowding Score, CCS)

## 2. 因子暴露度指數 (Factor Exposure Index, FEI)

### 2.1 資金淨流入率 (Net Capital Flow Rate, NCF)

**計算公式：**
```
NCF_t = Σ(w_i,t × r_i,t) / Σ|w_i,t × r_i,t| × 50 + 50
```

其中：
- `w_i,t` = 因子 i 在時期 t 的暴露度權重
- `r_i,t` = 因子 i 在時期 t 的收益率
- 調整至 0-100 範圍（50 = 中性）

**數據源：**
- 每日資金流數據（北向/南向、機構資金、主力資金）
- 因子暴露度矩陣

**解讀：**
- **NCF > 70:** 資金過度流入，過熱風險
- **NCF < 30:** 資金大幅流出，過冷風險
- **NCF 40-60:** 資金流動平衡

### 2.2 頭寸集中度 (Position Concentration, PC)

**計算公式：**
```
PC_t = (HHI_t - HHI_min) / (HHI_max - HHI_min) × 100
```

其中 HHI (Herfindahl-Hirschman Index)：
```
HHI_t = Σ(w_i,t)²
```

- `w_i,t` = 標的在因子 i 上的權重

**解讀：**
- **PC > 70:** 高度集中，擁擠風險極高
- **PC < 30:** 分散度高，擁擠風險低
- **PC 40-60:** 中等集中度

### 2.3 FEI 綜合計算

```
FEI = α × NCF + β × PC
```
其中 `α = 0.6`, `β = 0.4`（可調整）

## 3. 因子估值指數 (Factor Valuation Index, FVI)

### 3.1 相對估值百分位 (Relative Valuation Percentile, RVP)

**計算方法：**

1. **橫截面百分位（Cross-sectional）：**
   ```
   RVP_cross(i,t) = percentile(rank(Val_i,t), Val_all,t)
   ```
   - 將因子 i 的估值水平在所有因子中排序

2. **時間序列百分位（Time-series）：**
   ```
   RVP_time(i,t) = percentile(rank(Val_i,t), Val_i,1:T)
   ```
   - 將因子 i 的當前估值在歷史窗口中排序

3. **綜合估值百分位：**
   ```
   RVP = 0.5 × RVP_cross + 0.5 × RVP_time
   ```

**估值指標選擇：**
- **價值因子：** P/E, P/B, P/CF, EV/EBITDA
- **成長因子：** PEG, P/S
- **盈利因子：** ROE, ROA, 淨利率
- **動量因子：** RSI, MACD

**解讀：**
- **RVP > 80:** 高估值區，過熱風險高
- **RVP < 20:** 低估值區，過冷機會
- **RVP 40-60:** 估值合理

### 3.2 超預期程度 (Expectation Excess, EE)

**計算公式：**
```
EE = (Actual - Forecast) / |Forecast| × 100
```

應用於：
- **盈利預期：** 實際 EPS vs 分析師預測
- **成長預期：** 實際營收增長 vs 市場預期
- **政策預期：** 實際政策力度 vs 預期

**平滑處理：**
```
EE_smooth = EWMA(EE, λ = 0.94)
```

**解讀：**
- **EE > 20:** 顯著超預期，過熱
- **EE < -20:** 顯著低於預期，過冷
- **EE -10~10:** 符合預期

### 3.3 FVI 綜合計算

```
FVI = γ × RVP + δ × EE_adj
```
其中：
- `γ = 0.7`, `δ = 0.3`
- `EE_adj = (EE + 100) / 2`（調整至 0-100）

## 4. 因子波動率指數 (Factor Volatility Index, FVIol)

### 4.1 波動率偏離度 (Volatility Deviation, VD)

**計算公式：**
```
VD_t = |σ_t - σ_long| / σ_long × 100
```

其中：
- `σ_t` = 當前波動率（20日滾動）
- `σ_long` = 長期平均波動率（252日）

**波動率估計方法：**

1. **歷史波動率：**
   ```
   σ_hist = sqrt((1/(n-1)) × Σ(r_t - μ)²)
   ```
   - 使用 20 日窗口
   - 對數收益率

2. **GARCH(1,1) 波動率：**
   ```
   σ²_t = ω + α × ε²_{t-1} + β × σ²_{t-1}
   ```
   - 捕捉波動率集聚效應

**解讀：**
- **VD > 60:** 異常高波動，過熱/過冷
- **VD < 20:** 波動低，穩定
- **VD 30-50:** 正常波動

### 4.2 波動率結構指數 (Volatility Structure Index, VSI)

**計算公式：**
```
VSI = Correlation(σ_short, σ_long) × 50
```

其中：
- `σ_short` = 短期波動率（5日）
- `σ_long` = 長期波動率（60日）

**解讀：**
- **VSI > 70:** 短期/長期波動相關性高，結構穩定
- **VSI < 30:** 短期/長期波動分歧大，結構不穩
- **VSI 40-60:** 正常結構

### 4.3 FVIol 綜合計算

```
FVIol = ε × VD + ζ × VSI
```
其中 `ε = 0.7`, `ζ = 0.3`

## 5. 綜合擁擠度評分 (Comprehensive Crowding Score, CCS)

### 5.1 評分模型

**公式：**
```
CCS = w₁ × FEI + w₂ × FVI + w₃ × FVIol
```

**權重配置：**
```
w₁ = 0.4  （資金流權重最高）
w₂ = 0.35 （估值次之）
w₃ = 0.25 （波動率補充）
```

**權重可調整場景：**
- **牛市場景：** 增加 w₁ 至 0.5，降低 w₃ 至 0.2
- **熊市場景：** 增加 w₂ 至 0.4，降低 w₁ 至 0.35
- **高波動環境：** 增加 w₃ 至 0.3

### 5.2 評分等級

| 評分範圍 | 擁擠狀態 | 解讀 | 建議操作 |
|---------|---------|-----|---------|
| **0-20** | 極度過冷 | 因子被市場忽略，可能存在反轉機會 | 考慮逆向建倉 |
| **20-40** | 過冷 | 因子估值偏低，資金流出 | 觀察，逢低佈局 |
| **40-60** | 中性 | 因子價格合理，資金平衡 | 持有觀望 |
| **60-80** | 過熱 | 因子估值偏高，資金流入 | 考慮減持 |
| **80-100** | 極度過熱 | 因子過度擁擠，崩潰風險高 | 積極減持/避險 |

## 6. 早期預警機制

### 6.1 預警觸發條件

**一級預警（關注）：**
- CCS > 75 或 CCS < 25
- 持續 3 個交易日以上
- 行動：密切監控，評估風險敞口

**二級預警（警示）：**
- CCS > 80 或 CCS < 20
- 持續 5 個交易日以上
- 行動：調整倉位，控制風險

**三級預警（緊急）：**
- CCS > 90 或 CCS < 10
- 持續 7 個交易日以上
- 行動：大幅減倉或避險

### 6.2 預警確認機制

**多維度確認：**
```
預警確認 = (CCS 觸發) AND (
  (FEI 觸發 AND FVI 觸發) OR
  (FEI 觸發 AND FVIol 觸發) OR
  (FVI 觸發 AND FVIol 觸發)
)
```

**趨勢確認：**
- CCS 的 5 日均線穿過閾值
- CCS 變化率 > 5%（日）

## 7. 歷史回測驗證框架

### 7.1 回測目標

驗證擁擠度指標的預測能力：
1. **預測因子收益：** 高擁擠是否預示未來收益下降
2. **預測因子波動：** 高擁擠是否預示波動上升
3. **預測反轉時點：** 擁擠度極值是否標誌反轉

### 7.2 回測設計

**回測期間：**
- 全樣本：2010-01-01 至 2026-02-20
- 訓練集：2010-2019（參數校準）
- 驗證集：2020-2022（參數穩定性）
- 測試集：2023-2026（樣本外檢驗）

**回測頻率：** 日度

### 7.3 檢驗指標

#### 7.3.1 預測能力檢驗

**IC (Information Coefficient)：**
```
IC = Correlation(CCS_t, Return_{t+1,t+20})
```
- 檢驗 CCS 對未來收益的預測能力
- IC > 0.05：顯著正相關
- IC < -0.05：顯著負相關（過熱預示下跌）

**IR (Information Ratio)：**
```
IR = Mean(IC) / Std(IC)
```
- IR > 0.5：穩定的預測能力

#### 7.3.2 分組回測

**按擁擠度分組：**
- Group 1: CCS ∈ [0, 20]（極度過冷）
- Group 2: CCS ∈ (20, 40]（過冷）
- Group 3: CCS ∈ (40, 60]（中性）
- Group 4: CCS ∈ (60, 80]（過熱）
- Group 5: CCS ∈ (80, 100]（極度過熱）

**檢驗：**
- 各組未來收益差異（ANOVA）
- 各組未來波動差異
- 極值組反轉收益率

#### 7.3.3 預警命中率

**定義：**
```
命中率 = 正確預警次數 / 總預警次數
```

**正確預警：**
- 預警後 20 個交易日內，因子收益 < -5%
- 或因子波動率上升 > 50%

**目標：** 命中率 > 60%

### 7.4 模擬策略

**策略 A：擁擠度逆向策略**
- 當 CCS > 80 時，減持該因子
- 當 CCS < 20 時，增持該因子

**策略 B：擁擠度動態調整**
- 根據 CCS 線性調整因子權重
- CCS = 0 → 權重 +50%
- CCS = 100 → 權重 -50%

**策略 C：擁擠度避險策略**
- 當 CCS > 80 時，使用衍生品對沖
- 當 CCS 回落 < 60 時，解除對沖

**基準：** 等權組合

### 7.5 回測結果評估

| 指標 | 策略 A | 策略 B | 策略 C | 基準 |
|-----|-------|-------|-------|-----|
| 年化收益率 | - | - | - | - |
| 年化波動率 | - | - | - | - |
| 夏普比率 | - | - | - | - |
| 最大回撤 | - | - | - | - |
| IC 值 | - | - | - | - |
| 預警命中率 | - | - | - | - |

## 9. 風險控制與限制

### 9.1 模型風險

**過度擬合風險：**
- 在訓練集上表現優異，樣本外失效
- 緩解：使用交叉驗證，樣本外測試

**參數不穩定：**
- 權重參數隨時間漂移
- 緩解：定期重新校準，使用滾動窗口

### 9.2 數據風險

**數據質量：**
- 缺失值、異常值、延遲
- 緩解：數據清洗，異常值檢測

**前視偏差：**
- 使用未來信息
- 緩解：嚴格時間戳控制

### 9.3 使用限制

**不適用場景：**
- 新興市場因子（數據不足）
- 小眾因子（流動性差）
- 事件驅動因子（短期波動大）

**建議使用場景：**
- 主流風格因子（價值、成長、動量）
- 流動性好的市場因子
- 長期穩定的基本面因子

### 12.2 局限性

1. **市場結構變化：** 模型可能不適應市場結構的重大變化

3. **因子相關性：** 未考慮因子間的相關性影響
4. **流動性風險：** 未充分考慮流動性對擁擠度的影響

- **FEI (Factor Exposure Index):** 因子暴露度指數，衡量資金流動和頭寸集中度
