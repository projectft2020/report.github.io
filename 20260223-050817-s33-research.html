Detailed Analysis¶

Self-Distillation Theory¶

Self-distillation (SD) is a machine learning technique where a model trains itself by acting as both teacher and student. The process involves:

  1. Teacher Training: Initial training of a ridge regression model on ground-truth labels
  2. Self-Distillation: Retraining a student model on a mixture of ground-truth labels and the teacher's own predictions
  3. Same Architecture: The student uses the same architecture and training data as the teacher
  4. Mixing Weight: A key parameter ξ that controls the balance between ground-truth labels and teacher predictions

The unconstrained setting studied in this paper allows the mixing weight ξ to take values outside the traditional unit interval [0,1], which is crucial for achieving optimal performance.

Ridge Regression Methods¶

Ridge regression is a regularization technique for linear regression that addresses multicollinearity and overfitting:

Mathematical Formulation: - Standard ridge regression: β̂_ridge = argmin_β RSS(β) + λ‖β‖²₂ - Where RSS(β) is the residual sum of squares and λ is the regularization parameter - This L2 penalty shrinks coefficients toward zero, trading bias for reduced variance

Key Properties: - Introduced by Hoerl and Kennard in 1970 - Particularly useful for high-dimensional data with correlated features - Provides better generalization than ordinary least squares when multicollinearity exists - The regularization parameter λ controls the strength of shrinkage

Theoretical Guarantees¶

The paper provides strong theoretical foundations for the self-distillation approach:

1. Strict Improvement Guarantee: - For any λ > 0 where R'(λ) ≠ 0, the optimally mixed student strictly outperforms the ridge teacher - This applies to any squared prediction risk, including out-of-distribution settings - The improvement is guaranteed under the unconstrained mixing framework

2. Deterministic Equivalents: - Extension of standard second-order ridge deterministic equivalents to fourth-order - Derived using block linearization techniques - Valid in the proportional asymptotics regime (n,p → ∞ with p/n → constant)

3. Asymptotic Analysis: - Results hold under general anisotropic covariance structures - Applicable to deterministic signal settings - Provides precise characterization of risk improvements
