### 動態閾值系統
- **參數**：高分位數=90%，低分位數=10%，回看窗口=60 天
- **倉位規則**：
  - 相關性 < 低閾值（10%）：正常市場，100% 倉位
  - 低閾值 ≤ 相關性 ≤ 高閾值（10%-90%）：過渡市場，50% 倉位

### 相關性不對稱性驗證
- **完美驗證**：相關性上升快（1 個月），下降慢（3+ 個月）

- **策略優勢**：動態閾值能夠適應這種不對稱性，寬閾值（90/10）允許快速響應

#### 具體實現

# 雙重確認機制
def calculate_position(correlation, price, ma20, high_threshold, low_threshold):
    # 計算價格趨勢
    price_trend = "upward" if price > ma20 else "downward"

    # 雙重確認
    if correlation > high_threshold and price_trend == "downward":
        # 高相關性 + 下跌趨勢 → 危機市場
        return 0.0
    elif correlation > high_threshold and price_trend == "upward":
        # 高相關性 + 上漲趨勢 → 可能反彈
        return 0.25  # 而不是 0%
    elif correlation < low_threshold and price_trend == "upward":
        # 低相關性 + 上漲趨勢 → 正常市場
        return 1.0
    else:
        # 過渡市場
        return 0.5

# 固定閾值寬度 = 0.15
high_threshold = correlation.quantile(0.90)
low_threshold = high_threshold - 0.15  # 固定寬度
```

# 雙重確認機制
def calculate_market_state(correlation, high_threshold, low_threshold, price_trend):
    # 危機市場：高相關性 + 下跌趨勢
    if correlation > high_threshold and price_trend == "downward":
        return {"state": "crisis", "position": 0.0, "emoji": "🔴"}
    
    # 危機市場但上漲趨勢：可能反彈，倉位 25%
    elif correlation > high_threshold and price_trend == "upward":
        return {"state": "crisis_rebound", "position": 0.25, "emoji": "🟡"}
    
    # 正常市場：低相關性 + 上漲趨勢
    elif correlation < low_threshold and price_trend == "upward":
        return {"state": "normal", "position": 1.0, "emoji": "🟢"}
    
    # 過渡市場：其他情況
    else:
        return {"state": "transition", "position": 0.75, "emoji": "🟡"}
```
