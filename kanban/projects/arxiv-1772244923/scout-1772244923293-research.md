**論文 ID:** scout-1772244923293
**論文標題:** Efficient Online Learning in Interacting Particle Systems
**arXiv ID:** 2602.20875
**發布日期:** 2026年2月
**分析完成時間:** 2026年3月4日

## 執行摘要

本論文提出了一種新的在線參數估計方法，適用於隨機交互粒子系統（IPS）。該方法基於對系統中少量粒子（僅需3個）的連續觀測，遞歸更新模型參數。與需要觀察所有 N 個粒子的傳統方法相比，該方法在粒子數量龐大時（N ≫ 1）提供了顯著的計算優勢。理論分析證明了估計器在漸近對數似然函數穩定點處的收斂性，並在強凸條件下建立了 L² 收斂率和中心極限定理。數值實驗涵蓋了多個實際應用模型，包括系統性風險、FitzHugh-Nagumo 神經元、Cucker-Smale 鳥群等，驗證了方法的有效性。

## 1. 研究背景與問題陳述

### 1.1 交互粒子系統（IPS）

交互粒子系統是描述大量相互作用的粒子的隨機動力學系統，廣泛應用於：

**應用領域：**
- **統計物理**：氣體、液體、等離子體等複雜系統建模
- **多智能體系統**：機器人群體協調、無人機群控制
- **平均場博弈**：大規模經濟體系、交通流優化
- **數學生物學**：神經科學（神經元網絡）、種群動力學
- **社會科學**：意見動力學、協同行為
- **金融數學**：風險傳播、市場微結構
- **貝葉斯推斷**：大規模貝葉斯網絡
- **平均場神經網絡**：深度學習理論

### 1.2 數學模型

**交互粒子系統（IPS）：**

```
dx_t^{θ,i,N} = [-∇V(θ, x_t^{θ,i,N}) - (1/N) Σ_{j=1}^N ∇W(θ, x_t^{θ,i,N} - x_t^{θ,j,N})] dt + σ dw_t^{i,N}
```

其中：
- **V(θ, ·)**：限制勢場（confinement potential），驅動粒子趨向某個位置
- **W(θ, ·)**：交互勢場（interaction potential），描述粒子間相互作用
- **θ**：未知參數向量
- **N**：粒子總數
- **σ**：擴散係數
- **w_t^{i,N}**：獨立布朗運動

**平均場極限（Mean-Field Limit）：**

當 N → ∞ 時，單個粒子的行為可以由以下 McKean-Vlasov 隨機微分方程（MVSDE）近似：

```
dx_t^θ = [-∇V(θ, x_t^θ) - ∫_{ℝ^d} ∇W(θ, x_t^θ - y) μ_t^θ(dy)] dt + σ dw_t
```

其中 μ_t^θ 是粒子在時間 t 的分佈。

### 1.3 研究動機與挑戰

**統計推斷問題：**
- 傳統方法假設可以觀察 IPS 的所有 N 個粒子
- 或需要觀察多條 MVSDE 的獨立軌跡
- 當 N ≫ 1 時，這些假設不切實際或計算成本過高

**在線學習需求：**
- 現有方法主要是「離線」或「批量」方法
- 對於長時間跨度的大數據集不切實際
- 需要基於觀測數據流的遞歸更新

**核心問題：**
能否在只觀測單個粒子或少量粒子的情況下，實現 IPS 的在線參數估計？

**本文回答：** 肯定！

## 2. 核心思想與方法論

### 2.1 目標函數

研究聚焦於 IPS 的漸近（時間和粒子數）負對數似然函數：

**目標函數 ℒ(θ)：**

```
ℒ(θ) := ∫_{ℝ^d} (1/2)‖B(θ, x, π_θ0) - B(θ0, x, π_θ0)‖_{σσᵀ}² π_θ0(dx)
```

其中：
- **B(θ, x, μ)** = ∫ b(θ, x, y) μ(dy)
- **b(θ, x, y)** = -∇V(θ, x) - ∇W(θ, x - y)
- **π_θ0**：真實參數 θ0 下的不變測度

**關鍵性質：**
- 對所有 θ ∈ Θ，ℒ(θ) ≥ 0
- 在標準可識別性假設下，ℒ(θ) 在 θ0 處達到唯一最小值（等於 0）

### 2.2 連續時間梯度下降

**梯度流定義：**

```
dθ_t = -γ_t ∂_θ ℒ(θ_t) dt
```

其中 **γ_t**：學習率，是正的、非遞增函數，滿足：
- ∫₀^∞ γ_t dt = ∞（無限總和）
- ∫₀^∞ γ_t² dt < ∞（有限平方和）

**解釋：**
- 這是目標函數 ℒ 的最速下降曲線
- 包含學習率以控制收斂速度

### 2.3 目標函數的梯度

**命題（漸近對數似然的梯度）：**

```
∂_θ ℒ(θ) = ∫_{ℝ^d} H(θ, x, π_θ0) π_θ0(dx)
```

其中：

```
H(θ, x, μ) := G(θ, x, μ) (σσᵀ)⁻¹ [B(θ, x, μ) - B(θ0, x, μ)]
G(θ, x, μ) := ∂_θ B(θ, x, μ) = ∫ ∂_θ b(θ, x, y) μ(dy)
```

**挑戰：**
- 無法直接計算 ∂_θ ℒ，因為涉及未知的不變測度 π_θ0
- 需要構造隨機估計

### 2.4 漸近梯度的隨機估計

**方法 1：平均估計器（需要觀察所有粒子）**

由遍歷性和 μ_s^θ0 → π_θ0，以及混亂傳播，可得：

```
∂_θ ℒ(θ) ≈ G(θ, x_t^{i,N}, μ_t^N) (σσᵀ)⁻¹ [B(θ, x_t^{i,N}, μ_t^N) dt - dx_t^{i,N}]
```

其中 **μ_t^N = (1/N) Σ_{j=1}^N δ_{x_t^{j,N}}** 是經驗測度。

**估計器更新方程：**

```
dθ̄_t^{i,N} = -γ_t G(θ̄_t^{i,N}, x_t^{i,N}, μ_t^N) (σσᵀ)⁻¹ [B(θ̄_t^{i,N}, x_t^{i,N}, μ_t^N) dt - dx_t^{i,N}]
```

**局限性：**
- 需要觀察所有 N 個粒子
- 當 N ≫ 1 時計算成本過高

---

### 2.5 新方法：三粒子估計器

**核心洞察：**

目標函數和梯度可以表示為三元積分：

```
ℒ(θ) = ∫_{(ℝ^d)³} ℓ(θ, x, y, z, π_θ0) π_θ0^⊗³(dx, dy, dz)
∂_θ ℒ(θ) = ∫_{(ℝ^d)³} h(θ, x, y, z, π_θ0) π_θ0^⊗³(dx, dy, dz)
```

這意味著只需要三個獨立粒子即可估計梯度！

**隨機估計：**

```
∂_θ ℒ(θ) ≈ g(θ, x_t^{i,N}, x_t^{j,N}) (σσᵀ)⁻¹ [b(θ, x_t^{i,N}, x_t^{k,N}) dt - dx_t^{i,N}]
```

其中：
- **g(θ, x, y)** = ∂_θ b(θ, x, y)
- **x_t^{i,N}, x_t^{j,N}, x_t^{k,N}**：三個不同的粒子

**新估計器更新方程：**

```
dθ_t^{i,j,k,N} = -γ_t g(θ_t^{i,j,k,N}, x_t^{i,N}, x_t^{j,N}) (σσᵀ)⁻¹ [b(θ_t^{i,j,k,N}, x_t^{i,N}, x_t^{k,N}) dt - dx_t^{i,N}]
```

**優勢：**
- ✅ 只需觀察 3 個粒子（i, j, k）
- ✅ 與總粒子數 N 無關
- ✅ 當 N ≫ 1 時，計算成本顯著降低

### 2.6 擴展：多軌跡平均

為了提高精度，可以平均多個「主軌跡」：

**令 Π = {i₁, ..., i_M} ⊆ [N] 為 M 個粒子的集合**

**平均估計器的擴展：**

```
dθ̄_t^{N,M} = -γ_t (1/M) Σ_{i∈Π} [G(θ̄_t^{N,M}, x_t^{i,N}, μ_t^N) (σσᵀ)⁻¹ [B(θ̄_t^{N,M}, x_t^{i,N}, μ_t^N) dt - dx_t^{i,N}]]
```

**三粒子估計器的擴展：**

```
dθ_t^{N,M} = -γ_t (1/M) Σ_{(i,j,k)∈𝓒(Π)} [g(θ_t^{N,M}, x_t^{i,N}, x_t^{j,N}) (σσᵀ)⁻¹ [b(θ_t^{N,M}, x_t^{i,N}, x_t^{k,N}) dt - dx_t^{i,N}]]
```

其中 **𝓒(Π)** 是 Π 中的循環三元組。

**權衡：**
- 增加 M 提高精度（方差減少因子 M）
- 但計算成本增加
- 三粒子估計器的成本增加遠小於平均估計器

## 3. 理論結果

### 3.1 假設條件

**假設 1（矩條件）：**
初始分佈 μ₀ 具有所有矩：μ₀ ∈ 𝓟_k(ℝ^d)，∀ k ∈ ℕ

**假設 2（漂移項條件）：**
- ∇V(θ₀, ·) 和 ∇W(θ₀, ·) 二次連續可微
- 滿足 C(A, α) 條件或對稱凸性
- 局部 Lipschitz 連續，多項式增長

**假設 3（梯度條件）：**
- ∂_θ ∇V(θ, x) 和 ∂_θ ∇W(θ, x) 三次連續可微
- 滿足一致的多項式增長界

**假設 4（學習率條件）：**
γ_t 正值、非遞增，滿足：
- ∫₀^∞ γ_t dt = ∞
- ∫₀^∞ γ_t² dt < ∞
- lim_{t→∞} γ_t t^p = 0, ∃ p > 0

### 3.2 收斂性結果

**命題（固定粒子數下的收斂）：**

設 N ∈ ℕ 固定，i, j, k 互不相同。在假設 1-4 下，幾乎必然有：

```
lim_{t→∞} ‖∂_θ ℒ^{i,N}(θ̄_t^{i,N})‖ = 0
lim_{t→∞} ‖∂_θ ℒ^{i,j,k,N}(θ_t^{i,j,k,N})‖ = 0
```

其中 ℒ^{i,N} 和 ℒ^{i,j,k,N} 是有限粒子漸近負對數似然函數。

**定理 1（聯合極限下的收斂）：**

在假設 1-4 下，幾乎必然有：

```
lim_{N→∞} limsup_{t→∞} ‖∂_θ ℒ(θ̄_t^{i,N})‖ = 0
lim_{N→∞} limsup_{t→∞} ‖∂_θ ℒ(θ_t^{i,j,k,N})‖ = 0
```

**解釋：**
- 先取 t → ∞，再取 N → ∞
- 兩個估計器都收斂到漸近對數似然 ℒ(θ) 的穩定點

### 3.3 收斂率

**假設 5（強凸性 - 有限粒子）：**
ℒ^{i,N} 和 ℒ^{i,j,k,N} 強凸，常數 η_{i,N}, η_{i,j,k,N} > 0

**假設 6（學習率加強條件）：**
確保波動項消失，滿足條件類似於 γ_t = γ₀ (1+t)⁻ᵝ，β ∈ (1/2, 1)

**定理 2（L² 收斂率 - 固定 N）：**

對充分大的 t：

```
𝔼[‖θ̄_t^{i,N} - θ_0^{i,N}‖²] ≤ (K₁ + K₂) γ_t
𝔼[‖θ_t^{i,j,k,N} - θ_0^{i,j,k,N}‖²] ≤ (K₁† + K₂†) γ_t
```

其中 θ_0^{i,N} 和 θ_0^{i,j,k,N} 是 ℒ^{i,N} 和 ℒ^{i,j,k,N} 的唯一最小值。

**關鍵洞察：**

1. **平均估計器 vs 三粒子估計器：**
   - 平均估計器：θ_0^{i,N} = θ_0（無偏）
   - 三粒子估計器：θ_0^{i,j,k,N} ≠ θ_0（有偏，當 N 有限時）

2. **權衡建議：**
   - **N 小時**：三粒子估計器的偏移顯著，偏好平均估計器
   - **N 大時**：偏移可忽略，偏好三粒子估計器（計算成本低）

**定理 3（L² 收斂率 - N → ∞）：**

```
𝔼[‖θ̄_t^{i,N} - θ_0‖²] ≤ (K₁ + K₂) γ_t + K₃ ρ(N) + K₄ N^{-1/(2(1+α))}
𝔼[‖θ_t^{i,j,k,N} - θ_0‖²] ≤ (K₁† + K₂†) γ_t + K₃† ρ(N) + K₄† N^{-1/(2(1+α))}
```

其中 **ρ(N)** 是混亂傳播率：

```
ρ(N) = { N^{-1/4}           (d < 4)
         N^{-1/4}[log(1+N)]^{1/2}  (d = 4)
         N^{-1/d}           (d > 4)
```

**收斂階：**
- 學在粒子數收斂：O(ρ(N) + N^{-1/(2(1+α))})
- 時間收斂：O(γ_t)

### 3.4 中心極限定理

**定理 4（CLT - 固定 N）：**

```
γ_t^{-1/2} (θ̄_t^{i,N} - θ_0^{i,N}) → 𝒩(0, Σ̄^{i,N})
γ_t^{-1/2} (θ_t^{i,j,k,N} - θ_0^{i,j,k,N}) → 𝒩(0, Σ̄^{i,j,k,N})
```

其中協方差矩陣由泊松方程解決定。

**定理 5（CLT - N → ∞）：**

當 N = N(t) → ∞ 時，收斂率為 ρ(N) + N^{-1/(2(1+α))} = o(γ_t^{1/2})：

```
γ_t^{-1/2} (θ̄_t^{i,N} - θ_0) → 𝒩(0, Σ̄^{i})
γ_t^{-1/2} (θ_t^{i,j,k,N} - θ_0) → 𝒩(0, Σ̄^{i,j,k})
```

### 3.5 理論結果總結

| 性質 | 平均估計器 | 三粒子估計器 |
|------|-----------|-------------|
| 觀測要求 | 所有 N 個粒子 | 僅 3 個粒子 |
| 計算成本 | O(N) | O(1) |
| 固定 N 下收斂 | 無偏，收斂到 θ₀ | 有偏，收斂到 θ₀^{i,j,k,N} |
| N → ∞ 下收斂 | 收斂到 θ₀ | 收斂到 θ₀ |
| L² 收斂率 | O(γ_t) + O(ρ(N)) | O(γ_t) + O(ρ(N)) + 偏移項 |
| CLT | 成立 | 成立 |

**核心結論：**
- 兩估計器在 N → ∞ 時具有相同的漸近性質
- 三粒子估計器在計算效率上顯著優越
- 當 N 大時，偏移可忽略

## 4. 技術細節

### 4.1 數值實驗設置

**計算環境：**
- MacBook Pro 16" (2021)
- Apple M1 Pro chip
- 16GB RAM

**數值方法：**
- Euler-Maruyama 時間離散化
- 固定時間步長 Δt = 0.1

**學習率選擇：**
- 通常使用常數學習率 γ = (γ₁, γ₂, ...)ᵀ
- 對於某些模型，使用 RMSProp 等適應方法

### 4.2 實驗 1：二次限制勢場 + 二次交互勢場

**模型：**

```
dx_t^{θ,i,N} = [-θ₁ x_t^{θ,i,N} - (θ₂/N) Σ_{j=1}^N (x_t^{θ,i,N} - x_t^{θ,j,N})] dt + σ dw_t^{i,N}
```

**參數：**
- θ = (θ₁, θ₂)ᵀ
- θ₁：限制參數（趨向零的速率）
- θ₂：交互參數（粒子間交互強度）

**實驗設置：**
- 真實參數：θ₀ = (1.0, 0.2)ᵀ
- 初始估計：θ_init,1 ~ 𝒰[1.5, 2.5], θ_init,2 ~ 𝒰[0.5, 1.0]
- 粒子數：N = 50
- 迭代次數：T = 10,000
- 學習率：γ = (8×10⁻³, 5×10⁻³)ᵀ

**更新方程：**

```
dθ̄_t = -γ_t [-(x_t^{i,N}); -(x_t^{i,N} - x̄_t^N)]ᵀ (σσᵀ)⁻¹ [-(θ̄₁ x_t^{i,N} + θ̄₂ (x_t^{i,N} - x̄_t^N)) dt - dx_t^{i,N}]
dθ_t = -γ_t [-(x_t^{i,N}); -(x_t^{i,N} - x_t^{j,N})]ᵀ (σσᵀ)⁻¹ [-(θ₁ x_t^{i,N} + θ₂ (x_t^{i,N} - x_t^{k,N})) dt - dx_t^{i,N}]
```

**結果：**
- ✅ 兩估計器都收斂到真實參數
- ✅ 平均估計器方差略小
- ✅ 兩估計器表現在估計 θ₁ 時基本相同
- ✅ 在估計 θ₂ 時，平均估計器方差較小

**粒子數影響：**
- 平均估計器：L² 誤差基本與 N 無關
- 三粒子估計器：L² 誤差隨 N 增加而減少
- 符合理論預測

**可識別性問題：**
- 當 N 大時，θ₁ + θ₂ = θ₀,₁ + θ₀,₂ 的所有參數都有接近最大似然
- 需要謹慎選擇學習率
- 不當選擇可能導致收斂到錯誤值

### 4.3 實驗 2：雙井勢場 + 二次交互勢場

**模型：**

```
dx_t^{θ,i,N} = [-θ₁ (x_t^{θ,i,N})³ + θ₂ x_t^{θ,i,N} - (θ₃/N) Σ (x_t^{θ,i,N} - x_t^{θ,j,N})] dt + σ dw_t^{i,N}
```

**參數：**
- θ = (θ₁, θ₂, θ₃)ᵀ
- 假設 θ₃ 已知
- 估計 (θ₁, θ₂)

**相變現象：**
- 模型存在相變
- σ > σ_c ≈ 1.9：唯一不變分佈
- σ < σ_c：雙穩態（兩個穩定狀態）

**實驗設置：**
- 真實參數：θ₀ = (1.0, 2.0, 2.0)ᵀ
- σ ∈ {1.0, 2.0}（跨越相變點）
- 粒子數：N ∈ {3, 10, 50}
- 迭代次數：T = 5,000
- 學習率：γ = (2×10⁻³, 2×10⁻²)ᵀ

**結果：**
- ✅ 兩估計器收斂到真實參數
- ✅ 對 σ < σ_c（多穩態）和 σ > σ_c（唯一穩態）都有效
- ✅ 平均估計器在所有 N 下都收斂到 θ₀
- ✅ 三粒子估計器當 N 小時有偏移，N 大時偏移消失
- ✅ 方法在假設不成立（多穩態）時仍有效

**RMSProp 改進：**
- 對於雙參數估計，相對學習率很重要
- RMSProp 顯著改善收斂行為

### 4.4 實驗 3：隨機 FitzHugh-Nagumo 神經元模型

**模型：**

```
dx_t^{θ,i,N} = [θ₁(x_t^{θ,i,N} - (x_t^{θ,i,N})³/3 - y_t^{θ,i,N}) - (θ₂/N) Σ (x_t^{θ,i,N} - x_t^{θ,j,N})] dt + σ dw_t^{i,N}
dy_t^{θ,i,N} = [x_t^{θ,i,N} + θ₃ - θ₄ y_t^{θ,i,N}] dt
```

**應用：**
- 神經科學，建模神經元集群
- x：電壓，y：恢復變量
- 通過電突觸（線性平均場）耦合

**特殊性質：**
- 奇異（退化）模型：擴散只作用於第一個方程
- 無法使用 Girsanov 定理構造似然
- 修改目標函數：不再用 (σσᵀ)⁻¹ 加權

**實驗設置：**
- 估計前三個參數 (θ₁, θ₂, θ₃)
- θ₄ 已知固定為 1.0
- 粒子數：N ∈ {3, 50}
- 迭代次數：T = 10,000
- 學習率：γ = (1×10⁻³, 1×10⁻³, 1×10⁻³)ᵀ

**結果：**
- ✅ 兩估計器表現基本相同
- ✅ 都收斂到真實參數，與 N 無關
- ✅ 單個軌跡變化小
- **解釋：** 三粒子估計器的偏移在此模型中可忽略

### 4.5 實驗 4：隨機 Kuramoto 模型

**模型：**

```
dx_t^{θ,i,N} = -(θ/N) Σ_{j=1}^N sin(x_t^{θ,i,N} - x_t^{θ,j,N}) dt + σ dw_t^{i,N}
```

**應用：**
- 物理、化學、生物學中的振盪子同步
- 廣泛應用於耦合系統

**相變：**
- σ > σ_c：唯一不變分佈（均勻分佈）
- σ < σ_c：非平凡連貫平衡（同步狀態）
- 臨界耦合強度：θ_c = σ²

**時變參數實驗：**
- 真實參數隨時間變化
- 兩種情況：
  1. **變化點：** θ_{0,t} = {θ_{0,1}, t ∈ [0, 5000); θ_{0,2}, t ∈ [5000, 10000]}
  2. **線性插值：** θ_{0,t} = θ_{0,1} + (θ_{0,2} - θ_{0,1}) t/10000

**參數：**
- θ_{0,1} = 1.5, θ_{0,2} = 0.2
- σ = 1.0 ⇒ θ_c = 1.0
- 粒子數：N = 50
- 迭代次數：T = 10,000

**結果：**
- ✅ 估計器能準確追蹤時變參數
- ✅ 在 θ > θ_c（同步）和 θ < θ_c（非同步）都有效
- ✅ 展示在線學習相對於離線方法的優勢

### 4.6 實驗 5：隨機 Cucker-Smale 鳥群模型

**模型：**

```
dx_t^{θ,i,N} = v_t^{θ,i,N} dt
dv_t^{θ,i,N} = [-θ₁ x_t^{θ,i,N} - (θ₂/N) Σ ψ_t^{ij}(θ₃)(v_t^{θ,i,N} - v_t^{θ,j,N})] dt + σ dw_t^{i,N}
```

**通信函數：**
```
ψ_t^{ij}(θ₃) = ψ(θ₃, ‖x_t^{θ,i,N} - x_t^{θ,j,N}‖²) = (1 + ‖x_t^{θ,i,N} - x_t^{θ,j,N}‖²)^{-θ₃}
```

**應用：**
- 鳥群自組織行為建模
- 個體由位置和速度 (x_t, v_t) 描述

**特殊性質：**
- 退化模型：擴散只作用於速度
- 類似 FitzHugh-Nagumo，需修改目標函數

**實驗設置：**
- 真實參數：θ₀ = (0.2, 1.0, 0.5)ᵀ
- 分別估計 θ₂ 和 θ₃
- 粒子數：N ∈ {3, 5, 50}
- 迭代次數：T = 5,000
- 學習率：γ = (0.01, 0.005)ᵀ

**結果：**
- ✅ 平均估計器表現與 N 無關
- ✅ 三粒子估計器的漸近偏移隨 N 增加而減少
- ✅ 當 N 小時，三粒子估計器高估交互參數
- ✅ 當 N 大時，偏移消失

## 6. 局限性

### 6.1 方法論局限

**1. 學習率敏感性**
- 對學習率選擇高度敏感
- 錯誤選擇可能導致收斂到局部最優或錯誤值
- 多參數估計時相對學習率很重要
- 需要進一步的自適應方法研究

**2. 可識別性問題**
- 某些模型在 N → ∞ 時參數不可識別
- 例如：θ₁ + θ₂ = const 的所有參數都接近最優
- 有限 N 時雖可識別，但似然曲面平坦
- 需要更好的參數約束或正則化

**3. 偏移項（三粒子估計器）**
- 當 N 小時，漸近偏移顯著
- 偏移隨 N 收斂到零，但收斂速度較慢
- 需要權衡計算成本和偏移容忍度

### 6.2 理論假設

**1. 非退化擴散假設**
- 某些理論結果假設 σ 可逆
- 退化模型（如 FitzHugh-Nagumo）需特殊處理
- 需要擴展到更一般的奇異擴散情況

**2. 唯一不變測度**
- 某些假設要求唯一不變測度
- 實際模型可能存在多穩態（如相變）
- 數值實驗表明方法在此情況下仍有效，但理論未覆蓋

**3. 強凸性假設**
- L² 收斂率和 CLT 需要強凸性
- 實際目標函數可能不強凸
- 需要放寬到更一般的最小值條件

### 6.3 數值實驗局限

**1. 模型簡化**
- 實驗主要在低維和簡單模型上驗證
- 高維和複雜交互模型的行為未知
- 需要更廣泛的數值驗證

**2. 時間離散化誤差**
- 使用 Euler-Maruyama 方法
- 時間步長 Δt 的影響未系統分析
- 可能引入額外誤差

**3. 計算效率**
- 雖然理論成本為 O(1)，實際實現可能有常數因子
- 需要更詳細的計算複雜度分析
- 與現有方法的實際速度比較

### 6.4 擴展性問題

**1. 部分觀測（濾波問題）**
- 當前假設可以完全觀測粒子軌跡
- 實際可能只能部分觀測或帶噪
- 需要擴展到濾波設定

**2. 非參數/半參數設置**
- 假設漂移或擴散的函數形式已知
- 實際可能不知道函數形式
- 需要擴展到非參數估計

**3. 非高斯噪聲**
- 假設布朗運動驅動（高斯）
- 實際可能存在跳躍、有色噪聲
- 需要適應更一般的噪聲模型

### 6.5 實際應用挑戰

**1. 初始化問題**
- 參數初始化影響收斂速度
- 錯誤初始化可能導致收斂到錯誤值
- 需要更好的初始化策略

**2. 參數尺度**
- 不同參數可能有不同尺度
- 需要預處理或自適應學習率
- 否則收斂不均勻

**3. 實時系統中的穩定性**
- 實時系統可能有數值穩定性問題
- 需要魯棒的數值方案
- 誤差累積可能導致發散

2. **謹慎選擇學習率**：使用網格搜索或自適應方法
3. **監控收斂行為**：實時追蹤參數變化和目標函數值
4. **考慮多軌跡平均**：在計算資源允許時提升精度
5. **驗證魯棒性**：在不同初始條件下測試

**定理 1：聯合極限收斂**
```
lim_{N→∞} limsup_{t→∞} ‖∂_θ ℒ(θ̄_t^{i,N})‖ = 0

**定理 2：L² 收斂率**
```
𝔼[‖θ̄_t^{i,N} - θ₀^{i,N}‖²] ≤ (K₁ + K₂) γ_t
𝔼[‖θ_t^{i,j,k,N} - θ₀^{i,j,k,N}‖²] ≤ (K₁† + K₂†) γ_t
```

**定理 3：N → ∞ 時的 L² 收斂率**
```
𝔼[‖θ̄_t^{i,N} - θ₀‖²] ≤ (K₁ + K₂) γ_t + K₃ ρ(N) + K₄ N^{-1/(2(1+α))}

### C. 算法偽代碼

**算法 1：平均估計器**
```
輸入：觀測軌跡 {x_t^{i,N}}_{i=1}^N, 學習率 {γ_t}
初始化：θ̄₀^{i,N}

對 t > 0:
    計算經驗分佈：μ_t^N = (1/N) Σ_{j=1}^N δ_{x_t^{j,N}}
    計算梯度估計：G_t = G(θ̄_t^{i,N}, x_t^{i,N}, μ_t^N)
    計算預測誤差：ε_t = B(θ̄_t^{i,N}, x_t^{i,N}, μ_t^N) dt - dx_t^{i,N}
    更新參數：dθ̄_t^{i,N} = -γ_t G_t (σσᵀ)⁻¹ ε_t
```

**算法 2：三粒子估計器**
```
輸入：觀測軌跡 {x_t^{i,N}, x_t^{j,N}, x_t^{k,N}}, 學習率 {γ_t}
初始化：θ₀^{i,j,k,N}

對 t > 0:
    計算梯度估計：g_t = g(θ_t^{i,j,k,N}, x_t^{i,N}, x_t^{j,N})
    計算預測誤差：ε_t = b(θ_t^{i,j,k,N}, x_t^{i,N}, x_t^{k,N}) dt - dx_t^{i,N}
    更新參數：dθ_t^{i,j,k,N} = -γ_t g_t (σσᵀ)⁻¹ ε_t
```
