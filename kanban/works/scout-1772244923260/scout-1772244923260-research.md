## 研究摘要

本論文提出了一個名為「Defensive Generation」的新型算法，該算法能夠在線上設定中高效生成標量、多類別和向量值結果的生成模型。這些模型的關鍵特性是無法基於觀測數據和預先指定的計算測試集合進行偽造。這是首個能夠高效生成非伯努利結果的線上結果不可區分（OI）生成模型，對於無限類別的測試（包括檢查生成分布的高階矩）是不可偽造的。該方法運行時間接近線性，並實現了最優的 T^{-1/2} 生成誤差率。

## 詳細分析

### 1. 問題設定

線上結果不可區分性（Online Outcome Indistinguishability）是一種生成式建模的新範式。在傳統監督學習中，學習者的成功被定義為找到具有低過剩風險的預測規則。然而，結果不可區分性提供了不同的視角：學習者的成功在於找到一個無法基於觀測數據和預先指定的計算測試集合進行偽造的生成模型。

形式上，一個算法 𝒜 滿足相對於測試類別 ℱ 的線上結果不可區分性，如果它產生結果 𝒴 上的分布 μ_t 和統計量 p_t，使得對於任何 f∈ℱ：

```
OIGap_T(f) := |∑_{t=1}^T 𝔼_{p_t}[f(x_t,p_t,y_t)] - ∑_{t=1}^T 𝔼_{p_t,ỹ_t∼μ_t}[f(x_t,p_t,ỹ_t)]|
```

是 o(T)，無論自然選擇 (x_t,y_t) 如何。

直觀上，這意味著沒有計算過程能夠區分由自然產生的「真實」數據和由學習者模型產生的數據。

### 2. 技術方法

#### 2.1 多校準簡化

核心技術洞察是將結果不可區分性簡化為高維多校準。如果測試類別 ℱ 中的每個函數 f 都可以寫為：

```
f(x,p,y) = h(x,p)ᵀs(y)
```

其中 s(y) 是充分統計的向量，h(x,p) 是某個函數類別 ℋ 中的函數，那麼結果不可區分性等價於高維多校準。特別是：

```
OIGap_T = sup_{h∈ℋ} |∑_{t=1}^T 𝔼_{p_t}[h(x_t,p_t)ᵀs(y_t)] - 𝔼_{p_t,μ_t}[h(x_t,p_t)ᵀs(ỹ_t)]|
         = sup_{h∈ℋ} |∑_{t=1}^T 𝔼_{p_t}[h(x_t,p_t)ᵀ(z_t-p_t)]|
```

其中 z_t = s(y_t) 且 p_t = 𝔼_{μ_t}[s(ỹ_t)]。

#### 2.2 向量值 RKHS

為了表示豐富的測試類別 ℋ，論文使用向量值再生核希爾伯特空間。一個向量值 RKHS 是一個函數空間 ℋ ⊆ {𝒳×𝒵} → ℝ^d，它等於以下集合的閉包：

```
h(x,p) = ∑_{i=1}^n Γ((x_i,p_i),(x,p))θ_i
```

其中 Γ : (𝒳×𝒵)² → ℝ^{d×d} 是矩陣值核函數，θ_i 是向量。Γ 是對稱函數，其輸出為半正定矩陣，並滿足再生性質。

RKHS 的重要性在於：
- 提供實例相關的複雜度概念（RKHS 範數）
- 能夠表示豐富的函數類別，如所有多項式
- 可以高效學習
- 允許將非線性函數在高維擴展基中線性化

#### 2.3 防禦預測

防禦預測是由 Vovk 等人（2005）引入的預測算法模板，它將預測視為遊戲論策略。預測不是通過預測而是通過糾正過錯誤來導出。

在所有時間 t，防禦預測選擇分布 𝒟_t 以確保：

```
Z_T = ||∑_{t=1}^T 𝔼_{p_t∼𝒟_t}Φ(x_t,p_t)(z_t-p_t)||_ℋ
```

是次線性的，無論自然如何選擇 z_t 和 x_t。

遞迴展開顯示，只要 𝒟_t 被選擇使得：

```
𝔼_{p_t∼𝒟_t}[S_t(p_t)ᵀ(z-p_t)] ≤ 0 ∀ z∈𝒵
```

其中 S_t(p) = Φ(x_t,p_t)ᵀ∑_{τ=1}^{t-1} 𝔼_{p_τ∼𝒟_τ}Φ(x_τ,p_τ)(z_τ-p_τ)，那麼第三項可以被丟棄，產生：

```
Z_T² ≤ ∑_{t=1}^T ||𝔼_{p_t∼𝒟_T}Φ(x_t,p_t)(z_t-p_t)||_ℋ² ⇒ Z_T ≤ √∑_{t=1}^T ||𝔼_{p_t∼𝒟_T}Φ(x_t,p_t)(z_t-p_t)||_ℋ²
```

這保證了以 T 速率增長的多校準誤差。

#### 2.4 期望變分不等式（EVI）

關鍵算法步驟是構造一個滿足期望變分不等式問題的分布 𝒟_t。給定容差 ε>0，凸緊集 𝒵⊂ℝ^d，和算子 S:𝒵→ℝ^d，EVI 的目標是找到一個（有限支撐的）分布 𝒟∈Δ(𝒵) 使得：

```
𝔼_{p∼𝒟}[S(p)ᵀ(z-p)] ≤ 0 ∀ z∈𝒵
```

這類問題總是通過一個相當整潔的簡化到遺憾最小化，在 O(poly(d,1/ε)) 時間內找到算法。

### 3. 主要結果

論文證明了以下主要結果：

#### 定理 1（非正式）：關於 Defensive Generation 的以下陳述為真。此外，在每種設定中，它在時間步 t 運行時間為 O(poly(d)·log(t) 並且 OIGap_T ≤ O(√T)。

1. **多類別結果**（𝒴=[d]）：該程序輸出相對於任何形成 RKHS 的無限區分者集合 f(x,p,y) 的線上結果不可區分分布 μ_t，這些區分者完全訪問條件分布 μ_t。作為具體例子，這特別意味著相對於所有以 d√T 速率界定的線性函數的不可區分性。

2. **標量結果**（𝒴=[-1,1]）：Defensive Generation 產生相對於任何 RKHS 中僅檢查 μ_t 前 d 階矩的低階度測試的線上結果不可區分分布 μ_t。即 p_t=(1,𝔼_{μ_t}[ỹ_t],𝔼_{μ_t}[ỹ_t²],…,𝔼_{μ_t}[ỹ_t^d])。對於布林 x，這意味著可以產生標量 y~ 的分布，其前 d 個條件矩 𝔼[y~^j|c(x)=1] 對於 j=1 到 d，匹配觀測序列在所有由低深度決策樹 c(x) 計算的超立方體子集上的矩。

3. **高維結果**（𝒴={y∈ℝ^d:||y||≤1}）：它輸出相對於檢查 μ_t 均值和協方差的任何 RKHS 中區分者的線上 OI μ_t。即 p_t=(𝔼_{μ_t}[ỹ_t],𝔼_{μ_t}[ỹ_tỹ_tᵀ])。這特別意味著相對於能夠位於固定（非線性）特徵集 Φ(x,p) 線中的無限區分者集合的不可區分性。此外，如果區分者僅檢查前一階矩 p=𝔼_{μ_t}[ỹ_t]，那麼 Defensive Generation 算法對 ℝ^d 中的任何緊凸集都有效，不僅限於單位球。

### 4. 應用示例

#### 4.1 基於令牌的序列建模（語言建模）

在這種設定中，目標是給定令牌歷史 x_t 產生下一個令牌 y_t∈𝒴=[d] 的條件分布 μ_t。作為簡單例子，可以產生相對於嵌入歷史 Φ(x_t) 的線性函數的某些特徵嵌入（如凍結的 Transformer 或 BERT 表示）不可偽造的分布 μ_t。

Defensive Generation 保證，對於任何這樣的 h：

```
(1/T)·OIGap_T(f_h) = |(1/T)∑_{t=1}^T 𝔼[h(x_t)ᵀ(y_t - μ_t)]| ≤ 4||h||_ℋ√(G/T)
```

這意味著，無論令牌 y_t 如何生成，嵌入歷史 Φ(x_t) 的每個線性函數在以下兩種情況下看起來相同：1) 在令牌的經驗分布 ∑_{t=1}^T y_t 上，或 2) 在條件分布 ∑_{t=1}^t μ_t 上。

#### 4.2 美國各州降雨預測

考慮預測美國 50 個州每天的降雨量的問題。這裡 y_t∈ℝ^50 是向量，其中 y_t,i 表示第 t 天第 i 州的降雨量。標準化降雨單位使得 ||y_t||_2≤1。

Defensive Generation 輸出一個 ℝ^50 中向量的聯合分布 μ_t 以及描述降雨條件分布均值和協方差的輔助統計量 p_t=(v_t,Q_t)，其中 𝔼_{ỹ_t∼μ_t}[ỹ_t]=v_t 且 𝔼_{ỹ_t∼μ_t}[ỹ_tỹ_tᵀ]=Q_t。

使用線性核，算法保證（作為特例）相對於以下形式的所有函數的不可區分性：

```
f_θ(x_t,p_t,y_t) = y_t,i·θᵀx_t,
f_β(x_t,p_t,y_t) = y_t,i·y_t,j·βᵀx_t,
f_w(x_t,p_t,y_t) = v_tᵀw·y_t,i
```

展開不可區分性定義並代入 Defensive Generation 的保證，我們得到以下陳述：

1. (1/T)·OIGap(f_θ) = |(1/T)∑_{t=1}^T θᵀx_t(y_t,i-𝔼_{μ_t}[ỹ_t,i])| ≤ 4||θ||_2√((B²+2)/T)
   - 正確的每州均值

2. (1/T)·OIGap(f_β) = |∑_{t=1}^T βᵀx_t(y_t,i·y_t,j-𝔼_{μ_t}[ỹ_t,i·ỹ_t,j])| ≤ 4||β||_2√((B²+2)/T)
   - 正確的協方差

3. (1/T)·OIGap(f_w) = |(1/T)∑_{t=1}^T wᵀ𝔼_{μ_t}[ỹ_t](y_t,i-𝔼_{μ_t}[ỹ_t,i])| ≤ 4||w||_2√((B²+2)/T)
   - 自洽均值

第一組條件意味著，對於每個州 i，降雨的期望值與特徵 x 的任何線性函數不相關。也就是說，降雨的生成模型具有按此測試條件正確的均值。

第二個等式顯示，不僅每州的均值正確，條件分布還捕獲了任何州對之間的成對相關性。如果風暴襲擊麻薩諸塞州，它很可能也襲擊羅德島。第二組測試保證降雨的聯合分布通過所有這些成對檢查。它還斷言每州降雨分布的方差是正確的。

最後一組條件顯示，均值不僅是條件正確的，它們在均值分布本身的任何線性函數的誤差 𝔼_{μ_t}[ỹ_t,i]-y_t,i 不相關的意義上是自洽的。這是不可區分性保證的加強，不是由前兩個條件中的任何一個暗示的。

#### 4.3 線性動態系統學習

考慮找到相對於由線性動態系統生成的數據不可區分的生成模型的任務。考慮系統：

```
z_{t+1} = Az_t + ζ_t,  y_t = Cz_t + ν_t
```

其中 y_t∈ℝ^d 是觀測，z_t 是隱藏狀態，A 和 C 是矩陣，ζ_t 和 ν_t 是噪聲向量，它們可能是對手的（不是 i.i.d.）。

假設觀測 y_t 均勻有界，||y_t||≤B。這在隱藏狀態 x_0 有界、噪聲項 (ν_t,ζ_t) 有界且矩陣 A 的譜半徑嚴格小於 1（A 嚴格穩定）時為真。

Defensive Generation 算法產生 ℝ^d 中點的概率測度 μ_t 以及統計量 p_t=(𝔼_{μ_t}[ỹ_t],𝔼_{μ_t}[ỹ_tỹ_tᵀ])。

再次使用線性核，Defensive Generation 算法保證相對於以下形式的所有區分者的不可區分性：

```
f(x_t,p_t,y_t) = ∑_{i=1}^d x_tᵀα_i·y_t,i + ∑_{1≤i≤j≤d} x_tᵀβ_i,j·(y_t,i·y_t,j)
```

其中 α_i 和 β_i,j 都是 ℝ^d 中的向量。特別地，對於任何這樣的函數 f：

```
OIGap(f) ≤ (∑_{i=1}^d ||α_i||_2 + ∑_{1≤i≤j≤d} ||β_i,j||_2)√(4Tℓ)
```

這特別意味著 Defensive Generation 算法產生的概率分布 μ_t「看起來像」它們生成了觀測 y_t，至少從任何觀測截斷歷史的線性函數的角度來看。

這種概率測度 μ_t 的不可區分性保證對於任何截斷長度 ℓ 成立，並且對噪聲不做出隨機假設。此外，隱藏狀態 z_t 可以是無限維的，矩陣對 (A,C) 不需要滿足任何可觀測性條件。我們僅要求觀測 y_t 有有界範數。

#### 4.4 教育結果的生成模型

作為最後一個例子，提及本文開發的技術如何能夠為社會結果（如學生考試成績）的生成模型提供細粒度的不可區分性保證。算法可以生成標量結果的條件分布，即使在對豐富的特徵子集進行條件化並檢查高階矩時也有效。

使用多項式核，可以保證不僅均值，而且這些分布的任何固定數量的矩在所有由深度 r 決策樹可計算的子群體 c(x)⊆{±1}^n 上都準確：

```
(1/T)∑_{t:c(x_t)=1}^T y_t^j ≈ (1/T)∑_{t:c(x_t)=1}^T 𝔼_{μ_t}[ỹ_t^j]  對於所有 j=1,…,d
```

Defensive Generation 算法能夠高效產生這些分布。

## 應用價值評估

### 優點

1. **理論保證堅實**：提供無條件保證，不依賴於無法驗證的函數近似假設。這與依賴統計距離和函數近似的傳統生成建模方法形成鮮明對比。

2. **計算效率高**：算法在時間步 t 運行時間為 O(poly(d)·log(t))，對於大規模數據流實用。

3. **適應性廣泛**：適用於標量、多類別和向量值結果，覆蓋了廣泛的應用場景。

4. **對手魯棒性強**：不假設數據從任何分布 i.i.d. 生成，能夠處理對手選擇的數據。

5. **豐富的測試類別**：能夠處理無限類別的測試，包括檢查高階矩的測試，提供了比僅檢查均值的先前工作更強的保證。

6. **多損失同時最小化**：結果不可區分性（OI）隱含損失最小化不僅對於單一損失，而是同時對於許多損失（即全預測 omniprediction）。

### 局限性

1. **複雜度**：算法和理論相對複雜，需要深入理解多校準、RKHS 和期望變分不等式。

2. **實現挑戰**：雖然理論上高效，但實際實現可能需要小心處理數值穩定性和優化細節。

3. **分佈限制**：對於連續結果，算法不直接產生完整的條件分布，而是產生匹配前 d 階矩的分布。這可能不適合需要精確尾部分布的應用。

4. **特徵選擇**：性能依賴於選擇合適的核函數和特徵映射 Φ，這可能需要領域專業知識。

5. **實證驗證待進行**：作為最近的理論工作（2026年發布），缺乏廣泛的實證驗證和與現有方法的系統比較。

6. **常數因子**：理論界限中的常數可能很大，實際性能可能依賴於具體實現和數據特性。

## 未來研究方向

1. **實證驗證**：在實際應用中驗證算法的有效性和魯棒性，特別是在語言建模和高維預測任務中。

## 結論

Defensive Generation 代表了生成式建模的一個重要理論進步，提供了一個強有力的無條件保證框架，能夠在線上設定中高效生成相對於豐富測試類別不可區分的模型。該工作將結果不可區分性、線上多校準、博弈論預測和非線性優化等多個研究領域的見解巧妙地結合起來，為生成式建模提供了新的基礎方法。雖然仍需實證驗證和進一步改進，但該方法為未來的生成式建模研究開闢了有希望的方向，特別是在需要強理論保證和對手魯棒性的應用中。
