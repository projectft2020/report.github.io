## 🎯 使用方式

### 步驟 1：選擇模板
- 根據你的需求選擇適當的模板
- 查看模板列表中的「關鍵輸出」欄位

### 步驟 2：複製模板
- 打開對應的 `.md` 文件
- 複製提示詞內容

### 步驟 3：填入變數
- 將黃色括號中的變數 `[VAR]` 替換為你的具體內容
- 例如：`[INSERT TICKER SYMBOL]` → `AAPL`

### 步驟 4：發送到 Claude
- 貼上完整提示詞
- 等待 Claude 生成專業報告
