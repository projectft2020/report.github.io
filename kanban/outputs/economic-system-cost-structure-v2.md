# 經濟系統成本結構 v2.0
## 動態成本結構方案（2026-03-01）

---

## 一、背景問題

### 1.1 當前問題
- **初始本金**：100,000 TWD
- **應急儲備**：10,000 TWD（不動用）
- **實際可用**：90,000 TWD
- **固定成本**：3,000 TWD/月（電力 + 算力）
- **人工成本**：金額 × 0.5%（每次進/出）

### 1.2 成本壓力分析
| 項目 | 當前 | 佔本金比 | 年化 |
|------|------|----------|------|
| 固定成本 | 3,000 TWD/月 | 3.33% | **40%** |
| 人工成本 | 0.5%/次 | - | - |

**問題**：40% 年化成本對小資金（90,000 TWD）壓力過大

### 1.3 對比：如果本金是 1,000,000 TWD
| 項目 | 調整後 | 佔本金比 | 年化 |
|------|--------|----------|------|
| 固定成本 | 3,000 TWD/月 | 0.3% | **3.6%** |
| 人工成本 | 0.5%/次 | - | - |

**洞察**：固定成本應該是「百分比導向」，不是「固定金額導向」

---

## 二、動態成本結構方案

### 2.1 固定成本公式

```
固定成本 = min(3,000, 本金×0.3%/月)
```

**說明**：
- 上限：3,000 TWD/月（大資金時不超過此上限）
- 下限：0（本金為0時）
- 比例：0.3%/月（年化 3.6%）

**計算示例**：

| 本金 | 固定成本 | 佔比 | 每月釋放 |
|------|----------|------|----------|
| 50,000 TWD | 150 TWD | 0.3% | - |
| 90,000 TWD | **270 TWD** | 0.3% | **+2,730 TWD** |
| 200,000 TWD | 600 TWD | 0.3% | - |
| 500,000 TWD | 1,500 TWD | 0.3% | - |
| 1,000,000 TWD | 3,000 TWD | 0.3% | - |

### 2.2 人工成本公式

```
人工成本 =
  - 本金 < 100,000 TWD：0.2%
  - 100,000 TWD ≤ 本金 < 500,000 TWD：0.3%
  - 本金 ≥ 500,000 TWD：0.5%
```

**說明**：
- 小資金時降低人工成本（減少交易壓力）
- 大資金時恢復正常人工成本
- 分層調整，避免突變

**計算示例**：

| 本金 | 人工成本 | 交易 10,000 TWD | 交易 50,000 TWD |
|------|----------|----------------|----------------|
| 50,000 TWD | 0.2% | 20 TWD | 100 TWD |
| 90,000 TWD | 0.2% | 20 TWD | 100 TWD |
| 200,000 TWD | 0.3% | 30 TWD | 150 TWD |
| 500,000 TWD | 0.5% | 50 TWD | 250 TWD |
| 1,000,000 TWD | 0.5% | 50 TWD | 250 TWD |

---

## 三、調整頻率與機制

### 3.1 調整頻率
- **每月調整一次**：根據月底本金調整下月成本
- **觸發條件**：每月 1 日自動重新計算

### 3.2 調整流程
1. 每月 1 日，獲取當前本金（扣除應急儲備後）
2. 計算新的固定成本和人工成本
3. 更新經濟系統文件
4. 記錄到月度財務報表

### 3.3 本金變動處理
- **本金增加**：下月自動調整成本（無需確認）
- **本金大幅下降**（>20%）：人工檢查，確認是否需要調整

## 六、風險與注意事項

### 6.1 潛在風險
| 風險類型 | 說明 | 緩解措施 |
|---------|------|----------|
| 本金下降 | 成本降低導致服務品質下降 | 保留最低成本 270 TWD/月 |
| 交易頻繁 | 人工成本累積過高 | 監控月度人工成本總額 |
| 靈活性不足 | 無法應對緊急情況 | 保留 10,000 TWD 應急儲備 |

### 6.2 注意事項
- 本金計算應扣除應急儲備（10,000 TWD）
- 人工成本按每次進/出計算（不區分買入/賣出）
- 每月 1 日自動調整，無需手動干預
- 如本金大幅下降（>20%），需人工檢查

## 八、結論與建議

### 8.1 核心優勢
1. **降低小資金壓力**：40%/年 → 3.6%/年，降低 91%
2. **動態適應本金變化**：隨本金增長自動調整
3. **公平性提升**：百分比導向，而非固定金額
4. **激勵成長**：本金增加時成本可承受

### 8.2 實施建議
- ✅ **立即實施**：當前本金 90,000 TWD，壓力過大
- ✅ **每月自動調整**：無需手動干預
- ✅ **監控成本**：記錄月度成本，確保合理性

### 8.3 後續優化
- 根據實際使用情況調整 0.3% 的百分比
- 考慮加入 API 成本、數據成本等其他變動成本
