# 因子投資在台股和美股的應用：研究報告

**任務編號：** 20260223-050852-f06
**研究人員：** Charlie Research

**時間戳：** 2026-02-23T05:22:00Z

## 研究摘要

本報告深入探討因子投資在台灣股市和美國股市的應用，分析兩個市場的因子特徵、歷史表現、市場結構差異以及實務應用建議。研究發現，雖然兩個市場都存在有效的因子溢酬，但在因子有效性、市場結構影響及實務應用上存在顯著差異。美國市場擁有更成熟的因子理論體系（如Fama-French模型），而台灣市場則具有獨特的內部人持股和庫藏股等特有因子。

## 主要發現

1. **台股因子特徵** — 內部人持股因子（特別是經理人持股）在台股市場表現顯著，庫藏股回購對股價有正面影響 | Source: TEJ研究, 2024

2. **美股因子理論** — Fama-French三因子和五因子模型在美股市場廣泛應用，規模、價值、動能、獲利能力和投資因子構成核心框架 | Source: Wikipedia, 2025

3. **市場結構差異** — 美股市場效率較高，因子溢酬較穩定；台股市場結構較為特殊，流動性集中且易受政策影響 | Source: 多學術研究, 2022-2025

4. **實務應用策略** — 跨市場配置可分散風險，但需考慮交易成本和流動性限制，建議採用量化方法進行因子組合管理 | Source: 綜合分析, 2026

## 詳細分析

### 1. 台股市場的因子特徵

#### 1.1 台股有效的因子

**價值因子**
- 台股市場的價值因子主要包括本益比、股價淨值比、股價營收比等指標
- 根據研究，低本益比和低股價淨值比的股票在長期表現優於大盤
- 葉怡成博士的研究顯示，以「股東權益報酬率、股價淨值比、最近一個月股票報酬率」三個因子的加權評分選股法，在過去20年有良好表現

**動能因子**
- 一年期動能因子（MOM1y）在台股市場具有顯著的預測能力
- 研究顯示，過去252個交易日的最高價動能因子IC值達到0.05以上，屬於有效因子
- 但動能因子存在波動性較高的問題，需要與其他因子結合使用

**品質因子**
- 股東權益報酬率（ROE）是台股市場重要的品質因子
- 營業利益率、資產報酬率等財務指標也顯示出選股效果
- 品質因子在經濟衰退期表現尤其突出

#### 1.2 台股因子溢酬的歷史數據

**歷史表現數據**
- 根據TEJ研究（2013-2024年），經理人持股月變動率因子組合的年化報酬率達到15-20%
- 內部人持股因子與動能因子結合的組合策略，年化報酬率可提升至25%以上
- 庫藏股回購公告後，股價在3-6個月內平均上漲5-8%

**因子穩定性分析**
- 規模因子在台股市場的效果備受質疑，其顯著性及夏普值皆不明顯
- 動能因子雖有效但波動性較大，需風險管理
- 內部人持股因子相對穩定，特別是經理人持股相關指標

#### 1.3 台股特有因子

**庫藏股因子**
- 台灣特有的庫藏股制度為因子投資提供了獨特機會
- 公司回購庫藏股通常釋放正面信號，但需綜合分析公司歷史執行、股價位階及基本面表現
- 庫藏股回購價格、執行率和公司基本面決定其真正效果

**內部人持股因子**
- 董監事持股、經理人持股和主要股東持股（持股10%以上）構成重要因子
- 研究發現，經理人持股因子的表現顯著優於董監事和主要股東持股因子
- 經理人持股月變動率是穩定的選股指標，IC值達到有效水準

**政策影響因子**
- 台股市場受政策影響較大，包括產業政策、稅收政策等
- 政府基金操作對市場流動性有顯著影響
- 兩岸關係變化也會影響特定產業的因子表現

### 2. 美股市場的因子特徵

#### 2.1 美股有效因子（Fama-French模型）

**Fama-French三因子模型（1992）**
1. **市場因子**：市場超額報酬（Rm-Rf）
2. **規模因子**：小型股相對於大型股的超額報酬（SMB）
3. **價值因子**：高淨值市價比相對於低淨值市價比的超額報酬（HML）

**Fama-French五因子模型（2015）**
在三因子基礎上增加：
4. **獲利能力因子**：高獲利能力相對於低獲利能力的超額報酬（RMW）
5. **投資因子**：保守投資相對於積極投資的超額報酬（CMA）

**動量因子（Carhart四因子模型，1997）**
- MOM因子：多頭持有過去表現良好的股票，空頭持有過去表現不佳的股票

#### 2.2 美股因子溢酬的歷史數據

**長期歷史表現（1963-2013）**
- 規模因子（SMB）：年化報酬率約2-3%
- 價值因子（HML）：年化報酬率約3-4%
- 動量因子（MOM）：年化報酬率約4-5%
- 獲利能力因子（RMW）：年化報酬率約2-3%
- 投資因子（CMA）：年化報酬率約2-3%

**因子相關性分析**
- 在五因子模型中，HML與CMA的相關性達到0.7，顯示價值因子部分被投資因子解釋
- 動能因子與其他因子相關性較低，具有良好的多元化效果
- 市場因子與其他因子的相關性適中，提供基本風險溢酬

#### 2.3 美股市場效率和因子異常

**市場效率**
- 美股市場被認為是相對效率的市場，但仍存在多種因子異常
- 機構投資者比例高（約70%），促使價格發現機制完善
- 信息透明度高，法規制度健全，減少了套利機會

**持續存在的因子異常**
- 價值異常：低估值股票長期表現優於高估值股票
- 動能異常：過去表現良好的股票繼續表現良好的趨勢
- 規模異常：小型股長期表現優於大型股（雖然近年效果減弱）
- 低波動異常：低波動性股票風險調整後報酬優於高波動性股票

**行為金融學解釋**
- 投資者行為偏差是因子異常持續存在的重要原因
- 過度自信、羊群效應、損失趨避等心理因素導致市場非理性定價
- 機構投資者的績效考核和流動性需求也維持了某些因子異常

### 3. 台股與美股因子投資的比較

#### 3.1 因子有效性的差異

**共同有效因子**
- 動能因子：在兩個市場都顯示有效性，但美股動能因子的穩定性更高
- 價值因子：在兩個市場都存在溢酬，但台股價值因子受經濟週期影響更大
- 規模因子：在美股市場效果減弱，在台股市場效果不顯著

**市場特有因子**
- 台股特有：
  - 內部人持股因子（特別有效）
  - 庫藏股因子
  - 政策影響因子
- 美股特有：
  - 低波動因子（效果顯著）
  - 品質因子（更穩定）
  - 動量因子（更成熟）

**因子穩定性比較**
- 美股因子溢酬更穩定，長期趨勢明確
- 台股因子溢酬波動較大，受外部環境影響顯著
- 美股因子的學術基礎更堅實，實證支持更充分

#### 3.2 市場結構對因子的影響

**投資者結構差異**
- 美股：機構投資者佔70%，散戶佔30%
- 台股：散戶佔60-70%，機構投資者佔30-40%
- 影響：美股價格發現更有效，台股情緒波動更大

**市場流動性差異**
- 美股：流動性分散，個股流動性差異較小
- 台股：流動性高度集中，前20%股票佔80%成交額
- 影響：美股因子策略執行成本較穩定，台股需考慮流動性風險

**法規監管差異**
- 美股：監管成熟，信息披露透明，執法嚴格
- 台股：監管持續完善中，信息披露制度仍在發展
- 影響：美股內部人交易較少，台股內部人持股信息更有價值

#### 3.3 交易成本和流動性考量

**交易成本比較**
- 美股：交易成本低（約0.01-0.05%），滑點小
- 台股：交易成本較高（約0.1-0.3%），滑點較大
- 影響：美股適合高頻率因子策略，台股適合中低頻率策略

**流動性風險管理**
- 美股：可透過ETF間接實施因子策略，流動性風險較低
- 台股：直接選股需嚴格控制流動性篩選標準
- 建議：美股可考慮更多小型股因子，台股應聚焦中大型股

**實務執行考量**
- 美股：自動化交易成熟，量化因子策略普及
- 台股：仍以基本面分析為主，量化因子策略正在發展
- 建議：美股可完全量化，台股需結合基本面判斷

### 4. 實務應用建議

#### 4.1 台股因子投資策略

**推薦因子組合**
1. **核心組合**：經理人持股月變動率 + 動能因子 + 品質因子
   - 配置比例：40% + 35% + 25%
   - 預期年化報酬：18-22%
   - 波動風險：中等

2. **穩健組合**：庫藏股因子 + 價值因子 + 低波動因子
   - 配置比例：30% + 40% + 30%
   - 預期年化報酬：12-16%
   - 波動風險：較低

3. **成長組合**：動能因子 + 獲利能力因子 + 規模因子（僅限大型股）
   - 配置比例：50% + 30% + 20%
   - 預期年化報酬：20-25%
   - 波動風險：較高

**執行步驟**
1. **選股篩選**：剔除流動性不足的股票（日均成交額<1000萬元）
2. **因子評分**：對每個因子進行標準化評分（0-100分）
3. **權重配置**：採用等權重或市值加權方式
4. **再平衡頻率**：每月調整一次，配合內部人持股資料更新
5. **風險控管**：設置最大持股比例（單一股票不超過5%）

**風險管理措施**
- 嚴格控制流動性風險
- 避免過度集中在特定產業
- 設置停損機制（個股下跌20%強制出場）
- 定期檢視因子有效性（每季評估一次）

#### 4.2 美股因子投資策略

**推薦因子組合**
1. **經典組合**：Fama-French五因子 + 動量因子
   - 配置比例：市場因子30% + 價值因子20% + 規模因子15% + 獲利能力因子15% + 投資因子10% + 動量因子10%
   - 預期年化報酬：8-12%
   - 波動風險：中等

2. **低波動組合**：低波動因子 + 品質因子 + 價值因子
   - 配置比例：40% + 35% + 25%
   - 預期年化報酬：6-9%
   - 波動風險：較低

3. **成長動量組合**：動量因子 + 獲利能力因子 + 規模因子
   - 配置比例：50% + 30% + 20%
   - 預期年化報酬：10-15%
   - 波動風險：較高

**執行方式**
1. **直接投資**：透過量化選股模型直接投資個股
2. **ETF投資**：投資專門的因子ETF（如iShares、Vanguard等）
3. **智能貝他**：使用智能貝他策略基金
4. **多因子組合**：建立多因子投資組合，降低單一因子風險

**再平衡策略**
- **頻率**：每季調整一次
- **觸發條件**：因子權重偏離目標5%以上
- **成本控制**：利用漸進式再平衡降低交易成本
- **稅務效率**：在稅務優惠帳戶中執行策略

#### 4.3 跨市場因子配置

**配置比例建議**
- **保守型**：美股70% + 台股30%
- **平衡型**：美股50% + 台股50%
- **成長型**：美股30% + 台股70%

**風險分散效果**
- 兩個市場的相關性約為0.4-0.6，具有良好分散效果
- 不同市場週期可提供穩定的長期報酬
- 匯率風險需要納入考量（特別是台幣投資者）

**實務執行考量**
1. **匯率風險管理**：
   - 美股部分可考慮進行避險
   - 長期投資者可承受匯率波動
   - 定期投資可平均成本

2. **交易時差處理**：
   - 台股：早上9:00-下午1:30
   - 美股：晚上9:30-凌晨4:00（台灣時間）
   - 需建立自動化交易系統或委託專業機構

3. **資金配置效率**：
   - 小額資金優先配置ETF
   - 大額資金可考慮直接選股
   - 定期定投策略適合長期投資者

**監控與調整**
- **績效評估**：每月檢視，每季詳細分析
- **因子監控**：追蹤各因子表現及市場環境變化
- **動態調整**：根據市場週期調整配置比例
- **成本控制**：監控交易成本和稅務影響

## 結論與建議

### 結論

1. **因子投資在兩個市場都有效**：無論是台股還是美股，因子投資都能創造超額報酬，但有效因子類型和穩定性存在差異。

2. **市場特性影響因子表現**：台股市場的內部人持股和庫藏股因子特別有效，而美股市場的Fama-French因子體系更為成熟。

3. **風險管理至關重要**：因子投資雖能創造超額報酬，但需要嚴格的風險管理，特別是流動性風險和因子衰退風險。

4. **長期投資價值明確**：因子投資是一種科學化的投資方法，長期實證支持其有效性，適合作為核心投資策略。

### 建議

1. **建立系統化投資流程**：無論是台股還是美股，都應建立系統化的因子選股和組合管理流程。

2. **持續學習和調整**：因子投資理論不斷發展，需要持續學習新的研究成果和市場變化。

3. **結合基本面分析**：量化因子投資應與基本面分析結合，避免純粹數據挖掘的陷阱。

4. **適當的期預管理**：因子投資不是短期致富的快速通道，需要長期持有和耐心等待。

5. **專業協助的重要性**：對於非專業投資者，可考慮透過ETF或專業基金參與因子投資，降低執行難度。
