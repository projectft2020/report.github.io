## 摘要

本論文首次為貝葉斯 MIDAS（Mixed Data Sampling）迴歸模型開發了變分推斷算法。作者提出了一個坐標上升變分推斷（Coordinate Ascent Variational Inference, CAVI）算法，用於線性權重參數化的貝葉斯 MIDAS 迴歸。該模型通過正規化約束將影響係數與權重函數參數分離，創建了一個雙線性結構，這使得通用的 Hamiltonian Monte Carlo 採樣器不可靠，同時保留了可被 CAVI 利用的條件共軛性。每個變分更新都有閉式解：迴歸係數和權重參數為高斯分佈，誤差方差為逆伽馬分佈。算法通過二階矩在區塊間傳播不確定性，區別於樸素的插入近似。

---

## 1. 研究背景與動機

### 1.1 MIDAS 迴歸簡介

MIDAS（Mixed Data Sampling）迴歸由 Ghysels 等人（2006）引入，提供了一個簡約框架，將高頻數據整合到低頻預測方程中。該方法已成為宏觀經濟即時預測（nowcasting）和金融波動率預測的標準工具，其中日頻或日內觀測值包含關於月度或季度總量的信息。

### 1.2 貝葉斯處理的優勢

貝葉斯 MIDAS 模型通過先驗分佈提供自然的不確定性量化和正則化，這些特性當高頻預測變量的數量為中等至大型時特別有價值。

### 1.3 現有方法的挑戰

貝葉斯 MIDAS 模型的後驗推斷幾乎完全依賴於馬可夫鏈蒙特卡羅（MCMC）方法。這些方法漸近精確但計算密集：
- 每次後驗評估需要數千次採樣迭代
- 隨著參數空間的增長，混合效率下降
- 雙線性結構導致通用 Hamiltonian Monte Carlo（HMC）採樣器失效
- 區塊 Gibbs 採樣器在高維度下自相關性嚴重

### 1.4 變分推斷的機遇

變分推斷（VI）提供了一種根本不同的方法，將後驗近似轉化為優化問題：
- 不從後驗分佈生成相關樣本
- 尋找可處理分佈族中最接近的成員
- 通過最大化證據下界（ELBO）
- 產生確定性算法，在 MCMC 所需時間的一小部分內收斂

## 2. 核心技術貢獻

### 2.1 模型規範

#### 2.1.1 觀測方程

對於低頻因變量 y_t (t = 1, ..., T) 和 J 個高頻預測變量，MIDAS 迴歸模型為：

```
y_t = α + Σ[j=1 to J] β_j * x̃_t^(j)(θ_j) + ε_t, ε_t ~ iid N(0, σ²)
```

其中：
- α 是截距
- β_j 是預測變量 j 的影響係數
- x̃_t^(j)(θ_j) 是 MIDAS 聚合

#### 2.1.2 線性權重參數化

採用 Chan 等人（2025）的線性參數化：

```
w_j(k; θ_j) = Σ[p=1 to P_j] θ_j,p * φ_p(k) = φ(k)ᵀθ_j
```

主要規範使用 Almon 多項式：φ_p(k) = k^(p-1)

#### 2.1.3 正規化約束與重新參數化

權重總和為一的約束：

```
c_jᵀθ_j = 1, 其中 c_j = Φ_jᵀ1_K_j
```

通過零空間重新參數化處理：

```
θ_j = θ_j⁰ + N_j * η_j
```

其中：
- θ_j⁰ = c_j / ||c_j||² 是最小範數特解
- N_j 具有張成 ker(c_jᵀ) 的正交列
- η_j 是自由參數

### 2.2 先驗規範

獨立先驗分佈：
```
η_j ~ N(0, σ_η² I), σ_η² = 1
α ~ N(0, 100)
β_j ~ N(0, 10)
σ² ~ Inverse-Gamma(0.01, 0.01)
```

### 2.3 均場近似

使用以下均場分解近似後驗：

```
q(ξ, η_1, ..., η_J, σ²) = q(ξ) * Π[j=1 to J] q(η_j) * q(σ²)
```

其中 ξ = (α, β_1, ..., β_J)ᵀ

## 3. CAVI 算法

### 3.1 算法概述

坐標上升變分推斷（CAVI）算法通過交替更新每個參數區塊來最大化證據下界（ELBO）。

### 3.2 區塊 1：迴歸係數

**命題 1：** 在均場分解下，迴歸係數的最優變分分佈為：

```
q*(ξ) = N(μ_ξ, Σ_ξ)
```

其中：
```
Σ_ξ = (E_q[σ⁻²] * Σ[t=1 to T] E_q[x̃_t x̃_tᵀ] + Λ_ξ)⁻¹
μ_ξ = Σ_ξ * (E_q[σ⁻²] * Σ[t=1 to T] y_t * E_q[x̃_t])
```

Λ_ξ 是先驗精度矩陣。

**關鍵特性：**
- 包含方差校正項：r_t^(j)ᵀ Σ_η_j r_t^(j)
- 將 η_j 的不確定性傳播到 ξ 的更新中
- 區別於樸素插入方法

### 3.3 區塊 2：權重參數

**命題 2：** 對於每個預測變量 j，權重參數的最優變分分佈為：

```
q*(η_j) = N(μ_η_j, Σ_η_j)
```

其中：
```
Σ_η_j = (E_q[σ⁻²] * E_q[β_j²] * Σ[t=1 to T] r_t^(j) r_t^(j)ᵀ + σ_η⁻² I)⁻¹
μ_η_j = Σ_η_j * E_q[σ⁻²] * (μ_β_j * Σ[t=1 to T] r_t^(j) * ē_t^(-j) - Σ[t=1 to T] r_t^(j) g_tᵀ Σ_ξ e_{j+1})
```

**三個重要特性：**
1. 精度由 E_q[β_j²] 縮放，包含 β_j 的方差
2. 均值包含協方差校正項，因為 β_j 與其他 ξ 分量聯合分佈
3. 矩陣求逆維度為 (P_j - 1) × (P_j - 1)，通常為 2 或 3，計算量可忽略

### 3.4 區塊 3：誤差方差

**命題 3：** 誤差方差的最優變分分佈為：

```
q*(σ²) = Inverse-Gamma(ã, b̃)
```

其中：
```
ã = a₀ + T/2
b̃ = b₀ + (1/2) * Σ[t=1 to T] E_q[e_t²]
```

且：
```
E_q[e_t²] = y_t² - 2y_t E_q[x̃_t]ᵀμ_ξ + tr(E_q[x̃_t x̃_tᵀ] * (μ_ξ μ_ξᵀ + Σ_ξ))
```

所需的其他矩：
```
E_q[σ⁻²] = ã / b̃
E_q[log σ²] = log b̃ - ψ(ã)
```

### 3.5 算法收斂性

**命題 4（單調收斂）：** 算法 1 生成的 ELBO 序列 {ℒ^(t)} 滿足對所有 t 都有 ℒ^(t+1) ≥ ℒ^(t)，並收斂到固定點。

**證明：** 每個坐標更新在保持其他因子固定的同時最大化相應因子的 ELBO。由於 ELBO 被 log p(y) 上界限制，單調非遞減序列收斂。

### 3.6 計算複雜度

每次迭代的主導成本是區塊 2 中的 (J+1) × (J+1) 矩陣求逆，複雜度為 O(J³)。區塊 1 中的 J 次求逆各為 (P_j-1) × (P_j-1)，當 P_j ≤ 4 時可忽略。收斂需要 3-70 次迭代（J=1 時為 3 次，J=50 時高達 70 次）。

在 T=200, J=50, P=3 的情況下，標準筆記本電腦上完整的 CAVI 運行約需 1.4 秒。

### 3.7 均場欠離散

均場分解忽略了 Cov(β_j, η_j)，產生比真實後驗更集中的變分邊際（欠離散）。

**不對稱效應：**
- ξ 中的影響係數與所有 J 個權重向量同時相關
- 每個 η_j 主要僅與其自己的 β_j 相關
- 隨著 J 增長，ξ 的累積忽略相關性增加，但每個 η_j 大致保持不變

**部分緩解：**
通過二階矩耦合，E_q[β_j²] 將 β_j 的不確定性傳播到 η_j 的校準，但沒有類似的機制在單個係數水平上補償 β_j。

## 4. 蒙特卡羅模擬研究

### 4.1 實驗設計

跨越 21 個 DGP 配置評估 CAVI 與區塊 Gibbs 採樣器：
- **第一層：** J ∈ {1, 3, 5, 10, 25, 50}, T ∈ {50, 100, 200, 400}
- **第二層：** 權重輪廓、信噪比、基函數類型、頻率比率
- **第三層：** 壓力測試
- 每個配置 500 次重複（J=25, 50 時 Gibbs 為 50 次，因為計算時間過長）
- 總計：10,500 次 CAVI 擬合，9,500 次 Gibbs 擬合

對於 J > 1 的配置，一半影響係數設為零。

### 4.2 主要結果：J 的可擴展性

**表 1 結果摘要（T=200, K=9, P=3）：**

| J | CAVI 偏差 | Gibbs 偏差 | 偏差差異 | 加速比 | β 覆蓋率 (CAVI) | β 覆蓋率 (Gibbs) | η 覆蓋率 (CAVI) |
|---|----------|-----------|---------|--------|-----------------|------------------|-----------------|
| 1 | 0.001 | 0.002 | -0.001 | 1,772× | 89.4% | 85.2% | 97.2% |
| 3 | 0.003 | 0.005 | -0.002 | 1,084× | 79.8% | 84.9% | 95.8% |
| 5 | 0.004 | 0.003 | 0.001 | 716× | 62.3% | 87.1% | 94.1% |
| 10 | 0.008 | 0.009 | -0.001 | 423× | 56.7% | 88.3% | 93.2% |
| 25 | 0.012 | - | - | 107× | 55.8% | - | 92.6% |
| 50 | 0.025 | - | - | 53× (外推) | 55.0% | - | 98.0% |

**點估計：**
- CAVI 和 Gibbs 的偏差和 RMSE 在所有可用配置中可比
- 偏差差異最多為 0.03，確認變分後驗均值作為可靠點估計器

**速度：**
- 加速範圍從 107× (J=25) 到 1,772× (J=1)
- 優勢隨維度增長：在 J=25，CAVI 0.3 秒完成，Gibbs 34 秒
- 在 J=50，CAVI 1.4 秒完成，Gibbs 預估 60+ 秒

**覆蓋率：**
- **η 覆蓋率：** 在所有配置中保持在 92% 以上，在 J=50 時達到 98%
- **β 覆蓋率：** 從 89.4% (J=1) 下降到 55.0% (J=50)，下降集中在 J ≥ 5

**Gibbs ESS 退化：**
- 最小 ESS 從 J=1 時的 4,103 下降到 J=10 時的 539，J=25 時進一步下降到 134
- 表明隨著雙線性耦合涉及更多區塊，MCMC 鏈中自相關性增加

### 4.3 主要結果：樣本大小 T

**表 2 結果摘要（J=3, K=9, P=3）：**

| T | CAVI β 覆蓋率 | Gibbs β 覆蓋率 | 加速比 |
|---|--------------|----------------|--------|
| 50 | 58.4% | 68.7% | 223× |
| 100 | 71.6% | 81.2% | 537× |
| 200 | 79.8% | 84.9% | 1,084× |
| 400 | 86.9% | 92.8% | 1,504× |

**關鍵發現：**
- CAVI 對 β 的覆蓋率隨 T 增加從 58.4% (T=50) 提高到 86.9% (T=400)
- 接近 Gibbs 基準
- 在 T=400，差距縮小到 6 個百分點
- 加速在 T 大時最大：T=400 時為 1,504×
- CAVI 僅需 5 次迭代收斂，而 Gibbs 保持 5,000 次抽取

### 4.4 魯棒性和壓力測試

**權重輪廓（configs 2A-1 至 2A-3）：**
- 遞減、駝峰形和 U 形輪廓都被兩種方法準確恢復
- 對 η 的覆蓋率在所有形狀下都超過 92%

**信噪比：**
- 高 SNR (σ₀² = 0.25)：CAVI 對 β 的覆蓋率為 90%
- 低 SNR (σ₀² = 4)：CAVI 覆蓋率降至 75%
- Gibbs 覆蓋率也下降（95% 到 86%），表明模型固有的困難

**頻率比率：**
- 在 K=65（季度到日），CAVI 和 Gibbs 都掙扎
- β 覆蓋率降至 48-50%，確認這是 DGP/基函數限制而非算法失敗

### 4.5 與自動微分 VI 的比較

**表 3 結果摘要（J ∈ {1, 3, 5}）：**

| J | CAVI 偏差 | ADVI-MF 偏差 | ADVI-FR 偏差 | CAVI 速度 vs ADVI-MF | CAVI 速度 vs ADVI-FR |
|---|----------|--------------|--------------|---------------------|---------------------|
| 1 | 0.001 | 0.011 | 0.009 | 2,000× | 100,000× |
| 3 | 0.003 | 0.042 | 0.027 | 5,000× | 50,000× |
| 5 | 0.004 | 0.048 | 0.031 | 10,000× | 20,000× |

**關鍵發現：**

**準確性：**
- ADVI 偏差比 CAVI 大 7-14 倍
- 隨機梯度優化未能為 MIDAS 雙線性結構找到正確的後驗眾數
- ADVI 損失函數景觀可能受到困擾 NUTS 的相同漏斗幾何影響

**速度：**
- CAVI 比 ADVI 快 2,000-100,000 倍
- 優勢比 CAVI-Gibbs 比較更為戲劇性
- ADVI 需要數千次梯度評估，而 CAVI 利用閉式條件共軛性

**結論：**
這些結果確認了本文的貢獻不僅是相對於 MCMC 的計算加速，而且是利用通用隨機梯度方法無法利用的條件結構的模型特定推導。

### 4.6 實務覆蓋率指導

對於需要在 β 上進行不確定量化的從業者，基於完整的蒙特卡羅證據：

**J ≤ 3 且 T ≥ 200：**
- CAVI 可信區間覆蓋率超過 83%，對於大多數應用足夠
- 簡單的事後校準（將變分標準差乘以乘數 κ）可恢復名義覆蓋率
  - κ = 1.2 對於 J=1（校準覆蓋率 94.2%）
  - κ = 1.8 對於 J=3（校準覆蓋率 94.9%）

**J ≥ 5：**
- 均場近似對於區間估計變得不充分
- 即使 κ = 3.0 在 J=5 時僅達到 86%，在 J=50 時為 81%
- 因為欠離散在參數間不均勻

**建議：**
在這種設置中，建議使用 Gibbs 採樣器對 β 進行推斷，或者在計算約束排除 MCMC 時，採用 Giordano 等人（2018）的線性響應校正，使用 ELBO 的二階靈敏度調整變分協方差。擴展到能夠捕獲 Cov(β_j, η_j) 的結構化變分族是未來工作的自然方向。

## 5. 實證應用：實現波動率預測

### 5.1 設置

應用 Ghysels 等人（2006）的 MIDAS-RV 規範預測 S&P 500 指數的月度實現波動率：

```
log RV_t = α + β * Σ[k=0 to K-1] w(k; θ) * r_{t-k}² + ε_t
```

其中：
```
RV_t = Σ[d=1 to D_t] r_{t,d}²
```

使用：
- K = 22 日滯後
- Almon 多項式，P = 3
- J = 1 和 J = 3 規範
- J = 3 中，三個預測變量對應來自當前月（t）、前一月（t-1）和兩月前（t-2）的日平方回報

基準包括：
- HAR-RV（Corsi, 2009）
- AR(1), AR(4)
- 歷史平均值

### 5.2 預測準確性

**表 4 結果（187 個月外樣本）：**

| 模型 | MSE | MAE | 相對 MSE（vs HAR） | DM p 值 |
|------|-----|-----|-------------------|---------|
| MIDAS J=1 (CAVI) | 0.4123 | 0.4287 | 1.000 | - |
| MIDAS J=1 (Gibbs) | 0.4122 | 0.4286 | 1.000 | - |
| MIDAS J=3 (CAVI) | 0.4166 | 0.4332 | 1.011 | 0.864 |
| MIDAS J=3 (Gibbs) | 0.4165 | 0.4331 | 1.011 | 0.864 |
| HAR-RV | 0.4122 | 0.4288 | 1.000 | - |
| AR(1) | 0.4762 | 0.4853 | 1.155 | 0.045 |
| 歷史平均 | 0.5421 | 0.5512 | 1.316 | <0.001 |

**CAVI-Gibbs 一致性：**
- J=1 時 MSE 差異為 0.00006（相對差異：0.01%）
- 確認蒙特卡羅發現
- MIDAS J=1 (CAVI) 緊密跟踪實際 log-RV 序列，性能與 HAR-RV 相當

**預測性能：**
- MIDAS J=3 幾乎匹配 HAR-RV（相對 MSE = 1.011, DM p=0.864）
- MIDAS 規範在常規水平下不顯著劣於 HAR
- AR(1) 顯著劣於 HAR（p=0.045）
- 歷史平均被強烈支配（p < 0.001）

### 5.3 計算時間

- **CAVI：** 每月估計在 1 毫秒（J=1）或 9 毫秒（J=3）內完成
  - J=1 加速比：936×
  - J=3 加速比：264×
- **Gibbs：** 每月估計需要 1-2.5 秒

**實際意義：**
在單次傳遞估計中，絕對時間差異（Gibbs 為 1-2 秒）可能看起來微不足道。然而，速度優勢在應用工作中日益常見的三種設置中變得至關重要：

1. **大規模模型比較：** 掃描許多預測變量集、滯後結構或基函數規範
2. **實時即時預測管道：** 數百個候選預測變量，估計必須在發布周期內完成
3. **擴展窗口或滾動窗口評估：** 數百次重新估計，如本練習（187 個外樣本月 × 多個規範）

在 J=10 預測變量和 500 個窗口下，CAVI 在約 33 秒內完成，而 Gibbs 需要 2.2 小時。

### 5.4 估計的權重輪廓

**全樣本 J=1 估計：**
- CAVI 和 Gibbs 都產生單調遞減的輪廓
- 給最近的日觀測值分配最大權重
- 經濟上合理，因為最近的平方回報對當月波動率更提供信息
- 點估計幾乎相同
- CAVI 帶（紫色）比 Gibbs 帶（粉色）寬
  - 反映不同的不確定性傳播機制
  - CAVI 通過 (20) 中的 E_q[β²] 項膨脹有效方差，其中包括 β 的方差
  - Gibbs 帶條件於每個 β 的後驗抽取計算

**J=3 規範：**
- 預測變量 1（當月）保持相同的遞減輪廓
- 預測變量 2 和 3（月份 t-1 和 t-2）權重集中在零附近，可信帶寬
- 表明可忽略的邊際貢獻
- 與 J=3 超過 J=1 的預測收益一致

## 8. 應用價值評估

### 8.1 優勢

1. **計算效率：**
   - 相比 MCMC 有顯著加速（107× - 1,772×）
   - 特別適合大規模模型比較和實時應用
   - 擴展窗口評估的實際可行

2. **點估計精度：**
   - 後驗均值與 Gibbs 基準本質相同
   - 偏差差異 < 0.03
   - RMSE 在所有配置中可比

3. **權重參數校準：**
   - 覆蓋率始終 > 92%
   - 適合權重輪廓估計應用

4. **實現簡單性：**
   - 閉式更新易於實現
   - 不需要複雜的採樣器調整
   - 收斂判據清晰（ELBO）

5. **確定性結果：**
   - 無 MCMC 鏈的隨機性
   - 結果可重現
   - 便於調試和驗證

### 8.2 局限性

1. **均場欠離散：**
   - 影響係數可信區間覆蓋率下降
   - J ≥ 5 時顯著（55-60%）
   - 不適合需要精確不確定量化的高維應用

2. **維度依賴：**
   - 性能退化隨 J 增長
   - 低 T 時覆蓋率較差
   - 需要足夠數據量

3. **模型結構限制：**
   - 僅支持線性權重參數化
   - 不直接支持非線性規範（指數 Almon、貝塔多項式）
   - 需要擴展才能處理時變參數

4. **近似本質：**
   - 不是漸近精確的 MCMC
   - 依賴變分族的選擇
   - 潛在的局部最優值

### 8.3 適用場景

**推薦使用 CAVI：**
- 點預測應用
- 變量選擇（通過後驗均值或收縮）
- 權重輪廓估計
- 大規模模型比較掃描
- 實時即時預測管道
- 計算資源受限環境
- 快速原型開發

**建議使用 MCMC：**
- 需要精確可信區間的高維設置（J ≥ 5）
- 不確定量化是核心要求
- 計算時間不是約束
- 需要漸近精確性

**建議結合使用：**
- 用 CAVI 進行初步探索和變量選擇
- 用 MCMC 對最終模型進行精確推斷
- 用 CAVI 提供良好初始值

## 9. 結論

本論文首次為貝葉斯 MIDAS 迴歸開發了變分推斷算法，通過利用線性參數化模型的條件共軛性，產生了所有參數區塊的閉式更新。算法通過二階矩在區塊間傳播不確定性，包括針對 MIDAS 雙線性設置特有的協方差校正。

在跨越 21 個配置的蒙特卡羅研究中，CAVI 產生的後驗均值與區塊 Gibbs 採樣器本質相同，同時實現了 107× 到 1,772× 的加速。通用 ADVI 相比產生 7-14 倍更大的偏差，同時慢數量級，確認了模型特定推導對此模型類的重要性。

權重參數校準良好（覆蓋率 > 92%）。影響係數可信區間表現出均場欠離散，隨預測變量維度增長，覆蓋率從 89% 下降到 55%——這是一個誠實記錄的權衡。

實證應用確認了 CAVI 和 Gibbs 產生幾乎相同的預測，CAVI 在 10 毫秒內完成每月估計，而 Gibbs 需要 1-2.5 秒。在需要數百次重新估計的擴展窗口評估中，這一優勢變得至關重要。

對於點預測、變量選擇或權重輪廓估計的應用，CAVI 提供了一個引人注目的替代方案。對於高維設置中 β 的良好校準可信區間，應優先選擇 Gibbs 採樣或結構化 VI。

未來工作自然擴展到時變參數 MIDAS、全局-局部收縮先驗和隨機波動率，進一步增強變分推斷在混合頻率計量經濟學中的能力和適用性。

### B. 覆蓋率校準公式

對於需要校準的設置：

```
σ_calibrated = κ * σ_variational

其中 κ 根據 J 和 T 選擇：
- J = 1, T ≥ 200: κ = 1.2
- J = 3, T ≥ 200: κ = 1.8
- J ≥ 5: 校準效果有限，建議使用 MCMC
```

### C. 計算複雜度分析

**每次迭代複雜度：**
- 區塊 1 (J 個區塊): O(J * P³)，其中 P ≤ 4，可忽略
- 區塊 2: O(J³)，主導成本
- 區塊 3: O(T)

**總複雜度：**
- O(n_iter * (J³ + T))
- n_iter ∈ [3, 70]，隨 J 和 T 增長

**內存需求：**
- 存储基函數矩陣：O(J * K * P)
- 存储變分參數：O(J² + J * P)
- 中間計算：O(J * T)

**實現建議：**
- 對區塊 2 使用 Cholesky 分解進行穩定求逆
- 並行化區塊 1 的 J 個更新（獨立於區塊間）
- 預計算並緩存 r_t^(j) 和 c_t^(j)
- 使用稀疏矩陣技術（如果適用）
