## 研究摘要

本研究深入分析了1987年10月19日「黑色星期一」股市崩盤事件，從事件背景、崩盤原因、先行指標提取及與其他崩盤事件的比較等多個維度進行全面探討。研究結合了趨勢交易專案t003（失效監控）和風險管理專案s002（肥尾風險）的研究成果，為崩盤預測模型提供理論基礎和特徵工程輸入。

## 1. 事件概述

### 1.1 時間線詳情

**1987年10月19日（黑色星期一）詳細時間線：**

- **開盤前（美東時間上午9:30前）**：
  - 亞洲市場已經率先暴跌（香港恆生指數下跌45.8%）
  - 期貨市場開始出現大量賣單壓力
  - 投資者恐慌情緒開始蔓延

- **開盤（上午9:30）**：
  - 道瓊工業平均指數（DJIA）開盤即暴跌
  - 標普500指數95支成分股延遲開盤
  - 期貨市場持續拋售，與現貨市場脫節

- **交易日中（上午10:00-下午3:00）**：
  - 程序化交易系統觸發大量賣單
  - Portfolio Insurance策略自動執行賣出指令
  - 流動性枯竭，買單消失
  - 交易系統超負荷，訂單處理延遲超過1小時

- **收盤前90分鐘（下午2:00-4:00）**：
  - 出現最激烈的拋售壓力
  - DJIA在最後90分鐘內加速下跌
  - 交易商面臨巨額追加保證金壓力

- **收盤（下午4:00）**：
  - DJIA收盤報1,738.74點，較開盤2,246.74點暴跌508點
  - 單日跌幅達22.6%，創歷史最大單日百分比跌幅紀錄
  - 全球股市損失估計達1.71兆美元

### 1.2 全球市場影響

**主要市場表現：**
- **美國**：DJIA下跌22.6%，標普500下跌20.5%
- **英國**：FTSE 100指數兩日下跌23%
- **香港**：恆生指數下跌45.8%，期貨交易所關閉4個工作日
- **日本**：日經225指數下跌14.9%（被稱為「藍色星期二」）
- **紐西蘭**：股市下跌60%，經濟陷入長期衰退
- **其他**：全球23個主要工業國家中，19個跌幅超過20%

### 1.3 當時宏觀環境

**關鍵宏觀經濟指標：**
- **利率環境**：聯準會緊縮貨幣政策，3個月期國庫券收益率從5.30%升至7.19%
- **通貨膨脹**：通脹壓力上升，聯準德銀行加息引發美元匯率危機
- **貿易逆差**：美國貿易逆差擴大，美元走弱
- **盧浮宮協定**：1987年2月簽署，但市場對協定效力缺乏信心
- **股票估值**：P/E比率超過20，市場被認為嚴重高估

## 2. 崩盤原因分析（多維度）

### 2.1 程序化交易與Portfolio Insurance策略

**Portfolio Insurance機制：**
- 由Mark Rubinstein和Hayne Leland於1976年開發
- 設計目的：對沖投資組合風險，在市場下跌時自動賣出期貨
- **正反饋循環**：市場下跌→觸發賣出指令→加劇下跌→觸發更多賣出指令

**程序化交易的影響：**
- 交易量佔市場總交易量的20-30%
- 計算機模型根據價格變化自動執行交易，忽略基本面因素
- 當95支標普500成分股延遲開盤時，期貨價格與現貨價格嚴重脫節

### 2.2 期貨市場現貨價格套利失靈

**套利失靈原因：**
- 期貨市場正常開盤，現貨市場大量股票延遲開盤
- 現貨價格「陳舊」，未能反映即時市場情況
- 價格差距擴大，套利交易者難以有效執行策略

**後果：**
- 期貨市場價格失去價格發現功能
- 投資者無法通過套利交易穩定市場
- 進一步加劇市場混亂和不確定性

### 2.3 流動性枯竭與賣單積壓

**流動性枯竭機制：**
- 買單完全消失，市場單邊賣出
- 做市商無法應對巨額賣單壓力
- 195支股票出現交易延遲或暫停

**賣單積壓情況：**
- NYSE SuperDot系統超負荷運作
- 訂單處理延遲超過1小時
- 大額基金轉帳延遲，Fedwire系統多次中斷

### 2.4 市場情緒恐慌

**恐慌傳播機制：**
- 媒體大篇幅報道市場暴跌，加劇投資者恐慌
- 機構投資者和散戶同時拋售
- 「羊群效應」導致非理性賣出行為

**關鍵恐慌事件：**
- 10月17日（週六）美國財政部長James Baker威脅貶值美元
- 10月19日市場傳言NYSE即將關閉
- 10月20日傳言芝加哥商品期貨交易所結算公司破產

## 3. 先行指標清單（10-15個）

### 3.1 技術指標

#### **1. 價格/成交量背離指標（Price/Volume Divergence）**
- **計算方法**：
  ```
  PVD = (今日成交量 / 20日均量) × (今日價格變動% / 20日平均價格變動%)
  當PVD < 0.5且價格創新高時，發出警訊
  ```
- **數據來源**：每日價格和成交量數據
- **歷史驗證**：1987年10月16日，成交量激增但價格上漲動能減弱

#### **2. ADX趨勢強度衰減（ADX Trend Strength Decay）**
- **計算方法**：
  ```
  14日ADX值從高位（>40）持續下降3天以上
  同時+DI線從高位下跌，-DI線開始上升
  ```
- **數據來源**：每日高、低、收盤價
- **歷史驗證**：1987年8月ADX達到高位後持續下降

#### **3. MACD負背離（MACD Negative Divergence）**
- **計算方法**：
  ```
  價格創新高但MACD柱狀圖未能創新高
  MACD信號線從上方穿越MACD線（死亡交叉）
  ```
- **數據來源**：每日收盤價
- **歷史驗證**：1987年8月和9月出現明顯負背離

#### **4. 隱含波動率激增（Implied Volatility Spike）**
- **計算方法**：
  ```
  期權隱含波動率20日均值的2個標準差以上
  VIX（或其前身）單日上漲超過30%
  ```
- **數據來源**：期權市場數據
- **歷史驗證**：1987年10月16日隱含波動率從36.37%升至150.19%

### 3.2 微結構指標

#### **5. 期貨基差擴大（Futures Basis Widening）**
- **計算方法**：
  ```
  基差 = 期貨價格 - 現貨價格
  當基差絕對值超過歷史95%分位數時發出警訊
  ```
- **數據來源**：期貨和現貨市場即時價格
- **歷史驗證**：1987年10月19日期貨與現貨價格嚴重脫節

#### **6. 流動性指標下降（Liquidity Indicator Decline）**
- **計算方法**：
  ```
  LI = (買方深度 - 賣方深度) / 總訂單簿深度
  當LI持續3日為負且絕對值擴大時發出警訊
  ```
- **數據來源**：交易所訂單簿數據
- **歷史驗證**：1987年10月19日買單深度接近零

#### **7. 賣買比失衡（Sell/Buy Order Imbalance）**
- **計算方法**：
  ```
  SBOI = 賣單數量 / 買單數量
  當SBOI > 3.0且持續擴大時發出警訊
  ```
- **數據來源**：交易所訂單流數據
- **歷史驗證**：崩盤前SBOI達到歷史極值

### 3.3 情緒指標

#### **8. Put/Call比率異常（Put/Call Ratio Anomaly）**
- **計算方法**：
  ```
  PCR = 賣權成交量 / 買權成交量
  當20日PCR均值的2個標準差以上時發出警訊
  ```
- **數據來源**：期權市場成交量數據
- **歷史驗證**：1987年10月PCR達到歷史高位

#### **9. 融資融券餘額異常（Margin Debt Anomaly）**
- **計算方法**：
  ```
  MDA = (融資餘額 / 市值) / 歷史平均值
  當MDA > 1.5且融資餘額開始下降時發出警訊
  ```
- **數據來源**：監管機構融資數據
- **歷史驗證**：1987年融資餘額佔市值比例達到歷史高位

#### **10. 市場廣度惡化（Market Breadth Deterioration）**
- **計算方法**：
  ```
  MB = (上漲家數 - 下跌家數) / 總交易家數
  當MB持續5日為負且絕對值擴大時發出警訊
  ```
- **數據來源**：交易所個股漲跌數據
- **歷史驗證**：1987年10月市場廣度指標持續惡化

### 3.4 宏觀指標

#### **11. 利率變化加速（Interest Rate Acceleration）**
- **計算方法**：
  ```
  IRA = 3個月期國庫券收益率3個月變動率
  當IRA絕對值 > 100bps時發出警訊
  ```
- **數據來源**：美國財政部利率數據
- **歷史驗證**：1987年初至10月，利率上升189個基點

#### **12. 通脹數據意外（Inflation Surprise）**
- **計算方法**：
  ```
  IS = (實際CPI - 預期CPI) / 預期CPI
  當IS絕對值 > 0.5%時發出警訊
  ```
- **數據來源**：勞工統計局CPI數據
- **歷史驗證**：1987年通脹數據多次超出預期

#### **13. 貿易逆差惡化（Trade Deficit Deterioration）**
- **計算方法**：
  ```
  TDD = 貿易逆差 / GDP的年度變動率
  當TDD > 10%時發出警訊
  ```
- **數據來源**：商務部貿易數據
- **歷史驗證**：1987年貿易逆差大幅擴大

#### **14. 匯率波動率激增（Exchange Rate Volatility Spike）**
- **計算方法**：
  ```
  ERV = 美元指數30日波動率的年化值
  當ERV > 15%時發出警訊
  ```
- **數據來源**：美聯儲美元指數數據
- **歷史驗證**：1987年美元匯率大幅波動

#### **15. 信用利差擴大（Credit Spread Widening）**
- **計算方法**：
  ```
  CSW = (高收益債收益率 - 國庫券收益率)的3個月變動
  當CSW > 50bps時發出警訊
  ```
- **數據來源**：債券市場收益率數據
- **歷史驗證**：1987年信用利差開始擴大

## 4. 與其他崩盤事件的對比

### 4.1 主要崩盤事件比較

| 事件 | 發生時間 | 跌幅 | 主要原因 | 持續時間 | 通用信號 |
|------|----------|------|----------|----------|----------|

| 2000 互聯網泡沫 | 2000-2002 | 納斯達克78% | 估值泡沫、盈利預期落空 | 長期（2年） | 估值過高、成交量異常 |

| 2020 COVID崩盤 | 2020.03 | S&P 500 34% | 疫情恐慌、流動性危機 | 中期（1月） | 波動率激增、市場廣度惡化 |

### 4.2 通用崩盤信號提取

#### **技術層面通用信號：**
1. **價格與成交量背離**：所有崩盤前均出現價格創新高但成交量動能減弱
2. **波動率激增**：VIX或其前身指標均出現異常跳升
3. **趨勢指標衰減**：ADX、MACD等趨勢指標顯示強度減弱

#### **微結構通用信號：**
1. **流動性枯竭**：買單深度顯著減少，市場單邊賣出
2. **期貨現貨脫節**：期貨與現貨價格差距擴大
3. **訂單失衡**：賣單遠超買單，失衡指數達極值

#### **情緒層面通用信號：**
1. **Put/Call比率異常**：避險需求激增，賣權成交量大增
2. **融資槓桿過高**：市場槓桿率達到歷史高位
3. **市場廣度惡化**：下跌家數遠超上漲家數

#### **宏觀層面通用信號：**
1. **利率快速變動**：利率環境急劇變化
2. **流動性收緊**：央行貨幣政策轉向
3. **經濟數據意外**：關鍵經濟指標超出預期
