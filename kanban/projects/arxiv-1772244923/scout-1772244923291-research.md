## 執行摘要

本研究提出一種基於可交換多任務高斯過程（Gaussian Processes, GPs）的新穎因果推斷方法，用於評估面板數據中政策干預的因果效應。該方法特別針對兩種場景：單一受治療單位的一次性干預，以及多個受治療單位錯開採用干預的設計。通過在所有單位之間建立可交換的協方差結構，該模型能夠靈活地捕獲非線性時間趨勢和單位間的相依性，同時提供反事實路徑的概率預測和完整的量化不確定性區間。

**核心貢獻**: 將可交換高斯過程框架應用於面板數據因果推斷，為錯開採用設計提供一種計算高效且靈活的替代方案，能夠處理大量受治療和控制單位。

**關鍵優勢**:
1. 非參數/半參數建模，對趨勢形式假設較少
2. 自動量化反事實預測的不確定性
3. 可交換結構使模型能夠在單位間「借力」
4. 計算效率高，適用於大規模錯開採用數據集

**主要應用**: 論文通過兩個實證案例驗證方法：加州第99號提案（Prop99）菸草控制政策和希臘加油站監控設施政策評估。

## 一、研究背景與動機

### 1.1 政策評估中的面板數據挑戰

現代公共政策評估越來越依賴於追蹤多個單位（如州、地區、企業或個人）隨時間變化的縱向數據集。在許多應用中，核心問題是評估影響部分單位在特定時刻的政策干預的因果效應，使用剩餘單位作為控制組來重建如果干預從未發生時會觀察到的反事實路徑。

### 1.2 現有方法的局限性

**傳統方法及其問題**：

1. **雙重差分（DiD）**: 
   - 使用線性雙向固定效應模型
   - 加法假設過於嚴格，當效應隨時間變化或趨勢非線性時可能產生偏差
   - 公式: y_it = α_i + γ_t + β w_it + ε_it

2. **合成控制法（Synthetic Control）**:
   - 將受治療單位的反事實路徑估計為控制單位的加權和
   - 依賴於線性組合假設，當趨勢非線性時可能受限
   - 需要相當長的事前期

3. **廣義合成控制（GSC）**:
   - 擴展TWFE框架，允許時間特定和單位特定效應的交互
   - y_it(0) = γ_t + λ_t^⊤ μ_i + ε_it
   - 仍然有結構性假設

4. **貝葉斯結構時間序列（BSTS）**:
   - 通過狀態空間表示建模
   - 將無干預結果分解為潛在分量（局部水平、趨勢、季節性波動、迴歸項）
   - 潛在狀態遵循狀態轉移方程，對狀態方差和迴歸係數設置先驗

### 1.3 高斯過程方法的優勢

高斯過程先驗提供了一種靈活的非參數或半參數替代方案，用於建模時間序列動態和協變量關係：
- **靈活性**: 不受特定函數形式限制
- **非線性**: 能夠捕獲非線性趨勢和時變關係
- **不確定性量化**: 提供關於反事實路徑的可信賴區間
- **多任務學習**: 多任務GP模型將每個單位的結果軌跡視為一個「任務」並聯合學習

## 二、理論框架

### 2.1 面板結構與潛在結果

**符號定義**:
- 單位: i = 1, ..., m
- 時期: t = 1, ..., T
- 單位集合: ℐ = {1, ..., m}
- 時期集合: 𝒯 = {1, ..., T}
- 結果變量: y_it ∈ ℝ
- 協變量向量: x_it ∈ ℝ^p

**治療指定**:
- 政策干預在不同單位特定時間點引入，影響單位子集 ℐ₁ ⊆ ℐ
- 剩餘單位 ℐ₀ = ℐ \\ ℐ₁ 作為潛在控制組
- 治療指標: w_it ∈ {0, 1}
  - w_it = 1: 政策已在時間t應用於單位i
  - w_it = 0: 否則

**單一受治療單位設定**:
- 令 i⋆ 表示受治療單位，ℐ₁ = {i⋆}
- 將時間軸分為事前期和事後期:
  - 事前期: 𝒯₀ = {1, ..., T₀}
  - 事後期: 𝒯₁ = {T₀+1, ..., T}

### 2.2 潛在結果框架

每個 (i,t) 有兩個潛在結果：
- y_it(0): 無治療時會觀察到的結果
- y_it(1): 治療下會觀察到的結果

**實際結果**:
```
y_it = y_it(0) · 𝕀{w_it=0} + y_it(1) · 𝕀{w_it=1}
```

**核心問題**: 對於受治療單位 i⋆，我們只觀察到 y_i⋆t(1)，而無法觀察到 t > T₀ 時的 y_i⋆t(0)。主要任務是估計序列 {y_i⋆t(0): t ∈ 𝒯₁}，這給出了受治療單位的反事實路徑。

### 2.3 因果估計量

**個體因果效應**:
```
δ_{i,t} = y_{i,t}(1) - y_{i,t}(0)
```

**平均因果效應**:
```
τ_{i⋆,t} = 𝔼[δ_{i⋆,t} | 𝒟₀], t ∈ 𝒯₁
```
其中 𝒟₀ 是未受治療影響的結果和協變量集合。

**後驗方差**:
```
σ²_{i⋆,t} = 𝕍[δ_{i⋆,t} | 𝒟₀]
```

**累積治療效應**:
```
Δ_{i⋆} = Σ_{t=T₀+1}^T δ_{i⋆,t}
```

**平均治療效應**:
```
δ̄_{i⋆} = (1 / (T-T₀)) Σ_{t=T₀+1}^T δ_{i⋆,t}
```

### 2.4 核心假設

論文採用了五個關鍵假設（基於Menchetti等人2023年的工作）：

**假設1（單一尖銳干預）**
存在時間 T₀ ∈ {1, ..., T}，對於受治療單位 i⋆：
- w_i⋆t = 0 對於所有 t ≤ T₀
- w_i⋆t = 1 對於所有 t > T₀
- 對於所有控制單位 i ∈ ℐ₀，w_it = 0 在所有時間 t

**假設2（時間無干擾性）**
單位 i 在時間 t 的結果僅取決於其自身的治療路徑至時間 t，而不取決於其他單位的治療歷史：
```
y_it(w_{i,1:t}) = y_it(w_{1:m,1:t})
```

**假設3（協變量不受治療影響）**
對於所有單位和所有 t ≥ T₀：
```
x_it(1) = x_it(0)
```
模型中使用的協變量不受治療影響。

**假設4（結果的無預期）**
對於所有 t < T₀，單位 i 在時間 t 的結果僅取決於其過去和當前的治療歷史 w_{i,1:t}，不受該單位未來治療分配的影響：
```
y_it(w_{i,1:t}, w_{i,t+1:T}) = y_it(w_{i,1:t}, w'_{i,t+1:T})
```

**假設5（無預期治療）**
時間 T₀ 的治療分配僅取決於該單位的過去協變量和過去結果：
```
p(w_iT₀ | w_{1:m,1:T₀-1}, y_{1:m,1:T}, x_{1:m,1:T}) = p(w_iT₀ | y_{i,1:T₀-1}, x_{i,1:T₀-1})
```

## 三、可交換高斯過程模型

### 3.1 核心思想

可交換GP的關鍵思想是將每個單位特定的潛在函數 f_i(t) 分解為所有單位共享的公共分量和單位特定偏差：

```
f_i(t) = μ(t) + g_i(t), i = 1, ..., m, t ∈ 𝒯
```

這等價於帶有兩個潛在過程和固定載荷的線性核心化模型（Álvarez等人，2012）：
- 所有單位在公共過程 μ 上的載荷為 1
- 只有單位 i 在其自身偏差 g_i 上載荷，載荷也為 1

### 3.2 先驗設定

**公共過程**:
```
μ(·) ~ GP(0, σ_μ² · k(·,·))
```

**單位特定偏差**:
```
g_i(·) ~ i.i.d. GP(0, σ_g² · k(·,·)), i = 1, ..., m
```

**獨立性假設**:
- μ ⟂ g_i ⟂ g_j (i ≠ j)
- (μ, g_i) ⟂ ε_it

**核函數選擇**:
論文考慮了兩種時間核：
1. **Ornstein-Uhlenbeck (OU) 核**:
   ```
   k^{OU}(t,s) = exp(-|t-s| / ℓ)
   ```

2. **徑向基函數（RBF）核**:
   ```
   k^{RBF}(t,s) = exp(-(t-s)² / (2ℓ²))
   ```

兩種核都假設單位方差和長度尺度 ℓ。

### 3.3 協方差結構

**潛在函數的協方差**:
對於 i ≠ j:
```
Cov(f_i(t), f_j(s)) = σ_μ² · k(t,s)
```

對於 i = j:
```
Cov(f_i(t), f_i(s)) = (σ_μ² + σ_g²) · k(t,s)
```

**堆疊向量的分塊協方差矩陣**:
```
Cov(f) = 
⎡(σ_μ² + σ_g²)K   σ_μ²K   ...   σ_μ²K ⎤
⎢   σ_μ²K    (σ_μ² + σ_g²)K ... σ_μ²K ⎥
⎢   ...        ...      ⋱   ...    ⎥
⎣   σ_μ²K      σ_μ²K   ... (σ_μ² + σ_g²)K⎦
```

**Kronecker表示**:
```
Cov(f) = (1_m · 1_m^⊤) ⊗ (σ_μ² · K) + I_m ⊗ (σ_g² · K)
```

其中 1_m 是 m 維全一向量。這展示了分塊協方差矩陣的可交換結構：f 的聯合分布在單位索引的置換下是不變的。

### 3.4 觀測模型

**無治療潛在結果觀測模型**:
```
y_it(0) = f_i(t) + ε_it, ε_it ~ 𝒩(0, ω_i²)
```

其中：
- ε_it 在 (i,t) 之間相互獨立
- ε_it ⟂ f_j 對於所有 i, j
- 治療僅影響受治療單位的實現事后觀測值

**堆疊表示**:
```
y_i(0) = f_i + ε_i
y = f + ε
```

其中:
```
ε = (ε_1^⊤, ..., ε_m^⊤) ~ 𝒩(0, Ω)
Ω = diag(ω_1² · I_T, ..., ω_m² · I_T)
```

**邊際似然**:
```
y ~ 𝒩(0, K_f + Ω)
```

### 3.5 擴展：包含協變量

擴展分解以允許單位特定偏差依賴於協變量和時間：

```
f_i(t) = μ(t) + g_{i,1}(t) + g_{i,2}(x_it)
```

其中：
- g_{i,1}: 時間僅偏差
- g_{i,2}: 捕獲由協變量解釋的額外單位特定變異
- x_it: 單位 i 在時間 t 的協變量

**擴展的GP先驗**:
```
μ(·) ~ GP(0, σ_μ² · k_time(·,·))
g_{i,1}(·) ~ i.i.d. GP(0, σ_{g₁}² · k_time(·,·))
g_{i,2}(·) ~ i.i.d. GP(0, σ_{g₂}² · k_x(·,·)), i = 1, ..., m
```

所有過程相互獨立且與噪聲獨立。

**擴展協方差結構**:
```
Cov(f) = 
⏟共享均值 (1_m·1_m^⊤)⊗(σ_μ²·K_time) 
+ ⏟時間偏差 I_m⊗(σ_{g₁}²·K_time)
+ ⏟協變量偏差 diag(σ_{g₂}²·K_{x,1}, ..., σ_{g₂}²·K_{x,m})
```

### 3.6 不同長度時間網格的擴展

當單位具有不同長度的時間網格時，一般的協方差形式為：

```
Cov(f_i(t), f_j(s)) = σ_μ² · k_time(t,s) + 
                      𝕀{i=j} · [σ_{g₁}² · k_time(t,s) + σ_{g₂}² · k_x(x_it, x_is)]
```

單個潛在函數的方差：
```
Var(f_i(t)) = (σ_μ² + σ_{g₁}²) · k_time(t,t) + σ_{g₂}² · k_x(x_it, x_it)
```

## 四、錯開採用設計

### 4.1 問題設定

許多實證政策設定涉及：
- 多個受治療單位（通常是數百或數千個）
- 在常見面板中觀察，伴隨一組從未接受治療的控制單位
- 治療採用錯開：每個受治療單位 i ∈ ℐ₁ 在其自己的時間 T_{0i} 接受干預
- ℐ₀ 中的單位在整個過程中保持未受治療

**關鍵限制條件**: 無干擾條件——分配給一個單位的治療不影響其他單位的結果。

### 4.2 可交換GP的適應性

可交換GP模型天然適合錯開採用結構，因為它旨在跨單位匯總信息。原則上，可以將可交換多任務GP擬合到整個面板：
- 使用所有受治療單位的事前觀測值
- 使用從未接受治療的單位的所有觀測值
- 然後同時為所有受治療單位的未受治療潛在結果生成反事實預測

這種方法使用了所有可用的未受治療信息，並跨單位聯合學習共享潛在結構。

### 4.3 計算挑戰與解決方案

**問題**: 在精確GP的情況下，每次評估對數邊際似然及其導數都需要 n×n 矩陣的Cholesky分解，其中 n 是訓練觀測總數。

**解決方案**: 採用基於從從未接受治療的單位池中採樣的一個單位一次框架。

**算法**:
對於每個受治療單位 i ∈ ℐ₁：
1. 構建一個較小的訓練集，包括：
   - 該單位的事前觀測值
   - 從 ℐ₀ 中隨機採樣的 M 個未接受治療的單位
2. 如果 M 足夠大且採樣是隨機的，選定的控制近似完整的未受治療分布
3. 將此問題視為單一受治療單位反事實預測任務
4. 對每個受治療單位獨立重複此過程

**維度約簡**:
相關矩陣維度從 n 減少到：
```
n_i = T_{0i} + M · T
```
其中：
- T 是每個未接受治療單位可用的時間點數
- M 是控制單位數
- T_{0i} 是受治療單位 i 的事前觀測數

**計算複雜度降低**:
從 O(n³) 降低到 O(Σ_{i∈ℐ₁} n_i³)，使我們能夠在大規模錯開採用面板中反覆擬合可交換GP模型。

## 五、估計與實現

### 5.1 階層規範

模型的基本組成部分：
- 堆疊潛在無治療結果值：向量 f
- 超參數向量 θ，包括方差分量、觀測噪聲方差和不同的長度尺度參數

**階層規範**:
```
f | θ ~ 𝒩(0, K_f(θ))
y | f, θ ~ 𝒩(f, Ω(θ))
```

其中：
- K_f(θ): 可交換協方差矩陣
- Ω(θ): 對角噪聲協方差矩陣

### 5.2 超參數學習

**邊際似然**:
由於先驗和似然項都是高斯的，可以通過積分潛在向量獲得閉形式的邊際似然：
```
y | θ ~ 𝒩(0, Σ(θ))
Σ(θ) = K_f(θ) + Ω(θ)
```

**Type II 最大似然（經驗貝葉斯）**:
選擇點估計：
```
θ̂ = argmax_θ log p(y | θ)
```

在實現中，使用Python的GPflow庫。

### 5.3 預測

讓 y 是控制單位和受治療單位事前無治療潛在結果的完整堆疊向量，Z 是相應的輸入。假設我們希望預測在輸入集合 Z* 的無治療情況下事后潛在結果向量 y*。

**聯合高斯分布**:
```
[y; y*] | θ̂ ~ 
𝒩(𝟎, [[K(Z,Z;θ̂)+Ω, K(Z,Z*;θ̂)];
       [K(Z*,Z;θ̂), K(Z*,Z*;θ̂)+Ω*]])
```

其中：
- K(Z,Z;θ̂): 訓練輸入處潛在GP的協方差矩陣
- K(Z*,Z*;θ̂): 預測輸入處的相應矩陣
- K(Z,Z*;θ̂): 訓練和預測位置之間的潛在交叉協方差

**后驗預測分布**:
```
y* | Z, Z*, y, θ̂ ~ 𝒩(m*, V*)
```

其中：
```
m* = K(Z*,Z;θ̂) · [K(Z,Z;θ̂) + Ω]⁻¹ · y

V* = K(Z*,Z*;θ̂) + Ω* - K(Z*,Z;θ̂) · [K(Z,Z;θ̂) + Ω]⁻¹ · K(Z,Z*;θ̂)
```

**解釋**:
- 預測均值 m* 提供受治療單位的反事實路徑
- V* 量化相關的不確定性
- V* 僅依賴於 (Z,Z*) 和超參數 θ̂，完全獨立於實現的結果 y
- m* 是觀測結果的線性組合，權重由協方差結構和噪聲水平確定

### 5.4 因果效應計算

在GP模型與高斯似然下：
- 反事實結果 y_i⋆t(0) 的后驗預測分布是高斯分布，均值為 m_{0t}，方差為 v_{0t}²
- 觀測到的受治療結果 y_i⋆t(1) 是固定的

因此，差值 δ_i⋆t = y_i⋆t(1) - y_i⋆t(0) 在后驗下也是高斯的：
```
δ_i⋆t | 𝒟₀ ~ 𝒩(τ_i⋆t, σ²_i⋆t)
```
其中：
```
τ_i⋆t = y_i⋆t(1) - m_{0t}
σ²_i⋆t = v_{0t}²
```

可以直接從中導出時間特定效應的點估計和可信賴區間。

## 六、應用案例

### 6.1 應用1：第99號提案（Prop99）

**數據描述**:
- 使用Abadie等人（2010）合成控制應用中的數據集
- 加州於1988年實施菸草控制計劃

## 七、方法論比較與評估

### 7.1 與現有方法的比較

| 特性 | 傳統DiD | 合成控制 | GSC | BSTS | 可交換GP |
|------|---------|---------|-----|------|----------|
| 非線性趨勢 | ❌ | ⚠️ | ⚠️ | ✅ | ✅ |
| 不確定性量化 | ⚠️ | ❌ | ⚠️ | ✅ | ✅ |
| 多任務學習 | ❌ | ⚠️ | ⚠️ | ⚠️ | ✅ |
| 計算效率 | ✅ | ✅ | ✅ | ⚠️ | ✅ |
| 錯開採用 | ⚠️ | ⚠️ | ✅ | ⚠️ | ✅ |
| 協變量整合 | ✅ | ⚠️ | ✅ | ✅ | ✅ |
| 靈活性 | 低 | 中 | 中 | 中 | 高 |

**說明**: 
- ✅ = 良好支持
- ⚠️ = 部分支持或有限制
- ❌ = 不支持

### 7.2 優勢

1. **非參數靈活性**
   - 不對結果隨時間的演化形式施加具體的參數結構
   - 能夠適應廣泛的非線性趨勢和時變關係

2. **完整的概率框架**
   - 自動提供反事實預測的不確定性量化
   - 為治療效應提供可信賴區間

3. **多任務學習能力**
   - 聯合建模所有單位的結果軌跡
   - 預測受治療單位的未受治療路徑時跨單位共享信息
   - 類似於合成控制方法的「借力」，但是以非參數方式

4. **計算效率**
   - 可交換規範誘導的結構導致超參數較少
   - 一個單位一次的框架適用於大規模錯開採用面板

5. **可解釋性**
   - 潛在函數分解為單位特定和共享分量
   - 具有噪聲路徑的單位可以更多地匯總向公共均值
   - 有助於穩定反事實估計

### 7.3 局限性

1. **計算複雜度**
   - 雖然相對其他多任務GP規範高效，但對於非常大的數據集仍可能有計算挑戰
   - 每次優化步驟需要Cholesky分解

2. **核函數選擇**
   - 結果可能依賴於核函數的選擇（OU vs RBF）
   - 需要經驗選擇或交叉驗證

3. **假設依賴性**
   - 依賴於標準因果推斷假設（無干擾、協變量不受影響等）
   - 這些假設通常無法形式化驗證

4. **數據要求**
   - 需要相當長的事前期（特別是對於單一受治療單位設定）
   - 需要足夠的控制單位

5. **先驗敏感性**
   - 雖然使用非信息先驗，但GP模型對先驗選擇可能敏感
   - 超參數估計可能收斂到局部最優值

## 十、結論

### 10.1 總結

本論文提出了一種基於可交換高斯過程的新穎因果推斷方法，用於評估面板數據中政策干預的因果效應。該方法具有以下關鍵特點：

1. **靈活性**: 非參數/半參數框架，能夠適應廣泛的非線性趨勢和複雜單位間依賴性

2. **概率框架**: 自動提供完整的量化不確定性，包括反事實預測和治療效應的可信賴區間

3. **多任務學習**: 聯合建模所有單位，允許在預測受治療單位的未受治療路徑時跨單位共享信息

4. **計算效率**: 可交換結構和一個單位一次框架使其適用於大規模錯開採用數據集

5. **可解釋性**: 潛在函數分解提供解釋性，允許單位匯總到公共均值

### 10.2 主要貢獻

1. **方法論**: 將可交換多任務GP擴展到面板數據因果推斷，為錯開採用設計提供新穎方法

2. **計算**: 提出一個單位一次框架，使方法適用於大規模數據集

3. **應用**: 通過兩個實證案例（Prop99和希臘加油站）驗證方法的有效性

4. **不確定性量化**: 提供完整的概率框架，為政策決策提供可靠的量化不確定性

### 10.3 應用建議

**適用情況**:
- 有足夠長的事前期
- 有足夠的控制單位
- 趨勢可能是非線性的
- 需要不確定性量化
- 大規模錯開採用設計

**使用注意事項**:
1. 確保滿足因果假設（特別是無干擾和協變量不受影響）
2. 仔細選擇核函數
3. 使用安慰劑測試驗證預測準確性
4. 考慮與其他方法的比較
5. 解釋結果時考慮不確定性
