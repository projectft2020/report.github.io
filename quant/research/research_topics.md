**研究目標**：

**可用指標**：
- GVX 雙重移動平均線交叉
- GVX 通道突破
- GVX 隨機指標

**研究項目**：
- [ ] **關聯性分析**
  - [ ] GVX 與黃金價格的相關係數
  - [ ] 不同時間週期的關聯性
  - [ ] 極端事件的關聯性

- [ ] **波動率擴張效應**
  - [ ] GVX 飆升時的黃金價格反應
  - [ ] 均值回歸 vs 趨勢加強
  - [ ] 轉折點偵測

- [ ] **時間週期分析**
  - [ ] 日線 GVX vs 黃金價格
  - [ ] 週線 GVX vs 黃金價格
  - [ ] 月線 GVX vs 黃金價格

**預期輸出**：
- [ ] 關聯性分析報告
- [ ] 波動率擴張策略
- [ ] 最佳參數設定

#### 課題 B：GVX-Based 交易策略 ⭐⭐⭐⭐⭐

**研究目標**：
- 找出最佳 GVX-based 交易策略參數

**策略選項**：

1. **高波動時做空黃金（波動率 trading）**
   - [ ] GVX 飆升時做空
   - [ ] 參數優化（閾值、窗口期）
   - [ ] 風險管理

2. **GVX 背離信號（反轉）**
   - [ ] 價格新高但 GVX 未新高
   - [ ] 價格新低但 GVX 未新低
   - [ ] 背離有效性驗證

3. **多因子黃金策略**
   - [ ] 動能 + GVX + 移動平均的組合
   - [ ] 不同權重配置的回測結果
   - [ ] 優化權重分配

**研究項目**：
- [ ] **策略 1：波動率 trading**
  - [ ] GVX 超過平均線時做空
  - [ ] 參數優化（threshold、lookback）
  - [ ] 風險管理（止損、倉位）
  - [ ] 回測驗證

- [ ] **策略 2：背離信號**
  - [ ] 價格/GVX 背離偵測
  - [ ] 背離後回報分析
  - [ ] 背離有效性測試
  - [ ] 與傳統指標對比

- [ ] **策略 3：多因子組合**
  - [ ] 動能因子
  - [ ] GVX 因子
  - [ ] 移動平均因子
  - [ ] 組合權重優化

**預期輸出**：
- [ ] 3 個 GVX-based 策略回測
- [ ] 最佳參數報告
- [ ] 風險管理方案
- [ ] 策略比較報告
