## 執行摘要

本論文提出了目標導向影響最大化數據獲取（GOIMDA）算法，這是一種新的主動數據獲取方法，能夠在不需要顯式貝葉斯後驗推斷的情況下，實現不確定性感知的數據選擇。GOIMDA 通過最大化候選輸入對用戶指定目標函數的預期影響來選擇數據點，該目標函數可以是測試損失、預測熵或優化器推薦設計的價值等。在多個學習任務（包括圖像和文本分類）和優化任務（包括噪聲全局優化基準和神經網絡超參數調優）中，GOIMDA 能夠以比不確定性驅動的主動學習和高斯過程貝葉斯優化基線更少的標記樣本或函數評估，達到目標性能。

---

## 核心思想

### 動機與挑戰

主動數據獲取在許多深度神經網絡的學習和優化任務中至關重要，但仍然具有挑戰性，因為大多數方法依賴於難以可靠獲得的預測不確定性估計。在材料科學、生物系統識別和超參數優化等領域，每個候選的評估都極其昂貴和耗時，因此需要高效地選擇最有價值的數據點。

現有方法大多依賴於預測不確定性來指導數據選擇，這通常通過貝葉斯神經網絡或基於集成的方法獲得。然而，深度神經網絡的可靠不確定性估計仍然是一個重大挑戰——精確的貝葉斯推斷是不可行的，近似方法通常計算代價高昂、對建模選擇敏感，且在高維設定下校準性差。

### GOIMDA 的核心創新

GOIMDA 的核心思想是**避免直接量化預測不確定性**，而是基於候選數據點對用戶指定目標函數的預期影響來選擇數據點。目標函數可以靈活地定義，以適應不同的科學任務：

1. **全局優化目標**：最小化未知函數 f(x)，在材料設計和神經網絡超參數調優等應用中
2. **有監督學習目標**：在未標記目標集 𝒰 上最小化負對數似然或 focal loss
3. **熵基目標**：在 𝒰 上最小化模型預測分布的香農熵

## 技術細節

### 1. 影響函數理論

GOIMDA 利用一階影響函數來估計無限微調整候選示例時目標函數 𝒢 的瞬時變化。對於最小化目標 𝒢，影響定義為：

```
𝓘∘𝒢(θ(𝒟∪{(xc,yc)})) = -∂/∂ε 𝒢(θ^ε(𝒟∪{(xc,yc)}))|ε=0
```

其中 θ^ε(𝒟∪{(xc,yc)})) 是將 xc 以權重 ε 添加到當前數據集 𝒟 後的參數估計。

**命題 2** 證明了最大化影響得分提供了一種高效近似，可以在不重新訓練模型的情況下最大化 𝒢 的減少量。

### 2. 閉式影響得分

通過鏈式規則和標準影響函數參數，可以推導出閉式表達式：

```
𝓘~∘𝒢(θ(𝒟∪{(xc,yc)})) = [∇θ𝒢(θ(𝒟))]⊤ · Hθ^-1 · [∇θℓ(θ;(xc,yc))]
```

其中：
- ∇θ𝒢 是目標函數關於參數的梯度（目標梯度）
- Hθ^-1 是經驗訓練損失的 Hessian 矩陣的逆（曲率預條件器）
- ∇θℓ(θ;(xc,yc)) 是候選點對模型參數的敏感度

這個公式結合了三個關鍵組成部分：
1. **目標梯度**：促進目標導向的利用
2. **逆曲率項**：作為局部不確定性代理，鼓勵探索
3. **候選敏感度**：量化候選點對參數的影響

### 3. 實用且可擴展的實現

#### 3.1 代理集成模型

為了處理候選標籤的未知性，GOIMDA 引入了一個代理模型 ℳ~_φ，實現為 r 個具有不同隨機初始化並在不同訓練數據子集上訓練的神經網絡集成。為了增強預測穩定性，還採用了 Jackknife 重採樣技術。每個集成成員在可用數據的 Jackknife 子樣本上訓練，代理模型輸出所有 r 個網絡的平均預測。

#### 3.2 逆 Hessian-向量積計算

顯式構造和反轉 Hessian 矩陣的計算代價為 O(nm² + m³)，對於具有數百萬參數的神經網絡來說是不可行的。GOIMDA 使用以下方法來避免顯式 Hessian 反轉：

**共軛梯度（CG）方法**：解決：

```
min_u u^⊤(H² + λI)u - (Hv)^⊤u
```

其中解 u* 近似 H^-1v。經過 L 次 CG 迭代，計算代價為 O(nL)。

**LiSSA 算法**：通過遞歸估計逆 HVPs：

```
Ĥ_j^-1 v = v + (I - H)Ĥ_{j-1}^-1 v
Ĥ_0^-1 v = v
```

LiSSA 在每次迭代中使用小批量數據隨機近似逆 HVPs。使用批量大小 B 和 L 次迭代，計算複雜度為 O(nm + BLm)。

### 4. 理論性質：廣義線性模型下的分析

#### 4.1 幾何解釋

在廣義線性模型（GLMs）下，GOIMDA 可以分解為三個相互作用的組成部分：

```
𝖦𝖮𝖨(xc) = ⟨μ, ν(xc)⟩ × b(xc)
```

其中：
- μ = Hθ^-1/2 ∇θ𝒢(θ)：靈敏度向量，編碼哪些參數方向對改善 𝒢 至關重要
- ν(xc) = Hθ^-1/2 ∇θ ηθ(xc)：候選點在考慮局部曲率後的誘導更新方向
- b(xc) = A'(ηθ(xc)) - A'(η0(xc))：預測偏差，量化模型預測均值與數據生成過程的差異

幾何上：
- 目標對齊項 ⟨μ, ν(xc)⟩ 量化了 xc 建議的曲率感知更新方向對減少 𝒢 的影響程度
- 預測偏差項 b(xc) 將參數空間中的槓桿作用轉化為目標的預期改進

#### 4.2 與預測熵最小化的關係

**命題 3**：在具有典型鏈接的 GLMs 中，影響最大化近似於預測熵最小化。兩個目標的差異在於一個額外項，該項校準方向性參數偏差至多一個常數。

在典型鏈接假設下，最小化條件熵可以簡化為最小化：

```
E_yc[p0(·|xc)][log|Hθ_c^-1|] = log|Hθ^-1| - log(1 + xc^⊤ Hθ^-1 xc A''(ηθ(xc)))
```

這表明 GOIMDA 繼承了預測熵獲取中出現的相同 Hθ^-1 槓桿因子，同時通過目標對齊和預測偏差進行重新加權。

#### 4.3 偏差項的參數空間代理

通過對 A'(η0(xc)) 進行一階泰勒展開，可以得到可估計的參數偏差：

```
𝖦𝖮𝖨(xc) = ⟨μ, ν(xc)⟩ × ∇θ ηθ(xc)^⊤ (θ - θ0) × A''(ηθ(xc))
```

這將偏差歸因於通過 ∇θ ηθ(x) 具體化的參數空間方向，並恢復了 GLMs 的熟悉結構。

## 算法流程

### Algorithm 1: 目標導向影響最大化數據獲取

```
輸入：初始數據集 𝒟
重複：
  1. 在 𝒟 上更新主模型 ℳθ 和集成模型 ℳ~_φ
  2. 選擇 x_next = argmax_x E_y~pφ(·|x)[𝓘~∘𝒢_φ(θ(𝒟∪{(x,y)}))]
  3. 查詢 y_next = OBSERVE(x_next)
  4. 增廣數據集 𝒟 ← 𝒟 ∪ {(x_next, y_next)}
直到滿足終止條件（例如預算耗盡或達到期望目標）
```

## 實驗評估

### 1. 全局優化任務

在噪聲黑盒函數優化基準上，GOIMDA 與五種常用的高斯過程（GP）基於貝葉斯優化方法進行了比較：
- 概率改進（PI）
- 期望改進（EI）
- 上置信界（UCB）
- 最大值熵搜索（MES）
- 知識梯度（KG）

在 Branin、Ackley 和 Dropwave 基準上，在兩個噪聲水平（σ² = 0.01, 0.04）下：
- GOIMDA 在所有任務中始終更早地達到更低的遺憾值
- 優勢通常在較高噪聲水平下更加明顯
- 實體/虛線曲線顯示了運行之間的平均性能，陰影區域表示平均值的自助法 95% 置信區間

### 2. 學習任務

**圖像分類**：在 CIFAR-10 等數據集上，GOIMDA 在達到目標準確率時所需的標記樣本數量少於不確定性驅動的主動學習基線。

**文本分類**：在文本分類任務中，GOIMDA 展現了更快的收斂速度。

**超參數調優**：在神經網絡超參數調優任務中，GOIMDA 能夠以更少的函數評估找到更好的超參數配置。

## 應用價值

### 1. 科學與工程應用

**材料科學**：
- 在材料設計中，評估單個候選材料通常需要材料合成隨後進行實驗測試，每次函數評估都昂貴且耗時
- GOIMDA 可以指導材料發現，更快地找到具有卓越性能的材料

**生物系統識別**：
- 研究人員試圖描述系統對外部刺激的響應
- 預測模型通常在現有的刺激-響應對上訓練，用於指導新刺激的選擇
- GOIMDA 可以更高效地選擇預計能減少預測誤差的新刺激

**超參數優化**：
- 超參數控制模型架構和訓練動態，對性能有很大影響
- 評估單個配置需要完整的訓練和驗證週期
- GOIMDA 可以以更少的評估找到最佳超參數配置

### 2. 機器學習流水線

**主動學習**：
- 在標籤獲取成本高的場景下，GOIMDA 可以更高效地選擇需要標記的樣本
- 避免了貝葉斯神經網絡的複雜性和不確定性估計的困難

**貝葉斯優化**：
- 在黑盒函數優化中，GOIMDA 不需要 GP 假設，可以直接應用於神經網絡等複雜模型
- 在高噪聲環境下表現更好

## 局限性與挑戰

### 1. 理論假設

- 理論結果主要在廣義線性模型下推導，對於深度神經網絡的理論保證仍然有限
- 一階影響函數近似可能在大變化時失效

### 2. 計算複雜度

- 雖然通過 CG 和 LiSSA 避免了顯式 Hessian 反轉，但對於非常大的模型，計算開銷仍然顯著
- 需要訓練代理集成模型，增加了計算成本

### 3. 實現細節

- 集成大小和 Jackknife 子樣本的選擇需要調優
- CG/LiSSA 的迭代次數需要平衡計算效率和精度

### 4. 擴展性

- 在超高維參數空間（如大語言模型）中的擴展性有待驗證
- 對於大規模數據集，Hessian 計算的線性依賴仍然可能成為瓶頸

## 與現有方法的比較

### 與不確定性驅動的主動學習比較

| 方面 | 不確定性驅動方法 | GOIMDA |
|------|----------------|--------|
| 不確定性估計 | 顯式估計，依賴貝葉斯神經網絡或集成 | 隱式通過逆曲率實現 |
| 目標導向 | 通常作為代理標準（如預測熵） | 直接優化用戶指定的目標函數 |
| 貝葉斯後驗 | 需要 | 不需要 |
| 計算開銷 | 貝葉斯推斷計算成本高 | 一階近似，HVP 計算較高效 |

| 方面 | GP 基於的 BO | GOIMDA |
|------|-------------|--------|
| 模型假設 | 高斯過程 | 任意神經網絡 |
| 導函數 | 通常不使用 | 利用梯度信息 |
| 噪聲魯棒性 | 受 GP 假設限制 | 在高噪聲下表現更好 |
| 高維擴展性 | GP 在高維困難 | 通過神經網絡自然處理高維 |

## 創新點總結

1. **目標導向**：GOIMDA 允許用戶指定顯式的目標函數，而不是優化代理標準（如預測不確定性）
2. **無後驗推斷**：避免了顯式貝葉斯後驗推斷，通過逆曲率實現不確定性感知
3. **一階影響函數**：利用一階影響函數推導出可處理的獲取規則
4. **可擴展實現**：通過 Jackknife/深度集成代理和隨機逆 HVP 求解器，支持現代神經網絡
5. **理論連接**：在 GLMs 下，GOIMDA 近似於預測熵最小化，但通過目標對齊和預測偏差進行調節
