```python
# 修改前：硬編碼 5.0
self.dynamic_config["adaptive_threshold"] = max(4.5, 5.0 - reduction)

# 修改後：從配置文件讀取基準閾值
self.base_threshold = self.dynamic_config.get("adaptive_threshold", 5.0)
self.dynamic_config["adaptive_threshold"] = max(4.5, self.base_threshold - reduction)
```

- ✅ 配置文件的 adaptive_threshold (6.5) 現在生效
