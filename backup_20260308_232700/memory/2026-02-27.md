## 💡 關鍵學習：策略分類與評估標準（2026-02-27 20:45）

**David 指出：不同類型的策略評估方式完全不同！不能一概而論。**

### 策略類型分類

#### 1. 趨勢跟踪（Trend Following）
**代表策略：** Supertrend, MACD

**核心特點：**
- 漲了繼續漲，跌了繼續跌
- 在趨勢市場中表現優異
- 在震盪市場中會頻繁虧損

**評估標準（優先順序）：**
1. 總收益率 ⭐⭐⭐⭐⭐ - 核心目標
2. 最大回撤 ⭐⭐⭐⭐⭐ - 單次虧損可能很大
3. 夏普比率 ⭐⭐⭐⭐ - 風險調整後收益
4. 盈虧比 ⭐⭐⭐⭐ - 應 > 2.0
5. 趨勢持續性 ⭐⭐⭐⭐
6. 勝率 ⭐⭐ - 通常較低（30-50% 正常）

**典型模式：**
```
勝率：30-50%（正常）
盈虧比：2:1 到 4:1
交易頻率：低到中等
```

---

#### 2. 均值回歸（Mean Reversion）
**代表策略：** RSI, Bollinger, 協整對

**核心特點：**
- 價格偏離均值後會回歸
- 在震盪市場中表現優異
- 在趨勢市場中容易逆勢虧損

**評估標準（優先順序）：**
1. 勝率 ⭐⭐⭐⭐⭐ - 應 > 70%
2. 交易頻率 ⭐⭐⭐⭐ - 适中
3. 回歸速度 ⭐⭐⭐⭐ - 半衰期 5-20 天
4. 殘差穩定性 ⭐⭐⭐⭐ - ADF 檢驗 p-value < 0.05
5. 夏普比率 ⭐⭐⭐ - 風險調整後收益
6. 總收益率 ⭐⭐ - 穩定但不是爆發式

**典型模式：**
```
勝率：70-90%（正常）
盈虧比：1:1 到 1.5:1
交易頻率：中等
```

---

#### 3. 動量策略（Momentum）
**代表策略：** 動量輪動、強者恆強

**核心特點：**
- 強者恆強，弱者恆弱
- 基於相對表現選股
- 市場中性（多強空弱）

**評估標準（優先順序）：**
1. 動量持續性 ⭐⭐⭐⭐⭐ - 持續 3-6 個月
2. 換手率 ⭐⭐⭐⭐ - < 30% / 月
3. 信息比率 ⭐⭐⭐⭐ - 超額收益穩定性
4. 市場適應性 ⭐⭐⭐
5. 勝率 ⭐⭐ - 略高於隨機（55-65%）

**典型模式：**
```
勝率：55-65%（正常）
盈虧比：1:1 到 2:1
交易頻率：低到中等
```

**關鍵洞察：**

## 💻 倉位管理問題（2026-02-27 21:14）

**David 提問：當持股已經到假設 10 檔股票滿的時候，又有訊號出來，該如何處理？**

### 方案 1：固定倉位限制（最簡單）

**邏輯：**
```
最大倉位數：10 檔
新訊號出現時：
  ✅ 如果 < 10 檔 → 進場
  ❌ 如果已滿（= 10 檔）→ 忽略新訊號
```

**優點：**
- 簡單直接，易於執行
- 控制交易頻率
- 適合低頻策略

**缺點：**
- 可能錯過高質量訊號
- 倉位可能長期被「劣質」資產佔用

**適用情況：**
- 低頻策略（如 RSI，每月 2-3 筆）
- 資金有限
- 管理成本敏感

---

### 方案 2：動態替換（主動優化）

**邏輯：**
```
最大倉位數：10 檔
新訊號出現且已滿時：
  1. 評估現有 10 檔倉位
  2. 找到表現最差的倉位（如：虧損最大、止損觸發）
  3. 平倉該倉位
  4. 進場新訊號
```

**替換判斷標準：**
- 持倉虧損 > 5%
- 止損觸發
- 技術指標轉弱
- 持倉時間過長（> 3 個月）

**優點：**
- 保持倉位質量
- 適應市場變化
- 適合中高頻策略

**缺點：**
- 增加交易頻率
- 交易成本增加
- 可能過度交易

**適用情況：**
- 趨勢跟踪策略（如 Supertrend）
- 市場波動較大
- 需要快速調整

---

### 方案 3：優先級排序（最智能）

**邏輯：**
```
為每個訊號計算綜合得分：
  得分 = f(趨勢強度, 盈虧比, 風險, 資產質量)

新訊號出現且已滿時：
  1. 計算新訊號得分
  2. 計算現有 10 檔倉位得分
  3. 如果 新訊號 > 最低分倉位 → 替換
```

**優點：**
- 智能化決策
- 平衡機會和風險
- 可配置優先級

**缺點：**
- 實現複雜
- 需要調整參數
- 可能過度優化

**適用情況：**
- 高頻策略
- 多資產組合
- 需要自動化執行

---

### 推薦方案組合

#### RSI 策略（低頻）：固定倉位限制

## 🎯 Supertrend 訊號品質篩選（2026-02-27 21:16）

**David 問題：Supertrend 訊號出現時，如何篩選品質？**

### 核心問題

**Supertrend 只告訴你：**
- 趨勢方向（向上/向下）
- 交叉點 = 進場/出場訊號

**但沒告訴你：**
- 訊號「好不好」
- 價格是否已經追高
- 趨勢是否快要結束

---

### 實戰版三維篩選

#### 維度 1：趨勢持續性（最重要）

**指標：** Supertrend 方向連續不變的天數

**判斷標準：**
- **連續 ≥ 5 天** → 趨勢穩定 ✅
- **連續 3-4 天** → 趨勢剛形成 ⚠️
- **連續 ≤ 2 天** → 趨勢不穩 ❌

**代碼：**
```python
def trend_continuity(price_data, lookback=10):
    supertrend_direction = calculate_supertrend_direction(price_data)
    current_direction = supertrend_direction.iloc[-1]
    continuity = 0
    
    for i in range(len(supertrend_direction) - 1, -1, -1):
        if supertrend_direction.iloc[i] == current_direction:
            continuity += 1
        else:
            break
    
    return continuity
```

---

#### 維度 2：價格距離 Supertrend 線（避免追高）

**指標：** 價格距離 Supertrend 線的 ATR 倍數

**判斷標準：**
- **< 0.5 ATR** → 訊號剛產生 ✅
- **0.5-1.0 ATR** → 正常進場位置 ⚠️
- **> 1.0 ATR** → 可能追高 ❌

**代碼：**
```python
def price_distance_to_supertrend(price_data):
    supertrend_line = calculate_supertrend_line(price_data)
    current_price = price_data['close'].iloc[-1]
    current_supertrend = supertrend_line.iloc[-1]
    
    distance = abs(current_price - current_supertrend)
    atr = calculate_atr(price_data, period=14).iloc[-1]
    
    return distance / atr
```

---

#### 維度 3：成交量放大（確認）

**指標：** 當天成交量 vs 過去 5 天平均

**判斷標準：**
- **≥ 1.5 倍** → 成交量放大，訊號可靠 ✅
- **1.0-1.5 倍** → 成交量正常 ⚠️
- **< 1.0 倍** → 成交量萎縮，訊號可疑 ❌

**代碼：**
```python
def volume_spike(price_data, lookback=5):
    current_volume = price_data['volume'].iloc[-1]
    avg_volume = price_data['volume'].iloc[-lookback:].mean()
    
    return current_volume / avg_volume
```

---

### 綜合篩選邏輯

```python
def filter_supertrend_signal(price_data):
    """Supertrend 訊號品質篩選"""
    
    # 1. 趨勢持續性
    continuity = trend_continuity(price_data)
    if continuity < 3:
        return False
    
    # 2. 價格距離
    distance = price_distance_to_supertrend(price_data)
    if distance > 1.0:
        return False
    
    # 3. 成交量
    volume_ratio = volume_spike(price_data)
    if volume_ratio < 1.0:
        return False
    
    # 通過所有篩選
    return True
```

### 動態倉位分配

```python
def calculate_position_size(price_data):
    """根據訊號品質計算倉位大小"""
    
    # 1. 趨勢持續性（0-40 分）
    continuity = trend_continuity(price_data)
    continuity_score = min(continuity * 4, 40)
    
    # 2. 價格距離（0-30 分，反向）
    distance = price_distance_to_supertrend(price_data)
    distance_score = max(30 - distance * 30, 0)
    
    # 3. 成交量（0-30 分）
    volume_ratio = volume_spike(price_data)
    volume_score = min((volume_ratio - 0.5) / 1.5 * 30, 30)
    volume_score = max(volume_score, 0)
    
    # 總分
    total_score = continuity_score + distance_score + volume_score
    
    # 轉換為倉位大小
    if total_score >= 80:
        return 1.0  # 滿倉
    elif total_score >= 60:
        return 0.5  # 半倉
    elif total_score >= 40:
        return 0.2  # 觀察倉
    else:
        return 0.0  # 不進場
```

### 範例：2330.TW（台積電）

**場景 1：優質訊號**
- 持續：7 天
