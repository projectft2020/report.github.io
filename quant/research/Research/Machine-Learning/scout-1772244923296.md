## 執行摘要

本論文提出了 **Group Orthogonalized Policy Optimization（GOPO）**，一種完全源自 Hilbert 函數空間幾何學的新型大語言模型對齊算法。與在概率單純形上進行約束優化的現有方法（必然繼承 KL 散度的指數曲率）不同，GOPO 採用了一種根本不同的方法：將問題提升到關於當前參考策略的平方可積函數 Hilbert 空間 L²(πk)。

在這個函數空間中，作者做出了三個觀察，共同產生了一個原則性算法：
1. 概率單純形約束簡化為單一線性正交條件 ⟨v, 1⟩ = 0，定義了一個維數減一的閉子空間
2. 對齊目標自然從幾何最小距離原則出現，產生功-耗散泛函
3. 執行非負性邊界升級為有界 Hilbert 投影，產生精確稀疏性

實驗表明，GOPO 在數學推理基準上達到了競爭力的泛化能力（MATH Level 4 驗證準確率 47%，對比 GRPO/DAPO 的 44%），在基於裁剪的方法出現飽和的區域持續學習，並保持了最健康的梯度動態和熵保持。

## 核心思想與技術細節

### 1. Hilbert 空間視角

傳統的 RLHF 方法（PPO、DPO、GRPO）都通過 KL 散度正則化策略更新，繼承了 KL 的指數幾何曲率。當策略變得更加自信時（產生大的 log-ratios），梯度信號會指數衰減，導致梯度飽和。

GOPO 提出了一個根本性的變革：**將對齊問題從概率單純形轉移到 Hilbert 函數空間 L²(πk)**。

#### 1.1 Hilbert 空間的密度波動場

讓 πk 表示當前參考策略（通常是前一次迭代的策略）。我們定義環境空間為關於 πk 的平方可積函數 Hilbert 空間：

```
H = L²(πk), ⟨f, g⟩πk = Eπk[f(y)g(y)]
```

誘導範數為 ‖f‖πk = √⟨f, f⟩πk。

不再直接在單純形上優化概率向量 π(y|x)，我們轉移到一個居中坐標系。定義**密度波動場**：

```
v(y) = π(y)/πk(y) - 1
```

目標策略可以精確恢復為 π(y) = πk(y)(1 + v(y))。場 v 生活在 H 中，表示目標策略相對於參考的相對偏差。這個坐標變換是將非線性單純形幾何轉換為線性 Hilbert 空間幾何的關鍵步驟。

#### 1.2 概率守恆作為正交性

任何有效的概率分佈必須滿足 Σy π(y) = 1。在波動坐標中，這個基本物理定律顯著簡化：

```
Σy π(y) = Σy πk(y)(1 + v(y)) = 1 + Σy πk(y)v(y) = 1 + Eπk[v] = 1
```

因此，概率守恆等價於零均值條件：

```
Eπk[v] = ⟨v, 1⟩πk = 0
```

這是 H 中的一個單一線性約束，定義了一個維數減一的閉子空間：

```
H₀ = {f ∈ H : ⟨f, 1⟩πk = 0}
```

這是一個關鍵洞察：**概率守恆的約束從非線性歸一化條件變成了線性正交條件**。

### 2. 幾何原則：最小距離到無約束目標

假設暫時沒有物理約束。完全由外部度量調制場 gα 驅動，理想的無約束策略波動將簡單地與驅動力對齊，按順應性（或剛度）參數 μ > 0 縮放：

```
u* = gα/μ
```

然而，現實是任何有效的波動場 v 必須嚴格遵守概率守恆，將其鎖定在零均值超平面 H₀ 內。無約束目標 u* 通常違反這個約束，因為 Eπk[gα] ≠ 0。

#### 2.1 功-耗散泛函的出現

通過代數擴展這個純幾何距離，一個顯著的物理結構自然出現。代入 u* = gα/μ：

```
(μ/2)‖v - gα/μ‖²πk = (μ/2)‖v‖²πk - ⟨gα, v⟩πk + (1/2μ)‖gα‖²πk
```

最小化幾何距離因此嚴格等價於最大化 v 相關項的負值：

```
arg min v∈H₀ (μ/2)‖v - u*‖²πk ≡ arg max v∈H₀ (⟨gα, v⟩πk - (μ/2)‖v‖²πk)
```

經典的「功-耗散」泛函 J(v) = ⟨gα, v⟩πk - (μ/2)‖v‖²πk **不是任意構建的目標函數；它是在 Hilbert 空間中測量平方距離的代數必然性**。

#### 2.2 正交投影解

由於問題本質上是在閉子空間 H₀ 中找到最接近目標向量 u* 的點，我們不需要依賴拉格朗日微分。**Hilbert 投影定理**直接通過正交投影產生唯一的全局最優解：

**定理 4.1（通過正交投影的最優波動）**

最優概率守恆波動 v* 是無約束目標 u* 到 H₀ 的正交投影：

```
v* = PH₀(u*) = u* - ⟨u*, 1⟩πk 1 = (1/μ)(gα - Eπk[gα]·1)
```

這個定理表明，最優策略更新就是將無約束目標正交投影到概率守恆子空間。

### 3. 有界 Hilbert 投影與精確稀疏性

無約束投影可能產生 v*(y) < -1（對於具有強負驅動信號的動作），產生無效的負概率。為了執行物理約束 π(y) ≥ 0（等價於 v(y) ≥ -1），我們將投影擴展為**有界 Hilbert 投影（BHP）**。

**定理 4.3（有界投影解）**

有界解為：

```
v*(y) = max(-1, (gα(y) - λ*)/μ)
```

其中動態化學勢 λ* 是確保 Eπk[v*] = 0 的唯一值。

**推論 4.4（精確稀疏性）**

滿足 gα(y) < λ* - μ 的動作被分配 v*(y) = -1，即它們的目標概率精確為零：π(y) = πk(y)(1 + v*(y)) = 0。

BHP 因此提供了一個解析機制，通過一個硬的、數據自適應閾值來消除災難性差的輸出（例如幻覺），而不需要任何外部過濾啟發式。

這是一個深刻的理論結果：**GOPO 提供了一種機制，可以精確地將極差輸出的概率設置為零，這不是通過人為設計的啟發式方法，而是通過幾何優化的必然結果**。

### 4. 從全局到經驗：GOPO 目標函數

在全局空間 L²(πk) 中優化距離最小化需要訪問完整分佈，這對於自迴歸語言模型來說在計算上不可行。GOPO 通過將正交投影轉移到由群採樣引發的經驗離散子空間來填補這個空白。

對於給定的提示 x，採樣一組 G 個響應 G = {y₁, ..., yG} ~ πk(·|x)。定義經驗內積空間 L²(G) 度量：

```
⟨f, g⟩G = (1/G) Σi=1 to G f(yi)g(yi)
```

概率守恆約束自然地放鬆到經驗零均值條件：

```
Ĥ₀ = {v ∈ L²(G) : (1/G) Σi=1 to G v(yi) = 0}
```

#### 4.1 消失的化學勢

在標準 RLHF 管道中，原始獎勵被正規化為群相對優勢 Ai = ri - r̄，滿足 Σi=1 to G Ai = 0。因此，經驗驅動向量 A = [A₁, ..., AG]ᵀ 本質上居住在 Ĥ₀ 內。

因為驅動力已經與 1 正交，投影算子 PĤ₀ 在 A 上作用為恆等：

```
PĤ₀(A) = A - (1/G) Σi=1 to G Ai · 1 = A
```

**在經驗群層面，執行概率守恆的化學勢 λ* 精確消失**。

這是一個優雅的結構結果：**由於群歸一化優勢結構上和為零，它們已經居住在零均值子空間中，執行概率守恆的化學勢完全消失，將約束投影坍塌為無約束經驗損失**。

#### 4.2 GOPO 損失函數

用經驗優勢 Ai 替換驅動場 gα，並用參數化的密度比 ρθ(yi) = πθ(yi|x)/πk(yi|x) 替換解析波動 v(yi)，約束泛函最大化減少為：

```
ℒGOPO(θ) = -Ex~D [(1/G) Σi=1 to G (Ai ρθ(yi) - (μ/2)(ρθ(yi) - 1)²)]
```

#### 4.3 與 GRPO 的結構比較

```
ℒGRPO(θ) = -Ex~D [(1/G) Σi=1 to G (1/|yi|) Σt=1 to |yi| min(ρθ(yi,t)Âi,
          clip(ρθ(yi,t), 1-ε, 1+ε)Âi) - β DKL(πθ‖πk)]
```

兩個損失共享相同的根本動機——通過群採樣響應和歸一化優勢優化策略——但在三個根本方面不同：

1. **信託區域機制**：GRPO 通過硬裁剪令牌級別比率 ρθ(yi,t) 以及顯式 KL 懲罾來執行信託區域，引入了非平滑、分段線性景觀，在 |ρθ - 1| > ε 處產生零梯度平台。GOPO 用序列級別比率上的平滑二次懲罾 (μ/2)(ρθ - 1)² 替換這一點，連續懲罾對參考的偏差——比率可以自由移動，而恢復力線性增長，從不產生梯度死區。

2. **優化曲率**：GRPO 的裁剪替代項的 Hessian 要麼為 0（在平坦裁剪區域內），要麼未定義（在裁剪邊界）；其有效曲率是數據和狀態相關的。GOPO 的 Hessian 是恆定 μ，產生均勻凸景觀，具有可預測的收斂性。

3. **收斂處的梯度行為**：當 ρθ → 1 且策略接近參考時，GRPO 梯度由優勢加權策略梯度 Âi ∇θ log πθ 主導，它按優勢幅度縮放，但不考慮已經行駛的距離。GOPO 的梯度 -Ai + μ(ρθ - 1) 與距離平衡的位移成正比，當策略接近其目標 ρ* = 1 + Ai/μ 時提供自然減速。

**總結：GOPO 用軟幾何約束（投影）交換 GRPO 的硬組合約束（裁剪），在沒有額外計算成本的情況下獲得平滑性、恆定曲率和原則性平衡結構。**

### 5. 理論分析

#### 5.1 恆定曲率和解耦幾何

PPO、DPO 和 GRPO 的主要弱點是它們對指數散度 DKL 的依賴。基於 KL 的目標的 Hessian 按策略輸出非線性縮放，導致梯度飽和，必須通過啟發式裁剪來緩解。

**定理 5.1（恆定曲率優化）**

令 ℓ(ρ) = -Aρ + (μ/2)(ρ - 1)² 為點向未截斷的 GOPO 損失。關於密度比 ρ 的 Hessian 是恆定標量 μ，獨立於優勢信號 A、當前策略狀態或數據分佈。

**證明**：∇ρℓ = -A + μ(ρ - 1)，因此 ∇²ρℓ = μ。∎

**推論 5.2（結構解耦）**

在 GOPO 目標中，一階驅動力依賴於優勢信號：

```
∇ρℓ = -A + μ(ρ - 1)
```

而二階曲率 ∇²ρℓ = μ 獨立於 A。因此，改變優勢加權（採樣幾何）不改變優化景觀曲率（優化幾何），反之亦然。

這是一個關鍵的結構洞察：**GOPO 完全解耦了採樣幾何和優化幾何**。

#### 5.2 梯度動力學：非飽和保證

**DPO 中的梯度飽和**：對於邏輯損失，梯度幅度滿足 |∇mℓDPO| = |βσ(m)(1 - σ(m))| ≤ β/4，關鍵是，|∇mℓDPO| → 0 隨 |m| → ∞ 指數衰減。

**GOPO 中的非飽和梯度**：對於 GOPO 損失，梯度幅度為：

```
|∇ρℓGOPO| = |-Ai + μ(ρθ(yi) - 1)| = μ|ρθ - ρ*|
```

這完全與距離平衡點的距離成正比。對於任何非平衡狀態 |ρ - ρ*| ≥ δ：

```
|∇GOPO| ≥ μδ
```

梯度保持線性驅動力，在遠離平衡時從不消失，防止了基於 KL 的方法固有的飽和。

#### 5.3 死區動力學和梯度硬停止

BHP 截斷建立了梯度流動的隱式「死區」。有界 GOPO 損失的導數為：

```
∇θℒGOPO^BHP ∝ (1/G) Σi=1 to G I(ρθ(yi) > 0) · (-Ai + μ(ρθ(yi) - 1)) ∇θ ρθ(yi)
```

當動作接收極度負優勢 Ai ≪ 0 時，比率 ρθ(yi) 被快速抑制。一旦它接近物理邊界 ρθ → 0（等價於 v(yi) → -1），指示函數觸發硬停止：此動作的梯度精確為零。

這個機制防止網絡浪費表示容量於已經被抑制的輸出，將梯度帶寬專門重定向於區分較高質量的響應。

#### 5.4 與 χ² 散度的連接

Hilbert 空間框架提供了與統計散度的自然連接。

**命題 5.5（χ² 解釋）**

GOPO 泛函中的耗散項與 Pearson χ² 散度成正比：

```
(1/2) Eπk[v(y)²] = (1/2) Eπk[(π(y)/πk(y) - 1)²] = Dχ²(π‖πk)
```

因此，GOPO 目標可以等價地讀取為在 χ² 信託區域下最大化優勢加權功。

**命題 5.6（全變異界）**

通過 χ² 懲罾約束 E[v²] ≤ ε 保證，根據 Jensen 不等式：

```
TV(π, πk) = (1/2)E[|v|] ≤ (1/2)√E[v²] ≤ √ε/2
```

提供了在全變異距離中的分佈穩定性保證。

### 6. 算法與實現

#### 6.1 算法流程

**算法 1：Group Orthogonalized Policy Optimization (GOPO)**

輸入：初始策略 πθ，剛度 μ，群大小 G，學習率 η

對每個迭代：
1. 設置參考：πk ← πθ（on-policy 錨定）
2. 採樣群：對每個提示 x，採樣 G = {y₁, ..., yG} ~ πk(·|x)
3. 評分和歸一化：計算獎勵 ri；設置 Ai = ri - r̄（ΣAi = 0）
4. 計算比率：ρθ(yi) = πθ(yi|x)/πk(yi|x)
5. 計算損失：ℒ = -(1/G) Σi=1 to G [Ai ρθ(yi) - (μ/2)(ρθ(yi) - 1)²]
6. 更新：θ ← θ - η ∇θ ℒ

#### 6.2 實現注意事項

GOPO 與 GRPO 相比不引入額外的模型前向/後向傳遞。唯一的開銷是 (ρθ - 1)² 的計算，這可以忽略不計。參考策略 πk 設置為每次迭代開始時的策略（on-policy 錨定），確保信託區域 ρ ≈ 1 局部成立。不需要評論家網絡。

### 7. 實驗結果

#### 7.1 實驗設置

**模型和數據**：
- 使用 Qwen3-1.7B 作為基礎模型
- 在 4 × RTX 4090 GPU 上使用 VERL 框架訓練
- 刻意使用小型訓練集：約 10% 的 MATH Level 3 問題（使用種子 42 採樣）
- 驗證在 100 個保留的 MATH Level 4 問題上每 10 個訓練步驟執行一次——嚴格較高的難度層級，訓練期間未見過——以測量分佈外泛化

**基線**：
- OPO：完整的 Orthogonalized Policy Optimization，帶有重要性採樣重加權
- GRPO：令牌級策略梯度，帶有群歸一化優勢和 PPO 樣式比率裁剪（ε = 0.2）
- GSPO：GRPO 的句子級別變體，帶有步級優勢歸一化
- DAPO：高級基線，帶有不對稱裁剪邊界、未歸一化優勢和過長獎勵塑形

GOPO 使用剛度 μ = 0.5，護送指數 α = 0.5，和 on-policy 錨定。

#### 7.2 整體性能

| 方法 | MATH L4 驗證準確率 | 平均訓練獎勵 | 平均梯度範數 | 平均策略熵 |
|------|-------------------|--------------|--------------|------------|
| OPO  | 48% | 0.558 | 1.28 | 0.123 |
| **GOPO** | **47%** | **0.555** | **1.03** | **0.134** |
| GSPO | 48% | 0.551 | 0.62 | 0.089 |
| GRPO | 44% | 0.544 | 0.67 | 0.076 |
| DAPO | 44% | 0.550 | 0.21 | 0.082 |

**分析**：

1. **泛化 vs 訓練獎勵**：OPO 和 GOPO 在訓練過程中獲得最高的平均獎勵（0.558 和 0.555），這直接轉化為它們優越的泛化性能（48% 和 47% 驗證準確率）。相比之下，GRPO 在最低的平均獎勵（0.544）和最差的泛化（44%）上掙扎，難以從小型訓練集中高效學習。

2. **梯度動力學**：OPO 和 GOPO 維持了顯著更高的梯度範數（1.28 和 1.03）比比率裁剪方法（GRPO：0.67，GSPO：0.62，DAPO：0.21），經驗證實了定理 5.1 的非飽和預測。DAPO 嚴重減少的梯度（約 0.21）表明保守裁剪過於限制性。

3. **熵保持**：GOPO 在所有方法中維持最高的策略熵（0.134）。我們將此歸因於 GOPO 更新的序列級別性質：不像令牌級方法（GRPO、DAPO）強制密集每令牌監督導致快速模式坍塌，GOPO 對齊總軌跡概率而不微管理個別令牌。

#### 7.3 訓練動力學

**驗證準確率軌跡**揭示重要動力學：
- GOPO 顯示泛化的單調改善（38% → 46% → 47%），表明算法尚未飽和，將受益於更長的訓練
- OPO 開始強勁（步驟 10 時 49%）但表現非單調行為（50% → 44% → 48%），與完整 OPO 框架固有的重要性採樣方差一致
- GRPO 和 DAPO 在 44% 平台，儘管 GRPO 的較高訓練獎勵——這是獎勵駭客現象的特徵，其中裁剪方法利用訓練分佈而不是學習泛化推理模式

**梯度規範軌跡**：
- OPO/GOPO 在整個過程中維持健康範數，而 DAPO 顯示嚴重梯度飽和
- 這驗證了 GOPO 的恆定曲率設計確實防止了梯度飽和

**策略熵軌跡**：
- GOPO 保持了最優的多樣性，防止過早模式坍塌
- 這表明 GOPO 的序列級別正則化在保持探索性方面更有效

## 應用價值

### 1. 理論貢獻

**1.1 幾何學的統一框架**

GOPO 提供了一個原則性的幾何框架來理解策略優化，將對齊問題表述為：
- 無約束目標到可行概率分佈的正交投影
- 功-耗散泛函作為最小距離的代數必然性
- 有界投影提供精確稀疏性和硬停止機制

這個框架連接了：
- RLHF 與 Hilbert 空間投影
- χ² 信託區域與二次正則化
- 線性優化幾何與概率守恆

**1.2 採樣與優化幾何的解耦**

GOPO 完全解耦了兩個獨立的設計選擇：
1. **採樣幾何**：如何從人類偏好中提取信號（優勢函數）
2. **優化幾何**：如何正則化策略更新（曲率、信託區域）

在 KL 方法中，這兩個是糾纏的：改變優勢加權會改變 KL 曲率的影響。在 GOPO 中，優勢驅動一階項，而 μ 控制二階曲率，彼此獨立。

### 2. 實際應用優勢

**2.1 超參數調整更簡單**

GOPO 只有一個主要的超參數 μ（剛度），而不是需要調整多個啟發式參數：
- 裁剪邊界 ε
- KL 懲罾係數 β
- 熵懲罾係數
- 不對稱裁剪邊界等

**2.2 無需啟發式裁剪**

GOPO 不需要任何裁剪機制：
- 無需比率裁剪
- 無需漸近裁剪
- 無需 KL 懲罾
- 幾何投影自然地提供信託區域

**2.3 更穩定的訓練**

恆定曲率 μ 確保：
- 梯度永不飽和
- 收斂行為可預測
- 不存在裁剪邊界引起的梯度不連續
- 死區機制防止浪費容量於差輸出

**2.4 更好的泛化能力**

實驗顯示 GOPO 在分佈外泛化（MATH Level 4）上優於 GRPO/DAPO：
- 47% vs 44%
- 單調改善趨勢
- 更高的熵保持表明更好的多樣性

**2.5 計算效率**

GOPO 不需要額外開銷：
- 無需評論家網絡（像 GRPO）
- 只需要 (ρθ - 1)² 的計算，可忽略
- 與 GRPO 相同的群採樣流程

### 3. 潛在應用場景

**3.1 數學推理**

GOPO 在 MATH 基準上表現出色，特別適合：
- 需要長推理鏈的任務
- 需要高精度計算的任務
- 容易產生錯誤中間步驟的任務

**3.2 代碼生成**

精確稀疏性機制可以：
- 將不可運行的代碼片段的概率設置為零
- 防止累積錯誤
- 保持較高質量代碼路徑的探索

**3.3 長文本生成**

序列級別正則化：
- 防止過早模式坍塌
- 保持長距離依賴的探索
- 避免 token 級監督的過度約束

## 局限性與未來方向

### 1. 當前局限性

**1.1 超參數 μ**

GOPO 引入了剛度參數 μ，可能需要調整：
- μ 太小：正則化不足，可能不穩定
- μ 太大：過度正則化，學習緩慢
- 當前實驗使用 μ = 0.5，但可能因任務而異

**1.2 實驗範圍有限**

當前實驗集中在：
- 數學推理（MATH 基準）
- 單一模型（Qwen3-1.7B）
- 小型訓練集

未在其他領域驗證：
- 指令跟隨
- 代碼生成
- 對話系統
- 多模態任務

**1.3 BHP 實現差距**

理論推論 4.3 提供精確硬閾值：
```
v*(y) = max(-1, (gα(y) - λ*)/μ)
```

但當前實現使用軟 ReLU 近似：
```
max(0, -Ai ρθ(yi) + (μ/2)(ρθ(yi) - 1)²)
```

研究這個近似差距的影響是一個開放方向。

## 總結

### 核心貢獻

1. **幾何框架**：將對齊問題從概率單純形轉移到 Hilbert 函數空間，概率守恆變為線性正交條件

2. **功-耗散泛函**：從幾何最小距離原則自然出現，不是任意構建

3. **有界 Hilbert 投影**：提供精確稀疏性和硬停止機制，通過解析閾值消除災難性輸出

4. **消逝的化學勢**：在群層面，歸一化優勢已經在零均值子空間，化學勢精確消失

5. **恆定曲率優化**：Hessian 是恆定 μ，獨立於數據和策略狀態，確保非飽和梯度

6. **結構解耦**：採樣幾何（優勢）和優化幾何（曲率）完全解耦

### 實驗驗證

- MATH Level 4 驗證準確率：47% vs GRPO/DAPO 的 44%
- 最高的平均梯度範數：1.03 vs DAPO 的 0.21
- 最高的策略熵：0.134 vs GRPO 的 0.076
- 單調泛化改善趨勢

### 應用價值

- 理論原則性：基於 Hilbert 投影定理，不是啟發式
- 實用性：簡單的超參數（μ），無需裁剪
- 效率：無需評論家，與 GRPO 相同的開銷
- 泛化：在分佈外任務上優於裁剪方法

### 局限性

- 需要調整 μ
- 實驗範圍有限（僅數學推理）
- BHP 使用軟近似而非精確硬閾值

### 未來方向

- 自適應 μ
- 擴展到其他任務
- 精確 BHP 實現
- 與其他方法結合
- 理論擴展（離線設置、收斂速率）
