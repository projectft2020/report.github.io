## 研究摘要

本報告深入分析了arXiv論文《Empirically Calibrated Conditional Independence Tests》（經驗校準的條件獨立性檢測，ECCIT）。該論文提出了一種新的校準方法，用於解決條件獨立性檢測（CIT）在有限樣本或模型誤設定情況下的校準問題。ECCIT通過對抗性優化來識別最壞情況的校準誤差，並擬合一個單調校準映射來調整p值，從而恢復有效的錯誤發現率（FDR）控制。實驗結果表明，ECCIT在合成數據和真實基因表達數據上均優於現有方法。

## 詳細分析

### 1. 研究背景與動機

#### 1.1 條件獨立性檢測的重要性

條件獨立性檢測（Conditional Independence Tests, CIT）是因果發現和特征選擇的核心工具。在統計框架下，檢測因果關係等價於發現使變量獨立的條件集。數學上，存在因果邊緣的條件是變量X和Y在控制所有混淆因子Z後仍然依賴，因此無因果效應的零假設為：

H₀: X ⟂ Y | Z

#### 1.2 常見應用場景

**控制變量選擇問題：** 給定數據{(X₁,X₂,...,Xm,Y)ᵢ}ᵢ₌₁ⁿ，目標是找到變量子集S⊆[m]，使得Xⱼ⟂Y|X₋ⱼ當且僅當j∉S，即Y的馬爾可夫毯。如果不存在潛在混淆因子且所有Xⱼ變量已知不是由Y引起的（例如每個Xⱼ在Y之前被觀測），則S對應於Y的因果特征。

#### 1.3 現有方法的局限性

**理論限制：** Shah和Peters（2020）證明，理論上不可能創建一個能對所有可能的替代假設具有非平凡功效的方法。

**實踐挑戰：**
- **高維低樣本問題：** 當特征數量m相對於樣本量n較大時，許多方法難以控制錯誤率
- **非參數方法：** 收斂太慢，無法產生有效的頻域p值
- **參數假設：** 對可能的條件依賴結構做出參數假設提高了樣本效率，但容易受到模型誤設定的影響
- **雙重魯棒性方法：** 在有限樣本或模型誤設定時，往往無法提供理論保證，通常會失敗於控制第一類錯誤率

#### 1.4 科學應用的重要性

CIT在科學研究中的應用日益廣泛：
- 生物學和醫學數據分析
- 電子健康記錄中的模型開發
- 基因組學、轉錄組學和表觀基因組學數據庫分析
- 腫瘤、細胞系和類器官數據庫分析

科學數據集通常落入高維或低樣本區域，在這些區域中現有CIT程序通常失敗。

### 2. ECCIT方法詳解

#### 2.1 方法概覽

ECCIT（Empirically Calibrated Conditional Independence Tests）是一種包裝器方法，適用於廣泛的條件獨立性檢測程序。ECCIT接收一個現有的條件獨立性檢測或控制變量選擇程序，配合用戶希望應用該程序的數據集。

**核心思想：** 通過優化對抗性模型來最大化CIT程序在目標數據集上的錯誤率，然後將CIT程序應用於數據集時產生的p值進行校準，使其成為有效的p值。

#### 2.2 算法流程

**算法1：經驗校準（對抗性方法）**

**輸入：**
- 數據集 (X,Y)
- 測試 T:(X,Y)↦(p₁,...,pₘ)
- 對抗者類別 𝓕，包含f∈𝓕和Ỹ=f(X,ε)
- 校準度量 M(·,α)

**步驟：**

1. **擬合最壞情況對抗者 f⋆**
   - 如公式(2)所示，在對抗者類別𝓕中選擇使校準度量M最大的生成器

2. **定義零特征集**
   - 令 ℋ₀={j:γⱼ=0} 表示在f⋆下的零假設特征

3. **Bootstrap重採樣循環**
   - 對於 b=1,...,B：
     - 通過重採樣X的行來Bootstrap X(b)
     - 採樣 Ỹ(b)=f⋆(X(b),ε)
     - 計算 p(b)=T(X(b),Ỹ(b))

4. **構建校準器**
   - 根據 M(·,α) 從 {(p(b),ℋ₀(b))}ᵦ₌₁ᴮ 構建 Cal_α

5. **返回校準後的p值：**
   - p_cal = Cal_α(T(X,Y))

#### 2.3 對抗者優化

**目標函數：**

給定X和返回p值的測試程序 T(X,Y)，令𝓕為響應Y的數據生成機制類別，寫為 Y=f(X,ε) 且 f∈𝓕 和噪聲。給定校準度量𝓜和目標錯誤率水平α，ECCIT對抗者選擇：

f⋆ ∈ argmax_{f∈𝓕} 𝔼_X[𝓜(T(X,f(X,ε)), α)]

這個目標在對抗者類別和數據分布X的範圍內盡可能大地擴大校準度量。

**技術細節：**

**參數化：** 對抗者指定兩組參數：
1. 二元向量γ，對應於哪些特征Xⱼ將用於生成合成Ỹ（γⱼ=1），哪些是零變量（γⱼ=0）
2. 從非零特征到合成Ỹ的函數的參數θ

**響應模型：** 實踐中，將響應建模為加性誤差模型：

Ỹ = μ_θ(X·γ) + ε, ε~P(ε), 𝔼[ε]=0

其中μ_θ是具有可學習權重的均值函數。

**Gumbel-softmax技術：** 學習γ是一個非平滑問題，因為它是離散向量。使用Gumbel-softmax（Jang等，2016）來獲得二元掩碼的近似梯度。θ和γ聯合擬合。

#### 2.4 基礎條件獨立性檢測

ECCIT可以與任何返回p值的CIT程序配合使用。論文考慮了兩種具體方法：

**2.4.1 廣義協方差度量（GCM）**

**步驟：**
1. 固定j，令Z:=X₋ⱼ
2. 擬合條件均值估計器：
   - f̂ⱼ(Z) ≈ 𝔼[Xⱼ|Z]
   - ĝ(Z) ≈ 𝔼[Y|Z]
3. 形成殘差：
   - Rₓ^{(j)} = Xⱼ - f̂ⱼ(Z)
   - R_y^{(j)} = Y - ĝ(Z)
   - R^{(j)} = Rₓ^{(j)} ⊙ R_y^{(j)}（逐元素乘積）
4. 計算統計量：
   - R̄^{(j)} = (1/n)∑ᵢ₌₁ⁿ Rᵢ^{(j)}
   - s_(j)² = (1/n)∑ᵢ₌₁ⁿ (Rᵢ^{(j)} - R̄^{(j)})²
   - Tⱼ = √n · R̄^{(j)} / s_(j)
5. 計算p值：pⱼ = 2{1 - Φ(|Tⱼ|)}

**性質：** 當GCM正確規定時，它是速率雙重魯棒的：在零假設下，如果Xⱼ|Z和Y|Z的擬合回歸收斂足夠快，則測試統計量具有正確的極限分布。但在有限樣本或條件期望誤設時，可能會擴大錯誤率。

**2.4.2 留一隨機化測試（HRT）**

**步驟：**
1. 令Z:=X₋ⱼ，在訓練分割上擬合Y從X的預測器ĥ(·)
2. 計算保留集ℐ上的觀測損失：
   L_obs = (1/|ℐ|)∑ᵢ∈ℐ ℓ(Yᵢ, ĥ(Xᵢ))
3. 作為model-X方法，估計Xⱼ|X₋ⱼ的條件分布q̂ⱼ(·|Z)
4. 對於b=1,...,B：
   - X̃ⱼ,ᵢ^{(b)} ~ q̂ⱼ(·|Zᵢ), i∈{1,...,n}
   - 通過替換保留集上第j列形成X̃^{(b)}
   - 計算隨機化損失：
     L^{(b)} = (1/|ℐ|)∑ᵢ∈ℐ ℓ(Yᵢ, ĥ(X̃ᵢ^{(b)}))
5. 對每個特征計算L_obs和{L^{(b)}}ᵦ₌₁ᴮ
6. 右尾p值：
   pⱼ = (1 + ∑ᵦ₌₁ᴮ 𝟙{L^{(b)} ≥ L_obs}) / (B+1)

**計算效率：** 為了計算效率，使用{L^{(b)}}的樣本均值和標準差來近似L^{(b)}的零分布，並從此近似計算p值。

**性質：** 當q̂ⱼ正確規定且估計良好時，訓練-測試分割HRT產生有效的p值。

#### 2.5 校準度量

**2.5.1 Type-I校準度量**

令ℋ₀⊆{1,...,m}表示零假設的索引，令m₀=|ℋ₀|，令Ŝ⊆{1,...,m}表示被拒絕假設的集合。

**定義：** 零p值的經驗CDF：

F̂₀(u) = (1/m₀)∑ᵢ∈ℋ₀ 𝟙{pᵢ ≤ u}

固定截斷α∈(0,1]。在完美校準下，零p值是均勻的，因此在水平α下的實現Type-I錯誤是F̂₀(α)。

**校準誤差度量：**

𝒯(α) = F̂₀(α) - α

零表示精確校準，正值表示擴大的Type-I錯誤。

**2.5.2 FDP校準度量**

**假設：** 在大規模測試中控制家庭錯誤率（FWER）通常太嚴格，導致非常保守的閾值和功效的顯著損失。更常見的目標是錯誤發現率（FDR），它控制所有拒絕中錯誤發現的期望比例。

**Benjamini-Hochberg (BH)算法：**

令p_{(1)} ≤ ... ≤ p_{(m)}，定義：

t_BH = max{α·i/m : p_{(i)} ≤ α·i/m}

拒絕所有pᵢ ≤ t_BH的假設。

**假發現比例（FDP）：**

令R = #{i : pᵢ ≤ t_BH}和V = #{i∈ℋ₀ : pᵢ ≤ t_BH}，校準得分為FDP：

FDP = V / (R ∨ 1)

較低的值表示在水平α下對BH的校準更好。

#### 2.6 校準映射

給定對抗者的最壞情況度量，構建校準器，一個將原始輸出轉換為保守、良好校準輸出的固定單調映射。

**調整水平定義：**

α_cal = sup{t∈[0,1] : 𝜑_M(t) ≤ α}

即選擇最大名義水平，其在對抗者下的實現度量不超過目標α。

#### 2.7 有效性保證

**定理1：**

假設真實條件律 Y|X 存在於用於計算FDP度量函數𝜑_FDP(·)的對抗者類別中。那麼在水平α_cal下運行BH滿足：

FDR(BH(α_cal)) ≤ α

此外，α_cal ≤ α，因此當𝜑_FDP(α) > α時，調整是保守的。

**含義：** 通過對最壞情況進行校準，對同一類別中的任何其他生成器產生有效或保守的行為。

### 3. 實驗結果

#### 3.1 實驗設置

**評估目標：**
- 評估未校準和ECCIT校準版本的GCM和HRT的校準和功效
- 比較使用Type-I或FDP校準度量訓練和校準
- 在不同來源的校準誤差下評估：
  1. 正確規定模型下的小樣本
  2. 真實噪聲分布重尾時的外生噪聲欠規定
  3. 條件估計器和真實條件不匹配時的模型欠規定

**基準方法：** CONTRA（Sudarshan等，2021），一種用於提高CIT魯棒性的最先進方法。

**評估指標：** 在α=0.2（標準固定目標水平）下評估功效和實現FDR。使用**有效功效**（valid power），當FDR約束不滿足時設為零。

#### 3.2 合成數據實驗

**3.2.1 特征分布**

**獨立特征：**
- **正態：** Xᵢⱼ ~ 𝓝(0,1)
- **拉普拉斯：** Xᵢⱼ ~ Laplace(0, 1/√2)
- **Student-t：** Xᵢⱼ ~ t₁

**相關特征（潛在結構）：** 單因子結構：

X = γ·z·1ᵀ + √(1-γ²)·E

其中z ~ 𝓝(0,1)，Eᵢⱼ ~ 𝓝(0,1)，γ=0.5，產生≈γ²=0.25的成對相關性。

**3.2.2 響應Y**

**線性響應：** 稀疏線性模型，具有s個活動特征和高斯噪聲：

Y = X_Sβ_S + ε, β_S ~ 𝓝(0, I_s), ε ~ 𝓝(0, I_n)

**非線性響應：** 包含線性項和簡單非線性項（tanh函數）。

**3.2.3 模型回歸器**

**線性（嶺回歸）：** 為每個特征j擬合線性映射。

**非線性（MLP）：** 使用單隱藏層網絡帶ReLU激活函數替換線性映射。

**對抗者：** 對抗生成器使用相同的族來參數化Y的條件均值。

#### 3.3 單假設測試

雖然主要重點是多假設設置，但ECCIT也適用於單假設框架。相同的校準映射可用於校正單個測試統計量和目標水平的有限樣本Type-I錯誤。

**結果：** 校準顯著改善了Type-I錯誤控制，代價是功效損失。

#### 3.4 樣本和特征縮放

**3.4.1 正確規定設置**

在模型正確規定的情況下，校準主要由有限樣本噪聲驅動，並隨n增長而消退。

**發現：** 即使在模型正確規定時，兩個測試都表現出有限樣本偏差：
- 對於較小的n，殘差-乘積統計量有噪聲
- 產生的p值偏離均勻性，且隨m增大而增加

**校準效果：** 這裡的校準圍繞控制樣本噪聲以擁有校準測試。

**3.4.2 模型欠規定下的校準**

**設置：** 測試側回歸器可能欠規定：如果f̂ⱼ或ĝ的學習器無法捕獲Y對X₋ⱼ的依賴性，殘差結構保持，產生的p值有偏。

**方法：** 對抗者使用非線性回歸器訓練，而測試本身使用線性回歸器，並在從非線性響應Y生成的數據上評估。

**發現：**
- **Type-I度量：** 在[0,α]上應用更保守的校正。在考慮的Ground Truth非線性設置中，這種更強的校正沒有顯著懲罰功效
- **FDP度量：** 針對BH，選擇最小水平α_cal，其實現FDP不超過目標，有效地將閾值帶回限制。FDP校準通常在FDR約束下保留更多名義功效

**權衡：** Type-I度量在單個測試的水平校準，因此會降低名義功效。FDP校準保留更多功效，但可能在校準變化較大時魯棒性較差。

#### 3.5 分布和噪聲的魯棒性

**設置：** X中的潛在結構和重尾噪聲可能會降低測試使用的條件擬合，導致p值扭曲。通過變化特征分布和噪聲族來評估校準如何恢復FDR控制和功效。

**發現：** 在不同分布和噪聲下，能夠通過校準改善測試並獲得有效功效。

#### 3.6 基因表達數據比較

**應用場景：** 變量選擇的現實用例是基因表達分析中的變量選擇。基因表達數據的特征空間很大，且表現出強的跨基因相關性和潛在結構，原始計數通常建模為零膨脹負二項分布。

**數據集：** 從癌症藥物敏感性基因組學（GDSC）數據集構建半合成基準（Yang等，2013）。

**設置：**
- 癌症細胞系作為樣本（n）
- 基因作為特征（m）
- 對表達矩陣進行歸一化
- 應用方差穩定對數變換
- 對每個基因進行z-score標準化

**比較：** 將未校準和校準的GCM/HRT與CONTRA基線在線性和非線性結果模型下進行比較。對每次複現，繪製200×25切片，在非線性響應下合成結果，使用和不使用校準運行所有方法，在100次運行上聚合功效/FDR。

**結果：**
- **校準恢復FDR控制：** 對於GCM和HRT都恢復了FDR控制
- **有效功效增加：** 相對於基線方法如CONTRA增加了有效功效
- **CONTRA的問題：**
  - CONTRA-HRT變體在這個高維、相關的基因表達設置中掙扎
  - 其Xⱼ|X₋ⱼ的條件採樣器誤規定，且預測器不在零樣本下重擬合
  - 產生誤校準的p值和減少的有效功效
  - CONTRA-FASTCRT也表現不佳，因為它每個特征只擬合一個零模型並在零重採樣中重用固定統計量
  - 雖然這些測試提供非常高的名義功效，但它們有很高比例的錯誤發現，導致非常低的有效功效

### 4. 應用價值評估

#### 4.1 優勢

**4.1.1 測試無關性**

ECCIT是一種包裝器方法，可以與任何返回p值的條件獨立性檢測程序配合使用。這提供了極大的靈活性，用戶可以根據具體情況選擇最合適的基礎測試：
- 有大量未標記數據時使用model-X方法
- 難以近似高階矩時使用雙重魯棒方法（通常功效更高）

**4.1.2 理論保證**

定理1提供了FDR控制的有效性保證：當真實響應機制在對抗者類別中時，ECCIT校準保證在目標水平下的FDR控制。

**4.1.3 實踐改進**

**有限樣本校準：** 在有限樣本下恢復有效的p值分布
**模型魯棒性：** 在模型欠規定情況下恢復FDR控制
**功效保持：** 相比現有方法（如CONTRA）在保持FDR控制的同时具有更高的有效功效

**4.1.4 廣泛應用潛力**

**生物醫學領域：**
- 基因表達分析
- 生物標誌物選擇
- CRISPR篩選分析（如SCEPTRE）

**臨床應用：**
- 電子健康記錄分析
- 預測模型開發
- 因果推斷

**科學研究：**
- 因果發現
- 特征選擇
- 高維統計

#### 4.2 局限性

**4.2.1 魯棒性與功效的權衡**

**核心問題：** 對抗者類別的選擇是魯棒性和功效之間權衡的關鍵驅動因素。

**過於簡單的類別：**
- 可能導致校正不足
- 無法捕捉真實的響應機制
- 校準效果有限

**過於靈活的類別：**
- 最壞情況校準可能變得過度保守
- 導致功效下降
- 可能過度校正小p值，減少發現

**挑戰：** 如何在缺乏先驗知識的情況下選擇合適的對抗者類別是一個實踐挑戰。

**4.2.2 計算複雜度**

**Bootstrap重採樣：** 算法需要B次bootstrap重採樣，對於大型數據集可能計算密集。

**對抗者優化：** 優化對抗者模型需要訓練機器學習模型，增加了計算開銷。

**可擴展性：** 對於非常大規模的數據集（數百萬樣本和特征），計算成本可能成為瓶頸。

**4.2.3 依賴數據質量**

**對抗者訓練依賴：** 對抗者模型的有效性依賴於訓練數據的質量和代表性。

**Bootstrap假設：** 使用bootstrap重採樣來近似X的分布，假設觀測數據能夠代表真實分布。在數據有限或有偏差的情況下，這可能不成立。

**4.2.4 局部最壞情況**

**對抗者類別限制：** 校準針對特定對抗者類別𝓕中的最壞情況。如果真實響應機制不在𝓕中，校準保證可能不適用。

**過度保守風險：** 即使真實機制在𝓕中，但接近邊界，校準可能仍然過度保守。

**4.2.5 多重假設測試特定性**

**FDP度量的BH依賴：** FDP校準度量專門針對Benjamini-Hochberg過程設計。如果用戶使用其他多重測試校正方法（如Bonferroni），需要重新設計校準度量。

**Type-I度量的保守性：** Type-I校準度量可能對多重假設設置過於保守，因為它在單個測試水平上進行校準。

### 5. 與現有方法的比較

#### 5.1 現有改進魯棒性的方法

**Corrected Pearson Chi-squared CRT（Xu等，2024）：** 適應CRT以對數據中的協變量偏移魯棒。

**Maxway CRT（Li和Liu，2023）：** 同時建模P(X|Z)和P(Y|Z)，在理論和實踐上獲得優於基本CRT的優勢。

**CONTRA（Sudarshan等，2021）：** 使用真實X數據和估計的P(X|Z)樣本的混合來訓練Y的預測模型以執行CRT。通過混合真實和合成數據，預測模型被訓練使用在實踐中採樣的更現實版本的協變量。

**Zhang等（2025）：** 使用生成神經網絡模型估計兩個條件分布。

#### 5.2 ECCIT的獨特貢獻

**校準重點：** 上述方法都有助於減少CIT中的整體Type-I錯誤或FDR擴大，但沒有一個旨在產生正確校準的CIT。因此，到目前為止的差距是，使用這些方法進行CIT在某種程度上是魯棒的，但在實踐中不清楚如何校準CIT以在有限樣本中控制目標錯誤率。

**對抗性校準：** ECCIT是第一個明確設計用於產生校準CIT的方法，通過對抗性最壞情場景進行校準。

**測試無關性：** 與CONTRA等修改特定CIT的方法不同，ECCIT作為包裝器，可以與任何CIT配合使用。

### 6. 未來研究方向

#### 6.1 對抗者類別選擇

**自適應類別選擇：** 開發自動選擇適當對抗者類別的方法，在魯棒性和功效之間取得平衡。

**層次化類別：** 使用層次化的對抗者類別，從簡單到複雜，逐步增加靈活性。

#### 6.2 功效優化

**一組合理機制：** 改進在一組合理響應機制上的功效，而不是最壞情況。

**自適應校準：** 開發自適應校準策略，根據數據特徵調整校準強度。

#### 6.3 計算效率

**近似算法：** 開發更高效的近似算法，減少bootstrap重採樣次數。

**並行化：** 充分利用並行計算來加速對抗者優化和bootstrap過程。

#### 6.4 擴展應用

**其他多重測試方法：** 擴展到其他多重測試校正方法（如Storey's q-value）。

**因果發現：** 將ECCIT集成到因果發現算法中，如PC算法。

#### 6.5 理論擴展

**更廣泛的保證：** 擴展理論保證到更廣泛的設置和假設。

**漸近分析：** 提供更詳細的漸近分析，理解校準行為在大樣本下的表現。

### 7. 總結

#### 7.1 核心貢獻

ECCIT提出了一種實用的方法，用於在名義保證在實踐中失效時校準條件獨立性檢測。通過對抗性優化識別最壞情況的校準誤差，並擬合單調校準映射來調整p值，ECCIT在恢復校準的同时保持測試無關性。

#### 7.2 實踐意義

**易用性：** 作為包裝器方法，ECCIT易於與現有CIT方法集成。
**有效性：** 在廣泛的基準測試中，ECCIT恢復了有效的FDR控制，並具有比現有校準策略更高的功效。
**靈活性：** 用戶可以根據具體需求選擇基礎測試和校準度量。

#### 7.3 應用建議

**高維低樣本場景：** 當特征數量相對於樣本量較大時，ECCIT特別有用。
**模型不確定性：** 當對真實條件依賴結構不確定時，使用靈活的對抗者類別。
**功效優先：** 當功效優先時，考慮使用FDP校準度量而非Type-I度量。
**保守性優先：** 當嚴格的錯誤率控制優先時，使用Type-I校準度量。

#### 7.4 注意事項

**對抗者類別選擇：** 仔細選擇對抗者類別，平衡魯棒性和功效。
**計算資源：** 考慮計算成本，特別是對於大型數據集。
**驗證：** 在應用前驗證校準效果，檢查p值分布和FDR控制。
