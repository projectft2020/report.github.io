   - 場景：實現某個 API 功能
   - 沒有網絡：需要我（main）先搜尋 → 轉交給 developer
   - 有網絡：developer 自主搜尋文檔 → 直接實現
   - 影響：大幅提升開發效率

2. **Automation Agent 偶爾需要網絡 ⭐⭐**
   - 場景：自動化部署到 GitHub Pages

   - 不同 agent 可能搜尋到不同的資訊
   - 例如：developer 和 research 對同一個技術的理解可能不同

3. **錯誤率增加**
   - agent 可能搜尋到錯誤的資訊
   - 需要更嚴格的驗證機制
