## 1. 三因子模型（FF3）詳細解析

### 1.1 理論背景與發展

Fama-French 三因子模型由 Eugene Fama 和 Kenneth French 於 1993 年提出，是對傳統 CAPM 模型的重大擴展。該模型旨在解釋 CAPM 無法完全說明的股票報酬異常現象，特別是規模效應和價值效應。

### 1.2 數學公式

#### 1.2.1 CAPM 到 FF3 的擴展

**傳統 CAPM 模型：**
```
R_i - R_f = α_i + β_i(R_m - R_f) + ε_i
```

**Fama-French 三因子模型：**
```
R_i - R_f = α_i + β_i(R_m - R_f) + s_i·SMB + h_i·HML + ε_i
```

其中：
- `R_i`：資產 i 的報酬率
- `R_f`：無風險利率
- `R_m`：市場投資組合報酬率
- `R_m - R_f`：市場溢價（MKT 因子）
- `SMB`：規模因子（Small Minus Big）
- `HML`：價值因子（High Minus Low）
- `β_i`：市場風險係數
- `s_i`：規模風險係數
- `h_i`：價值風險係數
- `α_i`：異常報酬（Jensen's alpha）
- `ε_i`：誤差項

### 1.3 因子構建與經濟意義

#### 1.3.1 市場風險因子（MKT）

**數學定義：**
```
MKT = R_m - R_f
```

**構建方法：**
- 使用所有在 NYSE、AMEX、NASDAQ 上市的美國公司
- 市場報酬率為加權平均報酬率
- 無風險利率使用一個月期國庫券利率

**經濟意義：**
- 代表市場系統性風險
- 反映整體市場的波動性

#### 1.3.2 規模因子（SMB - Small Minus Big）

**數學定義：**
```
SMB = 1/3 (Small Value + Small Neutral + Small Growth) 
       - 1/3 (Big Value + Big Neutral + Big Growth)
```

**構建步驟：**
1. **規模分組：** 根據市值大小將股票分為小型（Small）和大型（Big）兩組
2. **價值分組：** 根據帳面市值比（Book-to-Market）將股票分為價值型（Value）、中立型（Neutral）、成長型（Growth）三組
3. **投資組合形成：** 形成 2×3 = 6 個投資組合
4. **因子計算：** 小型股票平均報酬減去大型股票平均報酬

**經濟意義：**
- 捕捉規模效應：小型股票平均而言比大型股票有更高的報酬
- 反映小公司的流動性風險和資訊不對稱性

#### 1.3.3 價值因子（HML - High Minus Low）

**數學定義：**
```
HML = 1/2 (Small Value + Big Value) 
       - 1/2 (Small Growth + Big Growth)
```

**構建方法：**
1. 使用相同的 6 個投資組合
2. 計算高價值股票與低價值股票的平均報酬差異

**經濟意義：**
- 捕捉價值效應：高帳面市值比股票（價值股）平均而言比低帳面市值比股票（成長股）有更高的報酬
- 反映市場對價值股的風險溢價

```python
import pandas as pd
import numpy as np
import statsmodels.formula.api as smf
import pandas_datareader.data as web

# 載入 Ken French 數據庫
def load_fama_french_factors(start_date='1970-01-01', end_date='2025-12-31'):
    """
    載入 Fama-French 三因子數據
    """
    # 載入三因子數據
    factors_3 = web.DataReader('F-F_Research_Data_Factors_daily', 'famafrench', start_date, end_date)
    factors_3.columns = ['MKT-RF', 'SMB', 'HML', 'RF']
    
    # 載入五因子數據
    factors_5 = web.DataReader('F-F_Research_Data_5_Factors_2x3_daily', 'famafrench', start_date, end_date)
    factors_5.columns = ['MKT-RF', 'SMB', 'HML', 'RMW', 'CMA', 'RF']
    
    return factors_3, factors_5

# 因子回歸分析
def fama_french_regression(returns, factors):
    """
    執行 Fama-French 回歸分析
    
    Parameters:
    returns: pd.Series - 資產超額報酬
    factors: pd.DataFrame - 因子數據
    
    Returns:
    回歸結果
    """
    # 合併數據
    data = pd.concat([returns, factors], axis=1).dropna()
    
    # 執行三因子回歸
    model_3 = smf.ols('returns ~ Q("MKT-RF") + SMB + HML', data=data).fit()
    
    # 執行五因子回歸（如果數據可用）
    if 'RMW' in factors.columns and 'CMA' in factors.columns:
        model_5 = smf.ols('returns ~ Q("MKT-RF") + SMB + HML + RMW + CMA', data=data).fit()
        return model_3, model_5
    else:
        return model_3

# 載入範例數據
factors_3, factors_5 = load_fama_french_factors()

# 使用 S&P 500 作為市場基準
sp500_returns = web.DataReader('SP500', 'fred', '1970-01-01', '2025-12-31')
sp500_returns = sp500_returns.pct_change().dropna() * 100  # 轉換為百分比

# 執行回歸分析
model_3, model_5 = fama_french_regression(sp500_returns, factors_5)
```

## 2. 五因子模型（FF5）詳細解析

### 2.1 理論背景

Fama-French 五因子模型於 2015 年提出，在三因子模型的基礎上增加了兩個新因子：盈利能力因子（RMW）和投資因子（CMA）。這個擴展旨在解釋三因子模型無法完全捕捉的報酬異常現象。

### 2.2 數學公式

```
R_i - R_f = α_i + β_i(R_m - R_f) + s_i·SMB + h_i·HML + r_i·RMW + c_i·CMA + ε_i
```

新增因子說明：
- `RMW`：盈利能力因子（Robust Minus Weak）
- `CMA`：投資因子（Conservative Minus Aggressive）
- `r_i`：盈利能力風險係數
- `c_i`：投資風險係數

### 2.3 新增因子構建與經濟意義

#### 2.3.1 盈利能力因子（RMW - Robust Minus Weak）

**數學定義：**
```
RMW = 1/2 (Robust_Small + Robust_Big) 
        - 1/2 (Weak_Small + Weak_Big)
```

**構建方法：**
1. **盈利能力分組：** 根據營業盈利能力（Operating Profitability）將股票分為強健（Robust）和疲弱（Weak）兩組
2. **投資組合形成：** 在規模分組的基礎上，形成 2×3×3 = 18 個投資組合
3. **因子計算：** 高盈利能力股票平均報酬減去低盈利能力股票平均報酬

**經濟意義：**
- 捕捉盈利能力效應：高盈利能力公司平均而言比低盈利能力公司有更高的報酬
- 反映市場對盈利穩定性的溢價

#### 2.3.2 投資因子（CMA - Conservative Minus Aggressive）

**數學定義：**
```
CMA = 1/2 (Conservative_Small + Conservative_Big) 
        - 1/2 (Aggressive_Small + Aggressive_Big)
```

**構建方法：**
1. **投資水平分組：** 根據投資資產比率（Asset Growth）將股票分為保守（Conservative）和積極（Aggressive）兩組
2. **投資組合形成：** 在規模和盈利能力分組的基礎上，形成更細分的投資組合
3. **因子計算：** 低投資公司平均報酬減去高投資公司平均報酬

**經濟意義：**
- 捕捉投資效應：保守型投資公司平均而言比積極型投資公司有更高的報酬
- 反映市場對投資效率的評價

### 2.4 FF3 與 FF5 的差異

| 特性 | 三因子模型 | 五因子模型 |
|------|------------|------------|
| 因子數量 | 3個 | 5個 |
| 新增因子 | 無 | RMW、CMA |
| 解釋能力 | 約90% | 約95% |
| 理論基礎 | 規模和價值效應 | 規模、價值、盈利能力和投資效應 |
| 實用性 | 適合一般分析 | 適合深度研究 |

### 2.5 Python 實作擴展

```python
def construct_fama_french_portfolios(crsp_data, compustat_data):
    """
    構建 Fama-French 投資組合
    
    Parameters:
    crsp_data: DataFrame - CRSP 報酬數據
    compustat_data: DataFrame - Compustat 財務數據
    
    Returns:
    包含所有因子的 DataFrame
    """
    # 1. 準備數據
    # 規模數據（6月份市值）
    size_data = crsp_data[crsp_data['date'].dt.month == 6].copy()
    size_data['sorting_date'] = size_data['date'] + pd.DateOffset(months=1)
    
    # 帳面市值比數據（12月份市值 + t-1年帳面價值）
    market_equity = crsp_data[crsp_data['date'].dt.month == 12].copy()
    market_equity['sorting_date'] = market_equity['date'] + pd.DateOffset(months=7)
    
    # 合併財務數據
    book_to_market = compustat_data.merge(market_equity, on=['gvkey', 'sorting_date'])
    book_to_market['bm'] = book_to_market['be'] / book_to_market['me']
    
    # 2. 投資組合分組
    def assign_portfolio(data, sorting_variable, percentiles):
        """分配投資組合"""
        breakpoints = data[sorting_variable].quantile(percentiles)
        assigned = pd.cut(data[sorting_variable], bins=breakpoints, 
                          labels=range(1, len(breakpoints)))
        return assigned
    
    # 規模分組（中位數分為兩組）
    size_data['portfolio_size'] = assign_portfolio(size_data, 'mktcap', [0, 0.5, 1])
    
    # 價值分組（30% 和 70% 分位數）
    book_to_market['portfolio_bm'] = assign_portfolio(book_to_market, 'bm', [0, 0.3, 0.7, 1])
    
    # 盈利能力分組
    book_to_market['portfolio_op'] = assign_portfolio(book_to_market, 'op', [0, 0.3, 0.7, 1])
    
    # 投資分組
    book_to_market['portfolio_inv'] = assign_portfolio(book_to_market, 'inv', [0, 0.3, 0.7, 1])
    
    # 3. 計算因子報酬
    def calculate_factor_returns(portfolios, factor_type):
        """計算因子報酬"""
        returns = portfolios.groupby(['portfolio_size', f'portfolio_{factor_type}', 'date']).apply(
            lambda x: np.average(x['ret_excess'], weights=x['mktcap_lag'])
        ).reset_index()
        
        if factor_type == 'size':
            # SMB 因子
            smb = returns.groupby('date').apply(
                lambda x: x[x['portfolio_size'] == 1]['ret'].mean() - 
                          x[x['portfolio_size'] == 2]['ret'].mean()
            )
            return smb
        elif factor_type == 'bm':
            # HML 因子
            hml = returns.groupby('date').apply(
                lambda x: x[x['portfolio_bm'] == 3]['ret'].mean() - 
                          x[x['portfolio_bm'] == 1]['ret'].mean()
            )
            return hml
        elif factor_type == 'op':
            # RMW 因子
            rmw = returns.groupby('date').apply(
                lambda x: x[x['portfolio_op'] == 3]['ret'].mean() - 
                          x[x['portfolio_op'] == 1]['ret'].mean()
            )
            return rmw
        elif factor_type == 'inv':
            # CMA 因子
            cma = returns.groupby('date').apply(
                lambda x: x[x['portfolio_inv'] == 1]['ret'].mean() - 
                          x[x['portfolio_inv'] == 3]['ret'].mean()
            )
            return cma
    
    # 計算所有因子
    smb = calculate_factor_returns(size_data, 'size')
    hml = calculate_factor_returns(book_to_market, 'bm')
    rmw = calculate_factor_returns(book_to_market, 'op')
    cma = calculate_factor_returns(book_to_market, 'inv')
    
    # 合併結果
    factors = pd.DataFrame({
        'SMB': smb,
        'HML': hml,
        'RMW': rmw,
        'CMA': cma
    })
    
    return factors
```

## 3. 模型數學推導

### 3.1 CAPM 到 FF3 的擴展理論

#### 3.1.1 CAPM 的局限性

傳統 CAPM 模型假設只有市場風險影響資產報酬：
```
E(R_i) - R_f = β_i[E(R_m) - R_f]
```

然而實證研究發現許多異常現象：
- 小公司規模效應
- 價值股溢價效應
- 動量效應等

#### 3.1.2 FF3 的理論基礎

Fama-French 提出這些異常現象反映了系統性風險，而非市場無效率。三因子模型將 CAPM 擴展為：

```
E(R_i) - R_f = β_i[E(R_m) - R_f] + s_i·E(SMB) + h_i·E(HML)
```

### 3.2 因子收益計算公式

#### 3.2.1 時間序列回歸

對於每個資產 i，我們執行以下回歸：
```
R_{i,t} - R_{f,t} = α_i + β_i(R_{m,t} - R_{f,t}) + s_i·SMB_t + h_i·HML_t + ε_{i,t}
```

#### 3.2.2 交叉截面回歸

在多資產分析中，我們可以進行交叉截面回歸：
```
E(R_i) - R_f = γ_0 + γ_1·β_i + γ_2·s_i + γ_3·h_i + η_i
```

### 3.3 統計顯著性檢驗

#### 3.3.1 單一係數檢驗

對於每個因子係數，我們執行 t 檢驗：
```
H_0: β_i = 0
H_1: β_i ≠ 0

t = (β̂_i - 0) / SE(β̂_i)
```

#### 3.3.2 整體模型檢驗

使用 F 檢驗評估模型整體顯著性：
```
F = (R²/(k-1)) / ((1-R²)/(n-k))
```

其中 k 為參數數量，n 為樣本數。

### 3.4 模型評估指標

#### 3.4.1 調整後 R²

```
R²_adj = 1 - [(1-R²)(n-1)/(n-k-1)]
```

#### 3.4.2 訊息比（Information Ratio）

```
IR = α̂ / σ(ε̂)
```

```python
def calculate_factor_exposure(returns, factors):
    """計算因子暴露度"""
    # 標準化數據
    returns_std = (returns - returns.mean()) / returns.std()
    factors_std = (factors - factors.mean()) / factors.std()
    
    # 計算相關係數矩陣
    correlation_matrix = pd.concat([returns_std, factors_std], axis=1).corr()
    
    # 提取因子暴露度
    factor_exposure = correlation_matrix.iloc[0, 1:]
    
    return factor_exposure
```

### 4.2 因子收益分解

```python
def decompose_returns(returns, factors, factor_betas):
    """分解報酬為因子貢獻"""
    # 計算每個因子的貢獻
    factor_contributions = {}
    for factor in factors.columns:
        factor_contributions[factor] = factor_betas[factor] * factors[factor]
    
    # 計算總因子報酬
    total_factor_return = sum(factor_contributions.values())
    
    # 計算特定風險報酬
    specific_return = returns - total_factor_return
    
    return factor_contributions, specific_return
```

### 4.3 回測方法

```python
def backtest_factor_model(strategy_returns, factors, lookback_period=252):
    """回測因子模型"""
    results = []
    
    for i in range(lookback_period, len(strategy_returns)):
        # 訓練期數據
        train_returns = strategy_returns.iloc[i-lookback_period:i]
        train_factors = factors.iloc[i-lookback_period:i]
        
        # 估計因子係數
        model = fama_french_regression(train_returns, train_factors)
        
        # 測試期預測
        test_factors = factors.iloc[i:i+1]
        predicted_returns = model.predict(test_factors)
        
        # 儲存結果
        results.append({
            'date': strategy_returns.index[i],
            'actual_return': strategy_returns.iloc[i],
            'predicted_return': predicted_returns.iloc[0],
            'model_r2': model.rsquared,
            'model_adj_r2': model.rsquared_adj
        })
    
    return pd.DataFrame(results)
```

## 5. 模型局限性和改進

### 5.1 FF3/FF5 的不足

#### 5.1.1 理論局限性

1. **因子定義主觀性**：投資組合構建方法（如分組點、權重計算）具有一定主觀性
2. **樣本依賴性**：因子表現可能受到特定樣本期間的影響
3. **時變性**：因子風險溢價可能隨時間變化

#### 5.1.2 實證局限性

1. **動量效應缺失**：FF3/FF5 無法完全解釋動量效應
2. **流動性忽略**：未考慮流動性風險溢價
3. **國際適用性**：因子在其他國家的表現可能不同

### 5.2 現代因子模型擴展

#### 5.2.1 Carhart 四因子模型

```
R_i - R_f = α_i + β_i(R_m - R_f) + s_i·SMB + h_i·HML + m_i·MOM + ε_i
```

新增動量因子（MOM）：
```
MOM = 過去 12 個月（扣除最近一個月）報酬最高的股票組合 - 
      過去 12 個月（扣除最近一個月）報酬最低的股票組合
```

#### 5.2.2 Fama-French 六因子模型

在五因子基礎上增加動量因子：
```
R_i - R_f = α_i + β_i(R_m - R_f) + s_i·SMB + h_i·HML + r_i·RMW + c_i·CMA + m_i·MOM + ε_i
```

#### 5.2.3 Q-Factor Model（Hou-Xue-Zhang）

基於投資理論的 q 因子模型：
- 市場因子（MKT）
- 規模因子（ME）
- 投資因子（I/A）
- 盈利能力因子（ROE）

```python
def analyze_factor_performance(factors, start_year=2000, end_year=2025):
    """分析因子表現"""
    # 篩選期間
    factors_period = factors[factors.index.year >= start_year]
    factors_period = factors_period[factors_period.index.year <= end_year]

    stats = pd.DataFrame({
        'Mean_Return': factors_period.mean() * 12,  # 年化報酬

        'Sharpe_Ratio': factors_period.mean() / factors_period.std() * np.sqrt(12),
        'Max_Drawdown': (factors_period.cummax() - factors_period).max(),
        'Skewness': factors_period.skew(),
        'Kurtosis': factors_period.kurtosis()
    })
    
    return stats
```

## 6. 結論與建議

### 6.1 主要發現

```python
import pandas as pd
import numpy as np
import statsmodels.formula.api as smf
import pandas_datareader.data as web
import matplotlib.pyplot as plt
import seaborn as sns
from datetime import datetime, timedelta
import warnings
warnings.filterwarnings('ignore')

class FamaFrenchModel:
    """Fama-French 模型完整實作類別"""
    
    def __init__(self, start_date='1970-01-01', end_date='2025-12-31'):
        self.start_date = start_date
        self.end_date = end_date
        self.factors_3 = None
        self.factors_5 = None
        self.market_data = None
        self.rf_data = None
        
    def load_data(self):
        """載入所有必要數據"""
        # 載入市場數據
        self.market_data = web.DataReader('SP500', 'fred', self.start_date, self.end_date)
        
        # 載入無風險利率
        self.rf_data = web.DataReader('DGS3MO', 'fred', self.start_date, self.end_date)
        
        # 載入因子數據
        self.factors_3 = web.DataReader('F-F_Research_Data_Factors_daily', 'famafrench', self.start_date, self.end_date)
        self.factors_5 = web.DataReader('F-F_Research_Data_5_Factors_2x3_daily', 'famafrench', self.start_date, self.end_date)
        
        # 標準化欄位名稱
        self.factors_3.columns = ['MKT-RF', 'SMB', 'HML', 'RF']
        self.factors_5.columns = ['MKT-RF', 'SMB', 'HML', 'RMW', 'CMA', 'RF']
        
        return self
    
    def analyze_portfolio(self, portfolio_returns, model_type='3-factor'):
        """分析投資組合的因子暴露度"""
        if model_type == '3-factor':
            factors = self.factors_3
            formula = 'portfolio_returns ~ Q("MKT-RF") + SMB + HML'
        else:
            factors = self.factors_5
            formula = 'portfolio_returns ~ Q("MKT-RF") + SMB + HML + RMW + CMA'
        
        # 合併數據
        data = pd.DataFrame({
            'portfolio_returns': portfolio_returns,
            **{col: factors[col] for col in factors.columns if col != 'RF'}
        }).dropna()
        
        # 執行回歸分析
        model = smf.ols(formula, data=data).fit()
        
        return model
    
    def factor_performance_analysis(self):
        """分析因子表現"""
        if self.factors_5 is not None:
            factors = self.factors_5.drop('RF', axis=1)
        else:
            factors = self.factors_3.drop('RF', axis=1)
        
        # 計算年化統計量
        stats = pd.DataFrame({
            'Annualized_Return': factors.mean() * 252,
            'Annualized_Std': factors.std() * np.sqrt(252),
            'Sharpe_Ratio': factors.mean() / factors.std() * np.sqrt(252),
            'Max_Drawdown': (factors.cummax() - factors).max(),
            'Skewness': factors.skew(),
            'Kurtosis': factors.kurtosis()
        })
        
        return stats
    
    def plot_factor_performance(self):
        """繪製因子表現圖表"""
        if self.factors_5 is not None:
            factors = self.factors_5.drop('RF', axis=1)
        else:
            factors = self.factors_3.drop('RF', axis=1)
        
        # 計算累積報酬
        cumulative_returns = (1 + factors/100).cumprod()
        
        # 繪製圖表
        plt.figure(figsize=(12, 8))
        for column in cumulative_returns.columns:
            plt.plot(cumulative_returns.index, cumulative_returns[column], label=column)
        
        plt.title('Factor Cumulative Returns')
        plt.xlabel('Date')
        plt.ylabel('Cumulative Return')
        plt.legend()
        plt.grid(True)
        plt.show()
        
        return cumulative_returns
    
    def backtest_strategy(self, strategy_returns, lookback_period=252):
        """回測投資策略"""
        results = []
        
        for i in range(lookback_period, len(strategy_returns)):
            # 訓練期數據
            train_returns = strategy_returns.iloc[i-lookback_period:i]
            train_factors = self.factors_5.iloc[i-lookback_period:i].drop('RF', axis=1)
            
            # 估計因子係數
            data = pd.concat([train_returns, train_factors], axis=1).dropna()
            if len(data) < 100:  # 確保有足夠數據
                continue
                
            model = smf.ols('train_returns ~ Q("MKT-RF") + SMB + HML + RMW + CMA', data=data).fit()
            
            # 測試期預測
            test_factors = self.factors_5.iloc[i:i+1].drop('RF', axis=1)
            if len(test_factors) > 0:
                try:
                    predicted_returns = model.predict(test_factors)
                    
                    results.append({
                        'date': strategy_returns.index[i],
                        'actual_return': strategy_returns.iloc[i],
                        'predicted_return': predicted_returns.iloc[0],
                        'model_r2': model.rsquared,
                        'model_adj_r2': model.rsquared_adj
                    })
                except:
                    continue
        
        return pd.DataFrame(results)

# 使用範例
if __name__ == "__main__":
    # 初始化模型
    ff_model = FamaFrenchModel(start_date='2000-01-01', end_date='2025-12-31')
```
