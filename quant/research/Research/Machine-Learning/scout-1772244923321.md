## 執行摘要

本篇論文提出了在線非單調 DR-Submodular（Diminishing-Return Submodular）函數在下閉凸集上最大化問題的突破性解決方案。作者證明了該類函數在精心設計的指數重參數化和代理勢函下具有 **1/e-Upper Linearizable** 性質，從而將非凸 DR-Submodular 優化約化為在線線性優化問題。這一結構性突破使得算法能夠：

- 達到 **O(T^(1/2))** 靜態遺憾率，僅需每輪 **O(1)** 次梯度查詢
- 首次獲得**自適應遺憾**和**動態遺憾**的理論保證
- 在半隨機、隨機和零階全信息反饋下顯著改善現有界限

## 一、研究背景

### 1.1 問題設定

**DR-Submodular 函數**是連續次模態函數的推廣，捕獲邊際效益遞減的性質。在機器學習、通信系統、運籌學和經濟學中有廣泛應用。

**線上優化設定：**
- 時間範圍：T 輪
- 約束集：K ⊆ [0,1]^d，下閉凸集（包含原點）
- 每輪 t：算法選擇 x̂_t ∈ K，敵手揭示目標函數 f_t
- 性能指標：α-遺憾，衡量與 α-近似最優解的差距

**下閉凸集**包括盒子約束、背包多面體、擬陣交集等，在資源分配和覆蓋問題中自然出現。

### 1.2 現有研究的局限性

| 方法 | 遺憾率 | 查詢複雜度 | 局限性 |
|------|---------|------------|--------|
| Thang & Srivastav (2021) | O(T^(3/4)) | T^β | 遺憾次優 |
| Zhang et al. (2023) | O(T^(1/2)) | T^(3/2) | 查詢複雜度過高 |
| Zhang et al. (2023) - 單查詢 | O(T^(4/5)) | O(1) | 遺憾次優 |
| Pedramfar et al. (2024) | O(T^(2/3)) | O(1) | 遺憾次優，無自適應/動態保證 |
| **本篇論文** | **O(T^(1/2))** | **O(1)** | **首次獲得自適應和動態遺憾保證** |

**核心挑戰：**
1. **非單調性**：梯度可能為負，算法需平衡上升和收縮
2. **下閉約束**：需在非凸景觀中導航
3. **查詢效率**：現有高效查詢方法遺憾率次優
4. **反饋限制**：多種反饋模式下缺乏統一框架

## 二、核心思想與技術創新

### 2.1 Upper Linearizability 框架

**定義（Upper Linearizable）：**
函數 f 在 K 上是 α-Upper Linearizable，若存在映射 h: K→K 和向量場 g: F×K→R^d，使得對所有 x, y ∈ K：

```
αf(y) - f(h(x)) ≤ β⟨g(f,x), y-x⟩
```

**直觀理解：**
- 將非凸優化約化為線性優化
- α 是近似/競爭因子（本論文 α = 1/e）
- β 是縮放參數（本論文 β = 1 - e^(-1)）
- h 是重參數化，g 是代理勢函

### 2.2 主要結構定理

**定理 1（Main Theorem）：**

設 K ⊆ [0,1]^d 是包含原點的下閉凸集，f: [0,1]^d → R_(≥0) 是非負、可微、非單調的 DR-Submodular 函數。

定義：
- 重參數化：h(x) = 1 - e^(-x)
- 代理函數：F(x) = ∫₀¹ [e^(z-1) / ((1-e^(-1))z)] [f(1 - e^(-zx)) - f(0)] dz
- 代理勢：g(f,x) = ∇F(x)

則對所有 x, y ∈ K：

```
(1/e)f(y) - f(h(x)) ≤ (1-e^(-1))⟨g(f,x), y-x⟩
```

**因此，f 是 1/e-Upper Linearizable！**

### 2.3 關鍵引理

**引理 1：**
設 f 是下閉凸集上的非單調連續 DR-Submodular 函數。對任意 x, y ∈ [0,1]^d 和 z ∈ [0,1]，令 h_z(x) = 1 - e^(-zx)，則：

```
f(h_z(x) ⊕ y) ≥ e^(-z)x̄ f(y) ≥ e^(-z) f(y)
```

其中 x̄ = max_j [x]_j，⊕ 是坐標概率和。

**意義：**建立了指數重參數化與目標 y 之間的嚴格下界關係，橋接了 DR-Submodular 性質與 1/e 近似因子。

## 三、算法設計

### 3.1 BQND：增強查詢算法

**算法 1：BQND (Boosted Query algorithm for Non-monotone DR-Submodular)**

```
輸入：點 x ∈ K，f 的一階隨機預言機 O
1. 採樣：根據概率密度 p(z) = e^(z-1) / (1-e^(-1)) 採樣 z ∈ [0,1]
2. 查詢：構造查詢點 u = 1 - e^(-zx)
3. 查詢預言機：獲取 ∇f(u) 的隨機樣本 v
4. 輸出：返回估計量 g = v ⊙ e^(-zx)
```

**引理 2（性質）：**
- E[g] = g(f,x) = ∇F(x) （無偏估計）
- ‖g‖ ≤ B₁ （有界）

**技術創新：**
1. **雅可比修正**：顯式納入指數映射的曲率到梯度查詢
2. **單次查詢**：構建非單調目標的無偏線性代理
3. **高效估計**：通過隨機採樣避免積分計算

### 3.2 主算法

**算法 2：在線非單調 DR-Submodular 最大化**

```
輸入：時間範圍 T，約束集 K，步長 η
初始化：
  基學習器 A：在線線性優化算法（如 SO-OGA）
  查詢算法 G：BQND（算法 1）

for t = 1,...,T:
  從基學習器 A 接收動作 x_t
  播放動作 x̂_t = 1 - e^(-x_t)
  敵手選擇函數 f_t 和查詢預言機 O_t
  運行查詢算法：o_t ← G(O_t, x_t)
  將 o_t 傳遞給 A 更新狀態
```

**設計理念：**
- **解耦非凸查詢與約束處理**
- **模組化設計**：基學習器可替換（投影、投影免費、非平穩）
- **繼承線性優化保證**：自動獲得基學習器的遺憾界限

## 四、理論結果

### 4.1 靜態遺憾

**命題 1（Static Regret）：**

設 {f_t}_{t=1}^T 是 M₁-Lipschitz 非單調 DR-Submodular 函數序列。實例化基學習器為 SO-OGA（投影免費），算法 2 達到：

```
E[(1/e) max_{u∈K} Σ_{t=1}^T f_t(u) - Σ_{t=1}^T f_t(x̂_t)] ≤ O(M₁√T)
```

**對比：**從 O(T^(2/3)) 改善到 O(√T)，達到線性優化的最優率！

### 4.2 自適應遺憾

**命題 2（Adaptive Regret）：**

```
AR_{1/e}(T) ≤ O(√T)
```

**意義：**首次為非單調 DR-Submodular 最大化提供自適應遺憾保證。自適應遺憾衡量任何連續區間上的最大遺憾，對變化環境更魯棒。

### 4.3 動態遺憾

**命題 3（Dynamic Regret）：**

設 P_T = Σ_{t=1}^T ‖x_t^* - x_{t-1}^*‖ 是最優解路徑長度，則：

```
DR_{1/e}(T) ≤ Õ(√(T(1+P_T)))
```

**意義：**首次為非單調 DR-Submodular 最大化提供動態遺憾保證，處理非平穩敵手。

### 4.4 限制反饋模式

| 反饋模型 | 靜態遺憾 | 自適應遺憾 | 動態遺憾 |
|---------|-----------|------------|----------|
| 一階全信息 | O(√T) | O(√T) | Õ(√(T(1+P_T))) |
| 半隨機 | O(T^(2/3)) | O(T^(2/3)) | Õ(T^(2/3)) |
| 零階全信息 | O(T^(3/4)) | O(T^(3/4)) | Õ(T^(3/4)) |
| 隨機 | O(T^(4/5)) | O(T^(4/5)) | Õ(T^(4/5)) |

**改進：**在所有反饋模式下嚴格優於現有方法。

## 十、技術細節補充

### 10.1 BQND 採樣分布

z 的採樣密度為：

```
p(z) = e^(z-1) / (1-e^(-1)), z ∈ [0,1]
```

這確保了：

```
E[g] = ∫₀¹ p(z) [v(z) ⊙ e^(-zx)] dz
      = ∫₀¹ [e^(z-1)/(1-e^(-1))] [∇f(1-e^(-zx)) ⊙ e^(-zx)] dz
      = ∇F(x)
```

### 10.2 基學習器實例

**SO-OGA (Online Gradient Ascent via Separation Oracle)：**
- 投影免費
- 自適應遺憾：O(√T)
- 適用於下閉凸集

**IA (Improved Ader)：**
- 處理非平穩環境
- 動態遺憾：O(√(T(1+P_T)))
- 追蹤變化最優解

### 10.3 積分計算

代理函數 F(x) 的積分：

```
F(x) = ∫₀¹ [e^(z-1) / ((1-e^(-1))z)] [f(1-e^(-zx)) - f(0)] dz
```

通過分部積分得到：

```
(1-e^(-1))⟨∇F(x), y-x⟩ = (1/e)f(y) - f(1-e^(-x))
```

這是 Upper Linearizability 的核心等式。
