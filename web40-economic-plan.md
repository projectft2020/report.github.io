### 計劃策略：恐慌逆勢（Panic Reversal）

#### 策略概念
利用市場恐慌情緒（VIX 高漲）的逆勢機會：
- 恐慌時：市場過度悲觀，資產價格被低估
- 逆勢：等待恐慌消退，反彈獲利

#### 配置計畫
```
分配資金：20,000 TWD（總資金的 20%）
標的：美國 ETF（QQQ, SPY, IWM）
指標：VIX, VIX3M, UVXY

倉位管理：
基礎：1/N 等權重（DeMiguel 2009）
增強：波動率目標化（VT/VP）
縮放：Kelly + VIX 混合

公式：
f_final = (1/N) × f_kelly × f_volatility × f_fractional

其中：
- f_kelly = expected_return / (volatility ** 2)
- f_volatility = target_vol / current_vol (限制 0.5-2.0)
- f_fractional = 0.5 (半 Kelly)
```

#### 入場條件
```
1. VIX > 30（恐慌閾值）
2. VIX3M > VIX（短期恐慌 > 長期恐慌）
3. 市場連續下跌 > 5 天
4. UVXY 暴漲 > 20%（恐慌指標確認）

入場方式：
- 分批建倉（3 批，33%）
- 時間間隔：3 天
- 總建倉期：6 天
```

#### 出場條件
```
1. VIX < 20（恐慌消退）
2. 市場連續上漲 > 3 天
3. 持倉盈利 > 15%（目標收益）
4. 持倉虧損 > 10%（止損）

出場方式：
- 分批出倉（分 2 次，各 50%）
- 第一次：VIX < 25，出 50%
- 第二次：VIX < 20，出剩餘 50%
```

## ⚠️ 風險管理

### 系統性風險
- **市場崩盤：** VIX > 50，全球市場恐慌
- **制度切換：** 市場規則改變，策略失效
- **黑天鵝事件：** 不可預測的重大事件

### 策略性風險
- **過度擬合：** 策略在樣本外失效
- **參數漂移：** 最優參數隨時間變化
- **策略過時：** 市場參與者學習，策略收益消失

### 執行性風險
- **滑點成本：** 市場衝擊導致執行價格劣化
- **流動性風險：** 無法以合理價格進出場
- **技術故障：** 系統故障導致交易失敗

### 風險控制措施
```
1. 最大回撤限制：< 20%
2. 單策略限制：< 30% 總資產
3. 單標的限制：< 10% 總資產
4. 緊急止損：< 15% 總資產
5. 流動性準備：> 10% 總資產
```
