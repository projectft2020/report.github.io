## 研究摘要

本論文《Conformal Risk Control for Non-Monotonic Losses》解決了共形風險控制算法在處理非單調損失函數時的局限性。傳統的共形風險控制僅適用於損失函數相對參數單調不增的情況，但許多實際應用場景中的損失函數並非單調。作者提出了一個基於算法穩定性（stability）的通用框架，能夠為非單調損失以及多維參數空間提供風險控制保證。該方法在選擇性圖像分類、腫瘤分割的FDR和IOU控制，以及累犯預測的多群組去偏等實際應用中得到了驗證。

## 主要發現

1. **穩定性驅動的風險控制框架** — 作者將共形風險控制的證明分解為兩個獨立部分：單調性意味著穩定性，而穩定性意味著風險控制。這使得原來僅適用於單調損失的證明可以擴展到更一般的非單調損失 | 來源: arXiv:2602.20151

2. **離散化方法適用於一般有界損失** — 通過將參數空間離散化，該方法能夠為任何[0,1]範圍內的有界損失提供風險控制，代價為 Õ(1/√n) 的偏差 | 來源: 命題 2.2.1

3. **連續Lipschitz損失的精確界限** — 對於滿足局部線性和強交叉點條件的平滑損失，該方法提供了更緊的風險控制界限 α + L/(m(n+1))，其中m是交叉點處的斜率，L是Lipschitz常數 | 來源: 命題 2.2.2

4. **選擇性分類的顯式穩定性表達** — 針對選擇性分類（帶棄權的分類），作者給出了穩定性參數的閉合表達：β = 2max{α,1-α}E[K]/(n+1)，其中K衡量了閾值變化的穩定性 | 來源: 命題 2.2.3

5. **經驗風險最小化的多維擴展** — 該文將穩定性框架擴展到多維參數的ERM算法，提供了損失尺度和梯度尺度的穩定性保證，並通過梯度平衡技術實現了分佈自由的多群組去偏 | 來源: 命題 2.3

6. **實驗驗證有效性** — 在ImageNet上的選擇性分類、腫瘤分割的FDR/IOU控制，以及累犯預測的多群組去偏實驗中，CRC-C方法（保守調整版本）在保持有效風險控制的同時，比LTT基線方法更具效率 | 來源: 命題 3

## 詳細分析

### 1. 核心思想與動機

**問題背景：**
- 共形預測是一種流行的無分佈假設的預測不確定性量化方法
- Conformal Risk Control (CRC) 是共形預測的擴展，不僅控制錯誤率，還可以控制任何單調損失函數的期望值
- 然而，CRC僅適用於損失函數 ℓ(x,y;θ) 相對參數 θ 單調不增的情況

**關鍵挑戰：**
- 許多實際應用中的損失函數並非單調，例如：
  - 選擇性分類：帶棄權的分類損失
  - 多目標優化：同時控制多個風險指標
  - 結構化預測：損失函數在參數空間中可能有局部最優值
- Angelopoulos et al. (2024b) 的命題1證明：對於非單調損失，標準CRC算法可能會嚴重失效

**創新解決方案：**
- 引入「算法穩定性」概念來替代「單調性」要求
- 定義算法 𝒜 相對於參考算法 𝒜* 是 β-穩定的
- 主要定理（定理1）：如果 𝒜 是 β-穩定的，且 𝒜* 在完整數據上能夠控制風險（至 α-β），則 𝒜 在子集數據上也能夠控制風險（至 α）

### 2. 理論框架

#### 2.1 基本定義

- **穩定性定義**：
  ```
  E[1/(n+1) Σ_{i=1}^{n+1} ℓ(X_i,Y_i; 𝒜(D_{-i})) ≤
  E[1/(n+1) Σ_{i=1}^{n+1} ℓ(X_i,Y_i; 𝒜*(D_{1:n+1}))] + β
  ```

- **參考算法** 𝒜*：在完整數據集 D_{1:n+1} 上運行的算法，通常已知能夠控制風險
- **目標算法** 𝒜：在子集數據集 D_{1:n} 上運行的算法，需要實際應用

#### 2.2 主要定理

**定理1（主要結果）：**
假設：
- 算法 𝒜 是對稱的且相對於 𝒜* 是 β-穩定的
- 數據 D_{1:n+1} 是可交換的
- 參考算法滿足：E[ℓ(X_{n+1},Y_{n+1}; 𝒜*(D_{1:n+1}))] ≤ α-β

結論：
```
E[ℓ(X_{n+1},Y_{n+1}; 𝒜(D_{1:n}))] ≤ α
```

**直觀解釋：**
- 想象一個「穩定的近似器」：如果一個算法能夠穩定地近似一個已知的良好參考算法，那麼近似器的風險不會超過參考算法的風險加上穩定性誤差
- 通過控制穩定性誤差 β，就能夠在實際可用的數據集上獲得有效的風險控制

### 3. 穩定性界限

#### 3.1 一般有界損失（離散化方法）

**算法定義：**
```
𝒜(D) = inf{θ ∈ Θ_m: R̂_D(θ) ≤ α}
```
其中 Θ_m = {0, 1/m, 2/m, ..., 1} 是離散化的參數空間

**命題2（穩定性界限）：**
```
E[R(θ̂_n)] ≤ α + Õ(1/√n)
```

**技術細節：**
- 通過離散化，將參數空間從連續變為離散的 m+1 個點
- 穩定性來自於離散化粒度：移除一個數據點最多使閾值移動一個離散單位
- 使用Lambert W函數來給出閉合表達
- 偏差項 Õ(1/√n) 隨樣本量增加而減小

**適用場景：**
- 任意[0,1]有界的非單調損失
- 不需要損失函數的平滑性假設
- 實現簡單：只需離散化參數空間

#### 3.2 連續Lipschitz損失

**算法定義：**
```
𝒜(D) = inf{θ ∈ ℝ: R̂_D(θ) ≤ α}
```

**正則條件（命題3假設）：**
存在 m, r > 0 使得：
1. 在左側交叉點附近有局部線性：R̂_{1:n+1}(θ̂_{n+1} ± δ) ≈ α ∓ mδ
2. 遠離交叉點後風險高於 α：R̂_{1:n+1}(θ) ≥ α + mr（當 θ << θ̂_{n+1}）

**命題3（穩定性界限）：**
```
𝒜 是 L/[m(n+1)]-穩定的
因此：
E[R(θ̂_n)] ≤ α + L/[m(n+1)]
```

**直觀理解：**
- **局部線性**：損失函數在交叉點附近足夠陡峭，這樣移除一個點不會導致閾值大幅跳動
- **遠離交叉**：如果損失函數在遠離交叉點的地方也接近 α，那麼會有許多「虛假」的交叉點，導致算法不穩定
- **Lipschitz常數L**：控制了損失函數隨參數變化的速率

**適用場景：**
- 平滑的損失函數（例如：交叉熵損失、平方損失）
- 具有良好幾何性質的風險函數
- 能夠驗證正則條件的情況

#### 3.3 選擇性分類

**問題設定：**
- 預測：Ŷ_i = ŷ(X_i)
- 置信度：P̂_i = p̂(X_i)
- 目標：選擇閾值 θ̂ 使得：P(Ŷ_{n+1} ≠ Y_{n+1} | P̂_{n+1} > θ̂) ≤ α

**損失函數：**
```
ℓ(x,y;θ) = 1{ŷ(x) ≠ y and p̂(x) > θ} - α·1{p̂(x) > θ} + α
```
這是一個非單調的階梯函數。

**命題4（穩定性表達）：**
```
β = 2max{α,1-α}·E[K]/(n+1)
其中 K = max_i |ĵ̂_{-i} - ĵ̂_{n+1}|
ĵ̂ 是閾值在排序後置信度數組中的索引
```

**直觀解釋：**
- K 衡量了當移除一個點時，閾值在排序位置上的最大變化量
- 當模型置信度排序良好時（高置信度的樣本更可能正確），K 值較小
- 係數 2max{α,1-α} 來自於損失函數的跳躍幅度

**命題5（K的界限）：**
```
E[K] ≤ Σ_{j=1}^{n+1} P(Ē_j ∈ (α+(1-α)/j, α+(2-α)/j])
```
其中 Ē_j 是前 j 個樣本的累積平均錯誤率。

**推論3（選擇性分類的風險控制）：**
```
P(Ŷ_{n+1} = Y_{n+1} | P̂_{n+1} > θ̂_n) ≥ 
  1-α - 2max{α,1-α}·E[K]/(n+1)
```

**實際應用：**
- 安全自動化：只在模型高度自信時才進行預測
- 醫療診斷：避免在低置信度時發出錯誤診斷
- 自駕系統：在場景不確定時保持「人不接管」

#### 3.4 經驗風險最小化（ERM）

**問題擴展：**
從一維根查找擴展到多維ERM：
```
𝒜(D) = argmin_{θ∈ℝ^d} [R̂_D(θ) + (λ/2)‖θ‖_2^2]
```

##### 3.4.1 損失尺度的穩定性

**命題6（凸損失）：**
假設：
- ℓ 相對於 θ 是凸的
- 存在 ρ(z) 使得 |ℓ(z;θ) - ℓ(z;θ')| ≤ ρ(z)‖θ-θ'‖_2

結論：
```
β ≤ 2E[ρ(Z_{n+1})^2] / [λ(n+1)]
```

**推論4（泛化界限）：**
```
E[ℓ(X_{n+1},Y_{n+1}; θ̂_n) + (λ/2)‖θ̂_n‖_2^2] ≤ 
  R* + 2E[ρ(Z_{n+1})^2] / [λ(n+1)]
```

**與標準結果的對比：**
- 與Bousquet和Elisseeff (2002)的經典穩定性結果一致
- 正則化參數 λ 起到了穩定化作用
- 強凸性假設可以比個別損失函數的強凸性更弱

##### 3.4.2 梯度尺度的穩定性

**多維穩定性定義：**
算法 𝒜 相對於 g（例如 ∇ℓ）是 β-穩定的，如果：
```
E[1/(n+1) Σ_{i=1}^{n+1} g(X_i,Y_i; 𝒜(D_{-i}))] ⪯ 
  E[1/(n+1) Σ_{i=1}^{n+1} g(X_i,Y_i; 𝒜*(D_{1:n+1}))] + β
```

**定理2（多維風險控制）：**
如果 𝒜 是 β-穩定的且 𝒜* 滿足 E[g(...)] ⪯ α-β，則：
```
E[g(X_{n+1},Y_{n+1}; 𝒜(D_{1:n}))] ⪯ α
```

**命題7（可微凸損失）：**
假設：
- ℓ 是凸且可微的
- ∇ℓ 是 ρ(z)-Lipschitz的
- R̂_{1:n+1} 是 μ-強凸的

結論：
```
β = (E[ρ(Z_{n+1})‖∇ℓ(Z_{n+1}; θ̂_n)‖_2] + 
       E[ρ(Z_{n+1})‖(1/n)Σ_{j=1}^{n}∇ℓ(Z_j; θ̂_n)‖_2]) / [(μ+λ)(n+1)] · 1_d
```

**直觀解釋：**
- 分子：測試梯度的大小 + 訓練梯度的大小（通常接近零）
- 分母：(μ+λ)(n+1) 反映了樣本量和強凸性
- 當 λ=0 時，第二項為零

##### 3.4.3 梯度平衡去偏

**調整算法：**
```
𝒜̃_γ(D) = argmin_{θ∈ℝ^d} [R̂_D(θ) + (λ/2)‖θ‖_2^2 + γ·1_d^⊤θ]
```

**命題8（梯度控制）：**
通過適當選擇 γ，可以實現：
```
E[∇ℓ(X_{n+1},Y_{n+1}; θ̂_n)] ⪯ 0
```

**應用案例（推論6）：**
**設置：**
- 特徵空間：X = {0,1}^d（分組指示器）
- 損失：ℓ(x,y;θ) = (1/2)‖f(x) + x^⊤θ - y‖_2^2
- 算法：無約束的普通最小二乘法

**結果（多群組不偏性）：**
對於每個群組 j：
```
E[Y_{n+1} | X_{n+1,j}=1] ∈ 
  E[f(X_{n+1}) + X_{n+1}^⊤θ̂_n | X_{n+1,j}=1] ± 
  (d^{3/2}·E[|f(X_{n+1}) + X_{n+1}^⊤θ̂_n - Y_{n+1}|]) / [μ(n+1)P(X_{n+1,j}=1)]
```

**意義：**
- 調整後的預測 f(X) + X^⊤θ̂_n 對所有群組同時近似不偏
- 偏差界隨維度 d 和樣本量 n 增加而縮小
- 需要 X_{n+1,j}=1 有足夠的概率觀測到（避免小樣本問題）

### 4. 穩定性參數估計

**方法：**
使用重採樣（bootstrap）從校準樣本估計穩定性參數 β。

**算法：**
1. 從經驗分佈 ℙ̂_n 繪製 B 個重採樣數據集 D^{(b)}
2. 對每個重採樣集，計算：
   ```
   Δ^{(b)} = 1/(n+1) Σ_{i=1}^{n+1} [ℓ(Z_i^{(b)}; 𝒜(D_{-i}^{(b)})) - ℓ(Z_i^{(b)}; 𝒜*(D^{(b)}))]
   ```
3. 估計 β：
   ```
   β̂_{def} = (Δ̄)_+ = max(0, 1/B Σ_{b=1}^{B} Δ^{(b)})
   ```

**理論基礎：**
- 在良好的條件下，重採樣均值一致估計相應的總體期望
- 實驗驗證了該方法的有效性
- 潛在限制：子採樣情況下的重採樣有效性

### 5. 實驗結果

#### 5.1 選擇性分類（ImageNet）

**設置：**
- 數據集：ImageNet（大規模圖像分類）
- 模型：預訓練的CNN，輸出預測和置信度
- 目標：控制在置信度閾值之上的錯誤率 ≤ α

**比較方法：**
1. **CRC-C**：保守版本，使用 α' = α - β̂_{def}
2. **CRC**：未調整版本，直接使用 α
3. **LTT**：Learn-then-Test，δ=0.1，提供高概率保證

**主要發現：**
- **有效性**：CRC和CRC-C都能夠在閾值之上的條件下控制錯誤率至目標水平 α
- **效率**：CRC-C比LTT更不保守，更高的選擇率（預測率）
- **閾值選擇**：CRC-C的閾值通常介於CRC和LTT之間
- **穩定性估計**：通過重採樣估計的 β̂_{def} 能夠有效反映算法的穩定性

**性能指標：**
- **選擇率**：P(P̂_{n+1} > θ̂)，模型選擇預測的比例
- **選擇性準確率**：P(Ŷ_{n+1} = Y_{n+1} | P̂_{n+1} > θ̂)
- CRC-C在保持有效風險控制的同時，獲得了更高的選擇率和準確率

#### 5.2 腫瘤分割的FDR和IOU控制

**應用場景：**
- 醫療影像分割：識別腫瘤區域
- 目標：控制虛假發現率（FDR）和交聯比（IOU）

**損失函數：**
- **FDR控制**：非單調損失，同時考慮分割錯誤和覆蓋率
- **IOU控制**：基於交聯比的非單調優化

**主要發現：**
- CRC方法能夠同時控制多個風險指標
- 對於結構化預測任務（如分割），非單調損失控制是必要的
- 與傳統單調方法相比，CRC-C提供了更緊的保證

#### 5.3 多群組去偏（累犯預測）

**問題設置：**
- 特徵：X ∈ {0,1}^d，表示多個重疊群組（如：種族和性別的組合）
- 標籤：Y ∈ {0,1}，表示是否累犯
- 目標：所有群組的預測誤差同時受控

**方法：**
- 使用ERM進行後處理調整
- 通過梯度平衡實現多群組不偏性
- 應用推論6的理論界限

**主要發現：**
- 調整後的模型在所有群組上近似不偏
- 群組預測的誤差界限：± O(d^{3/2} / [μn])
- 需要足夠的樣本覆蓋來避免小群組問題
- 與專門的多校準或多平衡方法相比，ERM方法更簡單且具有理論保證

## 應用價值與局限性評估

### 應用價值

#### 優勢

1. **通用性**：
   - 適用於廣泛的非單調損失，擴展了CRC的適用範圍
   - 可以處理多維參數空間，適用於複雜的模型

2. **理論嚴謹性**：
   - 提供了有限樣本保證，不依賴分佈假設
   - 穩定性界限是可計算的，可以直接用於實際算法設計

3. **實用性**：
   - 基於重採樣的穩定性估計容易實現
   - 實驗表明在多個實際任務中有效
   - 代價項（穩定性誤差）通常較小

4. **擴展性**：
   - 框架可以應用於新問題設置，只需：
     (1) 選擇適當的參考算法
     (2) 分析穩定性並給出界限
     (3) 使用重採樣估計穩定性參數

#### 潛在應用

1. **多目標優化**：
   - 同時控制多個風險指標（如FDR和FNDR）
   - 自駕系統中的複雜權衡

2. **公平機器學習**：
   - 多群組去偏、多平衡
   - 在重疊群組上的公平性保證

3. **醫療影像分析**：
   - 病變檢測的敏感度與特異度權衡
   - 分割品質的多指標控制

4. **自然語言處理**：
   - 翻譯質量與置信度權衡
   - 摘要任務的風險控制

5. **強化學習**：
   - 安全探索與利用的權衡
   - 多臂老虎機中的風險控制

### 局限性

#### 1. 理論限制

1. **穩定性誤差**：
   - 對於不穩定的算法，保證可能相當寬松（β值大）
   - 雖然CRC-C通過調整 α' = α - β̂_{def} 來補償，但過度保守會降低效率

2. **樣本複雜度**：
   - 穩定性界限通常隨 O(1/√n) 或 O(1/n) 縮小
   - 對於小樣本情況，保證可能不緊
   - 重採樣估計在樣本量小時可能不準確

3. **參考算法依賴**：
   - 需要找到一個「好」的參考算法 𝒜*
   - 對於某些問題，理想的參考算法可能難以構造或計算

4. **正則條件**：
   - 對於連續損失方法，需要損失函數滿足特定正則條件（如局部線性、強交叉點）
   - 違背這些條件的損失函數可能導致算法失效

#### 2. 實踐限制

1. **計算開銷**：
   - 重採樣估計需要多次運行算法
   - 對於大規模數據集和複雜模型，開銷可能顯著
   - 可以通過減少重採樣次數 B 來權衡精度和效率

2. **超參調整**：
   - 穩定性估計的偏差-方差權衡
   - 需要選擇合適的重採樣策略
   - 可能需要交叉驗證來確定穩定性估計的可靠性

3. **黑盒模型兼容性**：
   - 框架要求能夠評估損失 ℓ(x,y;θ)
   - 對於某些專有模型（如大型語言模型），可能難以獲得內部表示或修改模型

4. **離散化粒度**：
   - 離散化方法需要選擇合適的粒度 m
   - 太粗：損失控制不精確
   - 太細：計算開銷增加，但收益有限

#### 3. 比較分析

與現有方法的對比：

| 方法 | 類型 | 優點 | 缺點 |
|------|------|--------|--------|
| **傳統CRC** | 對單調損失精確且高效 | 不適用於非單調損失 |
| **LTT** | 提供高概率保證（1-δ） | 通常過度保守，選擇率低 |
| **CRC-C（本文）** | 適用於非單調損失，保證緊密 | 需要估計穩定性參數 |
| **多校準** | 專門設計的公平性方法 | 依賴分佈假設，缺乏有限樣本保證 |
| **多平衡** | 處理多個公平性指標 | 往往需要複雜的迭代算法 |

**CRC-C的定位：**
- 介於「通用但可能不適用」（傳統CRC）和「專門但可能過度保守」（LTT）之間
- 提供了分佈自由保證與實用效率的良好權衡
- 特別適合於需要風險控制的黑盒模型

### 改進方向

1. **更緊的穩定性界限**：
   - 對於特定類別的損失函數，開發更精確的穩定性分析
   - 利用損失函數的結構性質

2. **適應性穩定性估計**：
   - 根據算法的複雜度調整重採樣次數
   - 使用重要性採樣來降低變異
   - 結合交叉驗證

3. **參考算法的自動化選擇**：
   - 開發從損失函數形式自動構造參考算法的方法
   - 學習良好的參考算法

4. **在線學習擴展**：
   - 將框架擴展到增量學習場景
   - 動態更新穩定性估計

5. **多目標優化集成**：
   - 與Pareto優化方法結合
   - 處理衝突風險指標
