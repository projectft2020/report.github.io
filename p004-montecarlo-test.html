1.2 測試方法¶

幾何布朗運動模型 (GBM):

dS = μS dt + σS dW

其中:
- S: 資產價格
- μ: 預期收益率
- σ: 波動率
- dW: 維納過程增量 (標準布朗運動)
- dt: 時間步長

離散化實現:

S(t+dt) = S(t) × exp((μ - 0.5σ²)dt + σ√dt × Z)

其中 Z ~ N(0, 1) 為標準正態隨機變量

4. VectorBT 集成評估¶

4.1 集成方案¶

為什麼使用 VectorBT?

  1. 向量化優化: 自動並行化，支持多核加速
  2. 回測集成: 直接從模擬生成交易信號和績效指標
  3. 視覺化: 內置繪圖功能，生成分位數路徑圖
  4. 擴展性: 支持多資產、多策略模擬

擬議集成示例:

import vectorbt as vbt

# 創建價格路徑數組
price_matrix = geometric_brownian_motion(...)  # shape: (n_steps, n_simulations)

# 轉換為 VectorBT 格式
prices = pd.DataFrame(price_matrix)

# 簡單買入持有回測
pf = vbt.Portfolio.from_holding(
    prices,
    init_cash=10_000
)

# 計算統計
stats = pf.stats()
print(stats)

11.2 數學參考¶

幾何布朗運動 (GBM) 解析:

S(t) = S0 × exp((μ - 0.5σ²)t + σW(t))

其中 W(t) 是維納過程:
W(t) ~ N(0, t)

期望值與方差:

E[S(t)] = S0 × exp(μt)
Var[S(t)] = S0² × exp(2μt) × (exp(σ²t) - 1)
