### 2. ECDF Clustering: Core Innovation

**Fundamental Concept:**
ECDF clustering represents each set of LLM responses as an empirical cumulative distribution function of cosine similarities to reference answers. The ECDF $F_{\mathbf{x}_n}(x) = \frac{1}{n}\sum_{i=1}^{n}1_{(-\infty,x]}(x_i)$ shows the proportion of responses with similarity less than or equal to each value.

**Key Advantages:**
- **No Hyperparameters:** Unlike histograms requiring bin width decisions, ECDFs need no tuning
- **Length Agnostic:** Can compare response sets of different sizes directly
- **Distribution Preservation:** Maintains full distributional information rather than collapsing to single metrics
- **Mathematical Rigor:** Provides theoretically sound foundation for comparison

**Implementation Process:**
1. Compute cosine similarities between each response and correct answers using embeddings
2. Construct ECDF for each agent setting/configuration
3. Calculate L1 distances between ECDFs: $D_{ij} = \int_{-\infty}^{\infty}|\hat{F}_{\mathbf{v}_i}(x) - \hat{F}_{\mathbf{v}_j}(x)|dx$
4. Apply k-medoids clustering to identify similar response patterns

### 3. Response Distribution Analysis Methodology

**Similarity Computation:**
For each response $\hat{a}_{ij}$ in setting $i$, compute maximum cosine similarity to any correct answer:
$$v_{ij} = \max_{a \in \mathbf{a}_i} g(f_{\phi}(a), f_{\phi}(\hat{a}_{ij}))$$
where $f_\phi$ is the embedding function and $g$ computes cosine similarity.

**ECDF Construction:**
For each setting's similarity vector $\mathbf{v}_i = [v_{i1}, ..., v_{in^{(i)}_C}]$, construct ECDF:
$$\hat{F}_{\mathbf{v}_i}(x) = \frac{1}{n^{(i)}_C}\sum_{k=1}^{n^{(i)}_C}1_{(-\infty,x]}(v_{ik})$$

**Clustering Algorithm:**
- Use k-medoids (PAM algorithm) for clustering ECDFs
- L1 distance between ECDFs equals L1-Wasserstein distance between underlying probability distributions
- Provides interpretable medoids as cluster representatives

## Experimental Insights from Paper

**Key Findings:**
1. **Same Accuracy, Different Distributions:** Experiments show that settings with identical final accuracies can have dramatically different ECDFs, revealing hidden performance differences.

2. **Temperature Impact:** Higher temperature settings show more variable response distributions, while lower temperatures produce more consistent but potentially biased outputs.

3. **Persona Effects:** Different personas can lead to distinct response patterns, even when final accuracy is similar.

4. **Question Difficulty:** Certain question topics consistently produce better or worse response distributions across all settings.

**Implications for Trading Systems:**
