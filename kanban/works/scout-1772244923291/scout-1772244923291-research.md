## 執行摘要

本論文提出了一種基於可交換多任務高斯過程（Exchangeable Multi-task Gaussian Processes）的新方法，用於面板數據的因果推斷和政策評估。該方法能夠處理兩類場景：單一處理單位的一次性處理，以及多個處理單位的分期處理採用。研究通過分解共同均值過程和單位特定偏差，建立可交換的協方差結構，從而在保證單位間可交換性的同時，允許靈活的非線性時間趨勢。

主要貢獻包括：
- 提供完整的後驗預測分佈，用於構建反事實路徑
- 推導點狀和累積處理效應及可信區間
- 通過實證研究（加州煙草控制法案和希臘加油站審計）驗證方法的有效性
- 為分期採用設計提供了可擴展的計算框架

置信度：高（基於完整的arXiv論文內容）

### 1.2 研究背景

現代公共政策評估越來越依賴追蹤多個單位（如州、地區、公司或個人）隨時間變化的縱向數據集。在許多應用中，核心問題是評估政策干預對部分單位在已知時間點的因果效應，使用其餘單位作為對照組來重建若未發生干預時將觀察到的反事實軌跡。

傳統方法如合成控制法（Synthetic Control）、雙向固定效應模型（TWFE）、廣義合成控制（GSC）和貝葉斯結構時間序列（BSTS）在處理非線性趨勢、複雜協變量效應或單位間動態差異較大的情況時可能存在限制。

## 2. 核心思想與技術細節

### 2.1 方法論框架

#### 2.1.1 潛在結果框架

研究採用潛在結果框架：
- 對於每個單位 *i* 和時間 *t*，存在兩種潛在結果：
  - *yᵢₜ(0)*：無處理時的結果
  - *yᵢₜ(1)*：有處理時的結果
- 實際觀測結果：*yᵢₜ = yᵢₜ(0)I{wᵢₜ=0} + yᵢₜ(1)I{wᵢₜ=1}*
- *wᵢₜ* 是處理指示變量（0或1）

#### 2.1.2 因果估計量

研究定義了以下估計量：

1. **個體因果效應**:
   δᵢ,ₜ = yᵢ,ₜ(1) - yᵢ,ₜ(0)

2. **處理單位的平均因果效應**:
   τᵢ★,ₜ = 𝔼[δᵢ★,ₜ | 𝒟₀], t ∈ 𝒯₁

3. **累積處理效應**:
   Δᵢ★ = Σₜ=T₀+1ᵀ δᵢ★,ₜ

4. **平均處理效應**:
   δ̄ᵢ★ = (1/(T-T₀)) Σₜ=T₀+1ᵀ δᵢ★,ₜ

### 2.2 可交換高斯過程模型

#### 2.2.1 基本模型設定

每個單位的潛在無處理結果分解為：

**yᵢₜ(0) = fᵢ(t) + εᵢₜ, εᵢₜ ~ 𝒩(0, ωᵢ²)**

其中 *fᵢ(t)* 進一步分解為：

**fᵢ(t) = μ(t) + gᵢ(t), i = 1,...,m, t ∈ 𝒯**

- **μ(t)**：所有單位共享的共同過程
- **gᵢ(t)**：單位 *i* 的特定偏差

#### 2.2.2 GP先驗分佈

對共同過程和單位特定偏差分別放置獨立的GP先驗：

**μ(·) ~ 𝒢𝒫(0, σμ² k(·,·))**

**gᵢ(·) ~ i.i.d. 𝒢𝒫(0, σg² k(·,·)), i = 1,...,m**

其中 *k(·,·)* 是核函數，研究考慮：
- **OU核**: kᴼᵁ(t,s) = exp(-|t-s|/ℓ)
- **RBF核**: kᴿᴮᶠ(t,s) = exp(-(t-s)²/(2ℓ²))

#### 2.2.3 協方差結構

堆疊向量 **𝐟 = (𝐟₁ᵀ,...,𝐟ₘᵀ)ᵀ ∈ ℝᵐᵀ** 的協方差矩陣具有塊結構：

```
Cov(𝐟) = [(σμ²+σg²)K   σμ²K    ...   σμ²K
          σμ²K    (σμ²+σg²)K ...   σμ²K
           ...      ...         ...
          σμ²K     σμ²K    ... (σμ²+σg²)K]
```

或Kronecker形式：
```
Cov(𝐟) = (𝟏ₘ𝟏ₘᵀ) ⊗ (σμ²K) + Iₘ ⊗ (σg²K)
```

**可交換性體現**：聯合分佈在單位索引的置換下不變。

#### 2.2.4 包含協變量的擴展

模型可擴展以包含協變量：

**fᵢ(t) = μ(t) + gᵢ,₁(t) + gᵢ,₂(xᵢₜ)**

其中：
- *gᵢ,₁*：僅依賴時間的偏差
- *gᵢ,₂*：依賴協變量的偏差
- *xᵢₜ ∈ ℝᵖ*：單位 *i* 在時間 *t* 的協變量

對應的GP先驗：
- **μ(·) ~ 𝒢𝒫(0, σμ² kₜᵢₘₑ(·,·))**
- **gᵢ,₁(·) ~ 𝒢𝒫(0, σg₁² kₜᵢₘₑ(·,·))**
- **gᵢ,₂(·) ~ 𝒢𝒫(0, σg₂² kₓ(·,·))**

### 2.3 分期採用設計

#### 2.3.1 挑戰

在分期採用設計中：
- 多個處理單位在不同時間接受干預
- 計算複雜度隨觀測數量立方增長
- 精確GP每個邊際似然評估需要 *n×n* 矩陣的Cholesky分解

#### 2.3.2 計算解決方案

採用"一次一單位"框架：

對每個處理單位 *i* ∈ *𝐈₁*：
1. 構建較小的訓練集：該單位的預處理觀測 + 從 *𝐈₀* 中隨機採樣的 *M* 個未處理單位
2. 視為單一處理單位反事實預測任務
3. 對所有處理單位獨立重複此過程

矩陣維度從 *n* 降至：
**nᵢ = T₀ᵢ + M·T**

計算複雜度從 **𝒪(n³)** 降至 **𝒪(Σᵢ∈𝐈₁ nᵢ³)**

### 2.4 參數估計與預測

#### 2.4.1 邊際似然

給定高斯似然，邊際似然可解析計算：

**𝐲 | θ ~ 𝒩(0, Σ(θ)), Σ(θ) = K_f(θ) + Ω(θ)**

其中：
- *θ* 是超參數（方差分量、噪聲方差、長度尺度）
- *K_f* 是可交換協方差矩陣
- *Ω* 是對角噪聲協方差矩陣

#### 2.4.2 超參數學習

採用II型最大似然（經驗貝葉斯）：

**θ̂ = argmax_θ log p(𝐲 | θ)**

超參數包括：
- 方差分量：(σμ², σg₁², σg₂²)
- 單位特定噪聲方差：{ωᵢ²}ᵢ₌₁ᵐ
- 長度尺度參數：(ℓₜᵢₘₑ, ℓₓ)，ℓₓ = (ℓ₁,...,ℓₚ)ᵀ

#### 2.4.3 後驗預測

對於處理單位 *i* 在後處理時期的潛在無處理結果：

**𝐲★ | Z, Z★, 𝐲, θ̂ ~ 𝒩(m★, V★)**

預測均值：
**m★ = K(Z★, Z; θ̂)[K(Z, Z; θ̂) + Ω]⁻¹ 𝐲**

預測方差：
**V★ = K(Z★, Z★; θ̂) + Ω★ - K(Z★, Z; θ̂)[K(Z, Z; θ̂) + Ω]⁻¹ K(Z, Z★; θ̂)**

因果效應估計：
**δᵢ★,ₜ | 𝒟₀ ~ 𝒩(τᵢ★,ₜ, σᵢ★,ₜ²)**

其中：
- **τᵢ★,ₜ = yᵢ★,ₜ(1) - m★ₜ**
- **σᵢ★,ₜ² = v★ₜ**

### 2.5 基本假設

研究採用以下假設：

1. **單一尖銳干預**：存在時間 *T₀*，使得處理單位 *i★* 在 *t ≤ T₀* 時 *wᵢ★ₜ = 0*，在 *t > T₀* 時 *wᵢ★ₜ = 1*；所有對照單位在所有時間都無處理

2. **時間無干擾**：單位 *i* 在時間 *t* 的結果僅取決於其自身的處理路徑，而不依賴於其他單位的處理歷史

3. **協變量不受處理影響**：對於所有單位和 *t ≥ T₀*，*xᵢₜ(1) = xᵢₜ(0)*

4. **結果無預期**：在干預時間 *T₀* 之前，單位的結果不受未來處理分配的影響

5. **處理無預期**：時間 *T₀* 的處理分配僅取決於過去的協變量和結果

## 3. 實證應用

### 3.1 應用1：第99號提案（Proposition 99）

#### 3.1.1 數據集描述

- **處理單位**：加利福尼亞州
- **對照單位**：美國其他州
- **干預時間**：1988年
- **時間範圍**：1970-1999年（30年年度數據）
- **結果變量**：人均香菸銷售量
- **政策內容**：煙草控制法案，旨在減少香菸銷售

#### 3.1.2 分析設計

- 使用其他州的結果作為對照
- 結合觀測到的協變量構建加州1988年後 *y_CA,t(0)* 的後驗預測分佈
- 導出點狀和累積處理效應及可信區間
- 與合成差分-in-differences（SDID）方法進行基準比較
- 進行安慰劑式驗證：在預處理窗口內選擇"虛假"干預日期

### 3.2 應用2：希臘加油站審計

#### 3.2.1 數據集描述

- **處理單位**：數百個加油站
- **對照單位**：從未處理的加油站池
- **處理性質**：分審計（在不同時間點進行）
- **觀測頻率**：每周
- **時間範圍**：一年
- **政策內容**：審計旨在提高監測基礎設施的完整性和性能

#### 3.2.2 分析設計

- 採用一次一單位框架
- 對每個處理單位從未處理單位池中隨機採樣 *M* 個對照
- 獨立預測每個處理單位的反事實路徑
- 計算跨處理單位的匯總效應

## 4. 技術優勢與創新點

### 4.1 方法論優勢

1. **非參數/半參數靈活性**：
   - 不假設特定的函數形式
   - 通過核函數學習潛在的非線性動態
   - 可適應複雜的時間趨勢和單位間差異

2. **不確定性量化**：
   - 提供完整的後驗預測分佈
   - 自然導出可信區間
   - 不僅提供點估計，還提供不確定性度量

3. **可解釋性**：
   - 分解共同均值和單位特定偏差
   - 方差分量解釋不同來源的變異
   - 噪聲單位可以更強地匯聚到共同均值

4. **計算效率**：
   - 可交換結構減少了超參數數量
   - 固定載荷簡化了模型結構
   - 一次一單位框架適應大型數據集

### 4.2 與現有方法的比較

| 方法 | 優點 | 局限性 |
|------|------|--------|
| **雙向固定效應（TWFE）** | 簡單、易於解釋 | 假設可加性，處理效應隨時間變化或有非線性趨勢時可能偏差 |
| **合成控制（SC）** | 專注於單一處理單位，適合長預處理期 | 需要足夠長的預處理期，線性組合假設 |
| **廣義合成控制（GSC）** | 允許時間和單位特定效應交互 | 仍依賴線性結構假設 |
| **貝葉斯結構時間序列（BSTS）** | 狀態空間表示，靈活 | 需要指定潛在狀態結構，模型選擇複雜 |
| **可交換GP（本文）** | 非參數、靈活、量化不確定性、可擴展 | 計算複雜度高（雖有改進），需要選擇核函數 |

### 4.3 創新點

1. **將可交換GP應用於因果推斷**：首次將可交換多任務GP框架系統性地應用於面板數據的政策評估

2. **分期採用的計算框架**：提出了一次一單位的計算策略，使GP方法能夠處理大型分期採用數據集

3. **靈活的核函數選擇**：實現了多種核函數變體（OU、RBF、ARD），適應不同的數據特徵

4. **安撫劑驗證**：在預處理窗口內使用"虛假"干預日期進行模型驗證，評估預測準確性

   - 可處理不同地區在不同時間採取公共衛生措施的情況

4. **社會科學研究**：
   - 適合分析自然實驗和準實驗設計
   - 處理觀測數據中的因果推斷問題

### 5.2 方法論貢獻

1. **理論貢獻**：
   - 擴展了可交換GP的應用範圍
   - 為面板數據因果推斷提供了新的貝葉斯框架
   - 連接了機器學習和計量經濟學

### 5.3 適用條件

該方法最適用於以下情況：
- 有足夠的對照單位（通常需要多個）
- 預處理期相對較長，足以學習潛在動態
- 時間趨勢可能是非線性的
- 需要量化不確定性
- 處理採用是分期的或單一的

## 6. 局限性與挑戰

### 6.1 方法論局限

1. **計算複雜度**：
   - 儘管有一次一單位框架，GP方法仍然比參數方法計算量大
   - 對於極大規模數據集（如數十萬個單位），仍可能面臨挑戰
   - 可能需要進一步的近似方法（如稀疏GP、變分推斷）

2. **核函數選擇**：
   - 模型性能依賴於核函數的選擇
   - 不同核函數適應不同的數據特徵
   - 需要專業知識或交叉驗證來選擇合適的核函數

3. **超參數估計**：
   - 最大邊際似然可能有多個局部最優值
   - 初始化值可能影響最終結果
   - 可能需要多次運行以確保穩定性

### 6.2 假設限制

1. **無干擾假設**：
   - 處理分配不影響其他單位的結果
   - 在某些政策環境下（如區域間競爭）可能不成立

2. **協變量不受處理影響**：
   - 要求選擇的協變量不受政策影響
   - 在某些情況下難以找到合適的協變量

3. **單位間可交換性**：
   - 基本模型假設單位間可交換
   - 擴展模型放寬了此假設，但仍受限制
   - 對於高度異質的單位可能效果不佳

### 6.3 數據要求

1. **樣本量**：
   - 需要足夠的對照單位以學習共同結構
   - 對照單位太少時可能過擬合
   - 預處理期需要足夠長以學習潛在動態

2. **數據質量**：
   - 對缺失值和異常值敏感
   - 需要仔細的數據預處理
   - 測量誤差可能影響結果

### 6.4 解釋性挑戰

1. **黑箱性質**：
   - GP方法相對於線性模型更難解釋
   - 難以直觀理解每個參數的作用
   - 可能影響政策制定者的接受度

2. **結果呈現**：
   - 如何有效地呈現不確定性是一個挑戰
   - 可信區間的解釋需要統計素養
   - 可能需要額外的可視化工具

## 附錄：技術實現細節

### A.1 核函數選項

研究實現了以下核函數變體：

1. **時間核**：
   - Ornstein-Uhlenbeck (OU): k(t,s) = exp(-|t-s|/ℓ)
   - Radial Basis Function (RBF): k(t,s) = exp(-(t-s)²/(2ℓ²))

2. **協變量核**：
   - 標準RBF: k(x,x') = exp(-||x-x'||²/(2ℓ²))
   - ARD (自動相關決定): k(x,x') = exp(-Σⱼ (xⱼ-x'ⱼ)²/(2ℓⱼ²))

### A.2 超參數初始化

建議的初始化策略：
- σμ²: 數據標準差的平方
- σg²: 數據標準差的平方的0.1倍
- ℓ: 時間範圍的1/10
- ωᵢ²: 觀測噪聲方差的初步估計

### A.3 軟件實現

研究使用GPflow庫（Python）進行實現：
- GPflow是建立在TensorFlow之上的GP庫
- 支持自動微分和GPU加速
- 提供了靈活的核函數和似然選項

### A.4 驗證策略

**安慰劑驗證步驟**：
1. 在預處理期內選擇一個"虛假"干預日期
2. 將數據分為"預-虛假"和"後-虛假"時期
3. 使用"預-虛假"數據訓練模型
4. 預測"後-虛假"時期的反事實路徑
5. 計算預測誤差指標（RMSE、MAE、覆蓋率）
6. 重復多個虛假日期，評估穩定性

**覆蓋率評估**：
- 檢查95%可信區間的實際覆蓋率
- 理想情況下應接近95%
- 覆蓋率過低表示不確定性被低估
- 覆蓋率過高表示模型過度保守
