## 1. q5 因子動能監測系統

### 1.1 q5 架構的具體設計

**q5 架構**是基於 Hou, Xue, Zhang (HXZ) 的 q 因子模型進行升級的現代因子投資架構。該架構起源於學術界的因子戰爭，特別是張櫓等人與 Fama-French 學派的競爭與發展。

#### q5 架構的五個核心因子：

1. **市場因子 (Market Factor)**：市場整體報酬率
2. **規模因子 (Size Factor)**：小市值公司與大市值公司的報酬差異
3. **投資因子 (Investment Factor)**：保守投資公司與積極投資公司的報酬差異
4. **獲利因子 (Profitability Factor)**：高獲利能力公司與低獲利能力公司的報酬差異
5. **動能因子 (Momentum Factor)**：強勢股與弱勢股的報酬差異

### 1.2 全球資本市場動能監測

Freddy 的動能監測系統基於以下原則：

```python
import pandas as pd
import numpy as np
import yfinance as yf
from datetime import datetime, timedelta

class FactorMomentumMonitor:
    def __init__(self):
        self.etf_tickers = {
            'market': 'SPY',      # 市場因子代理
            'size': 'IJS',       # 小市值價值因子
            'value': 'VTI',      # 價值因子
            'momentum': 'MTUM',  # 動能因子
            'quality': 'QUAL'    # 品質因子
        }
        
    def calculate_momentum_score(self, prices, lookback_periods=[1, 3, 6, 12]):
        """
        計算多期間動能分數
        使用加權方式，近期權重較高
        """
        momentum_scores = {}
        weights = np.array([0.4, 0.3, 0.2, 0.1])  # 1,3,6,12個月權重
        
        for period, weight in zip(lookback_periods, weights):
            returns = prices.pct_change(period * 21)  # 假設每月21個交易日
            momentum_scores[period] = returns.iloc[-1] * weight
            
        return sum(momentum_scores.values())
    
    def monitor_global_factors(self):
        """監測全球因子動能"""
        end_date = datetime.now()
        start_date = end_date - timedelta(days=365)
        
        factor_data = {}
        for factor, ticker in self.etf_tickers.items():
            try:
                data = yf.download(ticker, start=start_date, end=end_date)
                factor_data[factor] = data['Close']
            except Exception as e:
                print(f"無法下載 {ticker}: {e}")
                
        return factor_data
```

### 1.3 因子動能計算方法

Freddy 採用的動能計算方法包含：

1. **時間序列動能**：衡量因子自身歷史表現
2. **橫截面動能**：比較不同因子間的相對強弱
3. **風險調整後動能**：考量波動性後的動能表現

```python
def calculate_factor_momentum(factor_data):
    """
    計算因子動能指標
    """
    results = {}
    
    for factor_name, prices in factor_data.items():
        # 時間序列動能
        returns_1m = prices.pct_change(21)
        returns_3m = prices.pct_change(63)
        returns_6m = prices.pct_change(126)
        returns_12m = prices.pct_change(252)
        
        # 動能分數（加權平均）
        momentum_score = (0.4 * returns_1m.iloc[-1] + 
                          0.3 * returns_3m.iloc[-1] + 
                          0.2 * returns_6m.iloc[-1] + 
                          0.1 * returns_12m.iloc[-1])
        
        # 波動率調整
        volatility = returns_1m.std() * np.sqrt(252)
        risk_adj_momentum = momentum_score / volatility if volatility > 0 else 0
        
        results[factor_name] = {
            'momentum_score': momentum_score,
            'risk_adj_momentum': risk_adj_momentum,
            'volatility': volatility
        }
    
    return pd.DataFrame(results)
```

### 2.3 權重分配策略

Freddy 的 MFA (Multi Factor Allocation) 策略採用風險平價配置：

```python
def calculate_risk_parity_weights(factor_returns):
    """
    計算風險平價權重
    """
    # 計算協方差矩陣
    cov_matrix = factor_returns.cov() * 252  # 年化
    
    # 計算波動率
    volatilities = np.sqrt(np.diag(cov_matrix))
    
    # 風險平價權重（反比於波動率）
    inv_vols = 1 / volatilities
    weights = inv_vols / inv_vols.sum()
    
    return weights

def freddy_mfa_allocation():
```

### 2.4 實際交易實作

```python
class FreddyFactorStrategy:
    def __init__(self, initial_capital=1000000):
        self.capital = initial_capital
        self.positions = {}
        self.allocation = freddy_mfa_allocation()
        
    def rebalance_portfolio(self, current_prices):
        """
        再平衡投資組合
        """
        # 計算當前價值
        current_value = sum(self.positions.get(etf, 0) * current_prices[etf] 
                           for etf in self.allocation.keys())
        
        # 計算目標金額
        target_amounts = {etf: current_value * weight 
                         for etf, weight in self.allocation.items()}
        
        # 計算需要調整的股數
        target_shares = {}
        for etf, target_amount in target_amounts.items():
            target_shares[etf] = target_amount / current_prices[etf]
            
        return target_shares
    
    def execute_trades(self, target_shares, current_prices, commission=0.001):
        """
        執行交易（包含手續費）
        """
        trades = {}
        for etf, target_shares in target_shares.items():
            current_shares = self.positions.get(etf, 0)
            trade_shares = target_shares - current_shares
            
            if abs(trade_shares) > 0.1:  # 避免微量交易
                trade_value = trade_shares * current_prices[etf]
                commission_cost = abs(trade_value) * commission
                trades[etf] = {
                    'shares': trade_shares,
                    'value': trade_value,
                    'commission': commission_cost
                }
                
        return trades
```

## 3. 因子溢酬利差分析

### 3.1 因子溢酬計算方法

因子溢酬 (Factor Premium) 的計算基於以下數學公式：

**單因子溢酬計算：**
```
Factor Premium = R_factor - R_market
```

其中：
- R_factor = 因子投資組合報酬率
- R_market = 市場基準報酬率

**多因子溢酬計算：**
```
R_i = α + β₁ * Market + β₂ * SMB + β₃ * HML + β₄ * MOM + ε_i
```

### 3.2 利差分析指標

Freddy 使用的關鍵利差指標：

1. **因子相關性矩陣**
2. **因子波動率比率**
3. **夏普比率差異**
4. **最大回撤比較**

```python
def factor_premium_analysis(factor_returns, market_returns):
    """
    因子溢酬分析
    """
    results = {}
    
    for factor_name, returns in factor_returns.items():
        # 因子溢酬（超額報酬）
        excess_returns = returns - market_returns
        
        # 計算各項指標
        results[factor_name] = {
            'annual_return': excess_returns.mean() * 252,
            'annual_vol': excess_returns.std() * np.sqrt(252),
            'sharpe_ratio': (excess_returns.mean() * 252) / (excess_returns.std() * np.sqrt(252)),
            'max_drawdown': calculate_max_drawdown(excess_returns),
            'correlation_with_market': np.corrcoef(returns, market_returns)[0, 1]
        }
    
    return pd.DataFrame(results)

def calculate_max_drawdown(returns):
    """
    計算最大回撤
    """
    cumulative = (1 + returns).cumprod()
    peak = cumulative.expanding().max()
    drawdown = (cumulative - peak) / peak
    return drawdown.min()
```

### 3.3 因子過熱/過冷判斷

Freddy 採用 Z-score 方法判斷因子過熱或過冷：

```python
def detect_factor_extremes(factor_returns, lookback_period=252, z_threshold=2.0):
    """
    檢測因子極端狀態（過熱/過冷）
    """
    results = {}
    
    for factor_name, returns in factor_returns.items():
        # 計算移動平均和標準差
        rolling_mean = returns.rolling(lookback_period).mean()
        rolling_std = returns.rolling(lookback_period).std()
        
        # 計算 Z-score
        z_score = (returns - rolling_mean) / rolling_std
        
        # 判斷極端狀態
        latest_z = z_score.iloc[-1]
        
        if latest_z > z_threshold:
            status = "過熱"
        elif latest_z < -z_threshold:
            status = "過冷"
        else:
            status = "正常"
            
        results[factor_name] = {
            'z_score': latest_z,
            'status': status,
            'signal': '賣出' if status == "過熱" else '買進' if status == "過冷" else '持有'
        }
    
    return results
```

## 4. 「去泡沫化」市場策略

### 4.1 市場「去泡沫化」的定義

**市場去泡沫化**是指市場從高估值、高成長預期向合理估值、基本面導向的轉變過程。Freddy 認為這是一種健康的市場調整機制，具有以下特徵：

1. **估值回歸均值**：高估值資產向歷史均值回歸
2. **資金再平衡**：資金從成長股流向價值股
3. **風險偏好降低**：投資者從追求高成長轉向追求穩定報酬
4. **基本面重要性提升**：企業基本面重新成為關鍵評估因素

### 4.2 泡沫識別方法

Freddy 採用多維度方法識別市場泡沫：

```python
def detect_market_bubbles(market_data):
    """
    識別市場泡沫
    """
    bubble_indicators = {}
    
    # 1. 估值指標
    pe_ratio = market_data['PE_Ratio']
    pe_percentile = pe_ratio.rank(pct=True).iloc[-1]
    
    # 2. 價格偏離基本面的程度
    eps_growth = market_data['EPS_Growth']
    price_growth = market_data['Price'].pct_change(252)
    deviation_ratio = price_growth / (eps_growth + 0.01)  # 避免0除
    
    # 3. 市場情緒指標
    volume_spike = market_data['Volume'] / market_data['Volume'].rolling(252).mean()
    
    # 4. 技術指標
    rsi = calculate_rsi(market_data['Price'])
    
    bubble_indicators = {
        'pe_percentile': pe_percentile,
        'deviation_ratio': deviation_ratio.iloc[-1],
        'volume_spike': volume_spike.iloc[-1],
        'rsi': rsi.iloc[-1],
        'bubble_score': (pe_percentile + 
                        min(deviation_ratio.iloc[-1] / 2, 1) + 
                        min(volume_spike.iloc[-1] / 3, 1) + 
                        max((rsi.iloc[-1] - 70) / 30, 0)) / 4
    }
    
    # 泡沫等級判斷
    if bubble_indicators['bubble_score'] > 0.8:
        bubble_level = "嚴重泡沫"
    elif bubble_indicators['bubble_score'] > 0.6:
        bubble_level = "中度泡沫"
    elif bubble_indicators['bubble_score'] > 0.4:
        bubble_level = "輕度泡沫"
    else:
        bubble_level = "無泡沫"
        
    bubble_indicators['bubble_level'] = bubble_level
    
    return bubble_indicators

def calculate_rsi(prices, period=14):
    """
    計算 RSI 指標
    """
    delta = prices.diff()
    gain = (delta.where(delta > 0, 0)).rolling(window=period).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(window=period).mean()
    rs = gain / loss
    rsi = 100 - (100 / (1 + rs))
    return rsi
```

### 4.3 Freddy 去泡沫化投資策略

Freddy 的去泡沫化策略核心原則：

1. **防守為主，攻擊為輔**：80% 防守配置，20% 進攻配置
2. **因子再平衡**：增加價值因子、品質因子權重
3. **流動性管理**：保持高流動性資產比例
4. **波段操作**：利用市場波動進行階段性操作

```python
class FreddyDebubblingStrategy:
    def __init__(self):
        self.defensive_allocation = {
            'bonds': 0.40,      # 債券 40%
            'value_etfs': 0.25,  # 價值型 ETF 25%
            'quality_etfs': 0.15, # 品質型 ETF 15%
            'cash': 0.20         # 現金 20%
        }
        
        self.offensive_allocation = {
            'growth_etfs': 0.10,  # 成長型 ETF 10%
            'momentum_etfs': 0.05, # 動能型 ETF 5%
            'special_situations': 0.05 # 特殊情況 5%
        }
    
    def adjust_for_bubble_conditions(self, bubble_level):
        """
        根據泡沫程度調整配置
        """
        if bubble_level == "嚴重泡沫":
            # 極度防守
            defensive_weights = {
                'bonds': 0.50,
                'value_etfs': 0.20,
                'quality_etfs': 0.20,
                'cash': 0.10
            }
            offensive_weights = {
                'growth_etfs': 0.00,
                'momentum_etfs': 0.00,
                'special_situations': 0.00
            }
            
        elif bubble_level == "中度泡沫":
            # 高度防守
            defensive_weights = self.defensive_allocation.copy()
            offensive_weights = {
                'growth_etfs': 0.05,
                'momentum_etfs': 0.02,
                'special_situations': 0.03
            }
            
        elif bubble_level == "輕度泡沫":
            # 中性防守
            defensive_weights = self.defensive_allocation.copy()
            offensive_weights = self.offensive_allocation.copy()
            
        else:
            # 正常配置
            defensive_weights = self.defensive_allocation.copy()
            offensive_weights = self.offensive_allocation.copy()
            
        return {**defensive_weights, **offensive_weights}
```

### 4.4 實證案例與績效

根據 Freddy 的歷史回測，去泡沫化策略在以下時期表現優異：

1. **2000年科技泡沫破裂**：策略跑贏大盤 25%
2. **2008年金融危機**：策略減損幅度比大盤少 40%
3. **2022年通膨危機**：策略跑贏大盤 18%

```python
def debubbling_backtest(market_data, start_date, end_date):
    """
    去泡沫化策略回測
    """
    # 模擬歷史數據
    dates = pd.date_range(start=start_date, end=end_date, freq='M')
    results = []
    
    for date in dates:
        # 計算當時的泡沫指標
        current_data = market_data[market_data.index <= date]
        if len(current_data) < 252:  # 需要足夠的歷史數據
            continue
            
        bubble_indicators = detect_market_bubbles(current_data)
        strategy = FreddyDebubblingStrategy()
        weights = strategy.adjust_for_bubble_conditions(bubble_indicators['bubble_level'])
        
        # 記錄結果
        results.append({
            'date': date,
            'bubble_level': bubble_indicators['bubble_level'],
            'bubble_score': bubble_indicators['bubble_score'],
            'bond_weight': weights['bonds'],
            'value_weight': weights['value_etfs'],
            'quality_weight': weights['quality_etfs'],
            'cash_weight': weights['cash']
        })
    
    return pd.DataFrame(results)
```

```python
    def daily_monitoring(self):
        """
        每日監控程序
        """
        # 1. 更新市場數據
        market_data = self.update_market_data()
        
        # 2. 計算動能指標
        momentum_analysis = self.momentum_monitor.analyze_momentum(market_data)
        
        # 3. 生成交易信號

        risk_metrics = self.calculate_risk_metrics(market_data)
        
        # 5. 生成監控報告
        report = self.generate_monitoring_report(
            momentum_analysis, trading_signals, risk_metrics
        )
        
        return report
    
    def generate_alerts(self, report):
        """
        生成重要警報
        """
        alerts = []
        
        # 檢查極端因子狀態
        for factor, signal in report['trading_signals'].items():
            if signal in ['強力買進', '賣出']:
                alerts.append({
                    'type': 'factor_signal',
                    'severity': 'high' if signal == '賣出' else 'medium',
                    'message': f'{factor} 因子出現 {signal} 信號'
                })
        
        # 檢查風險指標
        if report['risk_metrics']['portfolio_volatility'] > 0.15:
            alerts.append({
                'type': 'risk_alert',
                'severity': 'high',
                'message': '投資組合波動率超出警戒線'
            })
            
        return alerts
```

```python
def needs_rebalance(report):
    """
    判斷是否需要再平衡
    """
    # 檢查權重偏離程度
    current_weights = report['current_weights']
    target_weights = report['target_weights']
    
    max_deviation = max(abs(current_weights[w] - target_weights[w]) 
                       for w in current_weights.keys())
    
    return max_deviation > 0.05  # 5% 偏離閾值
```
