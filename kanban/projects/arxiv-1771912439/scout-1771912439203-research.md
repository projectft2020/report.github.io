## 論文摘要（中文）

本研究提出了一種名為「行為學習」（Behavior Learning，BL）的新型通用機器學習框架，其靈感來自行為科學。BL 框架能夠從數據中學習可解釋且可識別的優化結構，涵蓋範圍從單一優化問題到分層組合。該框架統一了預測性能、內在可解釋性和可識別性三大目標，廣泛適用於涉及優化的科學領域。

BL 通過參數化由內在可解釋的模組化區塊構成的組合效用函數，誘導出用於預測和生成的數據分佈。每個區塊都可以用符號形式表示為效用最大化問題（Utility Maximization Problem，UMP），這是行為科學中的基礎範式，也是優化的通用框架。BL 支援從單一 UMP 到分層組合的架構，後者用於建模分層優化結構。其平滑且單調的變體（IBL）保證了可識別性。

理論上，研究人員建立了 BL 的通用逼近性質，並分析了 IBL 的 M-估計性質。實驗證明，BL 展現了強大的預測性能、內在可解釋性以及對高維數據的可擴展性。代碼已在 GitHub 開源（MoonYLiang/Behavior-Learning），可透過 `pip install blnetwork` 安裝。

## BL 框架核心原理

### 1. 理論基礎：效用最大化問題（UMP）

BL 框架建立在行為科學中最基本的範式之一——效用最大化之上。該範式假設人類行為是通過解決效用最大化問題而產生的：

```
max_y∈Y U(x,y)
s.t. C(x,y) ≤ 0, T(x,y) = 0
```

其中：
- `U(x,y)`：效用函數，編碼代理人的內在偏好或目標
- `C(x,y)`：不等式約束，捕捉資源限制
- `T(x,y)`：等式約束，編碼內生信念一致性或外生守恆定律

**關鍵定理 2.1（UMP 的局部精確懲罰重構）：**
在溫和正則性條件下，UMP 可以重構為無約束的懲罰形式：

```
max_y λ₀·φ(U(x,y)) - λ₁ᵀ·ρ(C(x,y)) - λ₂ᵀ·ψ(T(x,y))
```

其中 `φ` 是嚴格增函數，`ρ(z)=max{z,0}`，`ψ(z)=|z|`。

**關鍵定理 2.2（UMP 的通用性）：**
任何形式的優化問題都等價於一個 UMP，這使得 BL 成為一個通用的建模框架。

### 2. 核心思想：組合效用函數

BL 的核心是構建一個組合效用函數 `BL(x,y)`，它由多個可解釋的模組化區塊 `B(x,y)` 組合而成。每個區塊提供單一 UMP 的懲罰公式，共同構成整體組合效用函數。

**基本區塊結構：**

```
B(x,y;θ) := λ₀ᵀ·φ(U_θU(x,y)) - λ₁ᵀ·ρ(C_θC(x,y)) - λ₂ᵀ·ψ(T_θT(x,y))
```

參數 `θ := (λ₀, λ₁, λ₂, θU, θC, θT)` 包含所有可學習參數。

### 3. 三種架構變體

BL 支援三種主要架構，從簡單到複雜：

#### (1) BL(Single)
- **結構：** 應用單一實例的 `B(x,y)`，無額外層
- **特點：** 可視為學習單一 UMP，提供最大的可解釋性
- **適用場景：** 簡單優化問題，需要高度可解釋性的場景

#### (2) BL(Shallow)
- **結構：** 使用 `B(x,y)` 作為基本模組構建淺層網絡
- **特點：** 引入一或兩個中間計算層
- **每層：** `𝔹_l` 堆疊多個並行的 `B_{l,i}` 區塊
- **適用場景：** 中等複雜度的優化結構

#### (3) BL(Deep)
- **結構：** 將 BL(Shallow) 擴展到超過兩層
- **特點：** 支援更豐富的 UMP 分層組合
- **適用場景：** 複雜的分層優化結構

### 4. 統一的數學形式

BL(Shallow) 和 BL(Deep) 可以用統一形式表示：

```
BL(x,y) := W_L · 𝔹_L(⋯𝔹_2(𝔹_1(x,y))⋯)
```

其中：
- `L ≤ 2`：淺層情況
- `L > 2`：深度情況
- `W_L`：最終的可學習仿射變換

## 技術實現細節

### 1. 數據分佈建模

BL 使用條件吉布斯分佈來建模數據：

```
p_τ(y|x;Θ) = exp(BL_Θ(x,y)/τ) / Z_τ(x;Θ)
```

其中：
- `τ > 0`：溫度參數，控制響應的隨機性
- 當 `τ → 0` 時，分佈收斂到 `arg max_y BL(x,y)` 的 Dirac 測度
- 這恢復了通過解組合 UMPs 獲得的確定性最佳響應

### 2. 學習目標

對於包含離散和連續分量的混合響應，採用組合損失函數：

**離散分量：** 交叉熵（Kullback & Leibler, 1951）
```
L_disc = 𝔼[-log p_τ(y^disc|x)]
```

**連續分量：** 去噪評分匹配（Vincent, 2011）
```
L_cont = 𝔼[‖∇_ỹ^cont log p_τ(ỹ^cont|x) + σ⁻²(ỹ^cont - y^cont)‖²]
```

**總損失函數：**
```
L(θ) = γ_d·L_disc + γ_c·L_cont
```

### 3. 函數實例化

默認實例化方式：

```
B(x,y) = λ₀ᵀ·tanh(p_u(x,y)) - λ₁ᵀ·ReLU(p_c(x,y)) - λ₂ᵀ·|p_t(x,y)|
```

其中：
- `tanh`：反映邊際效用遞減原理（Jevons, 2013）
- `ReLU`：引入約束違反的軟懲罰
- `|·|`：捕獲對稱偏差
- `p_u, p_c, p_t`：有界多項式特徵映射

### 4. 實現選項

#### (1) 多項式映射
- **BL(Single)：** 多項式映射結構可選
- **BL(Shallow/Deep)：** 每個 `B` 區塊使用仿射變換作為多項式映射
- 默認省略高階和交互項以提高計算效率

#### (2) 跳躍連接
- **密集跳躍連接：** DenseNet 風格（串聯）
- **殘差跳躍連接：** ResNet 風格（加法）
- **影響：** 提高表達效率，但會影響可解釋性

```bash
pip install blnetwork
```

## 可解釋性和可識別性的實現方式

### 1. 內在可解釋性

#### 符號化表示
每個 `B` 區塊都可以用符號形式表示為優化問題：
- `tanh` 項定義目標函數
- `ReLU` 項對應不等式約束
- 絕對值項對應等式約束

#### 分層解釋結構
- **BL(Single)：** 可直接表示為符號 UMP
- **更深架構：** 可解釋為 UMPs 的組合，每個區塊保持可解釋性
- **自底向上的解釋路徑：**
  ```
  原始輸入特徵 → 微觀層優化區塊 →
  宏觀層聚合或粗粒化行為構造 →
  宏觀層優化系統
  ```

#### 視覺化
多項式基礎確保與線性回歸相當的透明度：
- 可視化為計算圖（如圖 7）
- 每個輸入對每個 `B` 區塊的影響可通過組合路徑追蹤

#### 案例研究：波士頓房價數據集
- **BL(Single)：** 解釋為波士頓房市代表性買家的 UMP
- **效用項：** 房價中位數（MEDV）和房間數（RM）的主導作用
- **預算約束：** 低收入居民比例（LSTAT）反映隱性資源限制
- **信念項：** 犯罪率（CRIM）影響他人行為而非自身偏好

### 2. 可識別性（IBL）

#### 設計原理
為確保參數的唯一識別，IBL 施加更嚴格的結構約束：
- `φ^id` 和 `ρ^id` 嚴格單調增
- `ψ^id` 對 `|·|` 嚴格單調增
- 所有函數均為 C¹ 連續

#### 實例化
```
B^id(x,y) = λ₀ᵀ·tanh(p_u(x,y)) - λ₁ᵀ·softplus(p_c(x,y)) - λ₂ᵀ·(p_t(x,y))^⊙²
```

其中 `(·)^⊙²` 表示逐元素平方。

#### 理論保證

**定理 2.4（IBL 的可識別性）：**
在假設 B.1 下，IBL(Single)、IBL(Shallow) 和 IBL(Deep) 在參數商空間 `Θ̄` 中是可識別的。

**定理 2.5（損失可識別性）：**
- 若 `γ_c > 0`，在商空間 `Θ̄` 中有唯一最小化器
- 若 `γ_c = 0`，在尺度不變商空間 `Θ̃` 中有唯一最小化器

**定理 2.6（IBL 的一致性）：**
當樣本量 `n → ∞` 時，學習參數在機率上收斂到母體損失最小化器。若模型正確設定，估計器進一步收斂到真實參數。

**定理 2.7（IBL 的通用一致性）：**
即使在模型誤設定下，只要 IBL 架構容量隨樣本量增長，學習的預測分佈在 KL 意義下均勻收斂到真實條件分佈。

### 3. 統計性質

#### 漸近常態性
參數估計隨樣本量增加在分佈上收斂到常態分佈。

#### 漸近有效性
在額外正則性條件下，漸近方差達到有效的資訊界，證明了 IBL 的統計最優性。

## 應用案例分析

### 1. 標準預測任務

#### 數據集
在 10 個數據集上評估，涵蓋多種樣本量、特徵維度和科學領域：
- 波士頓房價數據集
- 其他分類和回歸數據集

#### 基線模型
比較 10 種基線模型，來自 5 個方法族：
- **神經網絡：** MLP
- **樹模型：** 決策樹（DT）
- **梯度提升：** LightGBM、XGBoost
- **貝葉斯方法：** Gaussian Process
- **線性模型：** 線性回歸

#### 結果
- **BL(Single) 和 BL(Shallow)：** 達到第一梯隊的預測性能
- **BL(Shallow)：** 超越 MLP，展示 BL 在不犧牲性能的情況下提供可解釋性
- **在可解釋模型中表現最佳**

### 2. 波士頓房價案例研究

#### BL(Single) 的符號形式
可解釋為代表性買家的 UMP，包含單一目標、不等式和等式項。

#### 效用項近似
```
p_u = -0.56·P² - 0.6·RM + 0.57·RM·P + R̃_u
    ≈ (1-P)(1+P-RM) + R̃_u
```

#### 關鍵發現
1. 房價中位數（MEDV）在所有項中主導
2. 房間數（RM）調節其邊際效應
3. 低收入居民比例（LSTAT）在預算約束中顯著
4. 犯罪率（CRIM）僅出現在信念項中

### 3. BL(Deep) 的分層解釋

#### BL[2,1] 架構（兩層）
- **第一層：** 識別兩個關鍵微觀層偏好類型
  - 經濟敏感型買家
  - 位置敏感型買家
- **第二層：** 將這兩個組件聚合為有效的代表性買家

#### BL(Deep) [5,3,1] 架構（三層）
- **第一層：** 恢復五種不同的微觀層房價偏好類型
- **第二層：** 識別三種宏觀層權衡類型
- **第三層：** 聚合為整體代表性買家

#### 科學一致性
- 與統計物理中的粗粒化原理（Kadanoff, 1966）一致
- 重構完整的微觀到宏觀優化層次
- 發現的偏好和權衡模式在古典經濟學文獻中有充分記錄

### 4. 高維輸入預測

#### 可擴展性
BL 在高維數據上展現良好的可擴展性：
- 參數量可控
- 運行時間合理
- 校準性能良好

#### 帕累托前沿下移
BL 在性能-可解釋性權衡中實現帕累托前沿的改善。

### 5. 約束執行測試：高維能量守恆

#### 實驗設置
在高維空間測試約束執行能力：
- 資料預處理
- 約束強制機制

#### 結果
BL 能夠有效執行約束條件，展示在科學建模中的應用潛力。

### 6. 廣泛應用領域

BL 不僅限於行為科學，適用於任何觀測結果是優化問題解的科學領域：

#### 經濟學
- 宏觀經濟學（Ramsey, 1928; Ljungqvist & Sargent, 2018）

## 與其他方法的比較

### 1. 可解釋性方法

#### 性能-可解釋性權衡
- **現狀：** 大多數方法在預測性能和可解釋性之間存在權衡
- **BL 的突破：** 統一高預測性能和內在可解釋性

- **現有方法的局限性：**
  - 與科學理論的對齊不足
  - 解釋的非唯一性（非可識別）
- **BL 的優勢：**
  - 基於優化問題的科學理論基礎
  - 保證可識別性（IBL）

### 2. 數據驅動逆向優化

#### 關係
BL 是一種通用的數據驅動逆向優化方法。

#### BL 的優勢
- 統一預測、解釋和生成
- 支援分層優化結構
- 提供理論保證

### 3. 基於能量的模型（EBMs）

#### 關係
BL 的組合效用函數類似於能量函數。

#### BL 的區別
- 明確的優化問題解釋
- 可解釋的模組化結構
- 可識別性保證

### 4. 其他可解釋機器學習方法

| 方法類別 | 代表 | 優點 | 局限性 | BL 的改進 |
|---------|------|------|--------|-----------|
| 線性模型 | 線性回歸、邏輯回歸 | 高度可解釋 | 表達能力有限 | 保持可解釋性，增強表達能力 |
| 樹模型 | 決策樹、隨機森林 | 直觀可解釋 | 深度增加時可解釮性下降 | 分層結構提供更好的可解釋性 |
| 神經網絡 | MLP、CNN、RNN | 強大表達能力 | 黑盒難解釋 | 模組化結構提供內在可解釋性 |
| 後驗可解釋方法 | SHAP、LIME | 可應用於任何模型 | 解釋可能不一致 | 內在可解釋性確保一致性 |

### 5. 與統計估計的比較

#### M-估計
- IBL 提供完整的 M-估計理論
- 包括一致性、漸近常態性、漸近有效性

#### 貝葉斯方法
- BL 提供頻率派的替代方案
- 在可識別性方面具有優勢

## 優缺點評估

### 優點

#### 1. 理論優勢
- **通用逼近性質：** BL 和 IBL 都具有通用逼近能力
- **可識別性：** IBL 保證參數的唯一識別
- **統計性質：** 完整的 M-估計理論支持
- **科學理論基礎：** 基於行為科學的 UMP 範式

#### 2. 實際優勢
- **預測性能：** 達到第一梯隊的預測性能
- **可解釋性：** 內在可解釋，支援符號化表示
- **可擴展性：** 支援高維數據
- **通用性：** 適用於多種科學領域

#### 3. 架構優勢
- **靈活性：** 三種架構變體（Single、Shallow、Deep）
- **模組化：** 可解釋的模組化區塊
- **分層能力：** 支援分層優化結構

#### 4. 實現優勢
- **易用性：** 簡單的 pip 安裝
- **開源：** 完整的開源實現
- **文檔：** 詳細的技術文檔

### 缺點和局限性

#### 1. 理論局限性
- **假設限制：** 某些理論假設可能在實踐中較難滿足
- **模型誤設定：** 雖然有通用一致性，但誤設定可能影響解釋性

#### 2. 計算複雜性
- **深度架構：** BL(Deep) 的計算成本較高
- **參數估計：** 可識別性約束可能增加優化難度

#### 3. 可解釋性的權衡
- **跳躍連接：** 雖然提高表達效率，但會影響可解釋性
- **仿射映射：** 用於計算效率時，符號粒度降低

#### 4. 實踐挑戰
- **超參數調優：** 溫度參數 τ、權重 γ_d, γ_c 需要調優
- **基礎函數選擇：** 多項式基礎的選擇影響性能和可解釋性
- **大規模應用：** 在超大型數據集上的擴展性有待驗證

#### 5. 科學應用的局限性
- **領域知識整合：** 如何更好地整合領域特定約束
- **不確定性量化：** 模型的不確定性量化能力有限
- **因果推理：** 本身不提供因果結構識別

### 3. 潛在風險

#### 解釋誤用風險
- 符號化解釋可能被過度簡化
- 分層解釋可能被誤解為因果關係

#### 統計風險
- 小樣本情況下可識別性可能不可靠
- 模型選擇偏差可能影響結果

## 結論

Behavior Learning (BL) 框架代表了可解釋機器學習領域的重要進展。通過將行為科學中的效用最大化範式與現代機器學習技術相結合，BL 成功地：

1. **統一了三大目標：** 預測性能、內在可解釋性和可識別性
2. **提供了科學基礎：** 基於優化問題的建模框架，與多種科學領域相容
3. **支援分層結構：** 從單一優化問題到複雜的分層組合
4. **建立了理論保證：** 通用逼近性、可識別性、統計一致性
5. **展示了實用性：** 在多個標準數據集上達到競爭性性能

BL 不僅是一個技術創新，更是一種範式轉變——從黑盒模型到可解釋的優化結構，從單純預測到科學理解。隨著進一步的理論發展和應用擴展，BL 有潛力成為科學建模和可解釋機器學習的重要工具。

然而，BL 仍處於發展初期，在計算效率、實踐應用和理論完整性方面仍有改進空間。未來的研究需要解決這些挑戰，並探索 BL 在更廣泛領域的應用。
