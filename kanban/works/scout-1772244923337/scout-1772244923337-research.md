## 研究摘要

本論文提出了一種基於圖形結構的自迴歸模型變量排序優化方法，通過學習底層數據的馬可夫隨機場（MRF）結構來構造優化的變量順序。研究以 Ising 模型為例，展示了結構感知的排序能夠顯著減少條件分佈的複雜度，從而提高生成樣本的保真度。數值實驗證明，與天真的變量排序相比，圖形感知的排序策略在鐵磁性和自旋玻璃模型上都能產生更高質量的樣本。

---

## 關鍵發現

### 1. **變量排序的關鍵作用** — 不同的變量順序會導致條件分佈的複雜度差異巨大，進而顯著影響自迴歸模型的採樣質量和準確性。良好的排序簡化條件分佈並減少誤差傳播，而糟糕的排序則會迫使模型學習不必要的複雜依賴關係。

### 2. **馬可夫隨機場結構的利用** — 通過學習底層圖形結構（MRF），可以利用馬可夫性質縮減條件集，將複雜的條件分佈簡化為僅依賴於父節點集合的簡單形式，從而顯著降低學習複雜度。

### 3. **父節點集合的概念** — 論文提出了父節點集合（parent set）的正式定義，通過圖形結構中的路徑連通性來確定每個變量的有效條件集，這是優化排序的理論基礎。

### 4. **對角線遍歷策略的優越性** — 在二維方格晶格上，提出的對角線遍歷策略比順序遍歷和棋盤格遍歷產生更低的採樣誤差，特別是在鐵磁性模型中優勢更為明顯。

### 5. **實驗驗證的廣泛性** — 研究在多種設置下驗證了方法的有效性，包括 5×5 和 10×10 方格晶格上的鐵磁性和自旋玻璃模型，以及來自 D-Wave 量子退火器的真實數據集。

## 詳細分析

### 一、問題背景與動機

#### 1.1 自迴歸模型的基本原理

自迴歸模型是現代 AI 架構中廣泛使用的強大分解方法，用於從學習的數據表示中產生精確樣本。其核心思想是通過祖先採樣（ancestral sampling）生成新樣本：

- 變量按照拓撲順序被訪問
- 每個變量 xᵢ 基於先前採樣的「父」變量 𝐱₋ᵢ 進行條件化
- 聯合分佈分解為：p(x) = ∏ᵢ p(xᵢ | 𝐱₋ᵢ)

這種方法的主要優點是精確採樣的計算成本為 O(N)，其中 N 是變量數量。這也是為什麼大語言模型（如 GPT-3、GPT-NeoX-20B）等應用廣泛採用自迴歸架構的原因。

#### 1.2 變量排序的重要性問題

在實際應用中，變量遍歷的排序很少被優化，通常是任意選擇或由數據的自然序列決定（例如文本中的單詞順序或圖像中的固定像素遍歷）。然而，**不同的排序可能導致條件分佈的複雜度差異巨大**：

- 良好的排序簡化條件分佈，減少誤差傳播
- 糟糕的排序迫使模型學習不必要的複雜依賴關係

#### 1.3 研究問題的提出

當聯合概率分佈以馬可夫隨機場（MRF）表示時，可以利用其結構構造優化的變量排序。一般情況下，條件 p(xᵢ | 𝐱₋ᵢ) 可能是一個複雜的分佈，但如果它依賴於所有先前採樣的變量。然而，利用馬可夫性質，有效的條件集可以縮減為底層無向圖形模型中鄰居節點的子集。

論文提出的核心問題是：**能否使用分佈 p(x) 的結構來選擇導致較低複雜度條件分佈的變量排序？**

### 二、理論框架與方法

#### 2.1 Ising 模型

研究聚焦於具有成對交互的二進制變量模型，即 Ising 模型。Ising 模型定義在無向圖 𝒢 = (𝒱, ℰ) 上，其中：

- 𝒱 = {1, ..., N} 是節點集
- ℰ ⊆ 𝒱 × 𝒱 是無向邊集
- xᵢ ∈ {-1, 1} 是與節點 i 相關的自旋
- 聯合概率分佈為：p(x) = (1/Z) exp(∑ᵢ∈𝒱 θᵢ xᵢ + ∑(i,j)∈ℰ θᵢ,ⱼ xᵢ xⱼ)

其中 θᵢ 和 θᵢ,ⱼ 分別是外磁場和成對交互的參數，Z 是配分函數（歸一化常數）。

#### 2.2 條件獨立性與父節點集合

**核心概念：父節點集合 Par(·)**

論文提出了父節點集合的正式定義，用於縮減條件集：

**定義 2.1：** 固定排列 σ = (σ(1), ..., σ(N)) 和無向強連通圖 𝒢 = (𝒱, ℰ)。

- Par(σ(1)) := ∅
- 對於每個索引 i ≥ 2，定義先前訪問節點集 V₋ᵢ := {σ(1), ..., σ(i-1)}
- σ(i) 的父節點集定義為：
  Par(σ(i)) := {σ(j) ∈ V₋ᵢ : 存在從 σ(i) 到 σ(j) 的路徑 π，使得 π ∩ (V₋ᵢ \ {σ(j)}) = ∅}

**直觀解釋：**
σ(j) ∈ Par(σ(i)) 當且僅當 σ(j) 在移除所有先前訪問節點（除了 σ(j)）後的誘導子圖中可從 σ(i) 到達。

**示例說明：**
考慮一個 5 節點圖形，排列 σ = (5, 1, 4, 2, 3)：
- Par(5) = ∅
- Par(1) = {5}（存在路徑 1-3-4-5）
- Par(4) = {1, 5}（存在路徑 4-3-1 和 4-5）
- Par(2) = {1, 4}
- Par(3) = {1, 4}

由於馬可夫性質，一般分解簡化為更簡單的形式：
p(x) = p(x₅) p(x₁|x₅) p(x₄|x₁,x₅) p(x₂|x₁,x₄) p(x₃|x₁,x₄)

#### 2.3 自迴歸分解中的縮減條件分佈

對於節點 k > 1，參數化條件分佈中的項數取決於其父節點集 Par(k) 的基數和交互階數。

設節點 k 有 dₖ := |Par(k)| 個父節點，並用最多 O 階的多項式交互參數化條件分佈。每個交互項的形式為 xₖ ∏ⱼ∈S xⱼ，其中 S ⊆ Par(k)。限制總階數最多為 O 意味著 |S| ≤ O-1。

涉及的交互項數為：
T = 1 + ∑ᵣ₌₁ᴼ⁻¹ (dₖ choose r)

其中 +1 對應於 S = ∅ 的單例交互項。

因此，條件分佈可以表示為：
p(xσ(i) | xPar(σ(i))) = (1/Zσ(i)) exp(xσ(i) ∑ₛ∈Sσ(i)⁽ᴼ⁾ θ̃σ(i),S ∏ⱼ∈S xⱼ)

#### 2.4 條件分佈的學習

論文使用 GRISE（Vuffray et al., 2020）的無約束版本来學習離散無向圖形模型的高階交互的條件概率。給定 m 個獨立同分佈（i.i.d.）樣本，節點 i 的 GRISE 估計通過求解以下最小化問題獲得：

min θ̃ᵢ,{θ̃ᵢ,S}ₛ∈Sᵢ⁽ᴼ⁾ (1/m) ∑ₗ₌₁ᵐ exp(-θ̃ᵢ xᵢ⁽ˡ⁾ - xᵢ⁽ˡ⁾ ∑ₛ∈Sᵢ⁽ᴼ⁾ θ̃ᵢ,S ∏ⱼ∈S xⱼ⁽ˡ⁾)

#### 2.5 優化排序的選擇

**馬可夫隨機場結構的學習：**

當只有數據可用且圖形 𝒢 = (𝒱, ℰ) 未知時，可以使用現有方法學習其結構。論文使用 RISE（Regularized Interaction Screening Estimator）（Vuffray et al., 2016）方法。

**選擇優化變量排序的標準：**

給定排列 σ，使用上述過程基於 MRF 邊集 ℰ 定義父節點集。設：

- d := maxₖ dₖ = maxₖ |Par(k)| 是自迴歸分解中父節點集的最大基數
- K 是分解中具有基數 d 的條件分佈的最大數量

根據 Vuffray et al. (2020) 的定理 2，這 K 個條件是最難學習的：保證給定類型的條件分佈固定誤差所需的樣本數量隨 d 指數增長。

**假設：** 給定固定數量的訓練樣本 m，比較兩個排列：

(i) 導致分解具有較小 d 的排列
(ii) 如果兩個排列的 d 相同，則具有較小 K 的分解應該產生更準確學習的條件分佈，從而產生更高質量的自迴歸模型

**額外優化：** 選擇排序可以通過在低階條件分佈中選擇強相關節點及其父節點，並利用最大階條件分佈中的空間相關衰減來進一步略微優化。

### 三、實驗結果

#### 3.1 精確訓練樣本：5×5 方格晶格

考慮 5×5 方格晶格（N=25），因為可以通過暴力計算產生任何 Ising 模型的 i.i.d. 樣本。研究了三種排序：

1. **順序序列：** 節點按行順序選擇。基準選擇涉及 K = O(L²) 個最大基數 d = L 的條件分佈。

2. **棋盤格序列：** 節點按棋盤格模式選擇。仍有 K = O(L²) 個最大基數 d = L 的條件分佈，但具有最大基數的項大約少一半，且每個條件分佈中的項之間相關性更小。

3. **對角線遍歷：** 為方格晶格設計的最佳優化變量排序。主要思想是通過直接對主對角線進行條件化使晶格的兩部分條件獨立。此策略不可避免地仍有最大基數 d = L 的項，但比前兩種策略少得多。

**結果：**
- 對於固定階（O=6）的模型學習，隨著訓練樣本數 M_l 的增加，採樣誤差 ε 減少
- 對角線遍歷（序列 3）在鐵磁性和自旋玻璃模型中都產生最低的採樣誤差
- 改進在鐵磁性情況下特別明顯
- 自旋玻璃模型對變量排序的選擇不太敏感，可能由於受挫自旋玻璃模型固有的採樣困難

**模型階數的影響：**
- 從真實分佈學習條件分佈（消除有限訓練樣本的誤差）
- 模型階數 O ∈ {2, 4, 6}
- 在所有情況下，序列 3 產生最低的採樣誤差
- 增加條件分佈學習中的模型階數並沒有導致顯著的性能改善，這是由於鐵磁性模型中高階項的值和影響較小

#### 3.2 高質量訓練樣本：10×10 鐵磁性模型

考慮更大的系統（L=10，N=100），使用 Gibbs 採樣生成數據。研究了模型階數和有限訓練樣本的結合效果。

**數據生成：**
- 使用兩個獨立的馬可夫鏈，分別初始化為全 +1 和全 -1 配置
- 捨棄初始預熱期，組合來自每條鏈的等量樣本
- 訓練數據集大小 M_l ∈ {0.5×10⁵, 2.5×10⁵, 5×10⁵}

**結果：**
- 模型階數在較大的系統中起著更明顯的作用
- 與 5×5 晶格相比，增加的系統規模導致更複雜的條件依賴關係，使條件模型的可表達性更為重要
- 較低階模型（O=2）的採樣誤差比較高階模型（O=4）更快飽和
- 較高階模型繼續從額外訓練數據中受益
- 在兩種模型階數下，序列 3 始終產生最低的採樣誤差

#### 3.3 真實數據用例：D-Wave 數據集

使用來自 D-Wave 2X 量子退火器的數據集評估變量排列對採樣算法的效果。

**數據集特點：**
- 62 個量子位
- 排列在非方形晶格上（大約 9×9 晶格，每個節點被一個稀疏連接的 4×2 子圖替換）
- 一些量子位損壞（連接性不規則）
- 參數均勻取自 [-2.0, -0.25] ∪ [0.25, 2.0]
- 對應於自旋玻璃模型

**方法：**
1. 首先重建底層 MRF 圖形拓撲
2. 基於恢復的邊權重，考慮兩種採樣序列：順序（基準）和交叉順序（優化）
3. 固定模型階數 O=3
4. 生成樣本數 M_s ∈ {1×10⁵, 2.5×10⁵, 5×10⁵, 10×10⁵}

**結果：**
- 採樣誤差隨 M_s 增加而減少，如預期
- 對變量排序選擇的敏感度較小（與合成實驗中的自旋玻璃模型一致）
- 這種行為與 D-Wave 數據集的強無序性質一致
- 結構感知的交叉順序始終顯示出對天真排序的優勢

1. **自旋玻璃的敏感性：**
   - 在強無序系統（如自旋玻璃）中效果較為有限

   - 最優排序策略基於規則晶格結構

1. **圖形學習的誤差傳播：**
   - 如果圖形結構學習不準確，優化排序的效果可能受限
   - 需要魯棒的圖形學習方法

2. **計算複雜度：**
   - 優化排序的計算成本可能在大型系統中成為瓶頸
   - 需要高效的排序算法

3. **通用性：**
   - 不同應用可能需要不同的排序策略
   - 缺乏通用的自動化排序方法

### 八、技術細節補充

#### 8.1 條件分佈的參數估計

對於節點 i，給定父節點集 Par(i)，條件分佈的參數估計涉及：

1. **設計矩陣：** 為每個樣本構造特徵向量
2. **優化目標：** 最小化經驗風險（通常使用最大似然）
3. **正則化：** 可能需要正則化以防止過擬合

#### 8.2 採樣誤差的計算

採樣誤差 ε 定義為經驗分佈 p̂ 和真實分佈 p 的前兩矩的差異：

ε = (‖E_{p̂}[x] - E_p[x]‖₂ + ‖Cov_{p̂}(x) - Cov_p(x)‖_F)^{1/2}

其中 ‖·‖₂ 是 L₂ 範數，‖·‖_F 是 Frobenius 範數。

#### 8.3 圖形學習方法

論文使用的 RISE 方法（Vuffray et al., 2016）是：

1. **互作用篩選：** 通過統計檢驗識別顯著的交互
2. **正則化：** 使用 L1 正則化選擇稀疏圖形
3. **一致性：** 在樣本數趨於無窮時確保一致性

### 九、相關工作對比

#### 9.1 與 Any-Order Autoregressive Models 的比較

- **Any-Order ARMs**（Germain et al., 2015a）：通過訓練支持多個排列的單個模型來減少對固定排序的敏感性
- **本文方法**：顯式利用科學數據底層的馬可夫隨機場結構構造最小化條件複雜度的排序

#### 9.2 與圖生成模型的比較

- **GraphRNN**（You et al., 2018）和 **GraphAF**（Shi et al., 2020）：依賴啟發式遍歷策略（如廣度優先或深度優先搜索）
- **本文方法**：基於底層圖形結構的系統性優化

#### 9.3 與 Bayesian 網絡結構學習的比較

- **Bayesian 網絡結構學習**（Kitson and Constantinou, 2024）：研究變量排序對學習圖形、模型複雜度和預測性能的影響
- **本文方法**：專注於自迴歸採樣而非結構學習

## 結論

本論文成功展示了變量排序在自迴歸模型中的關鍵作用，並提出了一種基於圖形結構的系統性優化方法。主要貢獻包括：

1. **理論框架：** 提出了父節點集合的概念和優化排序的量化標準
2. **實踐方法：** 開發了對角線遍歷等具體排序策略
3. **廣泛驗證：** 在多種設置下驗證了方法的有效性
4. **開源實現：** 提供了可複現的代碼

雖然研究主要在小型系統上進行，但結果清楚地表明變量排序對自迴歸採樣質量的系統性影響，這些影響預期在大型系統中會更加顯著。

方法的應用價值體現在統計物理、圖像生成、量子計算等多個領域，未來的擴展方向包括與神經網絡的結合、連續變量的擴展、以及更大規模系統的驗證。
