## 摘要

本論文深入研究平滑啟動函數（smooth activation functions）在神經網絡近似和統計學習中的理論優勢。作者證明了在恆定深度下，配備平滑啟動函數的神經網絡能夠實現「平滑適應性」（smoothness adaptivity）——僅通過增加網絡寬度即可達到極小化最優的近似誤差和估計誤差率（至多對數因子）。相比之下，非平滑啟動函數（如 ReLU）的平滑適應性在根本上是受限於深度的。這項研究揭示了啟動函數平滑性是實現 Sobolev 函數類最優速率的基本機制，與深度互為補充。

**論文來源:** [arXiv:2602.19691](https://arxiv.org/abs/2602.19691)  
**作者:** Yuhao Liu, Zilin Wang, Lei Wu, Shaobo Zhang（清華大學、北京大學）  
**發布時間:** 2026年2月28日（v2版本）

## 核心發現

### 1. **平滑啟動函數的平滑適應性** ⭐⭐⭐
- **發現:** 恆定深度（深度為6或7，取決於度量標準）的神經網絡，配備平滑啟動函數，能夠對任意平滑度 s > 0 的 Sobolev 空間函數實現最優近似率 O(N^(-s/d))
- **技術細節:** 網絡寬度規模為 M_ε ≲ ε^(-d/(2s))，參數範數被多項式控制為 B_ε ≲ ε^(-max{d^2/(2s)+d, d/s+2, ...})
- **意義:** 證明平滑啟動函數允許固定深度網絡適應目標函數的任意高階平滑度，無需增加深度
- **來源:** Theorem 4.1, Section 4

### 2. **非平滑啟動函數的深度瓶頸** ⭐⭐⭐
- **發現:** 恆定深度 ReLU 網絡的近似誤差下界為 Ω(N^(-min{L-1,s}))，當 s > L-1 時，速率飽和於 N^(-(L-1))
- **技術細節:** 對固定深度 L ≥ 2，平滑度參數 s > 0，存在常數 C_s,L 使得近似誤差下界為 C_s,L (M log M)^(-2 min{L-1,s})
- **意義:** 揭示了平滑和非平滑啟動函數之間的明確區分——ReLU 網絡在固定深度下存在本質性的平滑度上限
- **來源:** Proposition 6.1, Section 6

### 3. **最優估計率無需稀疏性約束** ⭐⭐
- **發現:** 配備平滑啟動函數的恆定深度網絡在經驗風險最小化（ERM）下，達到極小化最優估計率 O(n^(-2s/(2s+d)))（至多對數因子）
- **技術細節:** 網絡寬度 M_n ≂ n^(d/(4s+2d))，參數範數 B_n ≂ n^(max{d/2, 1, ...})
- **意義:** 成功移除了先前工作中必需的 ℓ^0-稀疏性約束，使 ERM 在實踐中可行
- **來源:** Theorem 5.1, Section 5

### 4. **多尺度近似框架** ⭐⭐
- **發現:** 提出一種新的多尺度近似框架，用於表示分段常數函數，避免稀疏性約束的需求
- **技術細節:** 使用粗細網格分割的分層結構，將 K^(2d) 個細化單元的分段常數函數用寬度 O(K^d) 的恆定深度網絡表示
- **意義:** 關鍵技術突破，實現了在不增加深度情況下控制網絡複雜度
- **來源:** Appendix B.4, Section 7.1

### 5. **數值實驗支持** ⭐
- **發現:** 使用隨機傅里葉特徵生成的平滑目標函數進行實驗，結果顯示平滑啟動函數（tanh、GELU）的泛化誤差衰減速率快於 ReLU
- **技術細節:** 兩層神經網絡，全批次 Adam 優化器，每個啟動函數調整學習率和 ℓ^2-正則化超參數
- **意義:** 實驗證據支持理論結果，表明平滑啟動函數在學習平滑函數時具有樣本效率優勢
- **來源:** Section 6, Appendix D.2

## 詳細分析

### 研究背景與動機

現代深度學習中，平滑啟動函數（如 GELU、SiLU、SwiGLU）被廣泛採用，但其相對於非平滑啟動函數（如 ReLU）的理論優勢尚未被充分理解。這項研究旨在回答一個基本問題：

> **平滑啟動函數相對於其非平滑對應物的理論優勢是什麼？**

這一問題的重要性體現在：
1. **實踐趨勢:** 平滑啟動函數已成為大型語言模型（如 GPT、LLaMA、DeepSeek）和現代架構的標準組件
2. **理論空白:** 大部分深度學習理論聚焦於非平滑啟動函數（特別是 ReLU）
3. **科學計算需求:** 在神經網絡求解 PDE 等應用中，高階導數是必需的

### 核心理論貢獻

#### 1. 近似理論：平滑性作為深度的替代機制

**主要結果：** 定理 4.1（L² 近似）和定理 4.4（L^∞ 近似）

- **L² 近似:** 對任意 f* ∈ W^(s,∞)([0,1]^d) 且範數 ≤ 1，對任意 ε ∈ (0,1)，存在恆定深度神經網絡 g ∈ H^(φ,L)(d,1,M_ε,B_ε) 使得 ||g - f*||_L²([0,1]^d) ≤ ε
  - 深度 L = 6
  - 寬度 M_ε ≲ (1/ε)^(d/(2s))
  - 參數範數 B_ε ≲ (1/ε)^(max{d²/(2s)+d, d/s+2, ...})

- **L^∞ 近似:** 深度 L = 7，寬度和範數界類似，但保證 ||g - f*||_L^∞([0,1]^d) ≤ ε

**技術創新：**
1. **多尺度分割構造（Appendix B.1）:** 創建粗細網格分層結構，允許高效表示分段常數函數
2. **Bramble-Hilbert 引理（Appendix B.2）:** 用於多項式近似誤差分析
3. **單項式近似（Appendix B.3）:** 通過有限差分近似導數實現
4. **加權疊加原理（Appendix B.8）:** 將局部近似保證提升到全局 L^∞ 界

#### 2. 學習理論：無稀疏性約束的最優風險

**主要結果：** 定理 5.1（風險界）

給定 i.i.d. 樣本 {(x_i, y_i)}_i=1^n，其中 y_i = f*(x_i) + ξ_i，ξ_i ~ N(0,σ²)：
- 經驗風險最小化（ERM）在 H^(φ,7)(d,1,M_n,B_n) 上執行
- 選擇 L = 7, M_n ≂ n^(d/(4s+2d)), B_n ≂ n^(max{d/2, 1, ...}), F = 2
- 期望風險界：E[||T_F \hat{f}_n - f*||_L²(ρ)²] ≲ n^(-2s/(2s+d)) log n

**關鍵洞察：**
- L^∞ 近似結果（定理 4.4）確保了分布無關（distribution-free）保證
- 不需要對輸入分布 ρ 施加額外假設（如密度有界）
- 構造性近似允許精確控制假設類的複雜度

#### 3. 非平滑啟動函數的深度限制

**主要結果：** 命題 6.1（恆定深度 ReLU 網絡的近似下界）

對固定深度 L ≥ 2 和平滑度 s > 0：
- 存在常數 C_s,L 使得對所有 M ≥ 2：
  sup_{||f*||_W^(s,∞)([0,1]) ≤ 1} inf_{g ∈ H^(ReLU,L)(1,1,M,∞)} ||g - f*||_L²([0,1]) ≥ C_s,L (M log M)^(-2 min{L-1,s})
- 轉換為參數數 N ≍ M² 後，近似率下界為 N^(-min{L-1,s})
- 當 s > L-1 時，速率飽和於 N^(-(L-1))

**直觀理解：**
- ReLU 網絡在固定深度下表示分段線性函數
- 線性區域數量的增長受寬度 M 的多項式約束，多項式次數由 L 決定
- 因此有效近似階數受到根本性限制，無法充分利用高階平滑性

**對比：** 允許深度隨參數數或平滑度增長時，ReLU 網絡可以恢復平滑適應性（如 Yarotsky, 2017）

### 技術方法論

#### 多尺度近似框架（核心技術）

**問題：** 如何用恆定深度網絡高效表示 K^(2d) 個細化單元的分段常數函數？

**傳統方法：**
- 直接分配每個單元一個神經元 → 寬度 O(K^(2d))
- 參數數過大，無法達到最優速率

**創新解決方案：**
1. **分層分解:** 將細化網格分為粗網格和子網格
2. **結構重寫:** 將 K^(2d) 單元的分段常數函數表示為：
   - 2^K 個粗網格分段常數函數的加權和
   - 乘以相應的細化單元指示函數
3. **寬度控制:** 每個分量可用寬度 O(K^d) 的網絡近似 → 整體寬度 O(K^d)

**結果：** 寬度從 O(K^(2d)) 降低到 O(K^d)，實現了最優近似率而無需 ℓ^0-稀疏性約束

#### 啟動函數假設

**假設 3.1（平滑性）:** φ 無限可微且不是多項式

**假設 3.2（Lipschitz 連續性）:** |φ(x) - φ(y)| ≤ ||φ||_Lip |x - y|

**假設 3.3（結構性質）:**
- **Heaviside-like:** 存在 C₁ > 0 使得 |φ(t) - H(t)| ≤ C₁ min{1, 1/|t|}
- **ReLU-like:** 存在 C₂ > 0 使得 |φ(t) - ReLU(t)| ≤ C₂

**實例：**
- **Heaviside-like:** sigmoid, φ_tanh(·) = (1/2)(1 + tanh(·))
- **ReLU-like:** GELU, SiLU
- 見 Appendix A.2 的詳細驗證

**維數擴展性：**

### 2. **實踐局限性**

**優化考量：**
- 理論結果未考慮優化難度
- 平滑啟動函數的梯度飽和問題可能影響訓練動力學
- 實踐中，優化算法和超參數調整會顯著影響最終性能

**假設分布：**
- 理論假設目標函數屬於特定 Sobolev 空間且範數有界
- 實際數據分布可能不滿足這些假設

### 3. **實驗驗證的局限性**

**實驗規模：**
- 僅提供有限的數值實驗支持
- 實驗規模相對較小（兩層網絡，合成數據）
- 缺乏在大規模真實數據集上的驗證

**泛化下界：**
- 未提供恆定深度非平滑啟動函數網絡的嚴格泛化下界
- 主要依賴實驗證據支持理論分離

## 技術細節深入

### 多尺度近似框架的數學細節

**問題設定：** 用恆定深度網絡表示分段常數函數
C(x) = Σ_{i∈[K]^d} Σ_{j∈[K]^d} c_{i,j} 𝟙_Ω_{i,j}^K(x)

其中 {Ω_{i,j}^K} 是細化分割，{Ω_i^K} 是粗分割。

**多尺度分解：**
1. 對每個粗單元 Ω_i^K，有 K^d 個細化子單元 {Ω_{i,j}^K}_{j∈[K]^d}
2. 定義參考點 a_i^K（如左下角）
3. 重寫為：
   C(x) = Σ_{j∈[K]^d} (Σ_{i∈[K]^d} c_{i,j} 𝟙_Ω_i^K(x)) 𝟙_Ω_{1,j}^K(x - Σ_{i∈[K]^d} a_i^K 𝟙_Ω_i^K(x))

**構造步驟：**
1. 用寬度 O(K^d) 的網絡近似函數 Σ_{i∈[K]^d} c_{i,j} 𝟙_Ω_i^K(x)（對每個 j）
2. 用寬度 O(K^d) 的網絡近似函數 Σ_{i∈[K]^d} a_i^K 𝟙_Ω_i^K(x)
3. 用寬度 O(K^d) 的網絡近似 K^d 個指示函數 𝟙_Ω_{1,j}^K(·)
4. 通過加法和乘法組合這些分量

**複雜度分析：**
- 每個分量的寬度：O(K^d)
- 總寬度：O(K^d)（因為並行構造）
- 總參數數：O(K^(2d)) = O(1/ε^(d/s)) → 最優

### 加權疊加原理（L^∞ 近似）

**目標：** 將 L^∞ 近似誤差控制到整個域 [0,1]^d

**構造：**
1. 構造 2^d 個網絡 {g_v}_{v∈[2]^d}，在內部區域 Ω_int^(K,v)(δ) 上實現近似精度 ε
2. 構造 2^d 個加權函數 {w_v}_{v∈[2]^d}，滿足：
   - 分割單位性：Σ_{v∈[2]^d} w_v(x) = 1 對 x ∈ [0,1]^d
   - 局部消失性：w_v(x) = 0 對 x ∈ Ω_band^(K,v)(δ)
3. 近似為：g(x) = Σ_{v∈[2]^d} g_v(x) w_v(x)

**誤差控制：**
- 在 Ω_int^(K,v)(δ) 上：|g_v - f*| ≤ ε，w_v 有界 → 誤差受控
- 在 Ω_band^(K,v)(δ) 上：w_v = 0 → 誤差消失
- 由於分割單位性，全域誤差 ≤ ε

**技術挑戰：** 需要額外一層實現乘法 g_v w_v，因此深度從 6 增加到 7

### 覆蓋數分析（泛化界）

**目標：** 計算網絡類 H^(φ,7)(d,1,M_ε,B_ε) 的對數覆蓋數

**結果：**
log N(τ, H^(φ,7)(d,1,M_ε,B_ε), ||·||_∞) ≲ (1/ε)^(d/s) (log(1/τ) + log(1/ε))

**應用：** 使用標準統計學習理論（Lemma C.1）得到：
E[||T_F \hat{f}_n - f*||_L²(ρ)²] ≲ ε² + (1/n)(1/ε)^(d/s) (log(1/τ) + log(1/ε)) + τ

**優化：** 選擇 τ = ε² 得到最優平衡

## 平滑啟動函數類型詳解

### 1. **GELU (Gaussian Error Linear Unit)**
- **定義:** GELU(x) = x · Φ(x)，其中 Φ(x) 是標準正態分佈的 CDF
- **性質:** 
  - 平滑且可微
  - 保留了 ReLU 的非線性特徵
  - 對負值有平滑過渡
- **應用:** BERT、GPT 等大型語言模型
- **理論依據:** 概率啟動，建模神經元激活的隨機行為

### 2. **SiLU (Sigmoid Linear Unit / Swish)**
- **定義:** SiLU(x) = x · sigmoid(x)
- **性質:**
  - 平滑且單調
  - 在負值區域非零，允許梯度流動
  - 非自門控特性
- **應用:** GPT-2、一些 Transformer 模型
- **歷史:** 最初稱為 Swish，通過神經架構搜索發現

### 3. **SwiGLU**
- **定義:** SwiGLU(x) = (x · W) · SiLU(x · V) · W'
- **性質:**
  - 門控機制
  - 增強表達能力
  - 計算成本較高
- **應用:** LLaMA、一些現代大型語言模型
- **注意:** 尾行為為二次增長，被假設 3.2 排除（但可擴展）

### 4. **Tanh**
- **定義:** tanh(x) = (e^x - e^(-x)) / (e^x + e^(-x))
- **性質:**
  - 古典平滑啟動函數
  - Heaviside-like（與單位階躍函數相似）
  - 有界輸出
- **應用:** 早期神經網絡、科學計算
- **限制:** 梯度飽和問題在深度網絡中較嚴重

### 5. **Sigmoid**
- **定義:** sigmoid(x) = 1 / (1 + e^(-x))
- **性質:**
  - Heaviside-like
  - 古典啟動函數
  - 梯度飽和問題
- **應用:** 歷史意義，現在較少使用
- **關聯:** 概率解釋（將輸入映射到 [0,1]）

## 與相關領域的聯繫

### 1. **近似理論**

**經典結果：**
- DeVore et al. (1989): 神經網絡的近似誤差界
- Mhaskar (1996), Pinkus (1999): 平滑啟動函數的近似理論
- Barron's theorem (1993): 維度無關的近似速率

**本研究貢獻：**
- 在顯式複雜度控制下分析平滑啟動函數
- 連接近似理論和統計學習理論
- 提供建構性證明

### 2. **深度學習理論**

**ReLU 網絡理論：**
- Yarotsky (2017): 深度 ReLU 網絡的最優近似
- Schmidt-Hieber (2020): ReLU 網絡的學習理論
- Telgarsky (2016), Vardi & Shamir (2020): 深度與表達能力

**本研究貢獻：**
- 揭示平滑性作為深度的替代機制
- 證明恆定深度即可實現最優速率
- 擴展到平滑啟動函數

### 3. **統計學習理論**

**極小化最優速率：**
- Stone (1982), Tsybakov (2009): Sobolev 空間的極小化最優速率
- 覆蓋數和 Rademacher 複雜度標準工具

**本研究貢獻：**
- 在無 ℓ^0 稀疏性約束下實現極小化最優速率
- 分佈無關的保證（通過 L^∞ 近似）
- ERM 的實踐可行性

### 4. **優化動力學**

**梯度飽和：**
- Hochreiter (1991), Bengio et al. (1994): 梯度消失問題
- ReLU 的優勢：避免梯度飽和

**平滑啟動函數的挑戰：**
- 平滑啟動函數可能面臨梯度飽和
- 但現代設計（GELU、SiLU）通過非線性增長緩解此問題

**本研究未涉及：**
- 理論結果未考慮優化難度
- 實踐中需結合優化動力學分析

## 數學符號與定義總結

### 基本符號
- ℕ = {0, 1, 2, ...}
- ℕ^d = {(α₁, ..., α_d): α_i ∈ ℕ, i = 1, ..., d}
- [K] = {1, 2, ..., K}
- 多指標 α = (α₁, ..., α_d), |α| = Σ_i α_i
- f(x) ≲ g(x) 表示 f(x) = O(g(x))
- f(x) = Õ(g(x)) 抑制對數因子
- f(x) ≂ g(x) 表示 f(x) ≲ g(x) 且 g(x) ≲ f(x)

### 函數空間
- Ω = [0,1]^d
- W^(s,∞)(Ω)（整數階）: L^∞ 函數，其所有弱導數 D^αu ∈ L^∞ 對 |α| ≤ s
- W^(s,∞)(Ω)（分數階）: s = m + ζ, m = ⌊s⌋, ζ ∈ (0,1)
  - W^(s,∞)(Ω) = {u ∈ W^(m,∞)(Ω): [u]_(C^(m,ζ)(Ω)) < ∞}
  - Hölder 半範數 [u]_(C^(m,ζ)(Ω)) = max_{|α|=m} sup_{x≠y∈Ω} |D^αu(x) - D^αu(y)| / ||x-y||₂^ζ

### 神經網絡類
- H^(φ,L)(d_in, d_out, M, B): 全連接網絡類
  - 深度 L ≥ 1, 寬度 M ≥ 1, 參數界 B > 0
  - 啟動函數 φ: ℝ → ℝ，逐元應用
  - 參數集 Θ = {(W_ℓ, b_ℓ)_(ℓ=1)^L: ...}
  - ||W_ℓ||_(∞,∞) ≤ B, ||b_ℓ||_∞ ≤ B

### 學習設定
- 樣本: {(x_i, y_i)}_i=1^n
- 模型: y_i = f*(x_i) + ξ_i
- 輸入分布: x_i ~ ρ (支持在 [0,1]^d)
- 噪聲: ξ_i ~ N(0, σ²), 獨立於 x_i
- 截斷算子: (T_F f)(x) = sign(f(x))F 如果 |f(x)| > F，否則 f(x)

## 關鍵公式總結

### 近似率
**L² 近似（定理 4.1）:**
- 寬度: M_ε ≲ ε^(-d/(2s))
- 參數數: N = O(M_ε²) = O(ε^(-d/s))
- 近似率: ||g - f*||_L² ≲ N^(-s/d)

**L^∞ 近似（定理 4.4）:**
- 深度: L = 7（相比 L² 的 L = 6）
- 其他參數規模類似

### 學習率
**風險界（定理 5.1）:**
- 寬度: M_n ≂ n^(d/(4s+2d))
- 參數範數: B_n ≂ n^(max{d/2, 1, (2s+d+4)/(2(2s+d)), s⌈s⌉/(2s+d), (4s+6)/(2s+d)})
- 風險界: E[||T_F \hat{f}_n - f*||_L²(ρ)²] ≲ n^(-2s/(2s+d)) log n
- 極小化最優速率: O(n^(-2s/(2s+d)))

### 下界
**ReLU 網絡（命題 6.1）:**
- 近似下界: sup_{||f*||_W^(s,∞) ≤ 1} inf_{g ∈ H^(ReLU,L)} ||g - f*||_L² ≥ C_s,L (M log M)^(-2 min{L-1,s})
- 參數化後: ≳ N^(-min{L-1,s})
- 飽和: 當 s > L-1 時，速率飽和於 N^(-(L-1))
