### 2. Theoretical Foundations

#### 2.1 Mathematical Formalization

**Definition 1: Normalized Gating Function Space**
The space of normalized gating functions $G_1$ is defined as the set of functions that satisfy:
- $\sum_{i=1}^K g_i(x) = 1$ for all inputs $x$
- $g_i(x) \geq 0$ for all $i$ and $x$
- Where $K$ is the number of expert models

**Minimax Game Formulation**
The problem is formulated as:
$$\min_{g \in G_1} \max_{\lambda \in \Lambda} \mathcal{L}(g, \lambda)$$

Where:
- $\mathcal{L}(g, \lambda)$ is the loss function
- $\lambda$ represents the worst-case data mixture
- $g$ is the gating function to be optimized

#### 2.2 Key Theoretical Results

**Theorem 1: Existence of Robust Gate**
Using Kakutani's fixed-point theorem, the authors prove the existence of a single robust gate that minimizes divergence to the worst-case data mixture.

**Theorem 2: Generalization Bounds**
Modularity acts as a strong regularizer, with generalization bounds scaling with the lightweight gate's complexity rather than the entire model complexity.

**Theorem 3: Performance Gap Characterization**
The modular approach can theoretically outperform models retrained on aggregate data, with the performance gap characterized by the Jensen-Shannon Divergence between data distributions.

### 6. Algorithm Descriptions

#### 6.1 Stochastic Primal-Dual Algorithm
The paper introduces a scalable algorithm for training the robust gating mechanism:

**Algorithm: Stochastic Primal-Dual Optimization**
1. **Input**: Pre-trained experts $E_1, ..., E_K$, training data $\mathcal{D}$
2. **Initialize**: Gating parameters $\theta_g$
3. **Repeat** until convergence:
   a. Sample batch $B$ from $\mathcal{D}$
   b. Compute expert outputs $h_i = E_i(x)$ for $x \in B$
   c. Compute gate predictions $g_i = g_\theta(x)$
   d. Update $\theta_g$ using stochastic gradient ascent/descent
4. **Output**: Trained gating function $g^*$
