## 研究摘要

本文提出了一個關於預訓練-微調（PT+FT）流程的端到端理論，研究了初始化參數如何影響神經網絡在微調階段的特徵重用和特徵學習能力。研究者在對角線性網絡中推導了通用化誤差的精確表達式，並識別出四個不同的微調機制，這些機制由其特徵學習和重用能力區分。關鍵發現是：較小的初始化尺度在早期層能夠使網絡同時重用和優化其特徵，從而在依賴預訓練特徵子集的微調任務中獲得優越的通用化性能。這些理論見解在 CIFAR-100 上訓練的非線性網絡中得到了實驗驗證。

## 主要發現

1. **四個微調機制的識別** — 研究識別出四個不同的微調機制：(I) 預訓練獨立的豐富機制、(II) 預訓練依賴的懶惰機制、(III) 預訓練獨立的懶惰機制、(IV) 預訓練依賴的豐富機制。每個機制都有不同的特徵學習和重用能力。| Source: arXiv:2602.20062

2. **相對初始化尺度的關鍵作用** — 相對初始化尺度（λPT）在進入預訓練依賴的豐富機制中起關鍵作用。較小的 λPT（負值）使得網絡能夠同時重用和優化特徵。| Source: arXiv:2602.20062

3. **ℓ-階數與預訓練依賴性的通用權衡** — 研究揭示了一個通用權衡：ℓ-order ∈ [1,2], PD ∈ [-1,0], ℓ-order+PD ∈ [1,2]。不可能同時達到 PD=-1 和 ℓ-order=1。| Source: arXiv:2602.20062

4. **ResNet 實驗驗證** — 在 CIFAR-100 上訓練的 ResNet 實驗表明，相同的初始化參數確實影響通用化性能，驗證了理論預測。| Source: arXiv:2602.20062

## 詳細分析

### 1. 研究背景與動機

預訓練和微調是現代機器學習系統的核心階段。在實踐中，特徵學習在這兩個階段都發揮重要作用：深度神經網絡在預訓練期間學習廣泛的有用特徵，並在微調期間進一步優化這些特徵。

儘管 PT+FT 在實踐中取得了成功，但控制預訓練和微調階段特徵學習的因素仍然不清楚。先前的工作表明，權重初始化——特別是其在層間的絕對和相對尺度——會將神經網絡訓練在固定特徵（"懶惰"）和特徵學習（"豐富"）機制之間轉移。然而，這些見解如何擴展到 PT+FT 等多階段訓練機制尚不清楚。

### 2. 理論框架

#### 2.1 網絡架構

研究使用對角線性網絡作為理論模型，這是一個簡單的單隱藏層網絡架構，參數化線性映射 f: ℝ^D → ℝ：

f_w,v(x) = β(w,v)^T x

其中 β(w,v) := v^+ ∘ w^+ - v^- ∘ w^- ∈ ℝ^D（∘ 表示逐元素乘法）。w^+, w^- ∈ ℝ^D 組成網絡的隱藏權重，v^+, v^- ∈ ℝ^D 組成其輸出權重。

對角線性網絡捕獲了實踐中使用的深度神經網絡的兩個關鍵方面：
1. 初始權重控制其在豐富（特徵學習）和懶惰（核）機制之間的轉換
2. 在這些機制之間通用化行為不同

#### 2.2 初始化參數

研究關注權重初始化的兩個關鍵方面：

1. **絕對尺度**（c_PT）：捕獲整體初始權重幅度
   c_PT := w_PT^±(0)^2 + v_PT^±(0)^2

2. **相對尺度**（λ_PT）：捕獲第一層和第二層之間的初始權重幅度差異
   λ_PT := (w_PT^±(0)^2 - v_PT^±(0)^2) / c_PT ∈ [-1,1]

在預訓練之後，重新初始化讀出權重到固定尺度 γ_FT ≥ 0：
   v_FT^±(0) := γ_FT

#### 2.3 數據生成模型

研究使用教師-學生設置，其中通過從聯合分布採樣來採樣地面真實預訓練和微調函數。重點關注尖峰和平板分布，其中：

- 預訓練任務的活動維度集：
  θ_PT ~ Bernoulli(ρ_PT), θ_PT ∈ {0,1}^D

- β_PT 通過為活動維度採樣隨機符號生成：
  β_PT = (σ/√ρ_PT) ∘ θ_PT, σ ~ Cat({-1,1})

- 微調任務的活動維度取決於預訓練任務：
  θ_FT|θ_PT=1 ~ Bernoulli(ρ_FT^shared/ρ_PT)
  θ_FT|θ_PT=0 ~ Bernoulli(ρ_FT^new/(1-ρ_PT))

其中 ρ_FT^shared ≥ ρ_FT^new 表示預訓練期間活動的維度在微調期間更可能活動。

### 3. 通用化誤差的理論表征

#### 3.1 隱性歸納偏置（定理 4.1）

對角線性網絡在梯度流收斂時的解由以下給出：

arg min_{β_FT} Q_k(β_FT) s.t. X_FT^T β_FT = y_FT

其中 Q_k(β_FT) = Σ_{d=1}^D q_{k_d}(β_FT,d)

q_k(z) = (√k/4) [1 - √(1 + 4z^2/k) + (2z/√k) arcsinh(2z/√k)]

k_d = [2c_PT(1+λ_PT)(1 + √(1 + (β̂_PT,d/c_PT)^2)) + γ^2]^2

這個定理表明網絡選擇的函數以非平凡方式依賴於 λ_PT、c_PT 和 γ_FT。重要的是，這個隱性偏置也依賴於 β̂_PT，意味著學習的解潛在地依賴於預訓練任務。

#### 3.2 通用化誤差的復制理論（命題 4.2）

在 NF_T → ∞ 和 NF_T/D → α_FT > 0 的極限下，估計問題解耦為標量問題：

β̂^sc(y;k,θ) = arg min_{β∈ℝ} {(y-β)^2/(2θ) + Q_k(β)}

其中 θ 和 θ_0 是"有效"正則化和噪聲參數，由固定點方程控制。

### 4. 四個微調機制

#### 4.1 三個極限機制

研究使用兩個指標來表征學習機制：

1. **ℓ-階數**：測量網絡是否受益於稀疏性
   ℓ-order = 1 表示稀疏歸納偏置（如 ℓ1-範數）
   ℓ-order = 2 表示非稀疏歸納偏置（如 ℓ2-範數）

2. **預訓練依賴性（PD）**：測量網絡是否受益於預訓練特徵
   PD = -1 表示懲罰與預訓練特徵幅度成反比
   PD = 0 表示懲罰不依賴於預訓練特徵

命題 5.1 識別了以下範圍：
- ℓ-order ∈ [1,2]
- PD ∈ [-1,0]
- ℓ-order + PD ∈ [1,2]

三個極限機制為：

**(I) 預訓練獨立的豐富機制**
- ℓ-order = 1, PD = 0
- 由 ℓ1-範數治理：Q(β_FT) → ‖β_FT‖_1
- 在 λ_PT → -1, γ_FT → 0 的極限下誘導
- 支持學習新維度

**(II) 預訓練依賴的懶惰機制**
- ℓ-order = 2, PD = -1
- 由加權 ℓ2-範數治理：Q(β_FT) → Σ |β_FT,i|^2/|β_PT,i|
- 在 c_PT → 0, λ_PT → 1 的極限下誘導
- 最小適應地重用特徵

**(III) 預訓練獨立的懶惰機制**
- ℓ-order = 2, PD = 0
- 由未加權 ℓ2-範數治理：Q(β_FT) → ‖β_FT‖_2
- 在 c_PT → ∞ 或 γ_FT → ∞ 的極限下誘導
- 不依賴於預訓練任務

#### 4.2 預訓練依賴的豐富機制（機制 IV）

研究識別了一個重要的中間機制：

**(IV) 預訓練依賴的豐富機制**
- ℓ-order ∈ (1,2), PD ∈ (-1,0)
- 同時支持特徵重用和繼續特徵學習
- 相對尺度 λ_PT 在進入這個機制中起關鍵作用

對於活動維度（β_PT,d^* = ±1/√ρ_PT），改變相對尺度 λ_PT 將網絡從懶惰、預訓練依賴機制（II）移到豐富、預訓練獨立機制（I），中間尺度產生豐富、預訓練依賴機制（IV）。

直觀地說：
- 負的 λ_PT 意味著第二層權重優於第一層（v > w）
- 正的 λ_PT 意味著相反（w > v）
- 當 λ_PT 大且為負時，即使在預訓練後，第一層表示也相對較小
- 將第二層重新縮放至 γ_FT = 0 後，這些大的第二層權重被減小，降低了網絡的整體尺度——這驅動系統走向豐富機制

### 5. 通用化性能分析

#### 5.1 從共享維度中受益？

如果懲罰實施預訓練依賴性，網絡應該從預訓練和微調之間的維度共享中受益。研究比較了：
- 完全共享（ρ_FT^shared = 0.1, ρ_FT^new = 0）
- 完全不同（ρ_FT^shared = 0, ρ_FT^new = 0.1）

發現：
- 當 c_PT 和 γ_FT 減小時，性能對任務重疊變得越來越敏感
- 當 λ_PT 減小時，從共享維度中受益的程度降低

#### 5.2 從新活動維度的稀疏性中受益？

對於非活動維度（β_PT,d^* = 0），可以處於豐富機制（從稀疏的新維度集中受益）或懶惰機制（同樣好地學習稀疏和非稀疏集）。

發現：當 c_PT 和 γ_FT 減小時，稀疏性對性能變得越來越有益。

#### 5.3 從共享活動維度的稀疏性中受益？

豐富學習機制也應該從預訓練維度的稀疏性中受益（β_PT,d^* = ±1/√ρ_PT）。

發現：當 λ_PT 減小時，網絡越來越受益於稀疏性。但如果 c_PT 或 γ_FT 非常大，這個效應消失。

### 6. 大規模視覺實驗

研究在 CIFAR-100 上訓練 ResNet-18，預訓練 98 個類別，然後微調剩餘的 2 個類別。

設計了三個實驗設置，將理論參數 λ_PT、c_PT 和 γ_FT 轉化為現實的深度網絡架構：

1. **相對尺度（κ）**：通過縮放嵌入層和前三個殘差階段來誘導網絡層間的不平衡（相當於 λ_PT）
2. **絕對尺度**：通過常數因子縮放所有網絡參數（相當於 c_PT）
3. **讀出初始化**：預訓練後縮放最終分類層的參數（相當於 γ_FT）

主要結果：

**圖 4a：** 用非標準小 κ 值（κ < 1）初始化的 ResNet 在微調期間表現出改進的通用化性能。

**圖 4b：** 增加 c_PT 的尺度也導致微調通用化性能下降，與理論預測一致。

**圖 4c：** 減小 γ_FT 的尺度對中間樣本數量改善微調通用化性能。

在附錄中，研究提供了這三個設置的表示級別分析，發現所有三個設置都誘導了預訓練依賴豐富機制的關鍵表示特徵。

### 7. 核心貢獻與創新點

1. **端到端理論**：首次為 PT+FT 流程提供了端到端理論，將通用化誤差分析地表達為權重初始化、任務參數和數據尺度的函數。

2. **四個機制的識別**：不僅識別了三個極限機制，還發現了重要的中間機制——預訓練依賴的豐富機制。

3. **相對尺度的關鍵作用**：揭示相對初始化尺度 λ_PT 在進入預訓練依賴豐富機制中起關鍵作用，這是一個重要的新見解。

4. **通用權衡**：證明了 ℓ-order 與預訓練依賴性之間的通用權衡，不可能同時達到最佳特徵重用和最佳特徵學習。

5. **實驗驗證**：在真實深度網絡（ResNet-18 on CIFAR-100）上驗證了理論見解。

### 8. 技術細節

#### 8.1 隱性正則化函數

隱性正則化函數 Q_k(z) 的形式為：

q_k(z) = (√k/4) [1 - √(1 + 4z^2/k) + (2z/√k) arcsinh(2z/√k)]

其中：
- 當 k → ∞ 時，q_k(z) → z^2/2（ℓ2-範數行為）
- 當 k → 0 時，q_k(z) → |z|（ℓ1-範數行為）

參數 k_d 為連接初始化和預訓練表示的橋樑：
k_d = [2c_PT(1+λ_PT)(1 + √(1 + (β̂_PT,d/c_PT)^2)) + γ^2]^2

這顯示了：
- 絕對尺度 c_PT 和讀出初始化 γ_FT 影響整體尺度
- 相對尺度 λ_PT 影響第一層表示的貢獻
- 預訓練表示 β̂_PT,d 通過 √(1 + (β̂_PT,d/c_PT)^2) 項調節懲罰

#### 8.2 復制方法

研究使用徑制方法在高維極限（D → ∞）下表征通用化誤差。雖然徑制方法是非嚴格的，但它提供了一個強大的工具，可以在不同懲罰下（包括 ℓ1）分析地表征估計誤差。

關鍵固定點方程：
- p = Σ_{j=1}^J π_j E[(β̂^sc(β*+η;k,θ) - β*)^2]
- χ = θ Σ_{j=1}^J π_j E[∂_y β̂^sc(β*+η;k,θ)]
- θ = (λ + χ)/α
- θ_0 = (σ_0^2 + p)/α

### 9. 應用價值

#### 9.1 理論價值

1. **理解機制**：提供了 PT+FT 流程的理論基礎，幫助理解為什麼某些初始化策略更有效。

2. **指導實踐**：為初始化參數的選擇提供理論指導，特別是相對尺度的調整。

3. **揭示權衡**：明確了特徵重用和特徵學習之間的權衡，幫助在實際應用中做出更好的決策。

#### 9.2 實踐價值

1. **初始化策略**：建議對於需要同時重用和優化特徵的任務，使用較小的相對初始化尺度（負 λ_PT）。

2. **層級設計**：強調了不同層級初始化尺度的相對重要性，為網絡架構設計提供了新視角。

3. **遷移學習**：為遷移學習中的微調策略提供了理論基礎，特別是在數據有限的情況下。

#### 9.3 潛在應用

1. **基礎模型微調**：對大規模基礎模型的微調具有重要指導意義，特別是在下游任務數據有限的情況下。

2. **神經科學**：為理解生物神經系統中的特徵學習和重用提供了理論框架。

3. **自動化機器學習**：可以集成到 AutoML 系統中，自動優化初始化參數。

### 10. 局限性

#### 10.1 理論模型的簡化

1. **線性網絡**：理論基於對角線性網絡，真實的神經網絡具有非線性和更深的架構。

2. **梯度流假設**：假設使用梯度流收斂到零訓練誤差，實踐中使用的是離散梯度下降。

3. **無噪聲假設**：主要關注無噪聲情況，真實數據通常包含噪聲。

#### 10.2 實驗規模

1. **CIFAR-100**：實驗主要在 CIFAR-100 上進行，更大規模的數據集（如 ImageNet）上的效果需要進一步驗證。

2. **ResNet-18**：使用的是相對較小的 ResNet-18 架構，對更大、更深的模型（如 GPT、ViT）的適用性需要研究。

3. **類別分割**：實驗使用 98/2 的類別分割，更複雜的遷移場景需要探索。

#### 10.3 理論擴展

1. **非線性激活**：將理論擴展到具有非線性激活的網絡是一個重要的開放問題。

2. **更深的網絡**：理論如何擴展到更深層的網絡架構需要研究。

3. **多任務學習**：當前關注預訓練-微調的兩階段設置，多任務學習的一般情況需要探索。
