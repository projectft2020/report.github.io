## 系統架構

### 1. 時間框架層級
```
1H  →  4H  →  12H  →  1D  →  1W
     ↑        ↑        ↑      ↑
   短期    中短期    中期    長期
```

### 2. 指標信號定義

#### MACD 信號
- **做多信號**：MACD 線 > 信號線 AND 直方圖 > 0
- **做空信號**：MACD 線 < 信號線 AND 直方圖 < 0
- **中立信號**：其他情況

#### RSI 信號
- **做多信號**：RSI > 50 且 RSI < 70（趨勢上漲但未超買）
- **做空信號**：RSI < 50 且 RSI > 30（趨勢下跌但未超賣）
- **超買警告**：RSI > 70（潛在回調風險）
- **超賣警告**：RSI < 30（潛在反彈機會）
- **中立信號**：其他情況

#### Supertrend 信號
- **做多信號**：supertrend_dir = 1（上漲趨勢）
- **做空信號**：supertrend_dir = -1（下跌趨勢）

```python
def generate_signals(df, timeframe):
    """
    為特定時間框架生成所有指標信號
    
    Args:
        df: 包含指標的 DataFrame
        timeframe: 時間框架標識 (e.g., '1h', '4h', '1d')
    
    Returns:
        dict: 包含所有信號的字典
    """
    latest = df.iloc[-1]
    signals = {
        'timeframe': timeframe,
        'timestamp': latest.name,
        'close': latest['close'],
        'macd': {
            'value': latest['macd'],
            'signal': latest['macd_signal'],
            'histogram': latest['macd_histogram'],
            'signal_type': get_macd_signal(latest)
        },
        'rsi': {
            'value': latest['rsi'],
            'signal_type': get_rsi_signal(latest),
            'zone': get_rsi_zone(latest['rsi'])
        },
        'supertrend': {
            'value': latest['supertrend'],
            'direction': latest['supertrend_dir'],
            'signal_type': get_supertrend_signal(latest)
        }
    }
    return signals
```

#### `check_resonance()` - 共振確認
```python
def check_resonance(all_signals, min_alignment=3):
    """
    檢查多時間框架共振情況
    
    Args:
        all_signals: 所有時間框架的信號
        min_alignment: 最小對齊數量（默認3個時間框架）
    
    Returns:
        dict: 共振分析結果
    """
    # 統計各方向信號數量
    long_signals = []
    short_signals = []
    
    for tf, signals in all_signals.items():
        # 每個時間框架需要至少2個指標同向才算有效
        long_count = sum(1 for ind in signals.values() 
                        if isinstance(ind, dict) and 'signal_type' in ind 
                        and ind['signal_type'] == 'LONG')
        short_count = sum(1 for ind in signals.values() 
                         if isinstance(ind, dict) and 'signal_type' in ind 
                         and ind['signal_type'] == 'SHORT')
        
        if long_count >= 2:
            long_signals.append(tf)
        elif short_count >= 2:
            short_signals.append(tf)
    
    # 判斷共振強度
    resonance = {
        'has_long_resonance': len(long_signals) >= min_alignment,
        'has_short_resonance': len(short_signals) >= min_alignment,
        'long_timeframes': long_signals,
        'short_timeframes': short_signals,
        'strength': calculate_resonance_strength(long_signals, short_signals)
    }
    
    return resonance
```

```python
def calculate_resonance_strength(long_signals, short_signals):
    """
    計算共振強度
    - 強共振：5/5 時間框架對齊
    - 中共振：3-4/5 時間框架對齊
    - 弱共振：2/5 時間框架對齊
    - 無共振：<2 時間框架對齊

    aligned_count = max(len(long_signals), len(short_signals))

        return 'STRONG'
    elif aligned_count >= 3:
        return 'MODERATE'
    elif aligned_count >= 2:
        return 'WEAK'
    else:
        return 'NONE'
```

```python
class SignalGenerator:

            return 'SHORT'

class TimeframeAnalyzer:
    """多時間框架分析器"""
    
    @staticmethod
    def analyze_multi_timeframe(data_dict: Dict[str, pd.DataFrame]) -> Dict[str, Dict]:
        """分析所有時間框架"""
        all_signals = {}
        for tf, df in data_dict.items():
            all_signals[tf] = SignalGenerator.generate_signals(df, tf)
        return all_signals
    
    @staticmethod
    def prepare_timeframe_data(raw_df: pd.DataFrame, timeframe: str) -> pd.DataFrame:
        """準備特定時間框架的數據並計算指標"""
        # 重採樣數據
        resampled = raw_df.copy()
        if timeframe == '1h':
            resampled = raw_df.resample('1H').agg({
                'open': 'first', 'high': 'max', 
                'low': 'min', 'close': 'last', 'volume': 'sum'
            }).dropna()
        elif timeframe == '4h':
            resampled = raw_df.resample('4H').agg({
                'open': 'first', 'high': 'max', 
                'low': 'min', 'close': 'last', 'volume': 'sum'
            }).dropna()
        elif timeframe == '12h':
            resampled = raw_df.resample('12H').agg({
                'open': 'first', 'high': 'max', 
                'low': 'min', 'close': 'last', 'volume': 'sum'
            }).dropna()
        elif timeframe == '1d':
            resampled = raw_df.resample('1D').agg({
                'open': 'first', 'high': 'max', 
                'low': 'min', 'close': 'last', 'volume': 'sum'
            }).dropna()
        elif timeframe == '1w':
            resampled = raw_df.resample('1W').agg({
                'open': 'first', 'high': 'max', 
                'low': 'min', 'close': 'last', 'volume': 'sum'
            }).dropna()
        
        # 計算指標
        return calculate_all_indicators(resampled)


class ResonanceChecker:
    """共振確認器"""
    
    @staticmethod
    def check_resonance(all_signals: Dict[str, Dict], min_alignment: int = 3) -> Dict:
        """檢查多時間框架共振"""
        long_timeframes = []
        short_timeframes = []
        neutral_timeframes = []
        
        for tf, signals in all_signals.items():
            # 計算該時間框架的傾向
            long_count = 0
            short_count = 0
            
            for ind in ['macd', 'rsi', 'supertrend']:
                sig_type = signals[ind]['signal_type']
                if sig_type == 'LONG':
                    long_count += 1
                elif sig_type == 'SHORT':
                    short_count += 1
                # 忽略 OVERBOUGHT/OVERSOLD/NEUTRAL
            
            if long_count >= 2:
                long_timeframes.append(tf)
            elif short_count >= 2:
                short_timeframes.append(tf)
            else:
                neutral_timeframes.append(tf)
        
        resonance = {
            'has_long_resonance': len(long_timeframes) >= min_alignment,
            'has_short_resonance': len(short_timeframes) >= min_alignment,
            'long_timeframes': long_timeframes,
            'short_timeframes': short_timeframes,
            'neutral_timeframes': neutral_timeframes,
            'strength': ResonanceChecker._calculate_strength(
                len(long_timeframes), len(short_timeframes)
            ),
            'alignment_score': ResonanceChecker._calculate_alignment_score(
                long_timeframes, short_timeframes
            )
        }
        
        return resonance
    
    @staticmethod
    def _calculate_strength(long_count: int, short_count: int) -> str:
        """計算共振強度"""
        aligned = max(long_count, short_count)
        
        if aligned == 5:
            return 'STRONG'
        elif aligned >= 3:
            return 'MODERATE'
        elif aligned >= 2:
            return 'WEAK'
        return 'NONE'
    
    @staticmethod
    def _calculate_alignment_score(long_tf: List[str], short_tf: List[str]) -> float:
        """計算對齊分數（考慮時間框架權重）"""
        weights = SignalGenerator.WEIGHTS
        score = 0.0
        
        for tf in long_tf:
            score += weights.get(tf, 1.0)
        
        for tf in short_tf:
            score -= weights.get(tf, 1.0)
        
        return score


class ReportGenerator:
    """報告生成器"""
    
    @staticmethod
    def determine_overall_signal(resonance: Dict) -> str:
        """確定整體信號"""
        if resonance['strength'] == 'NONE':
            return 'NO_SIGNAL'
        elif resonance['has_long_resonance']:
            return 'LONG'
        elif resonance['has_short_resonance']:
            return 'SHORT'
        return 'NEUTRAL'
    
    @staticmethod
    def calculate_confidence(resonance: Dict) -> float:
        """計算信心水平"""
        strength_map = {'STRONG': 0.85, 'MODERATE': 0.65, 'WEAK': 0.45, 'NONE': 0.2}
        base_confidence = strength_map.get(resonance['strength'], 0.2)
        
        # 根據對齊分數調整
        alignment_bonus = min(abs(resonance['alignment_score']) / 8.0 * 0.15, 0.15)
        
        return round(base_confidence + alignment_bonus, 2)
    
    @staticmethod
    def generate_entry_signal(resonance: Dict, all_signals: Dict) -> Dict:
        """生成入場信號"""
        overall = ReportGenerator.determine_overall_signal(resonance)
        
        if overall == 'NO_SIGNAL':
            return {
                'type': 'NONE',
                'reason': 'Insufficient timeframe alignment',
                'action': 'WAIT'
            }
        
        if overall == 'LONG':
            return {
                'type': 'LONG_ENTRY',
                'reason': f"Bullish resonance across {len(resonance['long_timeframes'])} timeframes",
                'strength': resonance['strength'],
                'action': 'BUY'
            }
        elif overall == 'SHORT':
            return {
                'type': 'SHORT_ENTRY',
                'reason': f"Bearish resonance across {len(resonance['short_timeframes'])} timeframes",
                'strength': resonance['strength'],
                'action': 'SELL'
            }
        
        return {'type': 'NEUTRAL', 'action': 'WAIT'}
    
    @staticmethod
    def generate_exit_signal(resonance: Dict, all_signals: Dict) -> Dict:
        """生成出場信號"""
        overall = ReportGenerator.determine_overall_signal(resonance)
        exit_signals = []
        
        # 檢查超買/超賣警告
        for tf, signals in all_signals.items():
            if signals['rsi']['signal_type'] == 'OVERBOUGHT':
                exit_signals.append({
                    'type': 'TAKE_PROFIT_LONG',
                    'timeframe': tf,
                    'reason': f"RSI overbought ({signals['rsi']['value']:.2f})",
                    'urgency': 'HIGH' if signals['rsi']['value'] > 75 else 'MEDIUM'
                })
            elif signals['rsi']['signal_type'] == 'OVERSOLD':
                exit_signals.append({
                    'type': 'TAKE_PROFIT_SHORT',
                    'timeframe': tf,
                    'reason': f"RSI oversold ({signals['rsi']['value']:.2f})",
                    'urgency': 'HIGH' if signals['rsi']['value'] < 25 else 'MEDIUM'
                })
        
        # 檢查趨勢轉向
        for tf, signals in all_signals.items():
            if overall == 'LONG' and signals['supertrend']['signal_type'] == 'SHORT':
                exit_signals.append({
                    'type': 'TREND_REVERSAL',
                    'timeframe': tf,
                    'reason': 'Supertrend turned bearish',
                    'urgency': 'HIGH'
                })
            elif overall == 'SHORT' and signals['supertrend']['signal_type'] == 'LONG':
                exit_signals.append({
                    'type': 'TREND_REVERSAL',
                    'timeframe': tf,
                    'reason': 'Supertrend turned bullish',
                    'urgency': 'HIGH'
                })
        
        return {
            'has_exit_signals': len(exit_signals) > 0,
            'signals': exit_signals,
            'action': 'HOLD' if not exit_signals else 'CONSIDER_EXIT'
        }
    
    @staticmethod
    def generate_warnings(all_signals: Dict) -> List[str]:
        """生成警告信息"""
        warnings = []
        
        # 檢查盤整區間
        rsi_values = [signals['rsi']['value'] for signals in all_signals.values()]
        rsi_range = max(rsi_values) - min(rsi_values)
        if rsi_range < 15:
            warnings.append("MARKET CONSOLIDATION: RSI range narrow, potential false signals")
        
        # 檢查趨勢分歧
        macd_signals = [signals['macd']['signal_type'] for signals in all_signals.values()]
        st_signals = [signals['supertrend']['signal_type'] for signals in all_signals.values()]
        
        if set(macd_signals) == {'LONG', 'SHORT'} or set(st_signals) == {'LONG', 'SHORT'}:
            warnings.append("TREND DIVERGENCE: Indicators showing conflicting signals across timeframes")
        
        # 檢查極端 RSI
        for tf, signals in all_signals.items():
            if signals['rsi']['value'] > 80:
                warnings.append(f"EXTREME OVERBOUGHT ({tf}): RSI = {signals['rsi']['value']:.2f}")
            elif signals['rsi']['value'] < 20:
                warnings.append(f"EXTREME OVERSOLD ({tf}): RSI = {signals['rsi']['value']:.2f}")
        
        return warnings
    
    @staticmethod
    def generate_recommendation(resonance: Dict, all_signals: Dict) -> Dict:
        """生成最終交易建議"""
        overall = ReportGenerator.determine_overall_signal(resonance)
        confidence = ReportGenerator.calculate_confidence(resonance)
        warnings = ReportGenerator.generate_warnings(all_signals)
        
        # 檢查是否有阻止條件
        blocked = False
        block_reason = None
        
        for warning in warnings:
            if 'MARKET CONSOLIDATION' in warning:
                blocked = True
                block_reason = "Market in consolidation phase"
                break
            if 'TREND DIVERGENCE' in warning and resonance['strength'] in ['NONE', 'WEAK']:
                blocked = True
                block_reason = "Strong indicator divergence"
                break
        
        if blocked:
            return {
                'action': 'WAIT',
                'reason': block_reason,
                'confidence': confidence,
                'conditions': ['Wait for clearer trend confirmation']
            }
        
        if overall == 'LONG' and confidence >= 0.6:
            return {
                'action': 'BUY',
                'reason': f"Bullish resonance with {resonance['strength']} strength",
                'confidence': confidence,
                'conditions': [
                    'Enter on pullback to support level',
                    'Stop loss below recent swing low',
                    'Take profit at resistance or next timeframe resistance'
                ]
            }
        elif overall == 'SHORT' and confidence >= 0.6:
            return {
                'action': 'SELL',
                'reason': f"Bearish resonance with {resonance['strength']} strength",
                'confidence': confidence,
                'conditions': [
                    'Enter on rally to resistance level',
                    'Stop loss above recent swing high',
                    'Take profit at support or next timeframe support'
                ]
            }
        
        return {
            'action': 'WAIT',
            'reason': f"Insufficient signal strength (confidence: {confidence})",
            'confidence': confidence,
            'conditions': ['Wait for higher confidence level']
        }
    
    @staticmethod
    def generate_full_report(all_signals: Dict, resonance: Dict) -> Dict:
        """生成完整報告"""
        return {
            'timestamp': datetime.now().isoformat(),
            'overall_signal': ReportGenerator.determine_overall_signal(resonance),
            'confidence': ReportGenerator.calculate_confidence(resonance),
            'resonance': resonance,
            'timeframe_breakdown': all_signals,
            'entry_signal': ReportGenerator.generate_entry_signal(resonance, all_signals),
            'exit_signal': ReportGenerator.generate_exit_signal(resonance, all_signals),
            'warnings': ReportGenerator.generate_warnings(all_signals),
            'recommendation': ReportGenerator.generate_recommendation(resonance, all_signals)
        }


class MultiTimeframeSystem:
    """多時間框架確認系統主類"""
    
    def __init__(self):
        self.timeframe_analyzer = TimeframeAnalyzer()
        self.resonance_checker = ResonanceChecker()
        self.report_generator = ReportGenerator()
    
    def analyze(self, raw_df: pd.DataFrame, timeframes: List[str] = None) -> Dict:
        """
        執行完整的多時間框架分析
        
        Args:
            raw_df: 原始 OHLCV 數據（最小時間粒度）
            timeframes: 要分析的時間框架列表，默認全部
        
        Returns:
            Dict: 完整的分析報告
        """
        if timeframes is None:
            timeframes = SignalGenerator.TIMEFRAMES
        
        # 準備所有時間框架的數據
        data_dict = {}
        for tf in timeframes:
            data_dict[tf] = self.timeframe_analyzer.prepare_timeframe_data(raw_df, tf)
        
        # 分析多時間框架
        all_signals = self.timeframe_analyzer.analyze_multi_timeframe(data_dict)
        
        # 檢查共振
        resonance = self.resonance_checker.check_resonance(all_signals)
        
        # 生成報告
        report = self.report_generator.generate_full_report(all_signals, resonance)
        
        return report
    
    def print_report(self, report: Dict):
        """打印格式化的報告"""
```

### 共振強度說明

| 強度 | 對齊時間框架數 | 信心水平 | 建議 |
|------|---------------|---------|------|
| STRONG | 5/5 | 85%+ | 可考慮入場 |
| MODERATE | 3-4/5 | 65-85% | 謹慎入場 |
| WEAK | 2/5 | 45-65% | 觀望為主 |
| NONE | <2 | <45% | 不入場 |

### 交易決策流程

```
開始
  ↓
檢查警告（盤整/分歧）
  ↓ [有警告]
觀望等待
  ↓ [無警告]
檢查共振強度
  ↓ [WEAK/NONE]
觀望等待
  ↓ [MODERATE/STRONG]
檢查信心水平
  ↓ [≥60%]
執行交易
  ↓ [<60%]
觀望等待
```

## 實際應用建議

### 1. 入場時機
- **STRONG 多頭共振** + 信心≥60% → 考慮做多
- **STRONG 空頭共振** + 信心≥60% → 考慮做空
- **MODERATE 共振** + 信心≥70% → 可謹慎入場

### 2. 出場時機
- RSI > 70（任何時間框架）→ 考慮減倉
- RSI < 30（任何時間框架）→ 考慮平倉
- Supertrend 轉向（短時間框架）→ 密切監控
- Supertrend 轉向（中長時間框架）→ 執行出場

### 3. 風險管理
- 止損設置：根據最近 swing low/high
- 止盈目標：根據 next timeframe 的阻力/支撐位
- 倉位大小：根據共振強度和信心水平調整
- 分批入場：MODERATE 共振可分批建倉

### 4. 避免的情況
- 盤整區間（RSI 範圍 < 15）
- 指標分歧嚴重
- 信心水平 < 60%
- 多個極端 RSI 警告
