## 系統修復：tasks.json 格式兼容性問題（2026-03-08 00:38 AM）

### 問題描述

**發現時間：** 2026-03-08 00:25 AM（心跳檢查時）

**錯誤信息：**
1. `backpressure.py`: `'str' object has no attribute 'get'`
2. `priority-rule-engine/apply_rules.py`: 執行失敗

**根本原因：**
- `tasks.json` 的實際格式是 `{"tasks": [...]}`
- 但多個腳本假設它是直接的一個列表 `[...]`
- 當 `json.load()` 返回字典時，腳本嘗試迭代字典或調用 `.get()` 方法時失敗

### 修復的組件

**1. backpressure.py（背壓機制）**
- 位置：`~/.openclaw/workspace/kanban-ops/backpressure.py`
- 修復：`_calculate_health()` 方法
- 修改前：直接使用 `tasks = json.load(f)`
- 修改後：檢查格式並提取任務列表

```python
# 修復前
tasks = json.load(f)
for task in tasks:
    status = task.get('status')

# 修復後
data = json.load(f)
if isinstance(data, dict) and 'tasks' in data:
    tasks = data['tasks']
elif isinstance(data, list):
    tasks = data
else:
    logger.error(f"未知的 tasks.json 格式: {type(data)}")
    return 1.0

for task in tasks:
    status = task.get('status')
```

**2. auto_spawn_heartbeat.py（自動任務啟動器）**
- 位置：`~/.openclaw/workspace/kanban-ops/auto_spawn_heartbeat.py`
- 修復：`load_tasks()` 函數
- 修改前：直接返回 `json.load(f)`
- 修改後：檢查格式並返回任務列表

```python
# 修復前
def load_tasks():
    with open(TASKS_JSON, 'r', encoding='utf-8') as f:
        return json.load(f)

# 修復後
def load_tasks():
    with open(TASKS_JSON, 'r', encoding='utf-8') as f:
        data = json.load(f)

        if isinstance(data, dict) and 'tasks' in data:
            return data['tasks']
        elif isinstance(data, list):
            return data
        else:
            log("ERROR", f"未知的 tasks.json 格式: {type(data)}")
            return []
```

**3. error_recovery.py（錯誤恢復）**
- 位置：`~/.openclaw/workspace/kanban-ops/error_recovery.py`

- 修改前：直接使用 `tasks = json.load(f)`
- 修改後：檢查格式並提取任務列表

**4. priority-rule-engine/scripts/apply_rules.py（優先級規則引擎）**
- 位置：`~/.openclaw/workspace/skills/priority-rule-engine/scripts/apply_rules.py`
- 修復：`load_tasks()` 和 `save_tasks()` 函數
- 修改前：
  - `load_tasks()`: 直接返回 `json.load(f)`
  - `save_tasks()`: 直接保存任務列表

  - `load_tasks()`: 檢查格式並返回任務列表
  - `save_tasks()`: 保持原始格式（dict 或 list）

```python

def load_tasks():
    with open(TASKS_FILE, "r") as f:
        data = json.load(f)

        if isinstance(data, dict) and 'tasks' in data:
            return data['tasks']
        elif isinstance(data, list):
            return data
        else:
            print(f"❌ 未知的 tasks.json 格式: {type(data)}")
            return []

# 修復後的 save_tasks()
def save_tasks(tasks):
    # 檢查原始格式
    if TASKS_FILE.exists():
        with open(TASKS_FILE, "r") as f:
            data = json.load(f)
            if isinstance(data, list):
                original_format = "list"
            else:
                original_format = "dict"
    else:
        original_format = "dict"

    # 保存時保持原始格式
    if original_format == "dict":
        save_data = {"tasks": tasks}
    else:
        save_data = tasks

    with open(TASKS_FILE, "w") as f:
        json.dump(save_data, f, indent=2, ensure_ascii=False)
```

```python
# obsidian_wrapper.py
import subprocess
import json

class ObsidianCLI:
    """Obsidian CLI Python 包裝器"""
    
    def __init__(self, vault_path=None):
        self.vault_path = vault_path
    
    def create(self, name, content, path=None):
        """創建新筆記"""
        cmd = ["obsidian", "create", f"name={name}", f'content="{content}"']
        if path:
            cmd.append(f"path={path}")
        subprocess.run(cmd, cwd=self.vault_path)
    
    def read(self, file):
        """讀取筆記"""
        cmd = ["obsidian", "read", f"file={file}"]
        result = subprocess.run(cmd, cwd=self.vault_path, capture_output=True, text=True)
        return result.stdout
    
    def append(self, file, content):
        """追加內容"""
        cmd = ["obsidian", "append", f"file={file}", f'content="{content}"']
        subprocess.run(cmd, cwd=self.vault_path)
    
    def search(self, query):
        """搜索"""
        cmd = ["obsidian", "search", f'query="{query}"', "format=json"]
        result = subprocess.run(cmd, cwd=self.vault_path, capture_output=True, text=True)
        return json.loads(result.stdout)
    
    def get_orphans(self):
        """找出孤立檔案"""
        cmd = ["obsidian", "orphans", "format=json"]
        result = subprocess.run(cmd, cwd=self.vault_path, capture_output=True, text=True)
        return json.loads(result.stdout)
    
    def get_backlinks(self, file):
        """獲取反向連結"""
        cmd = ["obsidian", "backlinks", f"file={file}", "format=json"]
        result = subprocess.run(cmd, cwd=self.vault_path, capture_output=True, text=True)
        return json.loads(result.stdout)
    
    def eval_js(self, code):
        """執行 JavaScript"""
        cmd = ["obsidian", "eval", f'code="{code}"']
        result = subprocess.run(cmd, cwd=self.vault_path, capture_output=True, text=True)
        return result.stdout
    
    def create_daily_note(self, content):
        """創建或打開今日日記"""
        cmd = ["obsidian", "daily"]
        subprocess.run(cmd, cwd=self.vault_path)
        if content:
            self.append("daily", content)
```
