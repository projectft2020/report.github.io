台股市場情緒分析（短期）¶

任務編號： q002a
分析師： Charlie Research
分析日期： 2026-02-21
數據截止： 2026年2月中旬

研究摘要¶

本次分析顯示台股市場短期情緒呈現「偏多謹慎」的特徵。三大法人買賣超呈現震盪格局，融資餘額略有減少，外資態度分化。AI相關板塊持續吸引資金流入，但市場波動性指標顯示投資人謹慎情緒仍在，短期內需注意技術性回檔風險。

1. 情緒指標分析¶

1.1 融資融券餘額變化¶

融資餘額（單位：億元） - 2026/02/11：3,680.5億元（減少45.4億元） - 2026/02/10：3,725.9億元（減少28.8億元） - 2026/02/09：3,754.7億元（增加14.8億元） - 2026/02/06：3,739.9億元（減少89.3億元）

融券餘額（單位：張） - 2026/02/11：248,353張（減少1,771張） - 2026/02/10：250,124張（減少9,864張） - 2026/02/09：259,988張（減少2,276張） - 2026/02/06：262,264張（減少4,443張）

分析： - 融資餘額呈現緩步下降趨勢，顯示散戶加碼意願減弱 - 融券餘額持續減少，空頭力道轉弱 - 融資減少、融券也減少，反映市場進入觀望期

1.2 外資買賣超（單位：億元）¶

近三日外資買賣超表現： - 2026/02/11：外資買超523.6億元 - 2026/02/10：外資買超557.11億元 - 2026/02/09：外資買超247.69億元

分析： - 外資連續三日買超，顯示國際資金對台股信心回溫 - 買超金額依日遞減，顯示追價意願有所收斂 - 外資成為近期市場主要支撐力道

1.3 大戶籌碼變化¶

三大法人買賣超總計（單位：億元）： - 2026/02/11：總計買超578.26億元 - 2026/02/10：總計買超632.04億元 - 2026/02/09：總計買超363.42億元 - 2026/02/06：總計賣超192.75億元

分解分析： - 外資：由賣轉買，且買超幅度擴大 - 投信：小幅買超，操作相對保守 - 自營商：買賣互見，避險部位增加

1.4 散戶指標¶

觀察指標： - 融資使用率：約11.6-11.8%（處於中等水平） - 散戶買賣超：缺乏具體數據，但從融資餘額減少推斷散戶偏保守 - 市場活躍度：日成交金額維持在6,500億元以上，顯示市場仍具流動性

2. 資金流向分析¶

2.1 三大法人買賣超結構¶

資金動向特徵： - 外資：轉為積極買超，AI半導體為主要標的 - 投信：保守操作，小幅買超為主 - 自營商：避險意識增強，自營避險部位增加

近期趨勢： - 外資由1月下旬的連續賣超轉為買超 - 三大法人合力買超，顯示機構投資人對短期行情偏多

2.2 類股資金流向¶

熱門板塊資金流向： - AI供應鏈：持續吸引大單買盤 - 半導體：資金持續匯入，設備股表現突出 - 記憶體：供需改善，資金回籠 - 光通訊：AI需求帶動，資金流入增加

分析： - AI相關產業鏈持續獲得資青睞 - 傳統產業資金流入相對緩慢 - 市場資金明顯偏好成長性產業

2.3 熱點板塊資金¶

六大熱門族群： 1. 記憶體：AI需求強勁，供需缺口擴大 2. 半導體廠務與設備：台積電擴產帶動 3. 光通訊：高速運算需求激增 4. 散熱：AI伺服器高功率需求 5. 電源供應：BBU供應鏈受惠 6. CCL/PCB：AI伺服器板需求增加

3. 市場情緒特徵¶

3.1 恐慌/貪婪指數¶

參考指標： - 台指VIX（VIXTWN）：反映台股期貨30天預期波動率 - MM台灣恐懼貪婪指數：綜合股價表現、跌幅、波動性等指標

當前狀態： - VIX指數：根據外圍市場參考，約26.42點（較前日上漲11.67%） - 市場情緒：由恐慌區域向中性區域轉換 - 投資人情緒：由悲觀轉為審慎樂觀

3.2 市場活躍度¶

活躍度指標： - 日成交金額：6,500-7,500億元之間 - 漲跌家數比例：上漲家數略多於下跌家數 - 熱門個股：信驊、鴻勁、川湖、台光電等持續熱門

評估： 市場活躍度維持在中等偏高水平，流動性充足

3.3 交易者情緒（樂觀/悲觀）¶

情緒分層： - 外資：轉為樂觀，主動買盤增加 - 投信：中性偏多，操作保守 - 自營商：中性偏空，避險意識強 - 散戶：觀望為主，加碼意願減弱

整體情緒： 機構投資人偏多，散戶相對保守

4. 短期情緒預測¶

4.1 情緒趨勢判斷¶

技術面指標： - 加權指數：33,072.97點（漲跌+668.35點，漲幅+2.06%） - 均線系統：短期均線呈現多頭排列 - 成交量：維持在攻擊量水平

情緒趨勢： - 短期情緒由中性偏多轉為偏多 - 外資態度轉變是關鍵因素 - AI產業持續發燒提供支撐

4.2 情緒極點識別¶

監控指標： - 融資使用率：接近11.8%，未達過熱警戒線（15%） - 融券餘額：持續減少，空頭力道減弱 - 市場波動率：溫和上升，未達恐慌水準

極點判斷： - 當前情緒未達極端樂觀或悲觀 - 需監控融資使用率是否快速攀升 - 注意外資買超是否持續擴大

4.3 情緒反轉預警¶

預警信號： - 正向信號： - 外資連續買超超過3日 - 融券餘額持續減少 - AI產業利多不斷

  • 負向信號：
  • 融資餘額開始縮減
  • 自營商避險部位增加
  • 外資買超幅度遞減

風險預警： - 技術性回檔壓力增大 - 獲利了結賣壓可能湧現 - 關注美國聯準會政策變化

5. 對股市的影響分析¶

5.1 短期影響（1-3天）¶

正面因素： - 三大法人合力買超形成支撐 - AI供應鏈持續發燒 - 外資態度轉向樂觀

負面因素： - 技術面漲幅過大，回檔壓力增加 - 散戶追價意願減弱 - 獲利了結賣壓可能出現

預期走勢： 震盪向上，但上檔壓力增加

5.2 中期影響（4-7天）¶

關鍵因素： - AI產業持續性 - 國際市場情緒變化 - 技術支撐位測試

風險評估： - 中性偏多，但波動性可能增加 - 需關注國際股市變化 - 觀察融資餘額變化趨勢

6. 結論與建議¶

6.1 綜合評估¶

台股市場短期情緒呈現「偏多謹慎」的格局。三大法人買賣超轉向買超，外資態度明顯轉變，AI相關板塊持續吸引資金流入。然而，融資餘額縮減、自營商避險意識增強，顯示市場仍存在不確定性。

6.2 操作建議¶

策略建議： - 短期可參與AI供應鏈相關個股 - 注意控制部位，避免過度追高 - 密切關注外資動向和融資餘額變化 - 預留現金等待技術性回檔機會

風險控管： - 設定停損點位 - 避免單一產業集中持有 - 關注國際市場變化對台股的影響
