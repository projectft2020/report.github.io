### 1. 優化動態閾值系統

#### 優化 A：增加暴漲檢測

```python
def check_correlation_surge(correlation_history, surge_threshold=0.2):
    """
    檢測相關性暴漲
    
    參數:
        correlation_history: 歷史相關性序列
        surge_threshold: 暴漲閾值
    
    返回:
        is_surge: 是否暴漲
        surge_magnitude: 暴漲幅度
    """
    if len(correlation_history) < 5:
        return False, 0.0
    
    # 計算 5 日變化
    recent_change = correlation_history.iloc[-1] - correlation_history.iloc[-5]
    
    # 檢測暴漲
    is_surge = recent_change > surge_threshold
    
    return is_surge, recent_change
```

#### 優化 B：結合 VIX 調整

```python
def get_vix_adjusted_threshold(vix, base_threshold):
    """
    根據 VIX 調整閾值
    
    參數:
        vix: 當前 VIX
        base_threshold: 基礎閾值
    
    返回:
        adjusted_threshold: 調整後的閾值
    """
    # VIX 低時（<20）：寬鬆閾值
    if vix < 20:
        return base_threshold * 1.1
    
    # VIX 高時（>40）：嚴格閾值
    elif vix > 40:
        return base_threshold * 0.8
    
    # VIX 中等（20-40）：線性調整
    else:
        return base_threshold * (1.1 - (vix - 20) / 20 * 0.3)
```
