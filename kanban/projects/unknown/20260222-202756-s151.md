## Research Summary

This research investigates asymptotically optimal sequential hypothesis testing for Markovian data, establishing tight non-asymptotic lower bounds on expected stopping times and proposing matching optimal procedures. The work addresses a fundamental gap in sequential testing theory under Markov dependence, improving upon existing asymptotic or sub-optimal bounds.

## Detailed Analysis

### Problem Framework

The research considers one-sided and α-correct sequential hypothesis testing for data generated by an ergodic Markov chain. The null hypothesis H₀: P ∈ 𝒫 states that the unknown transition matrix belongs to a prescribed set 𝒫 of stochastic matrices, while the alternative H₁: P ∈ 𝒬 corresponds to a disjoint set 𝒬. This is a composite versus composite testing problem where both hypothesis classes contain multiple transition matrices.

An α-correct, power-one sequential test is defined by a stopping time τ_α that satisfies:
- ℙ_P[τ_α < ∞] ≤ α for all P ∈ 𝒫 (Type I error control)
- ℙ_Q[τ_α < ∞] = 1 for all Q ∈ 𝒬 (power-one property)

### Main Technical Contributions

#### 1. Information-Theoretic Characterization

The fundamental hardness quantity is the stationary-weighted KL divergence:
Dℳ(Q,P) := ∑_{i∈[m]} π_i D_KL(Q(i,·), P(i,·))

where π is the stationary distribution of Q, and D_KL denotes the Kullback-Leibler divergence between row-wise transition distributions. The key quantity is Dℳ^inf(Q,𝒫) := inf_{P∈𝒫} Dℳ(Q,P).

#### 2. Instance-Dependent Lower Bound (Theorem 3.3)

For any ergodic Q ∈ 𝒬 and α-correct power-one test with stopping time τ_α:
𝔼_Q[τ_α] ≥ (log(1/α)/Dℳ^inf(Q,𝒫) - 2C_Q/min_i π_i)^+

where C_Q is a constant depending only on the pseudo-spectral gap and minimum stationary probability of Q. This bound is non-asymptotic and instance-dependent, incorporating both the stationary distribution and transition structure.

#### 3. Optimal Sequential Algorithm (Algorithm 1)

The proposed algorithm:
- Estimates the transition matrix from observed state transition counts
- Constructs a test statistic based on empirical KL divergences
- Uses a state-visitation-count-dependent boundary for stopping

The algorithm is proven to be α-correct and asymptotically optimal:
lim sup_{α→0} 𝔼_Q[τ_α]/log(1/α) ≤ 1/Dℳ^inf(Q,𝒫)

#### 4. Pseudo-Spectral Gap Analysis

The paper uses the pseudo-spectral gap γ_ps(P) := max_{k≥1} γ((P*)^k P^k)/k to control mixing properties, providing uniform bounds on solutions to the Poisson equation. This is crucial for handling the Markov dependence structure.

#### 5. Pinsker-Type Inequality (Proposition 4.3)

For ergodic P, Q with stationary distributions π_P, π_Q and any non-constant function g:
Dℳ(Q,P) ≥ 1/(2‖ω_{P,g}‖_∞^2) (𝔼_{π_Q}[g] - 𝔼_{π_P}[g])^2

where ω_{P,g} solves the Poisson equation for (P,g). This enables computationally tractable test statistics.

### Key Technical Innovations

1. **Composite Null Handling**: Unlike existing work that often assumes singleton null hypotheses, this framework handles composite nulls, requiring the test to rule out every P ∈ 𝒫 while maintaining uniform Type I control.

2. **Markovian Wald's Lemma**: The analysis develops a Markov-chain version of Wald's lemma to handle dependent data, expressing expected log-likelihood ratios in terms of the stationary-weighted KL divergence plus Poisson equation corrections.

3. **Uniform Poisson Control**: The paper establishes uniform control over Poisson equation solutions using spectral properties, enabling non-asymptotic bounds without restrictive assumptions like bounded likelihood ratios.

### Applications

#### 1. MCMC Misspecification Testing (Section 5.1)
The framework is applied to test whether an MCMC sampler's transition matrix preserves a target stationary distribution. The stationary-weighted divergence yields natural test statistics, and the lower bound provides fundamental detection limits.

#### 2. Linearity Testing in MDPs (Section 5.2)
For Markov Decision Processes, the method can sequentially test whether transition dynamics are linear, providing structural guarantees for reinforcement learning algorithms.

### Comparison to Existing Work

- **vs. i.i.d. setting**: The stationary-weighted KL divergence Dℳ(Q,P) generalizes the classical KL divergence to Markovian settings, reducing to it when Q has identical rows.
- **vs. simple hypotheses**: Most prior work assumes singleton null/alternative sets; this work handles composite vs. composite testing.
- **vs. asymptotic bounds**: Existing asymptotic bounds are improved to non-asymptotic, instance-dependent bounds.
- **vs. parametric assumptions**: Unlike SPEF (single-parameter exponential family) assumptions in bandit literature, this work applies to general ergodic Markov chains.
